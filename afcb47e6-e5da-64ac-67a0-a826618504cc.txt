　　第222章
<br/>　　西北大将林问天，是与尉迟离同一级别的人物，帝国西北防线的负责人。
<br/>　　在一年前的战争中，林问天玩忽职守，守城无力，最后做了逃兵。逃回京城后，明帝制了他重罪，全家满门抄斩。
<br/>　　如果今日杨龙停是帝国军人的骄傲，那么林问天就是帝国军人的耻辱。多少人提起他便咬牙切齿，恨不得生啖其尸。
<br/>　　不过也有人说，林问天是文官武将斗争的牺牲品，应该给的支援迟迟不到，林问天才无奈兵败。
<br/>　　程大雷在长安待了一年，却也明白，后者才是林问天死亡的真相。可程大雷没有想到，林少羽竟然是林问天的子嗣。
<br/>　　怪不得林少羽要刺杀明帝。
<br/>　　程大雷叹了口气，问：“林家还剩多少人？”
<br/>　　“还有几个逃脱的老弱，现在都隐姓埋名的活着，家里发生的事情，是一个老仆告诉我的。”林少羽咬着牙道：“家父为帝国抛头颅洒热血，竟然被昏君冤杀，我一定要向昏君讨个公道。”
<br/>　　“可就算杀了明帝，你父亲的冤屈也无法洗清。”
<br/>　　“呃……”
<br/>　　林冲和林少羽同时语结，程大雷摆摆头，这两个人的脑子怎么说呢，都有些拎不清。
<br/>　　“我定要为父亲洗刷冤屈，向那些昏君奸臣讨个公道。”
<br/>　　程大雷轻轻叹了口气，自己要向杨龙停讨公道，少羽要向明帝讨公道，可这世界真有公道两个字么。
<br/>　　“你们先躲在这里，地窖内有食物和水，这里是我给自己准备的，不会有人找到这里，等我离开长安的时候，咱们一起走。”
<br/>　　说着，程大雷提起铁枪往自己肩膀上狠狠一扎，提剑离开小庙。
<br/>　　他骑马走在回去的路上，此刻夜幕已降，星垂平野，前面忽然来了一队持火把的骑兵。
<br/>　　“是牛校尉么？”
<br/>　　看他们身上的装备，当是长安城的禁卫军，程大雷骑马走过去。
<br/>　　“是牛校尉，是牛校尉！”
<br/>　　“告诉大家，找到牛校尉了。”众人欢呼起来。
<br/>　　程大雷有些懵，我又那么受欢迎么。
<br/>　　“牛校尉抓到那贼人了么？”
<br/>　　“嘿，让他们逃走了！”程大雷颇为遗憾道。
<br/>　　“牛校尉好像受伤了？”
<br/>　　“喔，不小心挨了一枪，擦破点皮，不碍事。”
<br/>　　“快来人，牛校尉受伤了，快来人。”
<br/>　　这场面令程大雷摸不着头脑，自己冒充的牛三斤不过是个九品校尉，不入流的角色，可今天这些人则对自己很尊敬，甚至称得上谄媚。
<br/>　　……
<br/>　　未央宫，灯火通明。
<br/>　　明帝的寝宫外，跪倒一大片人，有帝国的皇子，宫中的嫔妃，以及朝内的文武将官。
<br/>　　刘阿吉走出寝宫，道：“陛下受了些惊吓，服过药已经睡下，各位请回吧。奉陛下口谕，今日之事勿要责罚任何人。”
<br/>　　“陛下仁慈。”
<br/>　　众人一一散去，刘阿吉走回房间，见明帝坐在榻上，端详着桌上的铁枪。
<br/>　　“陛下，人都走了。”
<br/>　　“嗯，那牛三斤找到没有？”
<br/>　　“禀陛下，已经被城防军找到，身上受了些伤，请太医去看过，倒是没有什么大碍。”刘阿吉躬身道：“陛下，此人今次救驾有功，要不要给他一些赏赐。”
<br/>　　“先莫急于赏他，有时间的话，我想亲自见见这个人。”明帝手拂过桌上的铁枪，问：“认得这柄枪么？”
<br/>　　刘阿吉沉默片刻，道：“看刺客出手的招式，应该是林家的破浪枪。”
<br/>　　“是呐，破浪枪。”明帝默默道：“林家还有人活着。”
<br/>　　……
<br/>　　程大雷已经感受到生活有些不同了，从自己卧床养伤开始，经常有人来看自己。都是朝中的大人物，文官武将，皇亲贵胄，而且来也绝不空手来。
<br/>　　“牛校尉，我有一匹西域千里马，正配得上牛校尉这样的英雄人物。”
<br/>　　“这柄精钢剑乃是我的心爱之物，嘿，谁让我我和牛校尉一见如故呢，就送给牛校尉了。”
<br/>　　……
<br/>　　如将军府，相府这样的大人物自然不会亲自来，但也差人送来了礼物。
<br/>　　一个九品校尉，怎么值得一品大员送礼呢，程大雷越琢磨越觉得不对劲，渐渐也就明白是怎么会事了。
<br/>　　所谓功高莫过救驾，罪极不过造反，程大雷毕竟替明帝接了一枪，先不说他这份本事，这份忠心就是难得的。明帝难道会忘了他，日后他的前程肯定小不了，就算封王拜相，也并不夸张。
<br/>　　那还不赶紧趁他现在没发迹来巴结他。
<br/>　　而在程大雷卧床这几天，帝国也发生几件事情，武科举已经开始，文试武试，一轮轮往下走，淘汰了一大批人。
<br/>　　当然，这些也和程大雷没有关系了，他现在是重点看护对象，大门不出二门不迈，专心在家里养伤。太医几乎是每天来一次，各种人参虎骨，程大雷都快喝吐了。
<br/>　　这天刘阿吉亲自带着一个太医过来，是一个老者，程大雷以前倒是从未见过。
<br/>　　“伤口已经愈合，在将养几日就能下床了。”老者道。
<br/>　　“嗯，以前我怎么没见过你，你贵姓？”程大雷躺在床上问道。
<br/>　　“姓李。”
<br/>　　“哦，倒是和皇族同姓。”
<br/>　　老者看了一眼刘阿吉，刘阿吉起身道：“牛三斤，我去院子透透气，你一切太医吩咐。”
<br/>　　说罢，他起身离开屋子，只有程大雷和大夫待在房间中。
<br/>　　“当时可是很危险，你奋不顾身，就不怕危险么？”
<br/>　　“这和你有关系么。”程大雷直接怼了回去。
<br/>　　“呃……”老者脸上表情变幻：“随便问问，随便问问，听说牛将军是从西北来的，那里情况好么？”
<br/>　　“好什么好，连饭都吃不饱，能活着就算不错了。”程大雷又没去过西北，他说的是幽州的情况。
<br/>　　“竟然这样惨么，那牛校尉觉得，我们和戎族打仗为何输多胜少呢？”
<br/>　　“那你去问明帝那个老头啊，何必问我。”
<br/>　　程大雷嘴里说着，心里琢磨刚才的事情，老者向刘阿吉递眼神，程大雷注意到了。以刘阿吉的身份，何必要听一个太医的话，那刘阿吉可是一人之下，万人之上，能让他听话的，除非……
