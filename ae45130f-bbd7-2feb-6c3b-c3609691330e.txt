　　“天儿。”龙啸暴喝一声。
<br/>　　龙啸的脸色，在那一瞬间冰冷到极点。
<br/>　　他的眼眸，瞪大着，直视萧逸的背影。
<br/>　　他万万没想到，仅仅是他愣神的几个呼吸间，他的独子龙天，便已在滔天紫炎之下被烧成虚无，就此身死。
<br/>　　“该死，易霄，本域主杀了你。”龙啸瞬间暴走。
<br/>　　轰…
<br/>　　惊人的气势，陡然震动长空。
<br/>　　龙啸脚踏木煞、雷煞、邪煞三条巨龙，破空而至。
<br/>　　木煞，早便在他掌控之中。
<br/>　　毒袍的操控与手段早已消散，雷煞、邪煞，自然也重新被他所掌控。
<br/>　　萧逸缓缓转过身，冷眼看向龙啸。
<br/>　　“龙啸，若你出手，易某会将你与邪修等同对待。”
<br/>　　“我能杀毒袍，同样也能杀你。”
<br/>　　淡漠而冰冷的话语，陡然落下。
<br/>　　龙啸破空而来的身影，瞬间停下。
<br/>　　萧逸那淡漠却又泛着底气十足的语气，让他瞬间迟疑了起来。
<br/>　　毒袍，是出了名的手段极多，又诡异莫测。
<br/>　　易霄，同样是声名在外，年纪轻轻，便一身滔天手段。
<br/>　　之前，六龙六煞毒袍掌控其四，本已胜券在握。
<br/>　　可最后，还是被易霄强行扭转了战局，甚至于，直接斩杀了掌控两条龙形煞气的毒袍。
<br/>　　连太荒之火，还有毒袍临死前引爆大阵，让得六龙六煞暴走的手段，最终也未能损伤易霄分毫。
<br/>　　那么，易霄的实力，到底强悍到何等地步？
<br/>　　再加上此刻那底气十足的语气，龙啸不得不瞬间停下身躯，迟疑起来。
<br/>　　他不知道易霄为何能将血煞、毒煞吸收，也不知易霄凭此那发挥何等实力。
<br/>　　但在他看来，易霄，起码是掌控了两条龙形煞气。
<br/>　　即便他此刻重新夺回了木煞、雷煞和邪煞，三条煞气的掌控权，可要说杀易霄，他毫无把握。
<br/>　　这时，萧逸再未理会龙啸，而是闪身到东方铁云身旁。
<br/>　　“东方殿主。”萧逸拱了拱手，“六龙地域这边，毒袍已死，原本的地域负责人虎护法，还有各分部聚集而来的十万邪修，已悉数身死。”
<br/>　　“故六龙地域这边，邪修之患已除。”
<br/>　　“易某也该离去了。”
<br/>　　“要走了？”东方铁云闻言，愣了愣，下一秒，大笑几声。
<br/>　　“易霄分殿主，果真如传闻那般，行事雷厉风行。”
<br/>　　“虽每每在各大地域斩尽邪修，留下大片声誉和惊人事迹，事后总挥挥衣袖便洒脱而离。”
<br/>　　“好，果真好。”
<br/>　　“不过。”东方铁云蓦地皱了皱眉，“易霄分殿主，老夫虽不知道怎么回事，感知不到你的气息还有情况。”
<br/>　　“但老夫从你眉宇间的苍白之色能看到，你现在身负不轻伤势吧？”
<br/>　　东方殿主，自然是感知不到萧逸的气息的。
<br/>　　但他是猎妖殿主殿主的同时，亦身兼药尊殿主殿主，自然一眼便能从萧逸眉宇间看出是否存在伤势。
<br/>　　萧逸也知瞒不过东方铁云这等绝世层次的炼药师，便也点了点头。
<br/>　　东方铁云闻言，皱眉道，“若如此，易霄分殿主何必急着离去，不如先养好伤势再走不迟。”
<br/>　　萧逸点点头，“那便谢过东方殿主好意，易某去你主殿叨扰一天，伤势好了便走。”
<br/>　　他倒没什么伤势，之前战斗的伤势，早就在借雷炼体下悉数恢复了。
<br/>　　现今的伤势，是刚才强行凝聚六色圆环，所受的些许反噬罢了。
<br/>　　到闭关室修养打坐半天左右便可无恙。
<br/>　　“易霄分殿主，这边请。”东方铁云做了个‘请’的手势，道，“至于邪修剩下来的其他祸患，我会谴麾下执法队处理的。”
<br/>　　萧逸点点头，转身便要离去。
<br/>　　十万邪修，还有里头强者，已悉数身死。
<br/>　　剩下的，也不过是些后续的麻烦，如一些在外的少数零散邪修，还有大战后的处理等等。
<br/>　　这些小事，猎妖殿的武者去处理即可。
<br/>　　恰在此时，龙啸横跨一步，挡住了去路。
<br/>　　“紫炎，你杀我独子龙天之事，尚情有可原，本域主也暂时不与你计较。”
<br/>　　“但…”龙啸的话语，陡然间强硬着。
<br/>　　“我龙家的血煞和毒煞，还请交还出来。”
<br/>　　龙啸将‘我龙家’三个字，咬得极重。
<br/>　　萧逸脚步一顿，淡漠的眼眸中，闪过一丝戏谑。
<br/>　　龙啸虽未明说，但他的这点心思，却瞒不过萧逸。
<br/>　　龙天之事，于理，龙啸确实无法与他萧逸计较。
<br/>　　但，若龙啸夺回了其他的龙形煞气，他将再度成为六龙地域之内无敌的存在。
<br/>　　届时，所谓规矩，所谓道理，仍旧是他龙啸一言蔽之。
<br/>　　是否要替龙天报仇，也将是他一念之间。
<br/>　　“要血煞和毒煞，没有。”萧逸摇了摇头，“要命，一条。”
<br/>　　“你…”龙啸双眼一眯，“这般说来，你堂堂猎妖殿、炎殿、药尊殿三殿分殿主，要觊觎我六龙地域的煞气，强行夺之了？”
<br/>　　“身兼三殿要职，却干起这等强夺重宝的勾当，易霄，此事传出去你也不怕惹天下人嗤笑？”
<br/>　　一旁东方铁云皱了皱眉，看向萧逸，“易霄分殿主，六龙六煞，毕竟是六龙地域的守护煞气。”
<br/>　　“龙家掌控之，保六龙地域全境安危。”
<br/>　　“若是强行夺了，确实…”
<br/>　　东方铁云后面的话没有说下去，但他的意思很清楚。
<br/>　　他也希望萧逸交回两条煞气。
<br/>　　“你听到了？”龙啸冷声道，“此事，即便你拿到总殿去说，也是你无理。”
<br/>　　“你若不还，我便去三殿总殿直接讨个说法。”
<br/>　　“按你八殿规矩，你们总殿主会亲自替本域主夺回来。”
<br/>　　“随你。”萧逸淡漠地吐出两个字。
<br/>　　“你…”龙啸眼眸一冷。
<br/>　　“易霄分殿主。”东方铁云亦皱了皱眉。
<br/>　　萧逸看向东方铁云，沉声道，“之前六煞狂猛而至，我为求保命，只能吸收其二。”
<br/>　　“如今，两条煞气，在我体内。”
<br/>　　“若要取回，只能杀了我，强行吸摄而出。”
<br/>　　“什么？那万万不可。”东方铁云脸色一变。
<br/>　　“至于你。”萧逸看向龙啸，“你大可去总殿告状，若总殿为此追杀我，即便杀了我也要为你六龙地域取回两条煞气，那易某也无话可说。”
