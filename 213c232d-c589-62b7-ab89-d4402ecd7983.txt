　　楚风眠修炼的乃是战龙决。
<br/>　　如今这傲恒龙皇的龙爪，正是在楚风眠的手中，才有着最大的作用。
<br/>　　“这龙皇龙爪，我必定要得到！”
<br/>　　楚风眠对着傲恒龙皇的龙爪，是势在必得，他的财富如此雄厚，不管是谁来争抢，楚风眠都要争上一番。
<br/>　　在这一道道议论声，逐渐消散的一刻，看台上的灵尊真人，在缓缓开口道。
<br/>　　“傲恒龙皇的龙爪，起拍价，一百万灵石！”
<br/>　　灵尊真人的语气，虽然平静，但是这其中的每一个字响起，都响彻所有人的心中。
<br/>　　起拍价百万，这足以是让许多巅峰圣者，倾家荡产，都无法换来的一个数字。
<br/>　　会场之中，暂时寂静下来。
<br/>　　但是在一刻，却是无数声音，同时响起。
<br/>　　“两百万！”
<br/>　　“三百万！”
<br/>　　“五百万！”
<br/>　　“一千万！”
<br/>　　地字包厢之中，无数声音便是猛然响起。
<br/>　　龙皇龙爪的价值，可不是区区百万灵石可以比拟的。
<br/>　　这价格，在瞬间，便是以破千万。
<br/>　　而且还是在节节攀升。
<br/>　　“两千万！”
<br/>　　在天字包厢之中，一道声音，猛然响起，这声音浑厚无比，是一尊真正的强者。
<br/>　　天字第七号。
<br/>　　楚风眠瞬间便是察觉出来，这声音，便是从天字第七号包厢中响彻而起的。
<br/>　　除了楚风眠所在的第十一号，跟古林所在的第十号之外，这还是天字包厢，第一次有人开口。
<br/>　　“真正的大人物，终于出手了。”
<br/>　　“龙皇龙爪，对生死境高阶的大人物来说，都是有着无比极大的价值。”
<br/>　　“这龙皇龙爪，只怕是会拍出天价。”
<br/>　　下方一尊尊武者，议论纷纷。
<br/>　　他们如今都是在看热闹的，这龙皇龙爪，自然不是他们可以争取的了的。
<br/>　　“这龙皇龙爪，两千万灵石，若是没有人要，就归我所有了！”
<br/>　　七号包厢中的声音传出来，这话中蕴含的，便是带有着几分威胁的意思。
<br/>　　谁现在敢出价，买下这龙皇龙爪，便是跟他为敌。
<br/>　　天字号包厢中的人，都是大人物，尤其是这开口之人，隐隐约约之间蕴含的力量，可都是达到了生死境高阶的地步。
<br/>　　这种大人物，可没有什么人想要去得罪。
<br/>　　“哦？此人是谁，如此嚣张？”
<br/>　　楚风眠听到这声音，也是可以从中听出其中的霸道威严，跟其中的威胁。
<br/>　　“七号包厢中，应该是天外楼的一尊客卿长老，玉门龙，此人据说得到过龙族奇遇，所以想要得到这龙皇龙爪，并不奇怪。”
<br/>　　李长老在一旁，出声解释道。
<br/>　　“玉面龙的实力，在生死境高阶之中，都是强者，楚公子还是不要去跟他争抢为好。”
<br/>　　李长老如今已经看得出来，楚风眠是一尊无法无天的人物了。
<br/>　　七大宗门，荒古门楚风眠都没有放在眼中，李长老可不想楚风眠去白白得罪玉面龙，让他们天商会，失去一个如此大的客户。
<br/>　　“哦？”
<br/>　　听到李长老的话，楚风眠悄然之间，催动灵识，便是看向这七号包厢之中。
<br/>　　穿透种种禁制，楚风眠便是一眼看出，在这七号包厢之中做着的一尊中年男子。
<br/>　　这中年男子的身上，蕴含有着淡淡的龙威，脖子上，甚至是还带有着几片龙鳞，显得极为诡异。
<br/>　　这玉面龙的实力，只怕是跟不死魔尊，都相差不远。
<br/>　　“虚空海中的强者，倒是不少。”
<br/>　　楚风眠点了点头道。
<br/>　　楚风眠本以为，北域中的强者，应该都是在北邙学院跟地魔门，现在看来，这虚空海中的势力中，也不是没有强者。
<br/>　　就在楚风眠思考之际，在外面，一声声音便是响彻起来。
<br/>　　“玉门兄，这龙皇龙爪，两千万灵石就想要得到，也未免有些太便宜了吧”
<br/>　　在旁边，六号包厢之中，又是传出了一道声音。
<br/>　　“我出三千万！”
<br/>　　这声音，同样也是霸道无比，话语中蕴含的气息，丝毫不比玉面龙要弱。
<br/>　　这又是一尊生死境高阶的强者。
<br/>　　生死境高阶的强者，在虚空海中，其实都是极为少见，如今在这拍卖会上，如今却是一个个显露了出来。
<br/>　　“四千万！”
<br/>　　又一座声音，来自天字包厢第三号中传了出来，这是一道女声。
<br/>　　楚风眠一眼望过去，便是看见天字三字包厢之中，坐着一名女子，在他的背后，两位中年男子，正站在她的背后，时时刻刻的守护着她。
<br/>　　这女子的实力，也就是跟落长恨等人相差不多，但是她背后的两尊中年男子，却可都是生死境高阶的强者。
<br/>　　能够让生死境高阶的强者来守护，这女子背后的势力，可想而知。
<br/>　　“五千万！”
<br/>　　七号包厢中，玉面龙再次开口道。
<br/>　　“六千万！”
<br/>　　这三号包厢中的女子，也是再度开口。
<br/>　　价格，在节节攀升，不过如今，也就只剩下这三家，还在争夺。
<br/>　　“差不多了。”
<br/>　　价格已经是涨到了八千万左右，楚风眠心中低声自语一句。
<br/>　　这价格，应该快要到头了。
<br/>　　“一个亿！”
<br/>　　楚风眠突然出声开口道。
<br/>　　短短的三个字，却是如同一刻惊雷，猛然在无数人的心中响彻。
<br/>　　一个亿灵石，是什么概念。
<br/>　　几乎是许多人从未想到过的一个天文数字。
<br/>　　“十一号包厢，又是十一号包厢！”
<br/>　　“此人的财富，莫非真的是无限的？居然硬生生的提升了两千万？”
<br/>　　十一号包厢内。
<br/>　　李长老听到楚风眠的报价，也差一点是要吓出一身冷汗来。
<br/>　　这样一个天价，已经不是得不得罪人的事了，就怕楚风眠根本就拿不出来。
<br/>　　一亿灵石，在他们天商会都不是一个小数字了，就算是楚风眠有些奇遇，得到财富，想要拿出一亿灵石来，也绝不容易。
<br/>　　“楚公子，您真的要花一亿灵石来买下这龙皇龙爪？”
<br/>　　李长老说话都有些结结巴巴的了。
<br/>　　一亿灵石，就算是他，都从未想过的一个数字。
<br/>　　“当然，这龙皇龙爪，我势在必得。”
<br/>　　楚风眠平静的开口道。
<br/>　　会场，一片寂静。
<br/>　　三号包厢，六号包厢，七号包厢，都是沉默，似乎也被这一个数字，镇住了。
<br/>　　“那好，这龙皇龙爪，便是属于十一号包厢了。”
<br/>　　灵尊真人大手一挥，这巨大的龙皇龙爪，便是直接向着十一号包厢飞了过来。
