
    奥黛丽敏锐觉察到情况不妙。<br/>
<br/>
    她脸色骤然变得凝重，明眸光芒一闪，轻呼道：“石岩，情况不妙”<br/>
<br/>
    此刻，那些天目族的族人，还没有意识到形势的复杂，只当这身下陨石朝着水族的水晶球接近，乃是石岩、奥黛丽故意为之，是故意要靠近水族的水晶球，好和水族族人交谈。<br/>
<br/>
    “嗯”<br/>
<br/>
    从沉思中惊醒，石岩一怔，露出不解之色。<br/>
<br/>
    奥黛丽指了指身下，又指了指前方的水族水晶宫殿，苦涩道：“快要撞上了”<br/>
<br/>
    此言一出，石岩眼神骇然，终于紧张起来，神情变得极为严峻。<br/>
<br/>
    “呃，你们不会无法驾驭这块陨石吧”玛希莎也看出微妙了，脸色一下子变得极为难看，“真要撞上去了”<br/>
<br/>
    她讲话的时候，身下陨石速度一点不减，朝着那水族的水晶球轰然而去。<br/>
<br/>
    那些原本坦然笑谈的水族俊男美女，一开始并没有将石岩他们的陨石当一回事，在这陨石接近的时候，他们还远远看着这块，见是天目族的族人，还露出笑容，因为他们和天目族有旧，以为伊夫林过来是要交谈。<br/>
<br/>
    待到他们发现那陨石离他们极为接近的时候，依旧没有减速，还是以极快的速度掠来，那些水族的男女，各个变脸，纷纷惊慌尖叫起来。<br/>
<br/>
    此刻，石岩身下陨石极快，离他们太过接近，水族族人这时候想躲避，好像都有点来不及了。<br/>
<br/>
    “不能撞上”<br/>
<br/>
    伊夫林大叫，“水族虽然性格温和。但他们并不好惹，他们和龙蜥族一直交好，得罪了他们，我们怕是连进入龙蜥族的领地都不能”<br/>
<br/>
    石岩双眸骤然变得猩红如血，一股澎湃的波动，从他脑海内荡漾出来，顷刻间，周边陨石涌现一股吸引力。最近的几块陨石如受着牵扯，猛然间堵在他们和水族的宫殿之间。<br/>
<br/>
    “都小心防护自己”石岩暴喝。<br/>
<br/>
    伊夫林和奥黛丽等人，闻言不及多想，都将神体外层光罩祭出，将力量凝炼全身，准备应付陨石撞击的毁灭波动。<br/>
<br/>
    然而，让他们瞠目结舌的事情，突地诡异发生。<br/>
<br/>
    那明明朝着一块块陨石撞击的身下三足巨鼎。在关键时刻，悄悄浮现一缕微弱的波动，旋即，他们身下陨石奇诡的扭动方向，险之又险的避过那些挡在他们眼前的陨石，妙至毫巅的避了过去。<br/>
<br/>
    天目族族人，还有那些水族族人，都惊出一身冷汗，表情都惊魂未定。<br/>
<br/>
    他们心中明白，石岩身下那块以三足玉鼎为核心的陨石。就算是没有冲撞在水族的水晶宫殿，就算只是撞击在两者间的陨石上，那大爆炸形成的恐怖余波，都可能导致两族族人死伤惨重。<br/>
<br/>
    陨石冲击的大爆炸，连不朽武者都很难承受，尤其是在这么近的距离，亚当斯、伊夫林都脸色发白，知道一旦真正碰上，他们很有可能也撑不住，被那些爆炸的碎片给绞成粉碎。<br/>
<br/>
    还好。最让他们担心的事情，并没有发生。<br/>
<br/>
    惊悸不安的众人，在陨石上渐渐逼近水族的水晶球，而此刻，那水晶球极速挪动，如被恶棍吓到的少女，紧急避开他们的陨石。生怕沾上一点点边。<br/>
<br/>
    他们当石岩等人疯了。<br/>
<br/>
    同时，一名俊美阴柔的男子，从那水晶球内骤然飞出。脸色森寒，浮动在他们陨石上方，兴师问罪道：“你们天目族疯了么是不是要和我们水族交战，不然怎么敢如此狂躁”<br/>
<br/>
    伊夫林满脸苦涩，连忙躬身行礼，道：“很抱歉，我们身下的陨石，忽然失去了控制，要不然不会如此紧张。你看，以我们的境界修为，这陨石真撞上你们，我们也活不了，我们没疯”<br/>
<br/>
    他和那水族的男子，都是用一种新的语言交流，那语言石岩、奥黛丽都听不懂，一头雾水。<br/>
<br/>
    “他们在说什么”奥黛丽以荒域语言问话。<br/>
<br/>
    “你们只懂得荒域语言，这样在域海很难与人交流，不介意的话，我分享我的语言印记，你们以灵魂攫取，如何”玛希莎微笑，“放心，语言间的分享，没有任何风险，我开放出来，你们只要以灵魂裹住即可。”<br/>
<br/>
    “没问题。”奥黛丽冲石岩点了点头，说道：“在我们冥皇族的奥义中，将灵魂记忆给剥离，也不是什么了不起的手段，我也懂得很多我们那里偏僻之地的语言，就是利用她说的方法。”<br/>
<br/>
    “嗯。”石岩神情自然，他也知道这只是小手段，不足为奇。<br/>
<br/>
    玛希莎第三目微微蠕动，一股灵魂磁场释放出来，旋即便见一团幽幽暗光，从她双眸内浮现出来，那暗光为记忆烙印，里面有许多不同的语言波动，只要融合脑海记忆，都能立即对域外语言有所认识。<br/>
<br/>
    石岩和奥黛丽两人，心神变幻间，分别释放出神识，将那两团记载着域海语言的暗团裹住，收拢进自己的脑海，变成自己的一份记忆。<br/>
<br/>
    一种种不同的语言，如电流一般在他们脑海扎根，变成他们的灵魂一部分。<br/>
<br/>
    然后，石岩、奥黛丽两人，便听懂了伊夫林和那水族族人的交流，他们使用的乃是一种域外通用语言，简单古朴，也最容易理解交流，各大种族在域外内活动，大多都是运用这种通用的语言。<br/>
<br/>
    “你们所讲的语言，可是荒域的语言，你们来自于荒域”<br/>
<br/>
    忽地，那名和伊夫林交涉的水族俊美男子，神情一动，露出惊诧之色，冷不防将目光落向石岩、奥黛丽两人，一副动容之色。<br/>
<br/>
    “不错，我们来自于荒域，你怎么这么惊讶”奥黛丽说道。<br/>
<br/>
    “以前有一名荒域的家伙，去过我们水族的领地，和我们水族颇有交情。不过后来，此人离开了，听说被魂族的一名强者给囚禁了，那名荒域的家伙，对我个人也有点恩惠，可惜魂族太强大，我无法给他任何帮助。”<br/>
<br/>
    这名俊美的过分的水族男子，喟然一叹，流露出失落之色。<br/>
<br/>
    他深深看向奥黛丽，仔细瞧了许久，忽然诧异道：“那家伙，和你应该来自于荒域同一种族。”<br/>
<br/>
    “哦”奥黛丽美眸一亮，“你可知他的姓名”<br/>
<br/>
    “冥鸿他叫冥鸿”那人肯定道。<br/>
<br/>
    奥黛丽身躯一震，明眸崭亮，露出难以置信的表情，“竟然，竟然是他，他原来一直在这里”<br/>
<br/>
    “他是谁”石岩讶然。<br/>
<br/>
    “记得我和你说过，我冥皇族的典籍记载，说有祖先在域海内走动，发现过异族族人，但却因为语言不同，没有交流的事情么”奥黛丽道。<br/>
<br/>
    石岩点头。<br/>
<br/>
    “那典籍，就是他记载的，算起来他应该是我祖辈了，根据我母亲说法，万年前嗜血未出之前，我们星域应该是我们冥皇族的时代，因为嗜血的横空出世，让我们冥皇族没落。”<br/>
<br/>
    “在那个时代，我们冥皇族很多强者陨灭，也有人消失不见，冥鸿先辈据说自知非嗜血之敌，远遁虚无域海苦修，力图来挽救我们冥皇族失利的大局，可惜他再也没有回来过，我们都当他早已陨灭了，他是我们冥皇族对虚无域海了解最深的一人，那记载，就是他以前在虚无域海活动时抒写的。”<br/>
<br/>
    “冥鸿前辈，其实还是我舅舅冥晧的第一个老师，就连冥晧这个名字，根据我母亲所说也是他取的。他和我祖父，好像还是堂兄弟，当年在我们冥皇族，也是一个极为了不得的人物。”<br/>
<br/>
    奥黛丽缓缓解释，话到后来，她脸色坚毅，道：“如果有可能，能找到冥鸿先辈，我们说不定就知道回去的路了”<br/>
<br/>
    石岩眼睛一亮。<br/>
<br/>
    按照她所言，冥鸿可谓是荒域进入虚无域海的先贤，数次进入过虚无域海，并且大多数全身而退，自然知道如何进来，如何成功返回荒域，要是能找到此人，他们的确就能认清方向了。<br/>
<br/>
    “劝你们别想了。”那名水族的族人，皱了皱眉头，说道：“他被魂族一名强者囚禁，变成那强者的九大凶魂之一，魂族在星海间太强了，那名强者也非常可怕，我都不敢招惹，更别提你们这些小家伙了。”<br/>
<br/>
    停顿了一下，他哼了一声，道：“看在你们和他来自于同一域界，你们鲁莽一事就算了，奉劝你们一句，最好将我的话忘掉，别去招惹那魂族的家伙，这次，听说他也会过来，他和龙蜥有旧，你们最好避的远远的。”<br/>
<br/>
    这番话落下，他就没有继续搭理石岩等人，板着冷脸，重返那水晶宫殿，吩咐族人继续行进。<br/>
<br/>
    “魂族，这是什么种族，很厉害么”石岩看向天目族众人。<br/>
<br/>
    那三人神情沉重点头，动作都一致，伊夫林回答：“星海间，种族成百数千，但实力都不相同，魂族，是星海间公认最强的七大种族之一，和魂族相比，我们天目族只是三流的小种族。”<br/>
<br/>
    “比龙蜥族如何”石岩再问。<br/>
<br/>
    “龙蜥族的先祖蜥早归墟了，如今的二代龙蜥没有突破域祖，所以龙蜥族也只能算是二流。但魂族，这个太初时代后和我们天目族一起出现的种族，却拥有真正的域祖级强者坐镇，有洞悉暗能量真谛的强者，魂族的底蕴，不是常人能度量的。”伊夫林敬畏道。
