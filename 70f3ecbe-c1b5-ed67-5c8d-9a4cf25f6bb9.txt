        &lt;h3&gt;第三卷第三百一十四章  乱七八糟的世间事&lt;/h3&gt;
<br/>
<br/>        潘青松听了陈子昂的建议，并没有对夏颜展开调查，第一，时间太长，人证物证都不太好找，第二，牵扯到了冯睿，冯睿和陈子建离婚了没有错，冯睿却有了俩个孩子。
<br/>
<br/>        陈子昂竟然说她已经知道了，并且不再计较，潘青松也知道了陈子昂的顾虑，便将这件事情按捺下来了。
<br/>
<br/>        他将目光放在了洛城，洛城他的同学和他的职位是一样的，他开着车亲自将陈子昂给他的资料交给了他同学徐佩云。
<br/>
<br/>        “我说你是不是管得太宽了啊，你的手都伸到洛城来了。”徐佩云笑着说。
<br/>
<br/>        “你看看你的辖区都是一些什么人，你还不管管，还嫌我多事，你说，这几年你都干了些啥？”潘青松也笑着说。
<br/>
<br/>        “吃饭睡觉看资料，没事可干按按腰。”徐佩云看着资料眉头皱在一起。
<br/>
<br/>        潘青松喝着茶等着徐佩云将资料看完。
<br/>
<br/>        “你这些资料可属实？”徐佩云语气不太好。
<br/>
<br/>        “属实不属实我不知道，这个要你们自己去调查，再说我只是给你提供线索。”潘青松淡淡的说。“要不是看在同学的份上我才懒得管这些破事，还不是怕这份资料被送到上面会影响你，到时候查起来你什么都不知道，你怕是要受过了。”
<br/>
<br/>        “这两件事情其实是一同件事情，性质是一样的，如此恶劣是在是可恶，我们也一直在关注这这两个项目，之前也有人给我们说过这件事情，苦于一直没有证据，还有就是和西城某些人关系，迟迟没动，你这倒是给我们提供了很多的证据，谢谢你啊，青松。只是能告诉我这些东西从哪里来的吗？”
<br/>
<br/>        “你我都是同一类型的人，何必问这么多？”潘青松笑眯眯的看着徐佩云。
<br/>
<br/>        “行，我不问了，谢谢你为我们提供了这么多资料，我为洛城的人们感谢你。”徐佩云由衷的说。
<br/>
<br/>        “谢我就不必了，我只想看到有结果。行了，我该走了，希望我的这些东西能起到作用。”
<br/>
<br/>        潘青松告别了徐佩云，他知道徐佩云的个性，天不怕地不怕，篓子捅得越大越开心的那种，可是毕竟他们都是小人物，篓子捅大了也有人会捂上。
<br/>
<br/>        从洛城回来就又跑去医院看了看老丈人，奇迹般的发现陈子建身边有了一个女人，俩个人在医院照顾着老丈人，病房竟然是欢声笑语一片。
<br/>
<br/>        “二哥，这位是？”
<br/>
<br/>        “青松，这是你二嫂，二秀。”冉灵卉为潘青松做着介绍。
<br/>
<br/>        “二，二嫂。”潘青松有点结巴，没听说陈子建结婚，这么快就有二嫂了？
<br/>
<br/>        “嗯。”二秀毫不犹豫的答应了。
<br/>
<br/>        陈子建已经和她确定关系了，老爷子也说了，等他出院之后就让他们举办婚礼。
<br/>
<br/>        “青松，这几天不忙了？”陈子建有些不好意思的挠挠头对潘青松打招呼。
<br/>
<br/>        “二哥，你这春天来得也太迟了点。”潘青松哈哈大笑，陈子建和冯睿在一起，陈子建完全就是虐受无主型，不知道和二秀在一起会怎么样？
<br/>
<br/>        不管怎么样，看起来这个二秀人不错，还能和陈子建一起伺候老爷子，从这一点来说，人品就比冯睿要好很多，看来，陈子建是老来得福了？
<br/>
<br/>        “只要有春天，管他迟或早！”二秀看着脸红红的陈子建开口了。
<br/>
<br/>        “二嫂说什么都是对的，二嫂和二哥这几天辛苦了。”潘青松一本正经的开口。
<br/>
<br/>        毕竟是第一次见二秀，可以挤兑挤兑陈子建，却不能对二秀不敬。
<br/>
<br/>        “子澜恢复得怎么样了？”二秀问潘青松。
<br/>
<br/>        “还不错，可以坐起来了，大夫说过一段时间就可以慢慢的下地走动走动了。”
<br/>
<br/>        “青松，你一会开车带着我和二秀去看看子澜。”冉灵卉开口。
<br/>
<br/>        “好嘞。”
<br/>
<br/>        “青松啊，这一段时间工作上怎么样啊？”陈元良靠在床上，他这一段时间感觉好了很多，不过后期要做化疗，要在医院呆一段时间。
<br/>
<br/>        “都很好。干我们这一行的就是讨人嫌。”潘青松拍了拍肩上的杠杠。
<br/>
<br/>        “谁嫌弃你，没有你们这些人维持秩序，岂不是乱套了，尽胡说。”冉灵卉笑着说潘青松。
<br/>
<br/>        “是啊，我们都应该感谢你们这些人呢，保家守平安。”二秀笑着插话：“我们去看看我们的大英雄子澜妹子吧，我来这么多天都没有去看看她。”
<br/>
<br/>        “好啊，那就一起走吧。”潘青松站起来扶着冉灵卉。
<br/>
<br/>        当他们走到门口的时候，冯睿和陈天天来了。
<br/>
<br/>        “爸，妈，我这一段时间有些感冒，爸做手术的时候我也没有来看，天天也是天天照顾我，没有时间来看你们，我这一好我就来了。”冯睿看着大家说。
<br/>
<br/>        但是她看见二秀立马心里就不好了，上次就是这个女人拉着陈子建跑了的。
<br/>
<br/>        “子建，这位是？.......”冯睿问陈子建。
<br/>
<br/>        “这个.......”陈子建支支吾吾。
<br/>
<br/>        “你叫我二秀就好，我是陈子建的媳妇儿。”二秀大大方方的对着冯睿，她是知道冯睿。
<br/>
<br/>        “媳妇？你们什么时候的事情？”冯睿难以置信，陈子建这么快就找到了媳妇？
<br/>
<br/>        “就这几天的事情，他们已经结婚了。”陈元良淡淡的开口。
<br/>
<br/>        “天天，来，爷爷看看你，爷爷都好久没有见到你了，你最近可好？”陈元良看着陈天天，这孩子从那天聚餐之后就玩消失，连人都看不到。
<br/>
<br/>        “爷爷，妈妈生病了，我就陪了她几天，我害怕感冒传染，所以就没有来看你，爷爷你不怪我吧？”陈天天毕竟从小在爷爷奶奶身边长大，见到爷爷生病在床，心里一酸哭出声了。
<br/>
<br/>        “天天，别哭，爷爷这不是好好的嘛，乖。”冉灵卉见陈天天哭了，准备出门的脚又缩了回来。
<br/>
<br/>        这孩子就是他们的心头肉。
<br/>
<br/>        “奶奶。”陈天天抱着冉灵卉抽抽噎噎。
<br/>
<br/>        “乖，别哭了，爷爷过一段时间就好了。”冉灵卉拍了拍陈天天的背，安慰陈天天。
<br/>
<br/>        冯睿冷眼看着他们，心里就后悔不该来医院看望陈元良。
<br/>
<br/>        潘青松看着二秀呆在这里会尴尬，便对二秀说：“二嫂，我们走？”
<br/>
<br/>        “妈，那我去看子澜了，你就在这里。”二秀甜甜的叫着妈，让冯睿听着心里冒酸水。
<br/>
<br/>        “行，那你去吧，天天来了，我和天天说会话。”
<br/>
<br/>        “冯睿，你坐啊，子建站着干嘛啊，给冯睿倒茶，来客人了都不知道招呼，你这孩子越来越傻了。”冉灵卉拉着陈天天的手，走到沙发上坐下。
<br/>
<br/>        “天天，告诉奶奶，最近都干了一些啥？”冉灵卉抚摸着陈天天的手问。
<br/>
<br/>        “上学，然后放学就回来陪着妈妈。”陈天天如实相告。
<br/>
<br/>        “家里谁做饭呢？”冉灵卉接着问。
<br/>
<br/>        “妈妈做呢。”
<br/>
<br/>        “妈妈感冒不是很严重吗？还能给你做饭？你这孩子，怎么这么不懂事呢，我是怎么教你的？要尊老爱幼，孝顺父母，你就是这样孝顺你妈妈的？”冉灵卉责怪陈天天。
<br/>
<br/>        陈天天心里一愣，冯睿天天说感冒很严重，每次放学回来她都把饭做好了，这些天除了偶尔咳嗽外也没有太大的问题。
<br/>
<br/>        她每次说要来看爷爷，冯睿总是推说自己感冒太严重，不能来医院，今天实在是推不过去了才和她一起来。
<br/>
<br/>        “奶奶，我知道了，我这几天陪着爷爷好不好？”陈天天觉得心里很愧疚，她要照顾爷爷一段时间。
<br/>
<br/>        “你不是还在上学吗？”冯睿开口了。
<br/>
<br/>        “没事，二妈，你感冒已经好了，你先回去，我每次放学过来就好了，这几天我就不回去家里了。”陈天天笑着对冯睿说。
<br/>
<br/>        陈天天的那一声“二妈”喊得冯睿的心一凉。
