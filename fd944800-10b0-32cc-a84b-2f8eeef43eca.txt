        一个婚礼，从决定到准备再到实行，居然只花了三天的时间，连婚礼当天的日子都是随便订的。
<br/>
<br/>        应傅老太爷的要求，婚礼必须在傅家大庄园里举办，仪式为传统婚礼，新娘必须身着凤冠霞帔地嫁进傅家。
<br/>
<br/>        李忆莎想要的那种西方婚礼无法实现，还万万没想到，傅夜沉的办事效率如此高，敷衍她的程度已经无法用语言去形容。
<br/>
<br/>        因为这要举行婚礼的所有事情，傅夜沉是一不过问、二不过目。
<br/>
<br/>        要说他唯一做了一件什么事情，那就是他自己主动发了一张电子请柬出去，还是给自己的弟弟傅昼景的。
<br/>
<br/>        其真正的目的是什么，别人就不得而知，但他傅夜沉心里倒是一清二楚。
<br/>
<br/>        他要见筱颖！
<br/>
<br/>        只是没想到，他只能在自己的婚礼上，以这样一种方式，再次见到自己心爱的女人。
<br/>
<br/>        别说傅夜沉意兴阑珊得不像一个准新郎，就连李忆莎，也开始怠慢，一副慵懒、无所谓的态度，更不像一个待嫁的新娘。
<br/>
<br/>        法式料理的高级餐厅，时间、地点，仍旧是落以柔一手操办的。
<br/>
<br/>        而就餐的男女主角，却是傅夜沉和李忆莎。
<br/>
<br/>        “我若不请冷爷爷出面，你是不是打算在婚前都不肯见我了？”李忆莎眼中夹杂着某种焦虑与愤怒的情绪。
<br/>
<br/>        傅夜沉淡定地拿着刀叉，切着盘子里的鹅肝，像是在切着玩一样：“我很忙。”
<br/>
<br/>        “今晚的法式鹅肝不合你胃口吗？”李忆莎又问。
<br/>
<br/>        这家有名的餐厅以法式鹅肝做招牌，每一克都不便宜，入口即化又美味，但傅夜沉几乎一口都没有动。
<br/>
<br/>        “我不饿。”傅夜沉勉强动了下叉子，但没把鹅肝送入口中。
<br/>
<br/>        “结婚那天我不想坐轿子。”李忆莎叹了口气后，又说道。
<br/>
<br/>        “你找小落谈就行了。”
<br/>
<br/>        “婚礼仪式可以是旧俗，但是接亲车队我要二十辆红色劳斯莱斯。”
<br/>
<br/>        “找小落谈。”
<br/>
<br/>        “之前挑选的结婚五金，我现在有些不喜欢那些款式了，要换。”
<br/>
<br/>        “找小落谈。”
<br/>
<br/>        “而且我近来瘦了，凤冠霞帔可能太大要改……”
<br/>
<br/>        “找小落谈。”
<br/>
<br/>        对于傅夜沉“找小落谈”这样的回答，李忆莎是真的有些听腻了，也厌恶了。
<br/>
<br/>        他什么事都交给那个落以柔，就连自己的终身大事都可以要落以柔来帮他搞定……
<br/>
<br/>        没有了落以柔，他会如何？
<br/>
<br/>        “如果没有那个落以柔，你是不是连这个婚都不知道要怎么结？”李忆莎语气中有些恼火。
<br/>
<br/>        傅夜沉看都不看李忆莎一眼，冷冷地回答道：“如果我的新娘不是你，那么这‘婚’，我就知道该怎么结了。”
<br/>
<br/>        “你的那个医生朋友来找过我。”李忆莎识趣地转开了话题。
<br/>
<br/>        傅夜沉惊诧地睨了李忆莎一眼，没有回话。
<br/>
<br/>        “为什么不拿着那个证据去冷爷爷和我爸爸那里揭穿我？”李忆莎若有所思地皱了下眉头。
<br/>
<br/>        傅夜沉垂着眸子，淡淡地回答道：“爷爷他很看好你，以死相逼，非得让我娶你。我已经做了让爷爷很气愤的事情了，爷爷他身体不好，不想再多一件事情来气他。”
<br/>
<br/>        “那天早上，我趁着你还没睡醒，有拿手机偷偷地拍了几张床照。回去后，故意设为手机桌面，特意让我爸爸无意间看到。然后我爸爸拿着照片和冷爷爷当面对峙。冷爷爷明知道照片是我摆拍的，但是冷爷爷并未在我爸爸面前拆穿我，反而护着我，还允诺我，一定会让你娶我。”李忆莎苦笑着感慨道。
<br/>
<br/>        那么多优秀的千金名媛上他们傅家门来“求嫁”，傅老太爷傅炎廷却从来都是持着中立的态度，唯独她李忆莎这次，傅老太爷是上心了，非逼着傅夜沉娶她不可。
<br/>
<br/>        顿时，傅夜沉剑眉微蹙。
<br/>
<br/>        李忆莎深吸了口气，接着说道：“夜沉，你那个朋友说得对！你和冷爷爷，都在为了我的私心买单。我算计得来的婚姻，是不会幸福的。所以，我想通了，这个婚，我不结了。”
<br/>
<br/>        “你……”傅夜沉身心一怔，完全没料到李忆莎会突然来个三百六十度的大转弯。
<br/>
<br/>        但她的话，着实让他心里松了口气。
<br/>
<br/>        “其实，在云城酒店的那天晚上，我去你房间找你的时候，有个女孩正从你房间里出来。她的情况看起来非常糟糕！她从你房间出来的时候，身上的裙子破了，头发凌乱，脖子上满是吻痕。她却跟我说，你没把她怎么样，希望我不要误会，还让我当什么也没看见。但我觉得她在撒谎。所以，你应该去对那个女孩负责。”李忆莎的表情忽然有些苦涩。
