　　蛰龙？
<br/>　　蛰伏起来休眠的恶龙吗？
<br/>　　“龙鹰！”
<br/>　　安东尼奥斯突然从背后叫住了蛰龙。
<br/>　　“嗯？”
<br/>　　走到门口的蛰龙，停下了脚步。
<br/>　　“你的那支小队也一起带走吧。”
<br/>　　安东尼奥斯苍老的声音从蛰龙身后传来。
<br/>　　听到安东尼奥斯的话，蛰龙脸上神情微变，但是他并没有转过身来，对着自己身后的安东尼奥斯问道：“他们忠诚于你，如果你想要让这个世界因为你而颤抖，你需要他们的力量。”
<br/>　　“好。”
<br/>　　蛰龙没有多说什么，轻轻点了点头，拉开房门走了出去。
<br/>　　蛰龙前脚刚离开，一道人影便是从房间屋梁上跳了下来，站在安东尼奥斯面前，问道：“鹰皇，为什么不把他留下来？”
<br/>　　“留得下来吗？”安东尼奥斯看了眼站在自己面前的刺客，说道：“当年就没能留下他，更何况是现在？你觉得你隐藏的很好，他没有发现你吗？并不是，他只是装作没有发现你而已。”
<br/>　　刺客沉默着没有说话。
<br/>　　“如果刚才你出现，你觉得你赢得了他吗？”安东尼奥斯看向沉默中的刺客，还没等这个刺客开口，安东尼奥斯就直接替他做了回答：“你赢不了他，即便是我们两个人联手也赢不了他。”
<br/>　　“可如果我们真的要留下他，也不是不能做到。”刺客执拗的说道，似乎是想为刚才安东尼奥斯那句“我们两个联手都赢不了他”找回一些面子。
<br/>　　“要想留下他，自然是可以做到，但是值得吗？”安东尼奥斯看看着自己面前的刺客，说道：“我们两个都会死，而为了留下他，‘刺客会’至少要损失一半的人手，但是不要忘了，他已经不是当年独自一人的龙鹰了。他敢这么明目张胆走进来，并且威胁我们帮他做事，自然是早有准备，他手下还有四个人，我相信，这四个人在他跟我谈判的时候，已经到了其他几个长老的房间里。除此以外，这里还有他的小队，这会让我们产生严重的损失并且得不到实际的利益。”
<br/>　　年迈的安东尼奥斯走到窗户边，看着院子中的蛰龙，说道：“我们都老了，现在是年轻人的天下了。”
<br/>　　“可即便如此，也不用让他把那支小队带走啊，他们都是精英。”刺客有些着急的说道。
<br/>　　“你知道那支小队是精英，我也知道那支小队是精英，可是我们能运用起他们吗？”安东尼奥斯朝着房间中的刺客反问了一句，随后径直回答道：“不能！他们努力训练保持自身的状态，使自己变得更加精锐，不是为了让自己为我们所用，而是为了等待他们的王归来的一天！即便是我不说，他也是会把他们这些人全都带走的，与其这样，还不如送他一个顺水人情，这样也不至于让他把我这个老师唯一一点感情都抹杀干净，这算是我欠他的。当年那一剑虽然没有能杀死琴魔，可还是让他本就残破的身躯伤上加伤……”
<br/>　　说到此处，安东尼奥斯忍不住叹了一口气，年迈的脸上浮现出一缕疲惫。作为“鹰皇”接掌“刺客会”这么长时间，安东尼奥斯自问自己没有做错什么，如果唯一有错，那就是当年通过了众位长老的意见，执意留下蛰龙，并且重伤了“琴魔”方牧南。
<br/>　　……
<br/>　　代码小镇，停车场。
<br/>　　David通过卫星电话跟公司高层取得了联系，在说明了自己当前的情况后，他要求立刻取消这次行动，降低损失；但是公司方面却拒绝了David的要求，明确表示，这次行动不会被取消，并且告诉David会有人来支援他们的行动。
<br/>　　“混蛋，这些该死的，无耻的混蛋！”
<br/>　　David挂断了电话，嘴里不停咒骂着，在他看来，所谓的有人会来支援，根本就是公司方面单方面的说辞罢了。
<br/>　　“公司那边怎么说？”
<br/>　　副队长Bob猫着腰来到David身边，从战术背心中拿出一个备用弹匣装入手中战术步枪中，向David询问公司那边的意见。如果国内公司方面同意他们撤退，他们会立刻退回到小镇中，只要进入了小镇，罗昊那边自然就不能再对他们攻击，不然他们就会遭到跟Henry一样的下场。
<br/>　　当然Bob并不会知道，只要有蛰龙这层关系在，即便是罗昊真的带着人冲进小镇，小镇里的“刺客会”刺客也不会出现；不仅Bob不会知道，就连罗昊自己也不会知道。
<br/>　　“what！？”
<br/>　　Bob皱眉看着David说道：“公司这是什么意思，就这样把我们丢在这里了？也没有说是谁会来支援我们，会什么时候来支援我们，说不定等到公司所说的支援赶到的时候，我们全都已经挂了。”
<br/>　　“呵呵，黑水安保部队也不过如此。”
<br/>　　就在这时一个突兀的声音出现在David和Bob身后。
<br/>　　David和Bob立刻转身把枪口对准自己身后的人，他们看到黑色兜帽衫和牛仔裤的毒天使站在距离他们不到五米的地方。David眼中露出警惕之色，在这警惕中还带着一丝后怕和恐惧，如果刚才对方没有出声，而是直接下杀手，那自己还有命吗？答案自然是否定的！
<br/>　　这停车场上打得热火朝天不假，可是以自己的警觉性竟然没有发现有人靠近自己身后，而且还要对方出声说话才发现，这个人实在太恐怖了！
<br/>　　David能想明白的事情，作为“十二主天使”之一的毒天使自然也是能想明白，嘴角勾起不屑，看着David和Bob说道：“如果我不是你们的援军，现在你们已经是死尸了。”
<br/>　　“你就是我们的援军？”Bob看着毒天使问道。
<br/>　　毒天使点了点头。
<br/>　　“开什么玩笑！？”Bob再次开口，语气中充满着难以置信，说道：“就你一个人能有什么用？”
<br/>　　“人多就一定有用吗？”毒天使看也不看Bob反问道：“如果现在给你们一百个娃娃兵，你们觉得就一定能打赢对面那些人吗？战争，兵员多的一定占有优势，但现在这并不是战争！”
