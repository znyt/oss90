        &lt;h3&gt;第三卷第三百九十四章  不知道该怎么办&lt;/h3&gt;
<br/>
<br/>        江俞轩还是不相信陈子昂走了，昨天下午他们还在冯睿家，今天陈子昂就飞欧洲了？
<br/>
<br/>        陈子昂是走了吗？为什么不给他说？
<br/>
<br/>        江俞轩挂了郝景文的电话，给陈子昂打电话。
<br/>
<br/>        电话里传来冰冷的女声：“你拨打的手机已经关机，请在滴声之后留言。”
<br/>
<br/>        江俞轩拿着电话痴痴的站着，陈子昂走了竟然连招呼都不给他打，她该是有多么伤心，她就这样走了，连给他解释的机会都没有了。
<br/>
<br/>        “俞轩，怎么了？”江俞琪从会议室走出来，已经晚上了，大家该下班了。
<br/>
<br/>        “陈子昂走了。”江俞轩深深的叹了口气，那种无力而失落的感觉侵蚀着他的心。
<br/>
<br/>        “走哪里去了？”江俞琪看着自己的弟弟，她的这个弟弟啊，智商情商都很高，就是一到自己感情上的事情就卡壳了。
<br/>
<br/>        “去欧洲了，全家都走了。”江俞轩说到全家都走了的时候心里一震，陈子昂以后还会回来吗？她会不会以后也去欧洲居住？
<br/>
<br/>        “全家都走了？”
<br/>
<br/>        “不是，是送.......”江俞轩突然就住口了，陈子昂有孩子的事情只有他们小范围的人知道。
<br/>
<br/>        “她去欧洲找她朋友，然后去甸城搞投资，那边有项目，可能一时半会儿不会回来的。”江俞轩转换了口气。
<br/>
<br/>        “那你一副伤心欲绝的样子，又不是生离死别，有那么难过吗？话说这么久都没有见到倩楠了，也没有见你这么难过。”江俞琪撇了撇嘴。
<br/>
<br/>        “姐，我会和张倩楠离婚。”江俞轩语出惊人。
<br/>
<br/>        “你？吃饱了撑得？你们出什么问题了吗？何苦呢？”
<br/>
<br/>        “不瞒你说，我和张倩楠这么久以来，连最起码的男女之间的亲昵的举动都没有，偶尔拥抱一下，碰碰额头、拉拉手。”江俞轩苦笑了一下。
<br/>
<br/>        “是你的问题还是倩楠的问题？”
<br/>
<br/>        “应该是我们俩个人都有问题，从过年在张家举行了简短的仪式，张倩楠就在疏远我，而我因为张倩楠的疏远，我发现了一个问题。”
<br/>
<br/>        “什么问题？”
<br/>
<br/>        “我喜欢陈子昂！”
<br/>
<br/>        “你是因为张倩楠的疏远才发现你喜欢陈子昂的吗？”
<br/>
<br/>        “不，应该是在张家和张倩楠举行仪式之后，我给陈子昂打电话听到陈子昂的哭声，我才知道我放不下她，我应该一直陪在她身边，陪着她走过今后的人生，而不是促成她和别人在一起。”
<br/>
<br/>        三月份的西城已经是春天了，院子里的花蕊在兴奋的冒着泡，微风吹在脸上，划过一丝温柔。
<br/>
<br/>        江俞轩和江俞琪面对面站着，江俞琪看着自己弟弟那帅气的脸微不可察的叹了口气，和陈子昂在一起工作六年，才发现自己喜欢人家，他是有多么迟钝啊。
<br/>
<br/>        “那你准备怎么做呢？”
<br/>
<br/>        “忙过这一段时间，我就像张倩楠提出离婚，重新去追回陈子昂，只要她不嫁人我就有机会。”
<br/>
<br/>        “那你知道陈子昂爱你吗？”江俞琪看着弟弟，“你可别一厢情愿，最后连朋友都没得做了。”
<br/>
<br/>        “她不反感我，我觉得她的心里是有我的。”此时的江俞轩不确定陈子昂会不会喜欢他，会不会爱他，如果她爱他，那么临走的时候怎么会不给他说呢？
<br/>
<br/>        “你觉得？”江俞琪带着疑问。
<br/>
<br/>        “姐，你是女人，如果一个女人在一个男人身边不做作，想哭就哭，想笑就笑，那么她是不是很喜欢这个男人？”江俞轩问江俞琪。
<br/>
<br/>        “陈子昂经常在你身边又哭又笑？”江俞琪的好奇心起来了。
<br/>
<br/>        “之前她不知道我身份的时候，我们在一起她会很自然的不自觉的抓住我的手，会很开心的和我说着一些事情，有时候喝醉了，会在我脸上“啵”那么一下。喝醉了的时候她会说要和我过一辈子。”
<br/>
<br/>        江俞轩记得有一次他们出去应酬，陈子昂又喝多了，喝多了的陈子昂抓住他的两只耳朵，眼睛亮晶晶的看着他说：“江俞轩，我要和你过一辈子，你答不答应？”然后在他脸上“啵”一下，不等他回答便昏沉沉的睡过去了。他替她整理好手包和衣服，让她舒适的靠在座椅上，送她回家。
<br/>
<br/>        想起往事，江俞轩心里是酸酸甜甜的感觉，嘴角泛起一丝甜蜜的笑。
<br/>
<br/>        “然后呢？”
<br/>
<br/>        “后来她知道我的身份和张倩楠的事情之后，她就疏远我了，而我不久也就到西城这边了，那一天我差点就说出“我喜欢她”了，她却拍了我一巴掌说我脸上有蚊子，然后我一生气冲着她喊：我喜欢陈天天。”
<br/>
<br/>        “陈天天是谁？”
<br/>
<br/>        “是她的侄女，比我们小了差九岁。”
<br/>
<br/>        “误会闹大了吧？”
<br/>
<br/>        “我们从北京回来的时候，她在机场抱着我哭，说怕我们都要离开她，我觉得我们的关系应该能缓和，可是昨天她又听到了我说我喜欢陈天天，她走到我身边骂我可渣了。”江俞轩苦恼的摇摇头。
<br/>
<br/>        “一次是误会，两次就不是误会了，那你到底喜欢谁？喜不喜欢陈天天？”
<br/>
<br/>        “陈天天是很讨人喜欢，但是我只当她是一个小辈，况且昨天是陈天天让我说的。”江俞轩不知道该如何解释了，他给姐姐江俞琪都解释不清楚，又如何能给陈子昂解释得清楚？况且陈子昂这一走，连解释的机会都没有了。
<br/>
<br/>        “江俞轩，你这左摇右摆，举棋不定，不知道自己该怎么办，合该你为情所困，现在还是单身狗一枚！”江俞琪越听越气。
<br/>
<br/>        “我哪有举棋不定啊！”
<br/>
<br/>        “怎么没有啊，你喜欢陈子昂吧，陈子昂也算是喜欢你，可是你在她身边呆了那么久，你竟然享受着陈子昂对你的感情还和张倩楠俩谈情说爱，现在说喜欢陈子昂？你早些时候干嘛去了？”
<br/>
<br/>        “你不明白情况嘛，那个时候，那个时候倩楠说张函和子昂俩个人，俩个人.......”江俞轩长叹一口气，好烦躁好复杂啊！
<br/>
<br/>        “张函和陈子昂俩个人怎么了？”
<br/>
<br/>        “他们俩个人连孩子都有了，张倩楠极力的想撮合张函和陈子昂复合，一直让我给他们汇报陈子昂的情况，我在陈子昂的身边就是他们的卧底，我就算是喜欢陈子昂又能怎么样？”
<br/>
<br/>        “你说什么？”江俞琪大惊，张函和陈子昂怎么会有孩子？
<br/>
<br/>        江俞轩在姐姐的紧逼一下透露了太多的信息给江俞琪，江俞琪感觉很震惊。
