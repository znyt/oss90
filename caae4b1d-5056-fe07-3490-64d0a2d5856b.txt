<br/>　　嘭！嘭！嘭！
<br/>　　闷响不断，电光闪过，立刻便有六名巫族天才脑袋爆炸，连惨叫都来不及发出一声，便惨死当场。

<br/>　　嘭嘭嘭！！
<br/>　　二十八道电光飞舞，一名名巫族天才的脑袋被洞穿，一名名巫族天才的脑袋炸裂。几乎无人能挡住本命灵剑一击。
<br/>　　就算这些巫族天才，战力超强，个个有武帝境的实力，也根本不是苏莫一招之敌。
<br/>　　几乎是呼吸之间，便有二十余名巫族天才，当场惨死。
<br/>　　“什么？”众巫族天才大惊失色，所以人都发现了这些漫天飞舞的电光，只不过，这些电光实在太快，快到只有一道模糊的影子，让人根本无法抵挡。
<br/>　　“哈哈哈！”
<br/>　　“杀！”
<br/>　　一众苍穹神宫之人，却是精神振奋，更加奋力的攻击了起来。
<br/>　　他们当然知道，这是苏莫的本命灵剑在攻击。
<br/>　　有了苏莫的帮助，他们灭杀这些巫族天才，不成问题。
<br/>　　而这时，不死鸟已经不再攻击巫天御，同样开始袭杀普通的巫族天才。
<br/>　　不死鸟实力极强，只是比苏莫稍弱一筹而已，其庞大的身躯遨游虚空，没一爪抓出，都能捏爆一名巫族天才的身躯。
<br/>　　一时间，一众巫族天才，开始了快速的陨落。
<br/>　　……
<br/>　　苏莫一心二用，一边控制本命灵剑袭杀巫族天才，一边开始向巫天御疯狂的攻击了起来。
<br/>　　唰唰唰！！
<br/>　　光影一闪，三具魔影分身再次出现在了苏莫的身边。
<br/>　　“斩！”
<br/>　　苏莫怒喝一声，和三具魔影分身齐齐出手，联手向巫天御攻击。
<br/>　　剑光闪耀，四道浩大的剑气，如同一片巨大的剑网，裹挟滔天威势，斩灭千里空间。
<br/>　　苏莫现在看似手段众多，实则他承受的压力非常的巨大。
<br/>　　他已经收回了大日武魂，因为他根本无法做到一心多用。
<br/>　　无论是控制二十八柄本命灵剑，还是催动十种剑武魂，亦是控制三具魔影分身，无一不是靠着他强大的精神力。
<br/>　　就算他天生精神力强大，远超常人，也吃不消了，感觉脑袋胀痛，心神疲累。
<br/>　　所以，他收回了大日武魂。
<br/>　　“可恶啊！”巫天御简直要疯狂了，苏莫一边和他对战，居然还能一边袭杀他的族人。
<br/>　　最主要的是，他还束手无策，短时间内也无法击败苏莫。
<br/>　　“给我死吧！”巫天御怒喝一声，身上气息突然大涨，瞬间暴涨了三、四倍。
<br/>　　他的气息变得无比的恐怖，如同爆炸的火山，将其渲染的似一位神灵，藐视天下，俯瞰苍生。
<br/>　　咻！咻！咻！！

<br/>　　这些枪影，每一道都比之前的枪影强大数倍，威势浩荡九天，仅仅是气息，便让整片空间被撕裂，出现了一道长达千里的空间裂缝。
<br/>　　无数枪影碾过虚空，几乎是瞬间便将苏莫和三具魔影分身的攻击碾成了碎片。
<br/>　　“不好！”
<br/>　　苏莫见此大惊失色，他的脚下立刻出现了一柄灵剑，身形急速暴退。
<br/>　　苏莫目前共有二十九柄本命灵剑，大星宿剑阵，只需要二十八柄，他还有一柄灵剑留着备用。
<br/>　　而三具魔影分身，则是直接向漫天枪影冲去，而后立刻自爆了。

<br/>　　三声惊天爆响，整动方圆数万里，猛烈的爆炸席卷八荒。
<br/>　　不过，三具魔影分身的自爆，根本挡不住漫天的枪影，狂暴的爆炸之力，被漫天枪影洞穿的千疮百孔。

<br/>　　对方显然动用了某种秘术，他若是不用，很难抵挡的住对方。
<br/>　　苏莫身上的气息同样开始了暴涨，无限拔高，狂暴不已。

<br/>　　这一点苏莫也曾想过，可能是修为越高，实力增加的比例越低吧！
<br/>　　不过，修为低之时，就算增加十倍的实力，也不如现在增加一倍来的大。
<br/>　　毕竟，以前增加的一倍实力，与现在增加的一倍实力，完全不可同日而语。

<br/>　　二十八柄灵剑的不断袭杀，再加上不死鸟的强大实力，片刻之间，众巫族天才便陨落了一半，还剩下不足四百人了。
<br/>　　局势，已经完全逆转了过来。
