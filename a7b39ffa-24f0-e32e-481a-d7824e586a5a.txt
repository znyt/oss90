　　见她着急的像只炸毛的小猫，慕清霄忍不住就想要调戏一下，小医仙心里乱糟糟的，捏起粉拳就在他胸膛轻轻的打了起来。
<br/>　　慕清霄握住她纤细的手腕，道：“先别闹了，他们进来了。”
<br/>　　闻言，小医仙急忙将目光投向石门口，果然是听见脚步声越来越响亮。
<br/>　　“呵呵，小医仙，真是多谢你们带路，看来我得到的这个消息，果然不假啊。”
<br/>　　话语刚落，十几道人影，逐渐从门外的黑暗中走进石室中，熟悉的淡笑声，在石室内部，颇为得意的响了起来。
<br/>　　听到这熟悉的声音，小医仙顿时银牙紧咬：“穆力！”
<br/>　　漆黑的石门处，十几道影子，逐渐从门外的黑暗中走了进来，最后将石门堵死。
<br/>　　随后，一道人影从后面走出，最后在月光石的照耀下，露出了其面目，正是狼头佣兵团的少团长，穆力。
<br/>　　目光先是在石室内部的几堆金光闪闪的金币上扫过，穆力眼中掠过一丝贪婪，舔了舔嘴唇，视线瞟了瞟已经被慕清霄两人打开的石盒，不由微笑道：“看来我运气不错啊。”
<br/>　　小医仙美眸中尽是愤怒，道：“你跟踪我们？”
<br/>　　“算不上跟踪吧，早在几日前，我便得到了小医仙寻找到宝洞的情报，不过因为不知道确切位置，所以只好由你亲自带路了。”
<br/>　　穆力含笑着说道，只是当他看见，慕清霄正握着小医仙的手腕后，脸色顿时阴沉如水，满含杀意道：“小子，你真是不识好歹，连我的女人都敢染指。”
<br/>　　慕清霄看了一眼小医仙，见她思索着什么，松开她的手腕，但右手却搂在她的柳腰上，冲着穆力微微一笑，随后直接无视了他。
<br/>　　小医仙俏脸一红，想要挣开，奈何慕清霄抱的太紧，只得愤怒的盯着穆力，道：“你是如何得知情报的？这件事我只跟住菲力提过，难道......菲力被你收买了？”
<br/>　　“呵呵，那女的挺傻，不过随便用了点花言巧语，便乖乖把什么东西都告诉了我。”
<br/>　　穆力虽然在笑，但却是冷笑，眼神盯着慕清霄，就像是在看一个死人。
<br/>　　“你简直是个混蛋。”柳眉倒竖，小医仙叱骂道。
<br/>　　“这些东西对我们狼头佣兵团太重要，只要用有了它们，我们便能吞并青山镇的所有势力，到时候就能向外发展，我的目光可不只是限于小小的青山镇。”
<br/>　　深情的望着小医仙，穆力声音柔和道：“把东西给我，小医仙，你应该清楚，我对你的感情，只要你跟着我，我往后绝不亏待你。”
<br/>　　“跟着你？我现在和你说话，都觉得恶心！”
<br/>　　红润的小嘴挑起一丝嘲讽，小医仙的声音中满是怒意，显然穆力收买她身边的人，让她极其的愤怒。
<br/>　　“是因为这小白脸吗？没事，等我把他杀了，在把你强行留在身边。”
<br/>　　穆力眼中满是阴冷，盯着慕清霄，道：“早就让你加入狼头佣兵团，要怪就怪你不识时务，敢染指我的女人，你别想活着走出洞府。”
<br/>　　“区区一个佣兵团，斗师团长，居然如此猖狂？”慕清霄不怒反笑，笑容中尽是不屑。
<br/>　　“那是因为我有猖狂的资本，就比如杀你易如反掌，把东西都交出来，我给你留个全尸。”
<br/>　　穆力眼中杀意凛然，站在石门中央，满脸戏谑的望着慕清霄，他就要看着小白脸能撑到什么时候。
<br/>　　“听说你天赋不错，不过翅膀没长硬就敢得罪我，你今天必须死在这里，否则我寝食难安啊！”
<br/>　　这时，小医仙抬起臻首，望着依旧是面带淡然微笑的慕清霄，美眸中闪过一丝温柔，这家伙何时何地都保持着这种让人舒心的笑容呢。
<br/>　　“待会我用催眠粉，你趁机逃走吧，说到底都是我害了你，他们不会拿我怎么样的。”
<br/>　　闻言，慕清霄眼底闪过一丝难以察觉的温柔，右手在她柳腰上捏了捏。不敢说话：“你真是太善良了，当初说你是菩萨心肠果然不错，可这话你自己都不相信吧？”
<br/>　　“相信我，区区一个佣兵团而已。”
<br/>　　关键时刻，小医仙也不忍责怪他轻薄自己，比起厌恶的穆力，慕清霄在她心目中，简直就是王子啊。
<br/>　　他平淡的话语中，却有极强的说服力，小医仙乖巧的点头，将希望都寄托在慕清霄身上，如果可以，她也不想落在穆力手上。
<br/>　　“动手，杀了那小子，注意别给我伤了小医仙，那可是我内定的女人！”
<br/>　　望着两人，在他眼皮子底下打情骂俏，穆力再也忍不住，手掌一挥，命令身后的斗者雇佣兵杀了慕清霄。
<br/>　　见五名雇佣兵满脸凶光的朝两人扑来，慕清霄只是随手一挥，一股恐怖的劲风，便将无名斗者掀飞出去，随后狠狠的撞在石壁上。
<br/>　　转眼间，五名斗者雇佣兵就生死不知。
<br/>　　见慕清霄一瞬间解决五名雇佣兵，小医仙玉手捂住粉唇，美眸中尽是不可思议。
<br/>　　穆力惊得后退一步，惊叫一声，道：“这小子隐藏了实力，你们一起上，给我杀他，等回去以后，我提拔你们做我的专属护卫！”
<br/>　　原本，见识到慕清霄的可怕，剩余的雇佣兵都有些惧意，但听到能够做穆力的专属护卫后，顿时野心滋生。
<br/>　　专属护卫的待遇，至少能让他们少奋斗大半辈子啊，而且跟着穆力吃香的喝辣的，还有女人玩，简直是肥差中的肥差啊！
<br/>　　见剩余的十几名雇佣兵目露凶光，直接朝自己扑来，慕清霄面色依旧平淡。
<br/>　　“不知死活。”
<br/>　　微微抬手，空中瞬间出现一排由灵气凝聚而成的小剑，小到肉眼都看不见，随手一挥，灵剑就破空而出。
<br/>　　瞬间，迎面而来的十几名雇佣兵都顿住了脚步，脸上的凶光凝固，最后莫名的倒在地上，致死都不知道自己怎么死的。
<br/>　　只是随手一挥，便解决十几名斗者雇佣兵，而且实力都在四星都五星斗者，穆力简直被吓坏了，脸色变得惨白，双腿都在打颤。
<br/>　　至于小医仙则是惊呆了，对于死去的雇佣兵不但不害怕，甚至有一丝快意。
