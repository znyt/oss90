　　“段惊天，怎么样了？父亲有没有收你为徒？他为何让我离开？是不是有什么秘密？”
<br/>　　见到苏莫的分身出来，一直在等待的沁儿，急忙迎了上来，小嘴如同连珠炮一般，不停的发问。
<br/>　　“没有！”分身摇了摇头。
<br/>　　“什么没有？是没有收你为徒？还是没有啥秘密？”沁儿急忙询问，又是连珠炮般问了三个问题。
<br/>　　“什么都没有，没有收我为徒，也没有什么秘密！”分身摇了摇头，大步而去。
<br/>　　“什么都没有……！”
<br/>　　沁儿看着大步离去的苏莫分身，一脸怔怔，不过，她倒是并未有什么意外。
<br/>　　毕竟，想要让他父亲收徒，那可不是一件容易之事。
<br/>　　只是，她想不明白，父亲为何要将她驱出大殿。
<br/>　　沁儿又回头看了看玄天神殿，随即转身向神殿中走去。
<br/>　　不过，到了神殿之中，她却是并未找到自己的父亲，最终只好无奈的离去。
<br/>　　……
<br/>　　苏莫分身回到惊天府之后，便静静的等待了起来。
<br/>　　一日无话，翌日清晨，分身离开了惊天府，再次向玄天神殿走去。
<br/>　　没过多久，他便到达了玄天神殿门前。
<br/>　　“拜见玄天长老！”苏莫向这紧闭的殿门，抱拳一拜。
<br/>　　嗡~~
<br/>　　殿门轻轻一颤，随即缓缓打开了。
<br/>　　分身抬目看去，只见神殿的大厅之中，赫然有两人。
<br/>　　主座之上，不是玄天长老，而是一名矮壮老者。
<br/>　　此人身穿褐色长衫，头发略显花白，面容肃穆，容貌和玄天长老很是相似。
<br/>　　而玄天长老，则端坐在老者的右下首。
<br/>　　看到这名老者，分身目光微闪，他猜到了此人是何身份。
<br/>　　能在玄天长老面前，端坐在主座之上，在人王宫只有一人。
<br/>　　那就是人王宫的宫主，被称为人德神王。
<br/>　　殿门打开了，老者和玄天神王，目光均是落在了苏莫分身的身上，如同四柄利箭，犀利无比。
<br/>　　分身心中一凛，这两人的目光，虽然没有杀气，但是，却让他如芒在背。
<br/>　　“拜见宫主，拜见玄天长老！”
<br/>　　分身走进殿内，向老者和玄天神王，郑重的行了一礼。
<br/>　　嘭！
<br/>　　这时，只见老者大袖一挥，大殿的殿门瞬间关闭，发出一声低沉的闷响。
<br/>　　“你叫段惊天？”老者凝视着苏莫分身，沉声问道，声音洪亮而威严，显得中气十足。
<br/>　　其没有否认宫主的身份，显然正是人王宫的宫主。
<br/>　　“正是！”分身点了点头，心中却又有些忐忑，结果如何，就看今日了。
<br/>　　“你可以留在人王宫，但是，我们不会给你多大的庇护！”宫主沉声开口，开门见山的说道。。
<br/>　　“嗯？”分身心中疑惑，对方此言何意？
<br/>　　可以留在人王宫，却不庇护他！
<br/>　　这让分身心中暗暗警惕，对方不会是想要安抚于他，然后找到他的本体吧！
<br/>　　“你、帝一魂、拓跋青云，你们三人之间的争斗，人王宫不会插手。”宫主继续道。
<br/>　　“宫主，在下不明白，难道你们就任由这样下去，难道你们不想让轮回天尊复生？”分身微眯着眼眸问道。
<br/>　　这一切，未免太耐人寻味了吧！
<br/>　　“想，不仅我们想，整个神域的所有巅峰强者都想！”
<br/>　　宫主面色沉静，眸光悠远，继续道：“不过，我们想的是，让天尊打破桎梏，而不是单纯的重生。”
<br/>　　“打破桎梏？”分身闻言，心中惊疑，这是说所有人都希望轮回天尊再进一步吗？
<br/>　　可是，就算要再进一步，也要先帮助轮回天尊复生吧！
<br/>　　不然的话，人都没有复活，怎么能再进一步？
<br/>　　“宫主，晚辈不明白，还请解惑！”分身郑重抱拳，面带期待之色。
<br/>　　他的确是有太多的不解了，不解轮回天尊，不解众人的态度，不解整个神域的态度。
<br/>　　帝一魂和拓跋青云的存在，不是秘密，相反还是人尽皆知。
<br/>　　但是，很多人都对于此事，闭口不提。
<br/>　　而且，若是其它势力希望轮回天尊复生，地王宫根本就保不住拓跋青云，而其它势力，若是压根就不想让轮回天尊复生，早就灭了帝一魂。
<br/>　　但是，这一切都平静无比，没有人对付帝一魂，也没有人对付拓跋青云。
<br/>　　“段惊天，你想要人王宫庇护你，但你还没有资本！”
<br/>　　宫主答非所问，继续道：“地王宫支持拓跋青云，那是因为拓跋青云乃是地王宫的嫡系血脉，而天王宫支持帝一魂，那是因为其和天王宫建立了密不可分的关系，而你，并不值得我们帮你！”
<br/>　　“如何才能值得你们帮我？”分身沉声问道。
<br/>　　“让我们看到希望，看到你战胜帝一魂和拓跋青云的希望！”宫主面无表情的说道。
<br/>　　“宫主，我若是战胜了他们，天尊岂不是永远无法复生了？”分身惊疑不定的问道。
<br/>　　他的心中，越发的疑惑了，既然希望轮回天尊复生，还希望轮回天尊打破桎梏，但是，他若是能战胜帝一魂和拓跋青云，轮回天尊还怎么复生？
<br/>　　这简直是不符合常理，其中定然有某种隐秘。
<br/>　　“不，你们谁能笑到最后，谁便是天尊！”宫主微微摇头说道。
<br/>　　若是眼前的段惊天，在修为和天赋之上，都超越帝一魂和拓跋青云，他自然愿意帮忙。
<br/>　　但是，可惜，眼前之人，修为实在太低下了，根本没有抗衡帝一魂和拓跋青云的希望。
<br/>　　“哦？”
<br/>　　分身闻言，顿时心中微震，皱眉问道：“宫主的意思是，无论谁是主魂，谁是分魂，谁是最后的胜利者，谁就是轮回天尊？”
<br/>　　“不错！所以，你没有希望成为天尊，我们自然不会帮你！”宫主微微点了点头。
<br/>　　当然，他不会帮苏莫，也不会去帮帝一魂和拓跋青云。
<br/>　　因为，他需要的不单单是所有分魂合一，轮回天尊复生。
<br/>　　而且，让轮回天尊，在复生的同时，能打破桎梏。
<br/>　　“我根本就不想成为轮回天尊！”分身心中腹诽，当然，这话他并没有说出来。
<br/>　　“那宫主为何不帮助帝一魂，融合拓跋青云？”分身再次问道。
<br/>　　“老夫已经说了！”宫主微微的摇了摇头。
<br/>　　分身见此，顿时眉头微皱，对方的确是说了很多，但是，却没有清晰的让他明白。
