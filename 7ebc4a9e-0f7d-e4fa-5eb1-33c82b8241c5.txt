　　“小鹿刚才说从地图上，看到了他家门口的大公鸡，所以，我和晓佳分析，小鹿家，就在公园对面的金鼎大厦。”
<br/>　　小惠展开一张地图，指着上面一个十分像公鸡的建筑，那个地方具体这里，距离并不算特别遥远。
<br/>　　“恩，看来我们的计划已经很明确了，放心，我一定会把你们安全送入撤离点的。”
<br/>　　记下了地图上的位置，白小飞眼神坚定，像众人保证道。
<br/>　　“好啦，先吃完饭再说，说起来，你们感觉我的手艺怎么样？”
<br/>　　眼睛弯成了月牙，晓佳催促着，这可是她除了父亲之外，第一次给别的男人做饭。
<br/>　　“还用感觉嘛，简直是太棒了！”白小飞竖起了拇指，夸赞着：“我们快点吃吧，吃完，一起去找小鹿的妈妈。”
<br/>　　众人应了一声，开始享用起桌上的食物。
<br/>　　……
<br/>　　太难吃了。
<br/>　　握着筷子的手有些颤抖，刘哔拼命压抑着胃中想要呕吐的感觉，那种剧烈的恶心感，像是喷泉一般一波波的翻涌了上来。
<br/>　　这只不过是……区区一碗面条啊！
<br/>　　与其面对这样的痛苦，他甚至宁愿去和实力与自己想通的能量体战斗。
<br/>　　但他必须要忍受这些，不仅仅是因为伪装，更是为了磨练自己。
<br/>　　一个男人，怎么能让这些东西给限制住。
<br/>　　只不过是一些食物而已，迟早他要和常人无异的将这些东西给吃下去。
<br/>　　“对了，晓佳，一会你给我找一把大一点的菜刀，还有胶带。”
<br/>　　没有注意到刘哔的异样，白小飞一边吃着饭，一边对晓佳交代着事情。
<br/>　　这毕竟不是他家，这些东西在哪，他还不太清楚。
<br/>　　“啊？”
<br/>　　闻言，晓佳有些疑惑，但也没多问，转身进厨房准备去了。
<br/>　　直至十分钟之后，她才明白，白小飞用这些东西，是要做什么。
<br/>　　“已经准备好了吧，那么，我们出发！”
<br/>　　看着全副武装的众人，白小飞单手握着自造的长矛，推开门口，一马当先的走了下去。
<br/>　　小区内，尸兄并不算多，有小胖和白小飞在，这些低级尸兄根本翻不起任何浪花。
<br/>　　只是丢失了自己的唐刀，小胖现在，也只能拿着白小飞为他制作的菜刀当做武器。
<br/>　　人群中，最为奇葩的就是刘哔，这货手中拿着一根黑不溜秋的棍子，如同烧火棍一般，卖相十分难看。
<br/>　　毫无阻碍的冲出来小区，众人都开始变的有些谨慎了起来，毕竟外面的街道上，尸兄的数量不在少数。
<br/>　　单纯的尸群，对他们来说都不会有太大的威胁，只是谁都不知道，其中会不会有实力强大的进化尸兄。
<br/>　　“走过这个大桥，就快到目的地了。”往上挪了挪装着小鹿的背包，白小飞停下了脚步：“有点不对劲，这个桥，好安静啊。”
<br/>　　经历了这么多天，他对一些特殊的情况也有了了解，像这样安静的地方，要么就是有别的幸存者刚刚经过，将这里的尸兄全给引走或者屠杀了，要么就是这里有强大的变异动物。
<br/>　　如果是前者倒勉强还好说，后者的话，那就很危险了。
<br/>　　“这是前往撤离点的必经之路，估计是有什么人，之前从这里经过了吧。”
<br/>　　嘀咕了一声，白小飞拄着长矛，小心翼翼的向前走去。
<br/>　　“叔叔，叔叔，你快看，美人鱼，美人鱼姐姐！”
<br/>　　小鹿的声音，将众人都吸引了过来，顺着小鹿的手指，众人看到了河中缠绕在一起的两道身影。
<br/>　　淡金色的头发，窈窕的身材，再加上水下若隐若现的鱼尾，怪不得让小鹿认成了美人鱼。
<br/>　　“应该是变异的尸兄吧。”眉头皱了一下，白小飞总感觉哪里不太对劲。
<br/>　　但很快，他就知道这股不对劲的感觉是来源于什么地方了，水下，一个巨大的阴影缓缓浮现，接近了两名变异尸兄。
<br/>　　“饿饿饿~”
<br/>　　“好饿~”
<br/>　　似乎嗅到了桥上几人身上的气味，尸兄转过头来，露出两张令人作呕的脸庞，但还不等它们有所动作，一个更加狰狞的巨大怪物，一口将其吞了下去。
<br/>　　“这，这是什么！”
<br/>　　小惠惊呼一声，刚才惊鸿一瞥的巨大身影，给她留下了难以磨灭的印象。
<br/>　　怎么会，有这么狰狞的怪物！
<br/>　　“河里全是变异的怪物，已经被污染了么？”冷汗从白小飞脸上落下，他已经感受到事情的不妙。
<br/>　　如果连海洋也都被污染了，那人类该怎么存活下去，所谓ZF，真的有能力将病毒给控制住吗？
<br/>　　这，也是所有人心中的疑问。
<br/>　　但谁都不敢主动提出，撤离点，ZF，是她们唯一的希望。
<br/>　　“啊……！！！！”
<br/>　　一阵凄厉的惨叫声，从大桥上猛然响起，众人心中都是一惊，转头看去，发现了前方一个背负长刀的男子。
<br/>　　“好不容易抓到个活人，却连瓶发胶都找不出来……真没用！”
<br/>　　不太清晰的话语被风声扯得粉碎，但众人却都听明白了背刀男子话语中的意思。
<br/>　　“我去，这家伙在干嘛！”白小飞脸色一变，语气有些愤怒：“他刚才，居然把一个活人给扔下去了？！”
<br/>　　“我认识他！”小惠惊呼着开口：“他是血莲教的一个头目，通缉很久了，可我们一直没有抓到他。”
<br/>　　顿了一下，小惠指了指背刀男子身后的长刀，继续说道：“而且，他的那把刀非常厉害，不知道有多少抓捕他的队伍，都死在了那把刀下。”
<br/>　　听到这么硕大的来头，众人脸色都是微微一变，异能人士，他们也听说过一些，这种人很稀少，但无一不是非常强大。
<br/>　　甚至就连ZF军队遇到这样来头的人，都占不到任何好处。
<br/>　　“咦？”
<br/>　　似乎是听到了这边的话语，背刀男子转过身来，走到了众人面前。
<br/>　　“小女孩，居然知道我的身份。”捋了一下脑后扎起来的头发，背刀男子在众人面前站定，虽然面无表情，但眼中的凶光，却是无论如何都遮盖不住：“那么，今天在场的各位，就都给我留在这吧！”
<br/>　　话语落音，男子抽出背后的长刀，随手放在地上，瞬间，一层蛛网般的裂缝，从桥面上缓缓裂开。
