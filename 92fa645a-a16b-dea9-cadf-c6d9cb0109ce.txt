　　第26章
<br/>　　飞虎寨，高飞虎坐在大树下，直到现在还气得不停颤抖。
<br/>　　“太无耻了，挖人竟然挖到我们飞虎寨，不行，拿我的大刀来，我要去找他们算账。”
<br/>　　“大哥息怒，大哥息怒。”高飞豹拼命摁住他。
<br/>　　“我能息怒嘛，这是明摆着上门欺负人呐！”高飞虎。
<br/>　　“一切等军师回来，一切等军师回来。”高飞豹。
<br/>　　高飞虎气鼓鼓的坐在那里，感觉自己胸腔像风箱一样发出呼呼的响声。
<br/>　　“你觉得是真的么，去了就有兵器啊？”
<br/>　　“还顿顿白面烙饼呢。”
<br/>　　“啧，想到我就流口水，要不咱们去碰碰运气。”
<br/>　　坐在大树下，时不时可以听到这样的议论声，虽然他们已经极力压抑声音，但还是难免传到高飞虎耳中。
<br/>　　“哼，还顿顿白面烙饼，我都没有这么奢侈。”高飞虎冷哼一声。
<br/>　　可高飞虎不得不承认，蛤蟆寨这个法子虽然无耻，倒是有效。现在山寨内已经人心浮动，如果人真的被蛤蟆寨挖走，该如何是好。
<br/>　　“大哥，我都是有个主意？”高飞豹道。
<br/>　　“咦，你竟然有主意！”高飞虎觉得今天出乎预料的事情一件接着一件。
<br/>　　“他不是想要从我们这里挖人嘛，我们干脆将计就计，派点弟兄过去，吃他们的，喝他们的，如果有机会的，直接一举将蛤蟆寨占领。现在，蛤蟆寨可是有不少粮食。”
<br/>　　说罢，高飞豹发现高飞虎睁大眼睛看着自己，满脸不可思议。
<br/>　　“大哥，怎么了？难道是我的法子不好么？”
<br/>　　“不是，我只是没想到，你竟然知道将计就计这个词。”高飞虎。
<br/>　　……
<br/>　　周小树今年十七岁，是飞虎寨的一名小喽啰。他的父亲是山贼，母亲是山贼，所以，他生下来就是山贼。
<br/>　　山贼的日子其实没那么好，说是大块吃肉大块喝酒，而周小树已经有一年多没吃过肉了。这两年年景不好，山寨种得田很多时候都是颗粒无收。没粮的时候，大家还要漫山遍野挖野菜。不过据说，山下的生活更惨，一整年要给财主做长工，还要交各种税赋。
<br/>　　这些都是军师说的，周小树其实连税赋是什么意思都听不懂。
<br/>　　他只是一个小山贼，最小的那种。
<br/>　　今天他和山寨二当家一起，去蛤蟆寨入伙，二当家说了，这是一条计策，属于反间计。其实他也不懂，他听二当家的。
<br/>　　等到了蛤蟆寨，才发现寨门口人满为患，许多山寨都来了人，那个姓黄的男人坐在一张桌子后，说是什么要登记。
<br/>　　大家都在排着队登记，二当家要插队，被人挡了回来。二当家这个人……有时候挺逗的。
<br/>　　等排到他们的时候，姓黄的男人递过来一张纸，据说是要签字。周小树不认字，二当家认字，知道得也不多。所以他也不知道上面写着什么，只学着前面的人在纸上画了个圈，圈画的不是很远，让周小树觉得有些丢人。
<br/>　　画过圈就代表自己是蛤蟆寨的人么？二当家已经说了，这次咱们是打入敌人内部，不管怎样，都得记得自己是飞虎寨的人。
<br/>　　结果有人给自己递过来一柄锯，让你干活去。干活，干什么活？一个长得很俊的姑娘说，好好干活，中午吃饺子。
<br/>　　饺子，那不是过年才吃的吗，记得去年自己吃了一碗，好几天都不想吃野菜饼了。她说得是真的么，估计骗自己吧，那能天天吃饺子。
<br/>　　既然让自己干活，自己就干活吧，二当家也没意见。不过自己分配的任务是伐树，二当家是去挖石头，他别的都没有，有把力气，虽然日头很毒辣，但对于七八岁就开始干活的周小树来说，不算什么。
<br/>　　只是，中午真的能吃到饺子嘛？
<br/>　　很忙碌的一个上午，所有人都在干活，却也不觉得累。不过日头没到正午，就有人让自己停下，说是开饭了。
<br/>　　午饭，真的是饺子。这一点真的出乎自己预料，白面野菜馅的饺子，里面好像还有肉，蛤蟆寨没有养猪，应该是野猪肉。自己见有一个大汉打猎回来，几个人扛着猎物，山鸡，狍子……这人打猎的手艺真好，如果自己也有他的手艺，每天估计都可以吃肉。
<br/>　　端着饺子自己还不敢吃，问二当家的意思，二当家说吃他的，吃得越多越好。
<br/>　　自己吃了三碗，真的咽不下去了，没想到，自己有一天还会咽不下饺子的时候。要知道自己过年的时候多吃两个，老爹就要骂娘了。
<br/>　　下午又是同样的伙计，有人指挥着，让自己做什么就做什么。这点活对他来说，却也不算什么。
<br/>　　晚上竟然是白面馒头，大锅里蹲着野菜，周小树真的被镇住了。这简直比过年还有奢侈啊，蛤蟆寨天天这么吃，会造天谴的吧。
<br/>　　不过，到晚上的时候，自己却被蛤蟆寨赶走了。难道是因为自己吃得太多了，后来才知道不是。
<br/>　　那个姓黄的男人说：你们现在还不是蛤蟆寨的人，只能说是合同工，目前是实习期，每天两顿饭，一人五斤粮食。如果想干活的话，明天还可以来。
<br/>　　『合同工』『实习期』自己都听不懂，不过那五斤小麦却不是假的。不过干了一些活，吃了人家两顿饭，临走还给自己粮食，这样连吃带拿的，不太好吧。
<br/>　　回到山寨后，大家听说今天的事，都怀疑自己说的是假的。可那五斤小麦却让他们不得不信，有人想用一柄刀给我换，被我拒绝了。虽然那柄刀自己很喜欢，但老爹和老娘也差不多一年没吃过白面馒头了。
<br/>　　只是，听说有人向大当家提意见去了，说为什么这样的好事不让他们去，大当家是不是偏心。
<br/>　　被他们这么一闹，自己明天就去不成了吧，这样的好事也不能天天是自己。可万一呢……实在不行，自己就偷偷去，一天就是五斤粮食啊，多干几天，冬天就不害怕了。
<br/>　　今年的年景可是不太好啊，老天已经很长时间没下雨了。
<br/>　　躺在炕上，胡乱想着这些事，周小树很快就沉入了梦想。
