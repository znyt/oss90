　　嗖！嗖！嗖！
<br/>　　破空声不绝于耳，不断的有太阴族内族之人，来到武斗场所在的空间，而后飞向北方看台。
<br/>　　整个武斗场，渐渐的安静了下来，所有人都望着这一幕。
<br/>　　内族之人到了，会武马上就要开始了。
<br/>　　苏莫紧紧的盯着入口处，期待青璇的到来，虽然对方来了之后，他也不能做什么，但还是希望能亲眼见到对方。
<br/>　　五年了！
<br/>　　从他离开苍穹世界，到如今整整五年没有见过青璇了。
<br/>　　而且，在苍穹世界之时，青璇便一直在黄泉魔宗的手上，可谓是长达十年都没有自由。
<br/>　　每每想到此处，苏莫都有些心痛，为青璇的遭遇感到悲哀。
<br/>　　虽然如今，青璇已经获得了自由，但以自己的记忆为代价，这个代价未免太过巨大了。
<br/>　　“怎么还不来？”苏莫心中略有些焦急，因为已经有不下数千名太阴族内族之人到来，其中居然没有青璇。
<br/>　　片刻之后，约莫来了近万名太阴族之人，而这个时候，入口处则是没有人再出现了。
<br/>　　“没来！”
<br/>　　苏莫暗叹一声，心中既无奈又郁闷，一瞬间，他已经没有任何继续待在武斗场的心情了。
<br/>　　就在此时，只见入口处人影重重，突然又进来了数十人。
<br/>　　这数十人，为首之人是一名高高瘦瘦的中年人，鹫鼻鹰目，眸光如电，身上气息极为庞大，赫然是一名武圣境的强者。
<br/>　　而在这名武圣境强者的身后，跟着数十名太阴族之人，有男有女，有老有少。
<br/>　　“青璇！”苏莫目光瞬间一凝，因为他在人群中看到了青璇。
<br/>　　只见在那名武圣境强者的侧后方，轰青璇款款而立，身着一袭湖蓝色的绒裙，身姿高挑，轻柔妙曼。
<br/>　　她的头发很长，如同黑色的瀑布一般，垂落到腰下。
<br/>　　此时的宏青璇，虽然容貌没有丝毫的改变，但气质却与以往大不相同，褪去了青涩，变得娴静、淡雅、高贵、脱俗，配合上其绝世的容颜，形成了一种绝代的风华，毫无意外，瞬间便吸引了很多人的目光。
<br/>　　此时青璇的气场，仿佛比前面那位武圣还要强大，成为了现场的焦点。
<br/>　　神女云悠悠，可是名动太阴星，名传明虚星河，集美貌和天赋于一身，自然能吸引绝大多数人的目光。
<br/>　　苏莫甚至有一种错觉，他发现青璇的气质，和其师尊芊寻月很像，甚至可以说是非常的相似。
<br/>　　那名武圣境的强者，一路不停，带着宏青璇等人，虚空踏步，径直走到了北方的看台，而后在看台的最上方端坐了下来。
<br/>　　“见过阴泽长老！”
<br/>　　“见过阴泽长老！”
<br/>　　“见过阴泽长老！”
<br/>　　东南西三方看台之上，所有人均是站了起来，向那位中年武圣抱拳行礼。
<br/>　　太阴族不像是其它的势力，长老很少，因为只有达到武圣之境，才能成为太阴族的长老。
<br/>　　所以，每一个长老地位都非常的崇高。
<br/>　　苏莫也同样起身抱拳，不过，他的目光却是一直落在青璇的身上。
<br/>　　此时，青璇坐在阴泽长老的身边，如水的目光亦是在环视整个武斗场。
<br/>　　对于苏莫的目光，宏青璇并未察觉，因为落在她身上的目光，实在是多不胜数。
<br/>　　苏莫深深的吸了口气，压下心中上前与青璇相认的冲动，他不能太贸然，不然只会死路一条。
<br/>　　“诸位免礼，请坐吧！”阴泽长老袖袍一挥，朗声轻喝，声音如雷霆降世，声震十万里。
<br/>　　随即，众人纷纷落座，等待阴泽长老的指示。
<br/>　　阴泽长老目光如电，环视全场，朗声道：“七大势力的会武，一直是我太阴族检校你们七大势力天才的盛会，每一届都有奖励，这一届也不例外。
<br/>　　现在，本长老宣布此次的奖励，第四名至第十名，可以获得免费在九重浮屠塔顶层修炼一个月的机会，以及一人一柄尊级中品宝器。
<br/>　　第三名，免费在九重浮屠塔修炼一个月的机会，以及一柄尊级上品宝器。
<br/>　　第二名，在第三名的基础上，奖励一门尊级上品武技。
<br/>　　第一名，在第二名的基础上，能得到本族虚神境大能的一次指点。”
<br/>　　阴泽长老声音洪亮，传遍全场，将此次会武的奖励，宣布了出来。
<br/>　　嘶！嘶！嘶！
<br/>　　阴泽长老话音一落，全场顿时响起了一阵倒吸冷气的声音。
<br/>　　这一届会武的奖励，居然如此之高！
<br/>　　不说浮屠塔免费修炼的机会，单是尊级上品宝器、尊级上品武技、虚神境大能的一次指点，这每一种奖励，对于武帝境武者来说，都是可遇而不可求的好处！
<br/>　　尊级上品的宝器和武技，比之尊级下品的宝器和武技，不知道珍贵多少倍！
<br/>　　像道宗的弟子，就算日后成为了道宗真传弟子，短时间之内，都不可能得到这种奖励。
<br/>　　而虚神境大能的指点，这更是重中之重，好处无穷。
<br/>　　当然，众人虽然感到心惊，但也没有太过意外，因为能在会武中得到前十名的弟子，将来达到武尊境之后，会被慢慢吸收进太阴族，最终成为太阴族的一份子。
<br/>　　吕罡、林皇、唐古，炎火、墨狼等等各大势力的顶尖天才，均是眸中精光闪烁，战意澎湃。
<br/>　　这次的会武奖励如此之大，他们就更要进入前十名了，哪怕没有宝器和武技的奖励，单单是九重浮屠塔免费修炼一个月，就是巨大的奖励了。
<br/>　　整个武斗场，七大势力的弟子，均是眸中战意澎湃，誓要在会武中取得好成绩，进入前十名。
<br/>　　唯有苏莫一人，对于奖励毫不关心，他目光望着青璇，心中暗暗的思索，如何能接近对方。
<br/>　　现在武斗场这么多人，青璇还和阴泽长老以及大量的太阴族内族之人坐在一起，他根本没有接近的机会。
<br/>　　这时，阴泽长老继续开口了，朗声道：“至于本次会武的规则，依旧是按照以往的规则，自由挑战；战胜一场，便得一分，最后以积分定名次，至于战败之人，败一场扣一分，败三场，将失去再挑战与被挑战的资格；还有就是，一轮比试，每人只能挑战一次，而且不得挑战本轮中已经战斗过的人！”
<br/>　　“明白了没有？”说完之后，阴泽长老朗声喝问，声音威严无比。
<br/>　　“明白！”七大势力的弟子齐齐应喝，这依旧是以往的规则，虽然不算公平，但胜在过程较为简单。
<br/>　　苏莫闻言讶然，居然使用这种比赛规则，那要比到什么时候！
