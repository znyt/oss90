　　返回真圣界耗费的时间，和去万界山差不多，约莫用了八、九天的时间。
<br/>　　当苏莫等人回到真圣界之时，又回到了原地，被传送离开的地方。
<br/>　　不远处，那巨大的旗杆依旧闪烁着九彩之光，在疯狂的吸收着灵气。
<br/>　　瞥了一眼那旗杆，苏莫心中暗想，这真圣界所在的这颗星辰，基本上已经是半废星辰，灵气很是稀薄，不会是都被这旗杆给吸收了吧？
<br/>　　“我们走！”
<br/>　　真贤道圣招呼一声，率先飞了出去，顺着那处虚幻的空间通道，飞到了通道之外，那一片名亮的空间中。
<br/>　　几人一路不停，直接就从那处明亮的空间之中飞了出去，飞到了真圣碑之外。
<br/>　　此刻的真圣界，已经没有半个人影，一片寂静，只有硕大的真圣碑屹立在虚空之中。
<br/>　　那考验的青色平台、圣门等，早已消失不见。
<br/>　　“真贤师兄！”几人停了下来，苏莫目光望向真贤道圣。
<br/>　　“苏莫，你暂缓离开，本道为你分身再施一法，随后你分身和姜神夜先行离开！”真贤道圣说道。
<br/>　　“好，多谢真贤师兄！”苏莫点了点头。
<br/>　　随即，真贤道圣仔细的打量苏莫和他的分身，看看还有哪些破绽之处。
<br/>　　大的破绽基本没有，分身的外表和气息完全和苏莫一样，没有任何分别，不仔细查探之下，不会发现有假。
<br/>　　巫族之人的族印，在巫人王本身的血气被压制之后，也消失不见了。
<br/>　　但是，大祭司巫仑可是武圣境武者，眼力何等毒辣，还是很容易被对方看出端倪。
<br/>　　这和当初苏莫冒充巫尘完全不同，他冒充巫尘之时，身体便是巫尘的身体。
<br/>　　现在则是用别人的身体，来冒充他自己，完全不是一个概念。
<br/>　　“苏莫，本道现在要施展一种秘法，名为衍神之法，需要抽取你一部分精气神！”真贤道圣说道。
<br/>　　“师兄动手吧！”苏莫没有拒绝，他相信对方不会害他，以对方的修为实力，没有理由暗中害他。
<br/>　　苏莫话音一落，真贤道圣动手了，大手一张，直接盖在了苏莫头顶之上。
<br/>　　随即，苏莫只感觉一阵头晕目眩，而后体内大概两成的精气神，被真贤道圣强行抽取了出来。
<br/>　　精气神仿若一团淡薄的雾气，被真贤道圣抽取之后，在其掌中不停的变化。
<br/>　　片刻之后，这团精气神化为了一个硕大的符印，弥漫着玄妙的气息。
<br/>　　随后，真贤道圣将这精气神化成的符印，直接从分身的天灵盖一巴掌排进了分身的体内。
<br/>　　分身全身一震，很快便吸收了苏莫的精气神。
<br/>　　“好了，一刻钟之内，就算是武圣境中期武者，也不会看出任何端倪！”真贤道圣轻笑道。
<br/>　　“多谢师兄！”苏莫和分身同时向对方抱拳道谢。
<br/>　　“嗯！”
<br/>　　真贤道圣微微颌首，随即目光看向分身，道：“准备好了吧？我送你和姜神夜出去！”
<br/>　　“准备好了！”分身点头说道。
<br/>　　随即，真贤圣师手捏印诀，就准备让苏莫分身和姜神夜传送出去，但立刻又停手了。
<br/>　　“对了，忘了告诉你们，你们加入了万界山，很快就会有第一个任务安排下来，快则三月，慢着一年。”真贤道圣说道。
<br/>　　“是！”苏莫和姜神夜点头。
<br/>　　随后，真贤道圣不再浪费时间，手掌飞快的变化，捏出玄妙的法诀。
<br/>　　随着法诀的生成，一股无形的力量笼罩在姜神夜和苏莫分身的身上。
<br/>　　下一刻，两人身影一闪，消失在了真圣界内，只留下真贤道圣和苏莫的本体。
<br/>　　真圣界外。
<br/>　　人影满山遍野，布满了方圆千里的丘陵地带，嘈杂的声浪此起彼伏。
<br/>　　“到底是怎么回事？”
<br/>　　“这都半个多月了，苏莫和姜神夜怎么还不出来？”
<br/>　　“两人不会死在了里面吧？”
<br/>　　“应该不可能，通过了考验之后，得真圣之气，能有什么危险？”
<br/>　　“真是奇怪，其他人早就出来了，他们两人居然滞留了半个多月！”
<br/>　　各种讨论声、疑惑声不绝于耳，很多人都是满脸费解之色。
<br/>　　因为其他真圣碑留名的人早已出来，苏莫和姜神夜两人，却在真圣界内滞留了半个多月，到现在还未出来！
<br/>　　有些不相干势力的人都已经离开了，虽然很多人都很好奇巫族如何对付苏莫，但苏莫迟迟不出现，他们也没有继续等下去。
<br/>　　此时，真圣界外的人，还剩下约莫数百万人。
<br/>　　逍遥剑圣和三神宗之人，面色稍稍凝重，因为以往基本上没有出现过这种状况。
<br/>　　按照正常的情况，融合一股真圣之气，最多也就十几个时辰，而一名天才，顶天了也只能融合三至四股真圣之气，万万不会耗费大半个月的时间。
<br/>　　“逍遥长老，姜师兄不会出事了吧？”一名三神宗的弟子，面色凝重的望着逍遥剑圣。
<br/>　　“继续等吧！”
<br/>　　逍遥剑圣微微摇了摇头，他哪里会知道有没有出事？按照最后出来的飞长云和黑锋的话，他们出来之时，苏莫和姜神夜都在融炼真圣之气，并没有什么变故。
<br/>　　一众神武学府之人，亦是等的焦急不已。
<br/>　　木璃、吴晋等一众导师，急的像是热锅上的蚂蚁，苏莫如此的妖孽人物，若是有事的话，是神武学府的巨大损失。
<br/>　　但他们毫无办法，无法进入真圣界，只能继续等待，若是再等一个月，苏莫还没有出来，他们只能先行返回神武学府了。
<br/>　　“死了最好！”
<br/>　　尤天韩和一些太史世家派系的人，则是丝毫不急，多是带着恶毒的心思，暗暗希望苏莫死在真圣界中。
<br/>　　尤天韩的伤势，经过半个多月的疗伤，已经好了小半，但想彻底恢复，还需要一段时间。
<br/>　　“尤师兄，这苏莫要是死了，你也算是报了仇了！”一名太史世家派系的弟子，轻声对尤天韩说道。
<br/>　　“嗯，只可惜不能亲眼看到他是怎么死的！”尤天韩叹道，苏莫将他打成如此重伤，让他无法得到真圣之气，此仇不可谓不深。
<br/>　　他自然希望苏莫死，只不过无法亲眼看到，也是一个不小遗憾。
