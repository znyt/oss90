　　所有能凝视着战台上的巫尘，眼眸一眨不眨。
<br/>　　巫尘的脸色已经苍白了下来，最多两个呼吸的时间，就要败了。
<br/>　　众人不禁感慨，强如巫尘都丝毫不是巫魔的对手，今年族中的少司选拔，巫魔有不小的希望啊！
<br/>　　时间缓缓而逝。
<br/>　　两个呼吸的时间转瞬而过，就在这时，原本弥漫整座战台的迷蒙雾气，突然剧烈的翻滚了起来，而后全部进入了巫尘的体内。
<br/>　　随即，面色呆滞的巫尘，眼眸之中立刻恢复了神采。
<br/>　　众人见此，蓦然一怔，个个面现不可思议之色。
<br/>　　怎么回事？
<br/>　　巫尘怎么醒了？难道逃脱了巫魔的梦境？这怎么可能？
<br/>　　苏莫苏醒之后，心中长长的出了一口气，这巫魔的战魂当真强大，构筑的梦境也堪称无敌。
<br/>　　不过，不论是什么战魂，都只是他吞噬战魂的养料。
<br/>　　现在，这不可思议的梦魔战魂，已经成为了他的战魂。
<br/>　　随即，苏莫抬眼看向对面犹如石化一般，已经没有生命气息的巫魔，抬脚向对方走了过去。
<br/>　　走到巫魔的身前，苏莫抬手摘下了对方手上的储物戒指，收了起来。
<br/>　　嘭！
<br/>　　这时，巫魔高瘦的身躯直挺挺的倒了下去，发出一声沉闷的响声。
<br/>　　整个武斗场死一般的寂静，巫魔身躯倒下发出的声音，虽然不大，却如同闷雷一般，在众人的心中炸响。
<br/>　　巫魔死了！
<br/>　　六个呼吸的时间，巫尘没死，巫魔却死了！
<br/>　　所有人都懵了，完全没有想到，会出现这种截然相反的结果。
<br/>　　巫魔为什么会死，巫尘又是如何战胜巫魔的，所有人都是百思不得其解
<br/>　　就连一些武尊境的长老人物，都是有些懵逼。
<br/>　　不过，武尊境的长老人物，比普通的巫族之人眼力要高明的多。
<br/>　　他们均是看出了一些端倪，巫尘有古怪啊！战魂绝对不简单。
<br/>　　适才，巫尘的身上分明出现了一股奇特的力量，将巫魔的梦魔战魂吸收了。
<br/>　　这是什么力量？
<br/>　　一些长老人物也想不明白，但他们知道绝对不简单，巫尘很可能有两种战魂。
<br/>　　“我就问一句，还有谁不服？”苏莫目光环视四方，朗声高喝。
<br/>　　不过，这次再也没有人上台了，连号称在黑耀城拥有同阶无敌实力的巫魔都死了，谁还敢上台？
<br/>　　巫源长老脸色铁青，心中愤然不已，这巫尘怎么可能如此强大？
<br/>　　但是，现在已经无人敢挑战巫尘了，他暂时也没有办法了，只能日后再找机会出手了。
<br/>　　“既然无人敢战，那便结束了！”足足等了半刻钟，也无人上台，苏莫淡淡的说了一句，便走下了战台，大步离开了武斗场。
<br/>　　直到苏莫离开了武斗场，武斗场中才哗然四起，嘈杂的议论声此起彼伏。
<br/>　　这一战，彻底奠定了巫尘的名声，一举进入超级天才的行列了。
<br/>　　“有意思！”巫天御静静的坐在看台上，微眯着眼眸，低声自喃。
<br/>　　他感觉巫尘隐藏的很深，身上仿佛隐藏了一层迷雾，深不可测。
<br/>　　这也激起他心中的战意，只不过，他现在才武帝境二重境界，不可能是巫尘的对手。
<br/>　　苏莫离开了武斗场之后，依旧是前往了地灵塔，再次来到了地灵塔的第二层空间之中。
<br/>　　接下来的时间，他不准备离开地灵塔了，准备一直闭关到巫族少司选拔开始。
<br/>　　坐在修炼室中，苏莫暗自思索，接下来的时间中，他不仅要将混天魔功第一层修炼精深，还要将本体的根基要巩固扎实，甚至突破到武帝境三重境界。
<br/>　　然后，他要将这具巫尘的肉身，炼化成本体的独立分身。
<br/>　　倒时候，他的分身成为巫族的少司，他的本体便可以返回神武学府了。
<br/>　　当然，临走之前，他必须想办法在巫族狠捞一笔。
<br/>　　太史轩和尤天韩两人，苏莫不会放过他们，敢派人杀他，就要承受他的报复。
<br/>　　随即，苏莫静下心神，准备尝试炼制独立分身。
<br/>　　手臂一挥，他将自己的本体放了出来，和他相对而坐。
<br/>　　而后，他意念一动，吞噬战魂直接返回了本体之中。
<br/>　　重新掌控本体，让苏莫稍稍有些不适应，毕竟离开的时间不短了，但很快他便恢复如常。
<br/>　　他要将巫尘的肉身炼化成分身，自然要回到本体，不然就会将本体练成分身了。
<br/>　　随即，苏莫仔细的思索了一番，要给分身什么战魂？
<br/>　　少倾之后，他意念一动，一片迷蒙的雾气从体内喷涌出，充满了整座修炼室。
<br/>　　这正是巫魔的梦魔战魂！
<br/>　　此战魂非常的逆天，若是给分身使用，绝对犀利无比，但是，这么强大的战魂，他却是有些不舍得给予分身。
<br/>　　最厉害的战魂，当然还是本体使用更好。
<br/>　　沉默了一番，他收回了梦魔武魂，决定将赵城的‘金刚猿战魂’，以及巫烈的‘烈焰石战魂’给予分身。
<br/>　　这两种战魂虽然单个不算多强，但毕竟是两种战魂，有了这两种战魂，分身可就相当于双生战魂的天才了。
<br/>　　当然，给予分身一种战魂，都有些难度，给予两种战魂，难度将更加的巨大。
<br/>　　少倾之后，苏莫开始尝试分化自己的神魂，他的神魂已经融于吞噬战魂之中，想要分化还要不少的时间。
<br/>　　但这并没有多大的难度，不多时，他便按照阎魔鬼圣说的方法，从吞噬战魂之中生生分化出了一缕神魂。
<br/>　　这缕神魂很微小，只有苏莫神魂的百分之一。
<br/>　　神魂分化成功，立刻让得苏莫脸色略显苍白了起来，神魂可是他的生命本源，分化出百分之一，都对于他有不小的伤害。
<br/>　　随即，苏莫立刻释放出了金刚猿战魂，并且控制着这缕分化的神魂，直接融入了金刚猿战魂之中。
<br/>　　而后，这缕神魂和金刚猿战魂缓缓相融，这个过程虽然很慢，耗费了两个时辰，但却是非常的顺利。
<br/>　　因为金刚猿战魂，已经被苏莫的吞噬战魂吞噬融炼，和他之间有无形的联系，气机几乎相同。
<br/>　　然后，苏莫一刻不停，立刻又释放出了烈焰石战魂。
<br/>　　接下来，他要让那缕融进金刚猿的战魂，和烈焰石战魂再次融合。
<br/>　　这一次，苏莫不再顺利，遇到了很大的困难。
