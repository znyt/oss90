　　玄天带着一批巫族回到人族祖地，与燧人氏三人商量了人巫联盟一事的具体事宜。同时，玄天也发现了人族之中的高端战力。
<br/>　　五百年的时间，燧人氏三祖自然不用多说，有人族气运加身，更有天道功德的庇护，修炼速度自然很快，都已经修炼出属于自己的天地法相，相当于洪荒大罗金仙。
<br/>　　人族中的洞天武尊境，也出现了一大批，足足有数百人，至于神通王者境，那就更多了，足足有近千人。
<br/>　　武道修炼前期只是基础，修行自然很快，但是从神通境以后，便会慢了下来，因为以后的修炼，是需要悟道与真气积累同时进行的。
<br/>　　人族中多出如此多强者，玄天自然高兴，但同时，也有许多人族丧生于危机四伏的洪荒之中，对此，玄天除了惋惜与沉痛之外，还能说什么。他知道，人族想要崛起，这些牺牲是在所难免的，以后甚至还会有更大的牺牲。
<br/>　　他只能将目光投向更远处，期待未来更加美好！
<br/>　　同时，随行来的巫族强者，一样在暗自惊骇人族出现的这些强者。据他们所知，人族出生才不到两千年时间，人族中竟然就会有如此多的强者，这简直是不可思议。还有，为何人族的武道修炼会与巫族的有些相似？
<br/>　　他们发现，人族的潜力，绝对是巨大的。不过，他们也没有多说什么，他们是为了与人族通婚而来，可不是为了其他目的而来。
<br/>　　“不周山乃洪荒中心，若是爆发大战，不周山定是首当其冲被毁灭，所以，我打算让人族迁往别处。”玄天看着不周山，对着燧人氏等人说道。
<br/>　　“如今洪荒哪里都一样，不过，长久来说，不周山的确是不安全。大哥，人族要迁往何处？”
<br/>　　燧人氏听说这个消息，有些不舍道，毕竟在不周山已经生活了两千年，难免对此处有些感情。
<br/>　　“我这些年游走洪荒，见过不少名山大川，我觉得首阳山不错，适合人族居住。”
<br/>　　“既然大哥觉得不错，那便搬走吧，留下不周山我也觉得不妥。”燧人氏思索了一番，轻声道。
<br/>　　“好，既然如此，那我们便准备准备，立刻搬走。”
<br/>　　玄天不再犹豫，说话做事雷厉风行，第二天便将有用的东西全部收走，有着巫族的护航，再加上人族四祖，以及这些年到洪荒中历练归来的武者们的保护，人族浩浩荡荡地向着首阳山搬迁而去。
<br/>　　首阳山靠近东海之滨，东海又有龙族盘踞，三清岛也隐匿在东海，那里倒是没有什么大凶大恶之辈，相对其他地方要平静许多。
<br/>　　迁徙之路，凶险无比，尤其是当一些妖族发现，吞噬人族，可以提升功力之后，更是将这份凶险提升无数倍。死亡的灾难，更是铺天盖地一般的席卷而来。
<br/>　　虽然人族之中，也有不少武者，但终究是修行时日尚短，即便有玄天、燧人氏等几人，但无论如何，也不能护住全部的人族。一时之间，人族遭遇前所未有的灾难，在迁徙途中，不断有妖族肆掠杀戮人族，而且伴随着越来越多的妖族知道，吞噬人族能够增长修为后，这种屠杀，更是一发不可收拾！
<br/>　　尸骨累累，人族一路之上，惨叫声，哭泣声，嘶喊声，绝望声……声声震耳！
<br/>　　还好，有巫族的护航，玄天等人族武者，更是在这一路之上，屠杀了不知多少妖族，其中，死在玄天手中的妖族，就有数万之多。
<br/>　　就这样，一路之上，不知屠杀了多少妖族，不知有多少人族死在这场迁徙之途中，最终，人族度过千险万难，终于踏入了首阳山范围。
<br/>　　在玄天清理完山阳山范围内的大部分妖族强者后，人族登上首阳山，就此建房修道，定居安家。
<br/>　　妖族依旧不断地前来人族之地肆无忌惮地屠杀人族，不过有着巫族与人族武者的不断巡逻与反抗，这种情况对比之前的迁徙途中遇到的，无疑是好了不知多少。
<br/>　　“记住，眼前这一幕幕，不久的将来，人族定要让他们还回来！”
<br/>　　有人目眦尽裂地看着妖族屠戮人族，对着自己的子孙这样说道。
<br/>　　“妖族，是人族的仇敌，将来强大了，我一定要将它们屠杀！”
<br/>　　这无疑是在人族的内心深处，买下了一颗复仇的种子，这颗种子经过仇恨的浇灌，早晚会茁壮成长。
<br/>　　“妖族，该死！”
<br/>　　“人族与妖族不死不休！”
<br/>　　看着无数族人在这场战争之中被妖族强者屠杀，人族的绝望，人族的疯狂，他们都将深深的记在心中，等待着将来疯狂的报复。
<br/>　　就这样，在人族安居下来后，人族之中，每一个人族心中都充满的干劲，武者数量在不断增加，人族中的强者也在不断增加。
<br/>　　同时，巫族也与人族开始联姻了，两族渐渐地开始融合。事实的确如玄天所说，两族联姻所生的生命，一半几率为人族，一半几率为人族，而且这些巫族之中，也可以修炼元神。
<br/>　　对比其他种族，人族别的不怎么样，但就这生育能力，人族绝对能够占据洪荒之中的前列。
<br/>　　不管死了多少人，人族很快便能够再生育出来。
<br/>　　看着人族后来者源源不断地涌现，一代接着一代的强者不断出现，玄天虽为人族武祖，也颇为兴奋。
<br/>　　人族在首阳山重建女娲庙，人祖殿，燧人氏等人族强者在人祖殿中闭关深修。玄天在首阳山选了一个地方，那是首阳山最高峰。
<br/>　　他在此建立武宫玄天殿，更采集首山之铜，以及一些这些年他游历洪荒收集到的仙材、神材，道金，炼制了一件九层宝塔，名为武塔！
<br/>　　武塔之中，每一层都收押了无数妖族，这些妖族是人类武者在洪荒中带回来的，他们被镇压其中，一生不得外出。而且每层塔中的妖兽实力也各不相同，越往上，实力越强。玄天在武塔最高层放了许多天才地宝，或者玄功仙法，神通秘笈等，供人族武者前来历练。
<br/>　　若是人族武者能斩杀一定妖族数量，通关成功后，能够根据完成程度到武塔之中兑换武者所需要的东西。
<br/>　　这相当于人族的一件藏宝库或者藏经阁，同时也不失为一件培养人族强者的宝贝。
<br/>　　人族中除了留在部落中看守族地的武者外，其他的武者依旧如之前那般，到了金丹境，便要出首阳山，到洪荒之中杀妖历练，只有这样，人族中的强者才会如涌泉一般不断涌现。
<br/>　　生于忧患，死于安乐！
<br/>　　只有在艰苦奋斗之中，更加艰难的环境中，才能培养出更高质量的强者。
<br/>　　就这样，洪荒之中又是匆匆而过数千年。
<br/>　　在这数千年年之中，人族强者不断涌现，人族已经有了一定的底蕴。如此快速的发展，已经引起了巫妖二族，以及其他种族的注意，不过，暂时还没有人来将这种平衡打破，小打小闹倒是不少。
<br/>　　同时，百年前，洪荒发生了一件大事件——
<br/>　　紫府被灭！
