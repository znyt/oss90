　　比赛虽然结束了，但是赛后关于荣耀蓝队的讨论却是一直没有停止过。
<br/>　　甚至就连许多国外的媒体，甚至就连韩国的媒体在看到荣耀蓝队在夏季赛揭幕战的时候轻取雷霆战队，这个时候也都是开始推测荣耀蓝队可能是今年最大的夺冠热门了。
<br/>　　而另外一个讨论的就是林雅楠说出的那个惊喜了，之前叶帝也在说夏季赛的时候会有惊喜，如今林雅楠旧事重提，这自然让大家期待万分了。
<br/>　　“怎么样，今天我们的发挥给力吧，说真的，没上场的时候我就知道已经赢了。”坐在一起吃晚饭的时候，蒋念在那边开口吹嘘着。
<br/>　　“然而并没有什么卵用，solo的时候你差点输给了小艺。”听着蒋念的吹嘘，叶帝点头开口称赞着。
<br/>　　听着叶帝的前半句，蒋念那边还很开心，然而当叶帝后半句说出来的时候，蒋念的脸色却是一下子垮了下来，和小艺solo这可能是自己人生中最大的败笔了。
<br/>　　“明天真的要让小艺上场？”一旁的秦霄好奇的开口问道，他的开口倒是缓解了蒋念的尴尬，同时将目光全部拉倒了叶帝的身上。
<br/>　　“肯定的啊，小艺现在已经完全有实力去担任白队的中单了，现在让她上场多攒些经验也好，老秦你也是时候让清浅多上场比赛了，打完雷霆，接下来你们B组没人能威胁到你们了，就算是清浅发挥失误，那蒋念他们也可以带赢的。”叶帝开口说道。
<br/>　　听着叶帝的话，坐在餐桌上吃饭的清浅顿时露出了一副期待的表情，就是连杨小艺这个时候也是变的有些激动了起来。
<br/>　　“行，那清浅你也好好准备准备，下一场蓝队的比赛你上来打辅助，正好磨炼磨炼，不过今年你是别想多打了，今年可是我最后一年的表演了，我还想多在场上待几天呢。”秦霄笑着开口说道。
<br/>　　虽然是笑着，不过在坐的众人都能够听的出来他内心中的那抹苦涩，身为一个职业选手，如果不是自己的状态不好了，谁又愿意离开这个舞台呢。
<br/>　　“那肯定的啊，今年肯定还是师傅们的舞台，我和小艺两人就是熟悉熟悉大赛经验，现在我们的实力还没强到那个地步呢，只有你们才能创造出蓝白的传奇时代。”清浅笑着开口说道。
<br/>　　虽然他的年龄不大，但是说出的话语却是分外暖心，也正是因为他这么单纯，所以叶帝他们几个都拿清浅当做弟弟来看待。
<br/>　　一天的时间很快便过去了，在无数粉丝的期待下，第二天终于到来，这一天荣耀白队将会迎战OG战队。
<br/>　　OG战队自从被荣耀战队这边收购之后，叶帝他们便是再也没有管过对方，实在是他们太忙了，没有那个时间去照顾他们，而且今年他们的主要目标是集体闯入世界决赛，所以更加的无法分心了。
<br/>　　好在OG战队也争气，在春季赛的时候还是拿到了一个比较好的名次，他们的排名在一线队伍中处在中游偏上的队伍，仅次于王朝那个梯队。
<br/>　　当叶帝他们一行人到达现场后，现场顿时爆发出一阵热烈的欢呼声，此刻所有的座位都已经坐满了人，大家都在那里等待着荣耀白队的首站。
<br/>　　“雅楠听说今天白队会给大家带来一个惊喜，现在他们已经到场了，这下你可以说出来了吧！”解说席上米粒一脸好奇的问道。
<br/>　　“还要等一等哦，反正比赛马上就要开始了，大家也不必太过着急。”林雅楠笑着开口说道。
<br/>　　“咦？今天白队好像多了一个人啊，怎么多了一个女孩，那是谁的妹妹么，好可爱啊！”突然间米粒的目光望向了跟在叶帝后面的杨小艺，一瞬间他的目光便是再也无法收回来了。
<br/>　　与此同时导播也是在第一时间将镜头抓向了杨小艺那里。
<br/>　　“我的天，这个女孩是哪里来的，像个瓷娃娃是的，真的太可爱了。”
<br/>　　“这个不会是大魔王的妹妹吧，大魔王所说的惊喜难道是要给他的妹妹找男朋友，厉害了我的大舅哥。”
<br/>　　“卧槽，大舅哥也是你可以叫的，那明明是我大舅哥好么。”
<br/>　　“6666666666666，国民大舅哥。”
<br/>　　“不对呀，那个女孩我好像见过，之前大魔王直播的时候她好像出现在大魔王的直播间里吧，就上次大魔王唱歌的时候。”
<br/>　　“好像还真是啊，我也有些印象，不过具体的我也不怎么了解了。”
<br/>　　看到杨小艺出现的一刹那，无论是现场的观众还是直播间里的观众这一刻都是抑制不住心中那激动的情绪，疯狂的在那里议论着。
<br/>　　今天的杨小艺确实是做过一些打扮的，原因无他，昨天晚上这家伙又通宵了，而且这一次为了打晋级赛，直到打到上午10点钟的时候，她才回到房间里去睡觉。
<br/>　　本来叶帝都已经不打算让她来打今天这场比赛了，可是谁知道这小家伙在12点的时候已经是睡眼惺忪的走了下来，非要来打比赛。
<br/>　　拗不过她，最终叶帝只好让杨小艺跟过来了，不过事先已经说好了，如果这一场杨小艺表现不好的话，那么接下来她可就要被叶帝禁赛了。
<br/>　　而来之前也是有专门的化妆师为杨小艺打扮了一番，如今叶帝他们已经是不折不扣的明星了，所以在公众面前他们自然是要保持着自己好的一面。
<br/>　　虽然杨小艺不怎么愿意，但是还是被打扮了一下，打扮过后的杨小艺完全是一个靓丽少女的样子，绝对秒杀大片宅男。
<br/>　　“她叫杨小艺，也算是大魔王的半个妹妹了。”林雅楠笑着开口说道。
<br/>　　虽然杨小艺叫叶帝师父，不过私下里大家其实都将她当做妹妹的，并没有人太在意那个备份什么的。
<br/>　　“大舅哥这个惊喜给的真是太好了，不过小艺看起来怎么像是一副睡不醒的样子啊。”米粒也是跟着开口调侃了起来。
<br/>　　听着米粒的话，一旁的林雅楠也是有些哭笑不得，谁能想到昨天都说的好好的今天要来比赛，结果这小家伙半夜还是通宵冲分，而且玩的更加的晚！
