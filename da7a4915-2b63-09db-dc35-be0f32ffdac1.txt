　　“少主，您跟剑祖荒交手了？”
<br/>　　九魔城宫殿之中，鲜血魔尊看到楚风眠归来，吃惊的问道。
<br/>　　他虽然没有追上去，观察战斗，但是这从远方爆发出来的剑意，他可是感觉的一清二楚。
<br/>　　在这天羽神域之中，能够拥有如此剑意的武者，除了楚风眠之外，也就唯有剑祖荒，才能够做到。
<br/>　　“不错，我在追杀小魔主的时候，正好碰到了剑祖荒的一缕剑意所化的化身。”
<br/>　　楚风眠开口道。
<br/>　　“不过我已经是斩了他这一道化身，小魔主也已经被我斩杀。”
<br/>　　“少主您斩杀了剑祖荒的剑意化身？”
<br/>　　鲜血魔尊有些吃惊的说道。
<br/>　　他在天羽神域中，对于剑祖荒的威名自然是知道的一清二楚，深深的知晓剑祖荒的强大。
<br/>　　就算是这一缕剑意化身，也要比起许多生死境巅峰的武者都要强大的多，楚风眠居然是斩杀了这一缕剑意化身。
<br/>　　鲜血魔尊想了一下，沉声说道。
<br/>　　“以剑祖荒的性格，少主您斩杀了他的剑意化身，他肯定不会善罢甘休的。”
<br/>　　“无妨，我既然足以斩了这一缕化身，剑祖荒本体来，我也丝毫不惧。”
<br/>　　楚风眠自信开口道。
<br/>　　他现在斩杀了小魔主，还有无数魔尊，他们的精血如今全部都被楚风眠得到，楚风眠感觉到，他现在的实力，很快便是足以再进一步。
<br/>　　更何况，现在楚风眠更是得到了天魔宝车，这天魔宝车，只要是楚风眠能够炼化，也就意味着得到了一件足以媲美祖龙至尊剑的灵器。
<br/>　　倒时候真正碰到这剑祖荒的本体，楚风眠也不惧。

<br/>　　鲜血魔尊突然开口道。
<br/>　　“据说在天羽神域之中，出现了一位先天神体的女子，如今七大宗门，天羽神域之中的无数势力，都在抓捕这女子，不知道少主的意思是？”
<br/>　　“先天神体的女子？”
<br/>　　楚风眠听到这个消息，心中不由一紧，追问道。
<br/>　　“是个什么样子的女子，你可知道？”
<br/>　　“具体情急，属下不知，不过在传闻中，这女子应该是一位剑修，一身剑术高超，如今已经是斩杀了不少七大宗门的弟子，现在还在被七大宗门追捕，一直在逃亡。”
<br/>　　鲜血魔尊恭敬的开口道。
<br/>　　“天剑宗的剑祖荒，圣王宗的鹿战神，荒古门的荒圣君，如今都想要得到此女，夺取了先天本源，一举突破生死极限，踏入碎命之境。”
<br/>　　“剑修，先天神体，女子。”
<br/>　　听到这里，楚风眠几乎便是确定这女子的身份，应该就是寒月漓。
<br/>　　除此之外，几乎找不到如此巧合的人。
<br/>　　现在看来，寒月漓也是来到了这天羽神域之中来，还不知道如何跟七大宗门的人发生了冲突。
<br/>　　先天神体，便是一个巨大无比的宝藏，谁能够得到其中的先天本源，都是一大造化。
<br/>　　现在寒月漓进入到天羽神域之中，简直便是如同羊入虎口一般。
<br/>　　不过幸好寒月漓现在也并非是没有任何力量，能够从七大宗门弟子的追杀中逃出，看来寒月漓在归墟天中，也曾经得到了不少的好处。
<br/>　　但是就算如此，一个人又如何跟七大宗门抗衡？
<br/>　　寒月漓再强，也会被逼入绝境之中。
<br/>　　“你可知道那个女子在哪里？”
<br/>　　楚风眠又问道。
<br/>　　“这个，属下不知，这女子一路躲藏，行踪缥缈，不过属下这就去打探情报。”
<br/>　　鲜血魔尊开口回答道。
<br/>　　“有此人的消息，立刻回来告诉我。”
<br/>　　楚风眠命令道。
<br/>　　“是。”
<br/>　　鲜血魔尊回答一声，身形一动，直接离开，去打探消息。
<br/>　　“可恶，七大宗门的人，既然敢追杀寒月漓，好，看来我不主动去杀他们，他们倒是来招惹我了。”
<br/>　　楚风眠眼神中，冰冷无比，他本来还没有想要立刻大开杀戒，但是现在看来，这七大宗门的人，果然该死。
<br/>　　如今楚风眠度过生死大劫，一切生死境的武者，楚风眠都不会惧怕，什么剑祖荒，鹿战神，谁招惹到楚风眠的头上，都唯有死路一条。
<br/>　　“不过如今，还要尽快找到寒月漓才行，在七大宗门的围攻之下，不知道她还能坚持多久。”
<br/>　　楚风眠心中暗自想到。
<br/>　　只是现在，毫无线索，楚风眠也只能是让鲜血魔尊出去打探消息。
<br/>　　毕竟在这天羽神域之中，鲜血魔尊也算得上是一位地头蛇，他打探消息比起楚风眠要快得多。

<br/>　　楚风眠心神很快平息下来，全身心中，来准备炼化这天魔宝车。
