　　小耀王寒夜如今身上的气息，比起上一次见到的时候，要强横了不少。

<br/>　　这也正是修炼霸主武道的艰难，一旦无敌之心被击碎，在这小耀王寒夜的心中，更是会留下一道心魔。
<br/>　　“小畜生，没想到吧，上一次你没有杀的了我，反而是让我得到了更加惊人的奇遇，这一次我一定要杀了你！铲除了我心中的心魔！”
<br/>　　小耀王寒夜看向楚风眠脸庞狰狞无比的说道。
<br/>　　他的心魔，就是曾经击败了他的楚风眠，只要是能够杀了楚风眠，他的心魔就会随时消散。
<br/>　　更有可能心神通畅，再一次重塑无敌之心，
<br/>　　他要亲手杀了楚风眠，扫除心魔。
<br/>　　轰隆隆！
<br/>　　耀王武道，在这一刻从小耀王寒夜的身上全然爆发而出，凝聚在了他手心的拳风之上，当即一拳向着楚风眠轰击而来。

<br/>　　小耀王寒夜狰狞的咆哮道，又是连连打出了数道拳风，每一道拳风的力量，都不亚于刚刚的那一道，一共五道拳风，同时向着楚风眠轰击而来。

<br/>　　隐隐约约之间，在这小耀王寒夜的拳风之中，都是带有着几分佛门的气息。
<br/>　　“佛门之中的金刚霸拳？”
<br/>　　楚风眠脑海之中思索，刹那之间看出看来了小耀王寒夜这一拳的来历，金刚霸拳，天级武技，在佛门之中也算是极为强横的武技之一了。
<br/>　　不过面对着这五道拳风的同时轰击而来，楚风眠的表情却是没有任何的变化，眼神之中更是露出了几分不屑之意。
<br/>　　在这五道拳风，轰击在楚风眠的身躯之上的一刻，楚风眠的身形动了。
<br/>　　剑锋！
<br/>　　在楚风眠的手心之中，祖龙至尊剑当即出鞘，一剑没有丝毫犹豫，斩杀而出。
<br/>　　这一剑之上，没有施展出任何的剑术来，只是平平常常的一剑，却是爆发出了震撼无比的力量。
<br/>　　咔嚓。
<br/>　　这五道即将轰击在楚风眠身躯之上的拳风，在这一剑之下，全部破碎。
<br/>　　随后楚风眠更是突然身形一动，猛然出手，一掌向着小耀王寒夜笼罩过去，强横的力量，压迫在小耀王寒夜的身躯之上。
<br/>　　只见楚风眠手上的五指之上，都是凝聚出了一股灵力，飞出去的一刻，顿时化为了五尊尊贵的远古战龙，狠狠的镇压在了这小耀王寒夜身躯之上。
<br/>　　刹那之间，这小耀王寒夜身上强横的武道，都是轰然破碎，他的身躯，被硬生生的打落在了地上，跪在地上，瑟瑟发抖这。
<br/>　　楚风眠身上的力量狂暴无比，在这一刻，他身上的全力，都给施展了出来。
<br/>　　这小耀王寒夜上一次就是在楚风眠的手中逃走了，这样的事情，楚风眠不会让他在出现第二次！
<br/>　　“你以为我杀不了你？上一次有人能救你，这一次我倒是要看看谁还能救得了你！”
<br/>　　“五龙秘术！”
<br/>　　楚风眠长啸一声，那五条镇压在小耀王寒夜之上的远古战龙，猛然之间咆哮一声，钻入到了小耀王寒夜的身躯之中去。
<br/>　　每一条远古战龙，都蕴含这不亚于小耀王寒夜的力量，这五条远古战龙，同时钻入到了小耀王寒夜的身躯之中。
<br/>　　几乎是让他的身体，都承受不住了这狂暴无比的力量，一下子开始膨胀了起来。
<br/>　　这转轮和尚救人的手段之多，楚风眠都忌惮无比，但是这一次，楚风眠可不会再给转轮和尚救人的机会。
<br/>　　“爆！”
<br/>　　楚风眠一声命令之下，在这小耀王寒夜身躯之中的五条远古战龙，突然开始膨胀起来，他要让这小耀王寒夜自爆。
<br/>　　一旦是武者自爆，哪怕是有人惊天的手段，让天人境的强者来，都救不活！

<br/>　　转轮和尚刚刚打算去收取那血邪魔帝骸骨，就听见了小耀王寒夜的惨叫之声，脸色猛然一变。

<br/>　　转轮和尚的身形，猛然冲了过来，又是一道佛光从他的手心之中打了出来，落到了小耀王寒夜的身上。

<br/>　　这五条远古战龙，打入到了小耀王寒夜的身躯之中，已经将他的灵力全部控制住。
