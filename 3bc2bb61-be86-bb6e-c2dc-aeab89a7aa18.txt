<br/>				    那赶过来的修士们瞪大了眼睛，一个个目瞪口呆的瞪着前面干了的湖，嘴里如同塞了鸭蛋般合不拢口。<br/>　　<br/>　　    那湖是这里岛心的水凝聚而成的，都不知形成多少年了，尤其是这湖泊也不浅，而且里面还有一条神兽级别的白蟒守着那些金莲，以至于让多少强者想要去摘那金莲也往往丧命于湖泊之中。<br/>　　<br/>　　    可如今，这金莲不仅没了，就连这湖中的水连同那条白蟒也没了，就剩下这么一个干枯了的大窟窿在这里，如何不叫他们傻眼？<br/>　　<br/>　　    可偏偏，这里除了他们之外，一个人也没有，而他们又是刚刚赶过来的，就算不是刚到这里，他们也没有那个能力可以让这湖泊干枯。<br/>　　<br/>　　    一时间，一个个修士站在那里大眼瞪着小眼，想不通这到底是怎么一回事？<br/>　　<br/>　　    空间之中，凤九还在进阶当中，这一次的进阶已经持续了好几天了，三道天雷落下后，那天空中的云层却还没散去。<br/>　　<br/>　　    而此时的凤九脑海中正回荡着一道浑厚的雷霆之声。<br/>　　<br/>　　    “天道三问，第一问，何为天道？”<br/>　　<br/>　　    凤九的脑海中重重的响起这话，那声音直达心灵深处，震得她的双耳有些发疼。<br/>　　<br/>　　    天道三问？第一问，何为天道？<br/>　　<br/>　　    她尽量主让心绪冷静下来，摒弃五官，不去理会那身体传来的椎心剧疼，而是认真思忖着：何为天道？修仙大道，长生之道，视为天道，然，仙道只有一条，但，每个修仙的人所走的路以及境遇却是不同的，不同的人生，有不同的答案，何为天道？答案有千千万，却只有一个是属于我自己的答案！<br/>　　<br/>　　    “以心正道，与天相争，何为天道？我即为天道！”<br/>　　<br/>　　    她听着自己清冷而蕴含着强大威压的声音传出，回荡在天空之中，就在那话语一落之时，天空中轰隆轰隆巨响，如同闷雷声翻滚而起，又似万兽奔腾而来，气势汹汹，震摄人心！<br/>　　<br/>　　    云层翻滚，雷声不断，天空中那声音似乎沉默了一下，下一刻，那云层雷声再度静止，那雷霆之声再度传来。<br/>　　<br/>　　    “天道三问，第二问，为何踏上修仙之道？”<br/>　　<br/>　　    那声音一落，凤九当即便说着：“为成为至尊强者！为护守我想守护的人！”<br/>　　<br/>　　    她的回答一落，又是天雷滚动的声音传出，轰隆作响，不多时，再次静了下来，那人的声音也第三次传来。<br/>　　<br/>　　    “天道三问，第三问，何为强者？”<br/>　　<br/>　　    “心坚志强，拥有绝对实力，能主宰命运，是为强者！”<br/>　　<br/>　　    天空中的云层因她的回答而散开，原本阴云密布的天空再次放晴，一道七色彩虹如拱桥般在她头顶的天空出现，隐隐有仙乐之声传来……<br/>　　<br/>　　    也就在这一刻，心境仿佛再次突破一个阶段，体内灵力气息一涌，从仙帝巅峰直接突破进入天神级别中阶实力，体内的青莲气息弥漫而开，迅速的修复着她受损的身体，伴随着身上的变化，她整个人的气息也在那一瞬间发生了天翻地覆的变化。<br/>　　<br/>
