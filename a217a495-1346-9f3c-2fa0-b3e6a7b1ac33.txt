        爹地傅夜沉虽然答应了撤掉那些暗中保护她的保镖，但是却强制给她戴上了监测生命体征的智能定位手环。
<br/>
<br/>        问题是，这手环防水防震还“防盗”。
<br/>
<br/>        如果她不回来，手环就没法取下来。
<br/>
<br/>        连星儿倒是没什么意见，反正她还能当手表用。
<br/>
<br/>        这丫头出门后，傅夜沉就有些坐立不安。
<br/>
<br/>        童筱颖不得不挽着他的臂弯，强行拉着他坐下来陪自己在家庭影音厅里看电影。
<br/>
<br/>        “难得你今天休息，臭丫头又不在家，你就好好地陪我看场电影好了。”童筱颖嘴角微扬地将脑袋靠在傅夜沉的肩头上。
<br/>
<br/>        傅夜沉想的却是女儿连星儿的事情：“我觉得，还是得派人跟着星儿。”
<br/>
<br/>        “大哥，你要相信我们的女儿，她会好好保护自己的。”童筱颖安慰道。
<br/>
<br/>        “万一她遇上小偷了怎么办？”傅夜沉像是在自言自语。
<br/>
<br/>        童筱颖十分淡定地回答：“她准备了两个手机，就算被偷了一个，还有一个可以用。”
<br/>
<br/>        “万一遇上图谋不轨的人呢？”
<br/>
<br/>        “你的小情人机灵得狠，一定会化险为夷。”
<br/>
<br/>        “那再万一……”
<br/>
<br/>        “要不，大哥你干脆派二十个保镖二十四小时轮流保护星儿？”童筱颖抬起头来看着傅夜沉打趣道。
<br/>
<br/>        傅夜沉纠结了一会儿，终于放宽了心：“算了，还是让她一个人开开心心地玩吧！”
<br/>
<br/>        其实，童筱颖能懂傅夜沉为什么如此紧张女儿。
<br/>
<br/>        因为，当年，星儿是在他俩没有做计划的情况下怀上的，而且星儿是早产儿，差点就死在了恒温箱里。
<br/>
<br/>        傅夜沉那段日子，没日没夜地在她和这个小丫头之间来回奔波。
<br/>
<br/>        当时，童筱颖并不知道星儿生命垂危，以为星儿只是早产需要住恒温箱而已，傅夜沉也没把星儿垂危的消息告诉过她。
<br/>
<br/>        那个时候，一切心理负担，他是一个人独自承担着。
<br/>
<br/>        现在回想起来，童筱颖反倒是心疼起他傅夜沉来。
<br/>
<br/>        “大哥。”童筱颖随之牵起傅夜沉的手，与他五指相扣，接着说道，“我们要一起慢慢变老。”
<br/>
<br/>        傅夜沉嘴角微扬，偏过头去，吻了吻童筱颖的额头，应了声：“好。”
<br/>
<br/>        前往云城的高速列车里，连星儿一手拿着车票，一手托着行李箱正寻找着自己的座位。
<br/>
<br/>        她给自己订的是商务舱靠窗的座位，当她找到自己的座位时，邻座正坐着一个戴着黑色眼罩，双手抱臂正仰躺着小憩的男人。
<br/>
<br/>        连星儿并未仔细去看这个男人，只是一个高乘小哥哥主动走过来，替她将行李箱放在了上头的置物架上。
<br/>
<br/>        从滨江城到云城要五个小时，连星儿打算先拿手机打一会儿游戏，再睡一会儿。
<br/>
<br/>        结果，她兴致勃勃地打了两局不仅被队友坑输了还掉了两颗星又降了等级后，耳畔忽然响起一个好听的男音。
<br/>
<br/>        “需要我帮你吗？”
<br/>
<br/>        连星儿看都不看男人一眼，二话不说就把自己的手机给了邻座这个男人，接着从手提包里掏出眼罩，戴上睡觉。
<br/>
<br/>        待到她睡醒后，这才瞥见一双修长而好看的大手，正拿着她的手机飞速操作。
<br/>
<br/>        连星儿惊怔了一番后，脑袋连忙凑过去，双眼直勾勾地盯着这双手，以及手机屏幕上那个被这双手操作的游戏人物。
<br/>
<br/>        不到十分钟，游戏结束，他们赢了！
<br/>
<br/>        男人也随之将手机还给了连星儿。
<br/>
<br/>        连星儿拿着手机一看，目瞪口呆了。
<br/>
<br/>        她的号不仅上了两个等级，还得了八颗星星。
<br/>
<br/>        “大神，求你的联系方式！”连星儿惊呼不已地抬起头来，看向这个男人的脸不到一秒的时间，脸上的笑容就僵硬了。
<br/>
<br/>        男人，哦，不，是这男生，剑眉星目，和那天一样穿着白衬衫黑色休闲裤，一身干净整洁，有一种出淤泥而不染的俊逸脱俗的气质。
<br/>
<br/>        半晌后，连星儿却塞好自己的手机，毫不客气地向这个男生伸出手来：“冰激凌奶茶二十块钱，我那朋友的衣服五百块。给钱！”
<br/>
<br/>        看着面前这个小女生，明司寒薄唇微扬，剑眉也跟着挑了挑：“追我的女孩子很多，用你这种方法追的，我还是头一次遇到。”
<br/>
<br/>        “？！”连星儿顿时一脸懵。
<br/>
<br/>        明司寒会心一笑，从黑色休闲裤的裤兜里掏出一个棕色的皮夹来，十分大方地掏了五百二十块钱递给连星儿。
<br/>
<br/>        连星儿这才反应过来，拿过钱，没好气地说道：“你别孔雀开屏了，这只是巧合而已！”
<br/>
<br/>        这都能遇上，还真是冤家路窄！
<br/>
<br/>        连星儿随之又掏出手机，给朱小溪转了五百过去，并告诉她，是那天那个肇事者给她的衣服的赔偿钱。
