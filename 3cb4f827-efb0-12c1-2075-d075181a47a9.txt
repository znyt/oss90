　　苏莫没有过多纠结，没有传开更好，对他没有影响。
<br/>　　“绫裳，去我的空间宝器之中，我有大礼送给你。”苏莫沉声说道。
<br/>　　“什么大礼？”绫裳疑惑的问道。
<br/>　　“进来你就知晓了！”
<br/>　　苏莫微微一笑，随即意念裹挟着绫裳，在对方没有反抗的情况之下，进入了空间宝器之中。
<br/>　　这是一个空间戒指，空间极大，大地郁郁葱葱，一望无际。
<br/>　　芊寻月的棺椁，就存放在这个空间宝器之中。
<br/>　　草地之上，苏莫和绫裳相对而立。
<br/>　　“什么礼物？”绫裳目光凝视着苏莫，绝美的面容之上，也露出一丝好奇之色。
<br/>　　苏莫如此神神秘秘，不由得让她也产生了好奇心理。
<br/>　　“你看！”苏莫微微一笑，随即意念一动，身上升起刺目的银光。
<br/>　　而后，一柄银色大剑元神，从他体内升腾而起。
<br/>　　银色的大剑，弥漫着森冷的锋芒，虽然不如万化神剑，但也远超一般的剑类元神。
<br/>　　这元神自然便是苏莫上次吞噬的元神，其主人的神魂，早已不在。
<br/>　　“这是……？”绫裳见此疑惑，这元神显然不是苏莫的，而是无主之物。
<br/>　　绫裳用青璇的记忆，知晓苏莫当年的武魂，有吞噬的威能，但是，具体的能力，就不是很清楚了。
<br/>　　“这元神赐予你！”苏莫凝视着绫裳如水的眼眸，郑重的说道。
<br/>　　“赐予我？”
<br/>　　绫裳闻言，顿时娇躯一颤，惊讶的望着苏莫，眸中闪过惊疑之色。
<br/>　　“对，赐予你，我能将这元神和你的神魂融合，成为你的元神！”苏莫郑重的点了点头。
<br/>　　“这……真的可以？”绫裳心中一惊，不仅瞪大了眼眸，因为这完全尤为常理。
<br/>　　但是，她心中又清楚，苏莫不是一个爱开玩笑的人。
<br/>　　“真的可以，与你自己天生的武魂一般无二！”苏莫面上带着笑意，眼眸中蕴含着坚定。
<br/>　　绫裳闻言沉默了，她没有料到，苏莫居然还有这种能力，融合元神的能力，这简直太逆天了。
<br/>　　“你能无限融合元神？”
<br/>　　少倾之后，绫裳凝声问出了一个关键的问题，若是能无限融合，那还得了？
<br/>　　“理论是可以，不过，在战斗之中催动元神之力，需要强大的精神力量，没有足够的精神力，再多的元神也是无用！”苏莫道。
<br/>　　若是不受精神力的限制，他吞噬无数的元神，早就无敌了。
<br/>　　“原来如此！”绫裳心中恍然，元神的运用，的确是需要精神力的配合，而精神力来自于神魂，神魂的强弱，是伴随着修为缓慢提升的。
<br/>　　“我来为你融合吧！”苏莫深深的吸了口气说道。
<br/>　　绫裳轻抿红.唇，略一沉吟，便默默的点了点头。
<br/>　　对于此事，她无法拒绝，也没有人会拒绝。
<br/>　　这一世的天赋，严重的限制了她的武道之路，若是能再有一个元神，那她将会一飞冲天。
<br/>　　当然，绫裳根本不怕苏莫害她，因为，以她的实力，轻易能镇杀苏莫。
<br/>　　随后，苏莫和绫裳两人，相对而坐。
<br/>　　苏莫意念一动，吞噬元神裹挟着银色大剑元神，直接冲进了绫裳的识海之中。
<br/>　　整个过程异常的顺利，比苏莫为司空炎等人融合战魂，还要容易十倍。
<br/>　　这是因为，绫裳的神魂，不知道什么原因，居然极度的强大，远超同阶武者数倍不止。
<br/>　　仅仅是一刻钟，一切便结束了，吞噬元神回归了苏莫的体内，银色大剑元神，成为了绫裳的元神。
<br/>　　“居然……真的可以！”绫裳低声自喃，美眸之中，闪过明亮的光彩。
<br/>　　即便她曾经身为神王，此刻也不禁有些激动了起来。
<br/>　　本来，她想要恢复到神王之境，起码需要千年的时间，但是，有了此元神，他只需要四五百年的时间，就足够了。
<br/>　　“这元神，还需要你重新融炼，才能与你彻底融合！”苏莫告诫道，此元神虽然和对方神魂融合了，但是，还需要对方自行祭炼，与体内的圣丹、气血等进行交.融。
<br/>　　“嗯！”
<br/>　　绫裳点了点头，随即抬头看向苏莫，一直亘古不变的清冷面色消失，而是展颜一笑，道：“苏莫，你真是个奇迹！”
<br/>　　苏莫见此，微微一怔，目光紧紧的盯着绫裳。
<br/>　　面对这种如沐春风的笑容，他有一种面对青璇的感觉，仿佛眼前之人，还是以前的青璇。
<br/>　　见到苏莫如此目光，绫裳突然有种不知如何面对的感觉，立刻微微的别过了脑袋。

<br/>　　“我正准备去！”苏莫闻言，点了点头。
<br/>　　“万道剑冢是宝地亦是险地，不过，以你的天赋，危险性微乎其微！”绫裳道。
<br/>　　“那就更好！”苏莫心中微松，没有什么大的危险，自然再好不过了。
<br/>　　“万道剑冢，可以磨炼你的剑道，让你更好的找到方向，不过，每一次开启，都有很多顶级妖孽，甚至一些万古天骄，你还是小心为妙！”绫裳叮嘱道。

<br/>　　树林之中，绫裳向苏莫告别，并叮嘱苏莫。
