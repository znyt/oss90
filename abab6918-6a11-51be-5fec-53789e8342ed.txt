　　“你是……？”
<br/>　　沁儿的眸中，带着惊喜，目光紧紧的望着苏莫，不过，她话未说完，便被苏莫抬手打断了。
<br/>　　苏莫自然明白，沁儿可能已经认出了他。
<br/>　　“沁儿姑娘，有缘再见！”苏莫向沁儿微微一笑道。
<br/>　　姬月松三人闻言，顿时眼眸一凝，此人和沁儿认识吗？
<br/>　　“你是段惊天？”裘蒙眸光一闪，沉声问道。
<br/>　　他心中惊疑不定，此人有可能就是苏莫，但是他还不能确定。
<br/>　　“你不需要知道！”
<br/>　　苏莫转头看向裘蒙，声音冷漠，他话音一落，瞬间就出手了。
<br/>　　长剑在手，剑威浩荡，战意无穷，犀利无比的剑气，直接斩向裘蒙。
<br/>　　此时此刻，苏莫已经能肯定，此人来者不善。
<br/>　　连姬月松等人，都没有在第一时间猜到是他，而这个不相干的人，却能猜到是他，对方的意图显而易见。
<br/>　　一剑出，风云变色，恐怖的剑威笼罩苍穹，无匹的战意如同火山沸腾。
<br/>　　这一剑，苏莫可谓是没有丝毫的保留，尽显他二品修为的实力。
<br/>　　由于距离很近，剑气一出，便杀到了裘蒙的身前。
<br/>　　“怎么可能？”
<br/>　　裘蒙面色大变，他知晓苏莫的实力强，比天尘山、姬月松都要强三分。
<br/>　　但是，此时此刻，苏莫展现的实力，根本不是那么回事。
<br/>　　本来以为，他的实力足以镇压苏莫，没想到苏莫居然又突破了修为。
<br/>　　来不及多想，杀机已然来临，电光火石之间，裘蒙急忙出拳抵挡。
<br/>　　轰！
<br/>　　裘蒙不愧是三品低级修为的强者，实力强大，拳出如电，狠狠的轰中了剑气。
<br/>　　然而，一切都是徒劳，剑气斩中，血雨腥风。
<br/>　　轰！
<br/>　　一声爆响，强大的剑气，瞬间将裘蒙撕成了碎片，肉身和元神全部被灭杀，彻底的陨落。
<br/>　　三品低级的裘蒙，根本不是苏莫一合之敌。
<br/>　　苏莫的修为，若是没有突破，或许不一定能打败裘蒙，但是，突破到了二品低级的他，战力早已不是当初可比。
<br/>　　劲力爆开，汹涌澎湃，伴随着血雾涌动，沁儿和姬月松四人，纷纷暴退，急忙拉开距离。
<br/>　　“你是段惊天？”骆妤惊呼，面带震惊之色。
<br/>　　“段惊天，你居然敢残杀同门？”褚彦怒声大喝。
<br/>　　“这么强大？”姬月松面带震惊之色，这个裘蒙的实力，比他都只强不弱，居然被段惊天一剑斩杀。
<br/>　　沁儿瞪大了眼眸，面带不可置信之色，这段惊天疯了吗？怎么突然暴起杀人。
<br/>　　“我叫苏莫，也叫段惊天！”
<br/>　　苏莫收起长剑，环视四人，此时此刻，他也没有在隐瞒，在他出手的时候，身份已经暴露了。
<br/>　　不过，他也无所谓了，天大地大，除了人德宫主知晓他去终极剑域之外，无人能知晓。
<br/>　　“段惊天，你敢残杀同门，这是重罪！”褚彦大声怒喝，他可不管苏莫叫什么，他心中暗喜，这段惊天杀了同门，以后别想在回去了。
<br/>　　不然的话，难逃人王宫最严厉的惩处。
<br/>　　“再多说一句，我连你也杀了！”苏莫转头看向褚彦，声音萧杀无比。
<br/>　　“你……！”褚彦顿时感觉心中一凉，立刻后退了数十丈。
<br/>　　“沁儿姑娘，多谢了！”苏莫向沁儿抱了抱拳，对方算是帮了他，又是人德宫主的孙女，理应道谢。
<br/>　　“你……你为何要杀人？”沁儿怔怔的问道。
<br/>　　“你日后自然会明白！”
<br/>　　苏莫叹了口气，随即轻声道：“告辞了！”
<br/>　　言罢，苏莫便不再多留，闪身向远处飞去。
<br/>　　“你什么时候回人王宫？”眼见苏莫即将远去，沁儿大声问道。
<br/>　　“当我能主宰生死，掌控一切之时，必会回来！”
<br/>　　苏莫的声音，从远处悠悠的传来。
<br/>　　听闻此言，沁儿不仅一怔，这是什么意思？
<br/>　　姬月松、骆妤、褚彦三人，并未阻拦苏莫，因为他们知晓，自己阻拦不住。
<br/>　　“月松哥，这个段惊天，简直无法无天，他残杀同门，必须禀报给长老！”褚彦脸色铁青的说道，这个段惊天仗着天赋妖孽，实在是太狂妄了。
<br/>　　“此事我会禀报，他的罪只有长老定夺。”姬月松道。
<br/>　　既然在高空中伫立了良久，最后也不再去时空界域，而是向人王宫折返。
<br/>　　毕竟，段惊天的突然出现，以及杀了一名同门，必须尽快回禀宗门。
<br/>　　……
<br/>　　苏莫与沁儿等人分别之后，便加速赶路，向终极剑域而去。
<br/>　　天池神朝地域辽阔，远非轮回神朝可比，苏莫将速度提升到了极致，争取早日到达终极剑域。
<br/>　　“不知道绫裳的修为，恢复到了什么地步？”苏莫心中暗道，在他想来，绫裳的修为定然很高了。
<br/>　　毕竟，神王转世重修，非同小可。
<br/>　　他现在，只希望绫裳能帮他，让紫霄神王出手帮助，一举救活芊寻月。
<br/>　　时间匆匆流逝，苏莫不停的赶路，一年半的时间，悄然而逝。
<br/>　　终于，苏莫来到了终极剑域的地界。
<br/>　　进入终极剑域，他在冥冥之中，能感觉到，就连空气之中，仿佛都弥漫着剑的锋芒。
<br/>　　这是一个剑道界域，九成的人都修炼剑道，让得整个界域，都流露出了剑之气息。
<br/>　　传闻，终极剑域，是由一位叫做终极剑尊的至强者所创。
<br/>　　终极剑尊，虽然声名不如轮回天尊、亘古魔祖、混沌神帝等太古至强者，但也是同境界的人物。
<br/>　　不过，终极剑尊已经消失了千万年，再也没有出现过。
<br/>　　而如今的神域，各大神朝、界域，那些至强者也都消身匿迹，几百万年也难得一见。
<br/>　　据闻，有的至强者已经陨落，但到底是如何陨落，就不是普通的真神境之人能知晓的了。
<br/>　　苏莫随意的打听了一番，便得知了终极神王殿的所在，是在一个叫神剑山的地方。
<br/>　　又过了数日，他接近了神剑山，只见他的前方视线之中，出现了一座笔直而雄伟的山峰。
<br/>　　此山，高约两万丈，上方略大，犹如是倒插在大地之上的巨大神剑。
<br/>　　距离神剑山还有数千里，苏莫便感受到了无形的剑威，浩浩荡荡，铺天盖地。
<br/>　　仿佛，这神剑山就是一柄神剑，自带无形的剑势，威慑九天十地。
<br/>　　“这神剑山，居然影响了整个终极剑域？”苏莫心中震惊，难怪整个终极剑域之内，都有无形的剑之锋芒，原来都是从神剑山上发出。

<br/>　　他的剑，到目前为止，根本没有什么正统的剑道，完全是野路子。
<br/>　　而在这里，他或许能参悟出，属于他自己的道。
<br/>　　随即，苏莫加快速度，向神剑山飞了过去。
<br/>　　越是临近神剑山，他的感受越发的真切，神剑山之上，散发的剑势，形成浩荡的剑威，浩大而纯粹。
<br/>　　就连他储物戒中的宝剑，都仿佛是感受到了这种剑威，发出轻吟之声。
<br/>　　但是，这种剑威，却是不会伤人，仿佛只有心神才能感受的到。
<br/>　　“居然如此繁华！”
<br/>　　随着苏莫飞近，他发现，在神剑山的四周，坐落着一望无际的建筑群。
<br/>　　这些建筑群，组成了一座又一座城池，数量多达数十座。
<br/>　　众多的城池互相连接在一起，将神剑山拱卫在了中央。
<br/>　　好似，神剑山是圣地，是所有人守护的地方。
