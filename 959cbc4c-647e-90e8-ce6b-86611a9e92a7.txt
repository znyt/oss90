<br/>　　巫森冷笑一声，手指陡然一点，而后巨大的地狱三头狼，如同一道黑色的闪电，向苏莫飞扑而去。

<br/>　　吼！
<br/>　　这地狱三头狼战魂，仿佛灵智极高，见到苏莫攻击，顿时狂吼一声，巨大的狼爪猛拍而出，犹如泰山压顶。
<br/>　　轰！
<br/>　　一声惊天动地的爆响，狼爪与剑光相击，顿时呈碾压之势，轰隆之间便震碎了所有剑光。
<br/>　　而后，巨大的狼爪狠狠的和帝皇剑硬撼在了一起。
<br/>　　铛！
<br/>　　一声金铁交鸣之声响起，震碎人的耳膜。狼爪仿若神兵利器，坚不可摧。
<br/>　　排山倒海的力量倾泄而出，势不可挡，让得苏莫全身剧震，直接被击飞了出去。
<br/>　　嗖！
<br/>　　耳边风声呼啸，体内气血逆流，苏莫倒飞千里，喉头一甜，嘴角忍不住流下一丝金色的血液。
<br/>　　“苏莫受伤了！”
<br/>　　“这就是战魂的力量吗？”
<br/>　　“好恐怖的战魂！”
<br/>　　有眼力较高的强者，发现苏莫受伤，顿时惊呼出声。
<br/>　　不是苏莫实力不强，而是这战魂太恐怖了！
<br/>　　所有人都瞪大了眼眸，不愿错过任何一丝战斗的细节。

<br/>　　唯有高阶武皇，才能具体看清两人的战况。
<br/>　　嗷嗷嗷！！
<br/>　　不给苏莫喘息之机，地狱三头狼嚎叫震天，再次向苏莫飞扑而来，滔天的凶威如冲天海浪，不断的涌动。
<br/>　　“战魂！”苏莫面无表情，心中亦是震惊这战魂的强大。
<br/>　　原来武皇晋升到战魂之后，还能直接用于攻击，飞天遁地。
<br/>　　那他的吞噬武魂晋升为战魂之后，会有什么能力，能不能直接用于攻击。
<br/>　　这一点苏莫不清楚，因为他的武魂太过特殊，与兽武魂完全是两码事。
<br/>　　不过对方想杀他，仅凭这战魂还不够。
<br/>　　“星宿剑阵！”眼见着地狱三头狼再次扑来，苏莫心神一动，无数剑光爆闪。

<br/>　　苏莫手指一点，其中一柄本命灵剑光芒一闪，划破虚空，向地狱三头狼迎面刺去。
<br/>　　犀利的剑光，一闪而没，莫测的星宿之力弥漫八方。
<br/>　　“虚宿！”
<br/>　　“井宿！”
<br/>　　“角宿！”
<br/>　　“房宿！”
<br/>　　“……！”
<br/>　　而后，苏莫手指连连点动，一柄柄本命灵剑不断的闪动，庞大的星宿剑阵，开始对地狱三头狼展开了绝杀。

<br/>　　爆响之声此起彼伏，本命灵剑不断的与地狱三头狼相击，猛烈的气劲不断的爆开，仿若有无数的核弹在不停的爆炸，一片末日般的景象。
