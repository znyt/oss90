　　其实当银背妖蟒出现时，萧逸就瞬间明白过来。
<br/>　　天材地宝中，一些奇特的灵果灵花必有妖兽守护。
<br/>　　而且妖兽往往一守就是多年，直至这株天材地宝成熟，可以吞服。
<br/>　　而往往，能让这只妖兽守候那么久，必然代表着这株天材地宝对它的意义非常大。
<br/>　　妖血炎心果，对人类武者有着焚炼经脉，提升修为之效；而对于妖兽，则有着进阶血脉之效。
<br/>　　如果没有猜错的话，银背妖蟒是想借这妖血炎心果，进阶为金背灵蟒。
<br/>　　金背灵蟒，乃是三级巅峰妖兽，起码要洞玄境武者才能将其收服。
<br/>　　而此时，银背妖蟒见自己守护多年的三颗妖血炎心果竟被一个人类采摘掉，瞬间暴怒。
<br/>　　偏偏，慕容千军采摘灵果后竟瞬间往萧逸这个方向跑来。
<br/>　　“混蛋。”萧景一惊，并怒骂一声。
<br/>　　萧逸也是脸色一变，慕容千军此举，将会让银背妖蟒认为他们是一伙的。
<br/>　　果不其然，银背妖蟒立刻疯狂攻击萧景和萧逸，同时打算拦下前方的道路，防止慕容千军逃跑。
<br/>　　但慕容千军早有准备，速度极快，一把越过萧景和萧逸，直往后发逃遁。
<br/>　　“拦下他。”探路队员们大喝一声。
<br/>　　只是，他们中最强的不过是一位后天一重的副队长，其余队员皆是凡境九重；萧壮萧子木等人更是不堪。
<br/>　　多人联手，却被慕容千军一掌震退，只能眼睁睁地看着慕容千军在他们身旁跃过。
<br/>　　“别管他，快跑。”萧逸大喝一声，让萧壮等人立刻逃跑。
<br/>　　他可不认为自己一行人能在暴怒的银背妖蟒下支撑多久。
<br/>　　萧壮和探路队员们也意识到这点，立刻转身便跑。
<br/>　　而萧景和萧逸，同样在快速奔跑，后方银背妖蟒紧追不舍。
<br/>　　银背妖蟒乃三级妖兽，实力不亚于先天一重武者。体型巨大，力量惊人，防御极高，唯一的缺点是不擅速度。
<br/>　　加之密林中巨大的树木很多，大大地限制了它庞大体型的移动。
<br/>　　如无意外，萧逸等人要逃跑还是没问题的。
<br/>　　可恰在此时，最前方慕容千军逃跑之处，却猛地跳出了十多道人影。
<br/>　　这些人尽皆手持刀刃，面露狞笑，齐齐地发出了十数道刀气，凌空朝萧逸等人打来。
<br/>　　“混账，铁刀你敢？”萧景大喝一声。
<br/>　　前方十数人竟是铁刀猎妖队的人，且队长铁刀，这个后天六重武者也在这。
<br/>　　若是在其它时候，萧景和探路队的队员自然不惧他们。
<br/>　　可现在这个时候，这些锋利的刀气却让萧景等人陷入了巨大的麻烦。
<br/>　　前方一同逃跑的萧壮等人被迫停了下来。
<br/>　　萧逸和萧景一跃而上，虽然挡下了这些刀气，却生生停顿了一下。
<br/>　　就是这一下的时间，让后面的银背妖蟒追了上来。而且这次银背妖蟒显然变聪明了，极长而粗壮的蛇身立刻拦住了去路。
<br/>　　此时，慕容千军和铁刀猎妖队的人双双快速离开。
<br/>　　“哈哈哈哈，萧家的蠢货，全部死在这里吧。”前方，传来了慕容千军肆意地大笑。
<br/>　　“卑鄙。”萧景怒喝一声。
<br/>　　“慕容千军，此事过后，我与你不死不休。”萧逸冷喝一声，满脸杀意。
<br/>　　“死到临头，还敢大言不惭，你们有命离开这里再说吧。”慕容千军的声音已经变得微弱，显然已经逃遁了。
<br/>　　“嘶嘶。”银背妖蟒那恐怖的信子快速申吐着，表明它对面前的人类起了极浓烈的杀意。
<br/>　　它幸苦守候了多年的妖血炎心果，被那可恶的人类抢走了，它愤怒到极点。
<br/>　　那可恶的人类已经远远逃遁了。它只能将恶气发泄到那人类的‘同伙’身上。
<br/>　　“该死。”萧景低喝一声，脸中尽是愧疚之色。
<br/>　　“如果我不是冲动，没有听少家主你的话，现在也不会陷入如此境地，还让慕容千军那群混蛋得逞了。”
<br/>　　萧景现在已经知道，他中了慕容千军的计，还让萧逸和其他人陷入了危险中。
<br/>　　这时，银背妖蟒那粗壮的蛇身狠狠拍来。
<br/>　　“小心。”萧景和萧逸同时喝了一声，双双后退。
<br/>　　轰的一声，蛇身直接将地面轰出了一个大洞。
<br/>　　“嘶。”两人同时倒吸一口凉气，这一击，要是打在他们身上，不死也重伤。
<br/>　　“少家主，你先走，我来拦住他。”萧景快速地说了一声，挡在萧逸身前。
<br/>　　萧逸喝道，“别废话，我们一起来的就一起走。”
<br/>　　就这短短的一句话语功夫，恐怖的蛇身再次袭来。
<br/>　　“少家主快走，我们不是它的对手，我去引开它。”萧景一把推开萧逸，独自迎上了银背妖蟒。
<br/>　　银背妖蟒封锁了去路，它的实力，不亚于先天武者。先天之下，没人能突破它的防御。
<br/>　　若不将它引开，没人能逃脱。
<br/>　　同一时间，萧壮、萧子木和探路队员，也立刻跟上，纷纷朝萧景而去。
<br/>　　萧逸站稳了身子后，大急道，“你们干什么，快回来！”
<br/>　　萧壮等人的动作，立刻引起了银背妖蟒的注意，并挪动了庞大的身躯，朝他们追去。
<br/>　　“少家主，你先跑，我们拦住这妖蟒。”萧壮等人大喝了一声，声音中没有任何犹豫。
<br/>　　轰的一声，蛇尾拍来，萧景和萧壮等人瞬间被拍飞。
<br/>　　“噗...”众人瞬间吐血，个个脸色苍白，显然受了重伤。
<br/>　　堂堂后天七重武者，外加一众武者，竟连银背妖蟒的一击都挡不住。
<br/>　　萧逸立刻打算救援。
<br/>　　萧景忍住伤势，喝道，“少家主快跑，别让我们白死。”
<br/>　　“混蛋，老子不承你们的恩惠。”萧逸大喝一声，脚步不停。
<br/>　　这时，蛇身的攻击再次到了。
<br/>　　萧景等人脸色一变，立刻跃过去，一把抱住了蛇尾。萧壮、萧子木等人也一样。
<br/>　　“少家主，你是萧家下一代的所有希望，不容有失。”萧景奋力地抱着蛇尾，着急道。
<br/>　　“少家主，没了我们，萧家还是那个萧家。但没了你，日后的萧家危矣。”萧壮等人同样奋力地抱住了蛇尾。
<br/>　　“你们...”萧逸一窒，他看着萧景、萧壮等人那视死如归的眼神，不惜一切都要帮他逃跑的脸色，心中头一次对这个家族无比的认可，头一次认可了这些族人们。
<br/>　　轰，就在他愣神的一瞬间，萧景等人已经被蛇尾拍飞，一个个大口大口地吐着鲜血，最后直接晕厥了过去。
<br/>　　“嘶嘶嘶。”银背妖蟒那恐怖的信子不断申吐着，蛇尾一卷，打算将萧景等人全部拍死。
<br/>　　萧景他们已然晕厥，蛇尾这一下打去，他们必死无疑。
<br/>　　“噬火百刃。”萧逸没有任何犹豫，瞬间出手。
<br/>　　他没有任何胜算，但他想尽力。
<br/>　　火刃重重地打在蛇身上，嘭的一声爆炸。然而，银背妖蟒却丝毫无损。
<br/>　　“孽畜，看这边。”萧逸大喝一声，又是一道火刃脱手而出。
<br/>　　嘭，火刃直接打到银背妖蟒的眼睛上，那里是它最脆弱的位置。
<br/>　　火刃爆炸，但，银背妖蟒的眼睛仅仅掉了点皮。
<br/>　　当然了，它虽然没受伤，但眼睛还是被打疼了。
<br/>　　“嘶吼。”银背妖蟒疼叫了一声。
<br/>　　它本来就处于暴怒状态，此时直接被萧逸激出了妖兽的凶性。
<br/>　　发狂的妖兽，是非常恐怖的。
<br/>　　它当即撇下已经昏厥的萧景等人不管，朝萧逸追击而去。
<br/>　　升龙，豹形。
<br/>　　萧逸立刻使用了升龙，将修为提升到后天三重。而后，冰火气丹调动真气，更使用了豹形，将自身的速度达到了极致。
<br/>　　银背妖蟒速度慢，但这是和三级妖兽和先天武者比。
<br/>　　若和一个后天武者比，它还是远超的。
<br/>　　萧逸不敢有任何怠慢，边跑边释放噬火百刃。
<br/>　　一道又一道火刃打出，可是却连银背妖蟒的皮都打不破，除了激怒它以外，没有任何作用。
<br/>　　“这孽畜，皮糙肉厚的，完全打不动。”萧逸暗暗着急，无论如何，他都想将萧景等人救下。
<br/>　　银背妖蟒一路狂追，愤怒下，将沿途的树木全部摧毁，但也远离了昏厥了的萧景等人。
<br/>　　萧逸看了眼那恐怖的攻击力，毫不意外自己若是被它打中，不死也残。
<br/>　　银背妖蟒终究是速度比萧逸快，渐渐追上了萧逸。
<br/>　　嗖，萧逸反应极快，瞬间消失在原地，转身跳上了蛇背。
<br/>　　“虎形。”萧逸一拳重重打出。
<br/>　　嘭，蛇背毫发无损，反倒是萧逸被反震得双臂发麻。
<br/>　　“嘶。”银背妖蟒蛇身一扭动，蛇尾反身拍向萧逸。
<br/>　　“蛇形。”萧逸身体诡异地扭动起来，堪堪躲过了蛇尾的拍击。
<br/>　　“啪”的一声。
<br/>　　萧逸吞了口口水，暗暗庆幸自己躲过。
<br/>　　战斗忽然间变得僵持。
<br/>　　银背妖蟒力量惊人，皮糙肉厚防御骇人，但速度却完全没有先天境的水准。
<br/>　　萧逸借着各种优势，速度快，但攻击却丝毫伤不了妖蟒。
<br/>　　但总的来说，萧逸还是处于劣势。
<br/>　　银背妖蟒随意被攻击也没事，可萧逸，只要被打中一次，他就彻底败了。
<br/>　　萧逸的神经完全绷紧，不敢放松，不断游走，时不时释放下释放武技攻击。
<br/>　　渐渐的，在这般高强度的战斗下，萧逸身上爆发出一股气势。
<br/>　　他突破了，修为重后天一重达到后天二重，在升龙的增幅下，实力达到后天四重。
<br/>　　同一时间，他心中一动，噬火百刃第二道也凝聚成功。
<br/>　　但他没有任何喜悦，神经仍旧绷得紧紧的。
<br/>　　“这个时候突破，希望能有用吧。”萧逸暗暗想着，两道火刃被他同时发出，朝银背妖蟒打去。
