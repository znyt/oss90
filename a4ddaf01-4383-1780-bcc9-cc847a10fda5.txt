　　听到这话的楚涵一愣：“谁跟你说我是五阶？”
<br/>　　邵咏早就在崩溃边缘，听到这话破口大骂：“你还想故弄玄虚？！你开领域是五阶，资料传递过来谁都知道是五阶，气息也是五阶！你不是五阶是几阶？！”
<br/>　　“哦，你误会了。”楚涵说着，直接拉开回炉系统面板，将赚来的百来个积分啪啪啪的一顿加在等阶上，下一秒他的气息便瞬间攀登至六阶巅峰，顺带着也将修罗战斧上的刻度改为了六。
<br/>　　五阶的他开启领域杀高阶丧尸，那数量多到吓傻一帮人，这些积分他都还没用呢！
<br/>　　而累积而起的积分，当然早已超过下一阶的等级提升，只是一时半会儿没想起来忘记升阶而已。
<br/>　　气息的瞬间变化让邵咏瞠目结舌，这是什么操作？
<br/>　　等阶还能说升就升的吗？！
<br/>　　楚涵这时候却盯着回炉系统面版，露出了一个诡异的笑容，由于专心致志的杀丧尸根本没有注意积分的具体数额，这时候一看才发现他距离七阶已经很近。
<br/>　　杀掉眼前之人再杀几个高阶丧尸，就能升级了！
<br/>　　看到楚涵忽然笑起来，邵咏内心冒出了一股撞鬼的感觉，他惊悚的后退好几步：“你不是人类，人类不可能会这么突兀的等级提升！你到底是什么东西？丧尸……还是异种！”
<br/>　　听到邵咏的话，楚涵露出玩味之色，这家伙也太能凭空想象。
<br/>　　“我要出去！我要告诉外面的人你不是人！内部有丧尸异种！”邵咏大喊大叫着，精神已经崩溃。
<br/>　　楚涵也懒得废话，六阶的他感受到自身力量更加熟悉，比之前的状态好了太多。
<br/>　　于是下一刻，带着极致能量风暴的修罗战斧，已经来到了邵咏近前。
<br/>　　恐怖的速度在领域内攀登到极限，邵咏本就如同溺水般受限，此时更加无处可躲。
<br/>　　再一次提升的战力让楚涵如鱼得水，修罗战斧的劈斩而下好似带着毁天灭地之能，一击重重的斩在邵咏头颅最上方。
<br/>　　哗！
<br/>　　迸开的血花和脑浆喷了一地，斧刃持续而下，竟然就这么将邵咏整个劈开成对称的两半。
<br/>　　画面是血腥无比令人作呕，楚涵却惊讶了一下刚刚自己的状态，准头又精进了一些，并且这种战斗力面对比自己高一阶的新人类，竟然依旧是秒杀。
<br/>　　不是邵咏太弱，是重新回到原点又一次逐步提升的楚涵，还在继续变强！
<br/>　　十阶已然是顶峰，竟然还能更强！
<br/>　　有所明悟的楚涵心中有所启发，一时半会儿便没有撤开领域，而是独自站在了其中感悟这一方环境的变化。
<br/>　　对比之前他熟悉的领域，亲和力更强，也不再需要他可以调动能量，这些微小的能量体凝聚在一起，就像是他身体的一部分，如同手足一般感受贴切。
<br/>　　楚涵的这么一感悟，时间不知不觉就拉长。
<br/>　　领域外不仅周军词赶到，还有不少前来支援尸潮战的新人类，他们都听到了消息，这种高价新人类之间的厮杀必然得到极高的关注。
<br/>　　陈正奇也在周军词到场后立即将事情汇报，同时目露担忧：“楚涵现在正以五阶独自对抗一个七阶暗杀者，这个人很强，强到我们都捕捉不到他的气息，更别说追上动作，如果不是楚涵及时开启领域隔绝，可能我和胡承泽都已经死了。”
<br/>　　胡承泽也补充道：“对方就是直接冲着灭口来的，没想过要留什么余地，也幸亏楚涵出手及时，不然恐怕我们连澄清情况的机会都没有！”
<br/>　　周军词听到这里一愣，目光冷然：“澄清情况，这是什么意思？难道这个暗杀者是我们营地的人？”
<br/>　　“是邵咏。”陈正奇说着，瞥了邵咏所在的队伍一眼。
<br/>　　邵咏一队的几个人后背都凉了一下，但很快一人就走出人群，冷笑道：“我看你们是故意陷害吧？邵咏怎么可能好端端的来刺杀你们？还是三个低他这么多的五阶新人类？要知道邵咏可是七阶强者，杀五阶用得着这么躲躲藏藏的？”
<br/>　　一番话说的极有道理，让很多人都迟疑起来，不知道到底哪方是对的。
<br/>　　陈正奇咬牙切齿：“反正你们是想好了，不管情况怎样，都可以用邵咏是高阶新人类，没必要残杀楚涵而开脱？”
<br/>　　“本来就是！”那人毫无惧意的回应：“这个世界本来就是强者为尊，更何况楚涵是个什么个性你们都看到了，今天刚结束一轮厮杀就杀了高远，本就有前科！要我说有问题的也是楚涵！”
<br/>　　“就是！”邵咏队伍中的另一人也站出来，嗤笑道：“哪怕真的是邵咏跑过来找楚涵麻烦，也是帮大家解决问题，楚涵这种人留在营地里只会是个祸害！”
<br/>　　“更何况邵咏根本不可能这么做，杀一个区区五阶的人对他有什么好处？”
<br/>　　“不信你们就等着吧，里面的人一定没有邵咏，不然我把头剁下来！”
<br/>　　他们知道邵咏善于隐匿，在战斗结束后的第一时间理所应当躲起来，而在场比邵咏等阶高的人根本没有，唯一的周军词也不会那么容易发现邵咏的踪迹。
<br/>　　一旦领域打开找不到邵咏的人，邵咏前来暗杀楚涵的这种言论自然就不成立。
<br/>　　几个人一唱一和，将事情彻底扭曲，气的陈正奇脸都涨红。
<br/>　　周军词皱眉，再看看依旧封闭的领域，心道这次恐怕真是栽了……
<br/>　　就在邵咏队伍中的众人都趾高气昂，他们想的很清楚，这一战楚涵是死定了，所有人第一时间关心的都是胜负和尸体，夜色这么黑没人会注意到那柄与众不同的斧头到底如何。
<br/>　　这样一来无论舆论有没有被他们带偏，好处都已经到手！
<br/>　　就在这帮人如此做想，狞笑着等待楚涵以尸体的状态出现时，前方那极致的黑色领域，终于打开了！
<br/>　　顿时，所有人都目光都超前方看去！
<br/>　　陈正奇等人担心之色溢于言表，邵咏队伍中人则是双眼放光神色兴奋。
<br/>　　很快，所有人都率先闻到了一股浓重的血腥味，地面上大`片的血色冲入眼帘，以及被斩开成半的尸体，已经彻底看不清面容。
<br/>　　死的也太惨了！
<br/>　　邵咏队伍中的人开始着急的在地面上寻找修罗战斧，但让他们没想到的是，顺着斧头形状一路看过去，竟然看到那柄斧头正被好端端的拿在一个人的手中！
