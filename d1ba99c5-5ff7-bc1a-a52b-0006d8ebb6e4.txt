        &lt;h3&gt;第三卷第七十五章  江俞轩为张函背锅&lt;/h3&gt;
<br/>
<br/>        这几年西城在大力发展中，一个古老的都市需要和现代化城市接轨，其需要改造的不单单是仪容仪貌，还有很多的方方面面。
<br/>
<br/>        而这座古老的城市容纳的人也愈来愈多，相对应的城市配套的实施基本都会改变。
<br/>
<br/>        江俞轩最近和姐姐一直在加班加点，在某江国际社区周围都是西城南郊的改造项目，西城将要将南郊改造城一个新的文化新区。
<br/>
<br/>        这是西城这几年来改造项目中的最大的一个项目，如果能拿下来，江家在西城就会是跑在前沿的企业，这附近有六个楼盘，一所中学，一所小学，这个项目在十月底就会进入招标流程，一点也马虎不得，所以江俞轩都没有时间参加公司的会议。
<br/>
<br/>        江家姐弟俩吃过饭，江俞琪才知道陈子昂用心良苦，他们后期项目上要接触的两个人，陈子昂都给她介绍了。
<br/>
<br/>        这一顿晚饭是吃得值。
<br/>
<br/>        冯睿在酒桌上问江俞轩和陈子昂什么时候结婚时，大家都怔了一下，陈子霖是知道江俞轩和陈子昂的关系的，他觉得二嫂有点事多，正准备回话。
<br/>
<br/>        江俞轩说：“我们结婚只是一个形式，什么时候结都行。”
<br/>
<br/>        “怎么能什么时候结都行呢，你们都多大了啊？”
<br/>
<br/>        “呵呵，二嫂这是关心子昂的生活吗？我们是先上车后买票，我和子昂的孩子都已经大了，上小学了呢。二嫂就没有必要担心了吧。”
<br/>
<br/>        “你说什么，你们孩子都大了，子昂，是真的吗？”冯睿惊问，陈子建也是一脸吃惊的看着陈子昂。
<br/>
<br/>        “我.....”陈子昂不知道怎么回答。
<br/>
<br/>        “吃虾。”江俞轩将手中剥的虾塞进了陈子昂的嘴里，陈子昂一个“我”字就被江俞轩给打住了后面要说的话。
<br/>
<br/>        “子昂到北京的那一年我们就在一起了，二嫂没有想到吧？我们不但有了孩子而且还是龙凤胎。”江俞轩骄傲的说。
<br/>
<br/>        “哎呀，恭喜，恭喜，你怎么这么大的事情都不告诉我们呢，当初和张函在一起就自作主张，现在还是这样，连孩子都有了啊。”冯睿有些恨铁不成钢的意思。
<br/>
<br/>        江俞轩一时没有回应过来冯睿会这么说。
<br/>
<br/>        “二嫂啊，现在的年轻人想怎么样就怎么样，当父母的和他们自己都不说什么，我们当哥哥姐姐的就不能说他们了吧？”江俞琪听见冯睿如此说陈子昂心里也很不舒服。
<br/>
<br/>        冯睿不是应该维护陈子昂吗？怎么还拆台呢？
<br/>
<br/>        “来，来，乔总，我们喝酒，喝酒，祝乔总生日快乐。”陈子建拿起酒杯，他真是后悔带着冯睿一起来了。
<br/>
<br/>        江俞琪想到饭桌上的情况，问江俞轩：“轩轩，你在吃饭的时候说的话都是真的？”
<br/>
<br/>        江俞轩正在开车，夜灯打在他的脸上忽明忽灭，江俞琪看不到江俞轩脸上的表情。
<br/>
<br/>        “张函最近在西城。”
<br/>
<br/>        “你怎么提起张函来了？”
<br/>
<br/>        “你以后会明白的，对了姐，和陈子昂的二嫂保持距离。这个人不地道。”
<br/>
<br/>        “我还说最近去拜访她呢，陈子建的外号可是宠妻如命呢。”
<br/>
<br/>        “眼睛被鹰啄了，你记住，正常的工作关系找陈子建就好，别的，没有必要去找他们。”
<br/>
<br/>        “听说，冯睿的背景很强，是陆家和张家呢。”江俞琪说。
<br/>
<br/>        “不管她的背景有多强大，你都不要去招惹她，听我的没错，这种人不知道什么时候就会给你挖个坑呢。上次停工的事情就是她和她弟弟搞的事情。”
<br/>
<br/>        “你调查出来了？”
<br/>
<br/>        “知己知彼方能百战百胜，你在西城这么多年了，怎么就不多了解了解西城的对手？”
<br/>
<br/>        江俞琪沉默了。因为有着爸爸在前面支撑着，她从来不屑于勾心斗角。
<br/>
<br/>        “姐，商场如战场，这个是千古不变的真理，我在陈子昂公司里的这几年什么事情都见过，小公司真的很锻炼人。”江俞轩叹了口气，他和陈子昂一起奋斗的日子快要结束了。
<br/>
<br/>        “我知道，可是......”
<br/>
<br/>        “没有什么可是的，一个的人道德感和责任感有可能会随着环境来改变的。不要把人想的那么美好。”江俞轩想到了冯睿。
<br/>
<br/>        “我问你和陈子昂的事情，你给我绕这么多。”江俞琪有些不满弟弟打岔。
<br/>
<br/>        “我和陈子昂之间清清白白，同事和朋友的关系，我要共度一生的人是倩楠。”江俞轩是认定了就不会放手。
<br/>
<br/>        “你今天这样说是有意的？”
<br/>
<br/>        “我看不惯冯睿那种高高在上的样子，在座的哪个人比她差？”
<br/>
<br/>        “你这样做，陈家怕是要误会了。”
<br/>
<br/>        “就是要让他们误会。”陈子霖安排座位是有一定的目的的，估计陈子霖会想到他二嫂会在吃饭的时候给陈子昂难堪？
<br/>
<br/>        陈子霖真是一个有趣的人，不知道后期他要怎么处理家里的事情？
<br/>
<br/>        “你最近在西城，你和伯安他们那边的事情怎么样了？”姐弟俩这几天都忙，忙着连吃饭都在聊项目上的事情。
<br/>
<br/>        “大哥带着大嫂出国考察去了，文文也是到处跑着考察，我们准备做一个新的项目了。”
<br/>
<br/>        “你们也真是有精力，身兼数职，要我就受不了。”
<br/>
<br/>        “我们都是把八个小时的时间当十六个小时来用。不趁着年轻折腾一把，老了就会后悔，当然，社会也需要我们这样的人，哈哈哈，不过，姐，你已经够优秀的了。”
<br/>
<br/>        郝景文给他发的邮件他看了，不知道陈子昂是怎么个打算，这几天忙的都没有时间和她聊天，今天虽然坐一起，但是不适合说别的话，还有，他说那番话的时候，陈子昂一直低着头，想着她那吃惊的表情，他心里爽得就像是一口气吃了八十颗小尖椒，热血沸腾。
<br/>
<br/>        嘿嘿，陈子昂，哥哥我在你身边隐藏了六七年，你就是一个小透明。
<br/>
<br/>        想到陈子昂这个小透明，江俞轩又是一阵苦恼，我走了，陈子昂这个粗线条怎么办？
<br/>
<br/>        “我佩服你们几个。明明有很好的出身还要自己去打拼。”江俞琪感慨的说。
<br/>
<br/>        “谁也改变不了出身时带来的一切，如果你能改变自己，让自己脱离出身所带来的桎梏，按照自己的生活方式去生活，那才叫本事。”对于陈子昂那个小透明，他还是挺佩服的。他们算是同一类人吧？
<br/>
<br/>        “真是不理解你们的想法。”江俞琪其实也很能干，但是她将她的能力归功于家。
<br/>
<br/>        “不理解没事的，你只管当好我的姐姐就是，哈哈哈。”他姐姐也是个厉害的角色，要不然能主管一方业务？
<br/>
<br/>        “明天就休息一天，你陪陪倩楠吧。”江俞琪说。
<br/>
<br/>        因为他们太忙，说好的婚期推迟了，张倩楠也到西城了，张家算是通情达理的，要不然这事情说不过去。
<br/>
<br/>        “他和张函忙着有事情呢。你明天休息一天，我联系一下大哥和文仔。我这几天都没有和他们联系了。”
<br/>
<br/>        张函将姐姐送到家之后，就回到了他和张倩楠暂时的家。
<br/>
<br/>        夜，柔情似水，张倩楠披着齐肩短发正在低着头写写画画。
