　　他们想过无数种可能，可是怎么也想不到，堂堂的冥家大长老，居然被这个元婴境小子给虐成了死狗！
<br/>　　而且这一切发生的实在太快，就连金老和那位万杀门的火云天君，一时间也没有反应过来。
<br/>　　等他们回过神来，冥家大长老，已经像死狗一样，瘫软在了地上，彻底被废掉了。
<br/>　　至于那大公子冥沧山，望着自己最大的靠山，被叶尘轻松暴虐，整个人都快要崩溃了，
<br/>　　“这不可能！这怎么可能？怎么可能？”
<br/>　　冥沧山大呼小叫了一阵之后，这才意识到自己此刻的处境，慌忙连滚带爬，来到那火云天君的跟前，
<br/>　　“火云前辈！您可是已经收了我的钱的，您要救救我啊！！”
<br/>　　开玩笑，对方连大长老都可以轻松完虐，杀他还不是小菜一碟？
<br/>　　火云天君望着眼前苦苦哀求的冥沧山，又望了一眼远处地上，犹如烂泥一般的冥家大长老，还有那一脸云淡风轻的叶尘，不由得嘴角微微一抽，眼眸中透着浓浓的忌惮之色。
<br/>　　到了此时此刻，他哪里还敢再小瞧这个只有元婴境四重的年轻人？
<br/>　　甚至，他有一种感觉，眼前这个透着神秘的年轻人，实力远远不止于此！
<br/>　　虽然他的理智告诉自己，这种想法很荒谬，对方只有区区元婴境四重，再厉害又能厉害到哪里去？
<br/>　　但是他的直觉，却告诉自己，眼前这个人，不可招惹！
<br/>　　万杀门的名誉虽然重要，但也不及自己的老命重要。
<br/>　　一念及此，火云天君直接一脚将冥沧山踹开一旁，冷冷一笑，道：
<br/>　　“大公子，实在抱歉！你提供的信息似乎不太准确，本次任务难度太大，我们万杀门决定取消这次任务！撤！”
<br/>　　撂下这句话，火云天君转身便走，其余万杀门的杀手们，早就被之前叶尘狂虐大长老的一幕吓傻了，如今看到自家老大都走了，他们哪里还敢多待，于是全都一股脑的溜了。
<br/>　　冥沧山顿时傻眼了，愣了片刻之后，转身拔腿便要逃窜。
<br/>　　可惜，叶尘又岂会轻易放过他？
<br/>　　“让你走了吗？给我，滚回来！”
<br/>　　叶尘随手一抓，
<br/>　　呼！
<br/>　　冥沧山只有区区元婴境初期的实力，哪里有什么抵抗之力，整个人完全不受控制的，被叶尘一把拉扯了回来。
<br/>　　“沧荨，杀了他！”
<br/>　　叶尘将冥沧山擒回，直接丢到冥沧荨的跟前，淡淡命令道。
<br/>　　“我……”
<br/>　　冥沧荨顿时有些不知所措，
<br/>　　纵然冥沧荨恨极了自己这个哥哥，可此时的冥沧荨，还没有成长为前世的冥帝，只是一个阳光可爱的小女孩，让她出手杀了自己的哥哥，她还真有点做不到。
<br/>　　那冥沧山似乎也看出了一点，原本都已经绝望的眼眸深处，重新燃起了希望，当即趴在地上，拼命的磕头起来，
<br/>　　“好妹妹，我错了！我真的知道错了！求求你，不要杀我！我发誓，以后再不和你争夺家主之位了，我可是你的亲哥哥啊！你就饶了我这一次吧！……”
<br/>　　冥沧荨看到这一幕，俏脸上的不忍之色，顿时愈发浓郁，不由得望向一旁的叶尘，低声道：
<br/>　　“叶先生，他虽然不是什么好人，但毕竟是我的哥哥，要不就饶他一次吧……”
<br/>　　叶尘顿时眉头一皱，原本想随手一掌，直接结果了冥沧山，不过他转念又一想，
<br/>　　此时自己如果杀了冥沧山，冥沧荨嘴上不说，心里只怕多少会有些怨气，
<br/>　　这丫头还太年轻，根本不知道人心险恶，倒不如利用这冥沧山，给她好好上一课。
<br/>　　“也罢，我尊重你的选择，只是希望你不要后悔！”
<br/>　　叶尘淡淡瞥了一眼地上的冥沧山，不再多说什么。
<br/>　　冥沧山本以为自己肯定必死无疑了，没想到叶尘竟然这么好说话，顿时大喜过望，连忙又是磕头，又是发毒誓，不过眼眸深处，却透着一抹浓浓的阴毒之色。
<br/>　　冥沧荨丝毫没有察觉到，自己这个哥哥，根本就是贼心不死，还以为他真的已经改过，摆了摆手道：
<br/>　　“冥沧山，今天的事情，我可以当做什么也没有发生过，今后我们两个井水不犯河水，希望你不要再来招惹我，带上你的人，你走吧！”
<br/>　　“是是是！好妹妹，你放心，我以后再也不敢了！”
<br/>　　冥沧山如蒙大赦，当即连滚带爬，带着他的那帮手下，屁滚尿流的跑了。
<br/>　　冥沧山一走，于是便只剩下金老，和那几名随着冥沧荨一起来的护卫。
<br/>　　“沧荨小姐，对不起！”
<br/>　　金老和一众护卫，全都纷纷低下了头颅，个个满脸的羞愧之色。
<br/>　　他们身为冥沧荨的护卫，在刚刚危急时刻，竟然将自己的主子丢下，实在有些说不过去。
<br/>　　“你们走吧！我再也不想看到你们！”
<br/>　　冥沧荨淡淡瞥了一眼几人，俏脸发寒道。
<br/>　　她能够接受冥沧山想要杀她，毕竟两人一向不和，但是却不能容忍自己人的背叛。
<br/>　　“小姐，之前的事情，确实使我们不对！不过眼下，其他三大家族的人，肯定都已经进来了，就让我们留下，继续保护您吧！”
<br/>　　金老有些不甘心，继续开口道。
<br/>　　他是冥家的客卿长老，若是不能争取到冥沧荨的原谅，今后他便没办法继续在冥家待下去了，只能沦为无依无靠的散修，这个结果，可不是他愿意看到的，
<br/>　　然而，冥沧荨却丝毫不为所动，
<br/>　　“我说了，我以后都不想再看到你们！”
<br/>　　说着，冥沧荨直接朝丛林深处走去，叶尘不由得摇了摇头，也跟了上去。
<br/>　　在叶尘看来，金老这些人，固然不值得原谅，但是冥沧荨的做法，显然还是太天真了，很有可能会把他们，逼向冥沧山的阵营之中。
<br/>　　而且，出了这种事情，冥沧荨应该第一时间返回冥家，将事情的经过，告知家族长辈，这样才能取得主动权。
<br/>　　若是让冥沧山恶人先告状，她到时候可就麻烦了！
<br/>　　不过，这个结果，反而是叶尘想要看到的。
<br/>　　他要利用这些人，让冥沧荨看清楚人心险恶，所以并没有出言提醒这丫头。
