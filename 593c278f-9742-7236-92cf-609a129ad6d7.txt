夏夕绾站在厨房里准备早餐，“陆先生，我们早餐吃面条好不好，你刚苏醒，饮食还是要清淡一些。”
<br/>
<br/>        陆寒霆倚靠在门框上看着她在厨房里忙碌的小身影，他到现在还觉得自己像是在做梦，“好。”
<br/>
<br/>        “陆先生，那你帮我拿一个碗来。”
<br/>
<br/>        陆寒霆走过去拿碗，但是碗拿在手里，他英俊的眼睑动了一下。
<br/>
<br/>        这个碗是青花碗，清代皇宫里的御用宫碗，现在已经成了私藏的古董，拍卖会上拿来拍卖的那一种。
<br/>
<br/>        这一个碗，价值连城。
<br/>
<br/>        陆寒霆抬眸看了一下，这里不止有御用宫碗，筷子也是象牙白的，一双就在六位数。
<br/>
<br/>        陆寒霆刚苏醒，一开始并没有注意到这些，现在他认真的环顾了一下这个屋子，紫檀木雕刻的桌椅，复古宫灯，就连案板上的刀都是德国工艺，说成白话就是这个屋子入眼都是真金白银堆砌的低调奢侈，奢侈程度简直令人咂舌。
<br/>
<br/>        “陆先生，你拿着一个碗在发什么呆，你喜欢这个碗吗，那我送给你啊，你拿去吧。”这时夏夕绾看了过来。
<br/>
<br/>        陆寒霆一张俊脸上并没有什么情绪起伏，只是淡淡的勾了一下薄唇不动声色的问道，“这个碗…还有这里的装潢用具都是谁准备的？”
<br/>
<br/>        “不知道啊，我被送到乡下的时候这些都有了。”
<br/>
<br/>        她什么都不知道。
<br/>
<br/>        陆寒霆打量着她长大的这个地方，她九岁就被送到了乡下，是被这里的一方水土养大的。
<br/>
<br/>        除了这个表面简单平凡实际遍地奢华的屋子，她的屋外就是大片的园林，在物欲横流的都市之外，这里宁静美好的像是一处仙境。
<br/>
<br/>        外面那片园林太大了，一望无际，陆寒霆开腔，“外面那些蔬菜水果都是你种植的？”
<br/>
<br/>        “那个呀…”夏夕绾迅速露出了羞赧之色，“你觉得我一个人可以搞定那些么，都是邻居大婶们帮我在打理的。”
<br/>
<br/>        陆寒霆看到了，园林里有几个中年妇女的身影，她们手脚干练勤快的在除草施肥。
<br/>
<br/>        “蔬菜水果有了，那如果你想吃肉怎么办？”
<br/>
<br/>        “大婶家的大叔们自家养了猪鸭鸡牛羊之类的，我想吃的话，他们会分我一点。”
<br/>
<br/>        “那如果你想吃海鲜呢？”
<br/>
<br/>        “大叔们会出海啊，鱼虾贝类，什么海鲜都有。”
<br/>
<br/>        “…”
<br/>
<br/>        陆寒霆无话可说，他出身豪门，从小就是含着金汤匙长大的，夏夕绾看不懂的，但他一眼就能看出来。
<br/>
<br/>        她喝的水是山间流淌下来的，蔬菜水果纯天然，想吃肉了杀头猪，想吃海鲜了有人为她出次海…
<br/>
<br/>        陆寒霆看向忙碌的女孩，他一直觉得自己没有看懂她，她身上有很多谜，比如是谁在世外给她建了这么一方娇软奢华天地，护她风雨不侵，将她娇养着养大？
<br/>
<br/>        “绾绾，你回乡下是你爸的安排？”
<br/>
<br/>        “是啊。”夏夕绾突然想到了什么，她回眸看着陆寒霆，“也不算是吧，也跟苏希有关。”
<br/>
<br/>        苏希？
<br/>
<br/>        她的前未婚夫？
<br/>
<br/>        那个苏希竟然这么多年不动声色的将她呵护成了世间最娇嫩的玫瑰花，一手将她养大。
