<br/>				<br/>　　陈默送走了心情高兴的南宫亮，并且还在之前答应了，在自己回来以后，还可以按照同样的方式，炼制其他的丹药！<br/>　　<br/>　　南宫亮提供丹方和药材，陈默炼制。并且依然按照武道界的规矩，十份保三份的原则！这个原则要比给特管局炼制要好的多。<br/>　　<br/>　　当然，条件也不能这么看，乍看上去，好像给南宫世家炼制丹药，陈默要占便宜一些，但是特管局的条件要更加的好。<br/>　　<br/>　　无论是武技的提供，在武道界中，没有那个世家能够将武技随便的提供给别人，而且在世家看来，宁愿花钱和花费一定代价，获得丹药，也比用武技换取丹药要好的多。<br/>　　<br/>　　对于特管局倒是没有这种保护，所有的武技都是长久以来从不同渠道获得，只要是有贡献的人员，就能得到奖励，这也是特管局能够经久不衰的体现。<br/>　　<br/>　　转身，去了趟葫芦谷，召集所有的管理人员开了个会！现在，葫芦谷里在医院的旁边，有个三层小楼作为办公室用，大小会议室和各种用途的功能房间，以及办公室等等，全部都设置在这里。<br/>　　<br/>　　管理人员，经过现在的一些确认以后，基本上由齐亚成负责疗养院，陈萍负责酒厂，陈海，也就是大海负责养殖和种植，还有丁兰花负责财务这一块！<br/>　　<br/>　　所有的人暂时这样划分了，等到时候如果有可能，或者说拓展新的事业了，在做调整。其他一些人，比如陈金贵，姚爷等等，各自负责一块，到时不用过来开会。<br/>　　<br/>　　陈默在会议中交代了一下，自己要出去一阵，时间不定，但是所有的人，相互之间配合做好所有的工作。在自己离开期间，这里的工作由陈萍居中协调等等。<br/>　　<br/>　　对于陈默来说，现在疗养院也好，菜地也好，酒厂也好，其实对他来说并没有放在心上。主要是现在父母都在，而且他也需要个收入，所以才会搞这些事情。<br/>　　<br/>　　随着他的实力渐渐升高，其实光是炼制丹药就能换取很大的利润，而且还能辅助修炼。所以现在的这些事情，能放手的他已经放手了，管理的反而有些松散。<br/>　　<br/>　　好在现在这些管理人员，都是他亲近的人员，而且也不是大撒手，所以还能维持下去。要是换成其他人，早就开始发生什么中饱私囊，挖空财务等等事情了！<br/>　　<br/>　　将所有的事情交代了一遍，也将近期的工作布置了一下之后，陈默三言两语的就宣布散会，留下了姐姐陈萍。<br/>　　<br/>　　“姐！我不在的时候，你可要多看看多操心一下！”陈默说道。<br/>　　<br/>　　“没有问题！不过二娃，你这是要做什么去，要这么长时间？”陈萍问道。<br/>　　<br/>　　刚才在会上，陈默说了一下离开的时间，最长半个月时间。<br/>　　<br/>　　“哦！主要是去一下东南亚！”陈默说道。他没有说什么事情，所以陈萍也就不好在追着问了！<br/>　　<br/>　　“难道非要你去，不能安排其他人替代你么？”陈萍问道。<br/>　　<br/>　　“嗯！事情有些麻烦，必须我一个人过去。”陈默说道。<br/>　　<br/>　　“好吧！我知道了！”陈萍回答道。<br/>　　<br/>　　不过，接着陈萍又说道：“二娃，你这里还是要找个专业的管理人员，不然我们真的管不了！如果你的事业要发展，还是走正规化比较好！”<br/>　　<br/>　　“咦！老姐，你也想到了？”陈默本来就有这种想法，不过从那里找专业的管理人员，也是个问题，所以事情就拖了下来，现在姐姐倒是提出了如此的问题，也让他有些惊讶。<br/>　　<br/>　　主要是老姐以前并没有管理过企业，而且问话程度不高。在离婚前主要是在婆家忙里忙外的，现在说出这样的话，说明姐姐已经开始用管理者的思维，在思考问题了！<br/>　　<br/>　　“这个事情我一直在考虑，不过由于还没有找到合适的人员，再说了我一直在家里待着，也就放松了这一块！”陈默笑着说道。<br/>　　<br/>　　“这可不行！我上次看书，书上说道管理一个企业，不是你是企业的创始人，就能管理好一个企业！企业要发展，就要引进正规制度，引进管理人才，这样才能让企业大好基础！无论是你在不在，都应该引入管理人才！”姐姐陈萍说道。<br/>　　<br/>　　哈！三日不见当刮目相看啊！没有想到姐姐现在能够对企业发展侃侃而谈，倒也出乎陈默的意料！<br/>　　<br/>　　“好的！老姐，这件事情等我回来，会好好计划一下！”陈默说道。<br/>　　<br/>　　其实陈默一直有个人选，但是这个人选可能还看不上他这里。这个人就是席志涵，这个女人的管理能力毋庸置疑，很厉害！<br/>　　<br/>　　但是她现在手头一大堆的事情，而且她家里还有建筑公司等着她去管理，恐怕自己这边就会看不上的。而且现在爽肤水销售的这么厉害，带给她的利益，也是爽歪歪，更加不会来陈家村，给他管理这些事情了！<br/>　　<br/>　　因此，管理人员，还是要慢慢寻找。<br/>　　<br/>　　等陈默和姐姐分开以后，走回别墅的时候，就碰到三个身穿黑色衣服的人员。浑身冒着彪悍气息不说，身体上的气血之气，也是让陈默知道，这三个人都是武者，而且等级都在后天六层以上。<br/>　　<br/>　　“陈默，你好！”来人见到陈默以后，就上前答话！<br/>　　<br/>　　“嗯你们好！请问……？”陈默早就发现这三个人了，但是这三个人没有露出敌意，所以就装作奇怪的问道。<br/>　　<br/>　　“你好，我是上市特管局的外勤人员，接到宁局的命令，想你移交物品，请你签收一下！”说着，领头的人拿出一张纸，递给陈默。<br/>　　<br/>　　陈默接过一看，就明白是药材，这是宁永志准备炼制黄龙丹的药材。<br/>　　<br/>　　“请随我来！”领头的人，带着陈默来到路边停着的一辆军用越野车边，然后挥手示意的了一下，身后另外两个人就开始从车里取出合金制的几个箱子。<br/>　　<br/>　　四个箱子，都有着大型拉杆箱的大小，上面都有贴着货物的编号。<br/>　　<br/>　　“请核对一下单号！货物总共四个箱子，每个箱子的编号都在货单上有写！另外这些箱子是密码箱，想要打开它们，还请和宁局联系，他会给你提供密码！”领头的人，一脸严肃的说道。<br/>　　<br/>　　陈默手里拿着单据，一扫之下就能核对好，根本不需要专注的去一一核对，直接点头说道：“好的！数量正确，箱号也没有问题！”<br/>　　<br/>　　“那好，这些箱子你要放哪里，我们给你放过去，就算是全部移交给你了！”领头说道。<br/>　　<br/>　　陈默表示感谢，然后带着他们将箱子一一放入房间中。东西放好以后，三个人转身就走，倒也干脆利落！富品中文<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
