　　吞噬武魂释放而出，汹涌的吞噬之力，顿时席卷天地，笼罩八荒，充斥在玉镯空间的每一个角落。
<br/>　　方圆三百里的空间，完全被强大的吞噬之力笼罩，没有任何的例外之处。
<br/>　　霎时之间，那些黄泉魔宗、阴阳神宫和北堂世家之人的噩梦，来临了。
<br/>　　嘭嘭嘭！！
<br/>　　那些修为不到武皇境的武者，在强大的吞噬之力下，纷纷暴体而亡，掀起漫天气浪，汹涌澎湃。
<br/>　　啊！啊！啊！
<br/>　　惊恐的惨叫声不绝于耳，除了那些纷纷暴体而亡的武者，那些低阶的武皇，个个体内玄力和气血疯狂向外流动，根本就阻止不住。
<br/>　　就算是大魔老、二魔老、殷棕、阳鼎天等准帝强者，面对苏莫的吞噬之力，都感觉到自己体内的血液和玄力急速流逝。
<br/>　　“攻击！大家一起出手攻击他！”二魔老放声大喝，他们无法抵抗苏莫的吞噬之力，只能出手攻击释放出吞噬之力的苏莫。
<br/>　　“一起出手！”
<br/>　　“快出手！”
<br/>　　“联手杀了他！”
<br/>　　众人放声大喝，纷纷出手，恐怖的玄力浩荡九天，无数的攻击遮天蔽日，浩浩荡荡向苏莫镇杀而去。
<br/>　　但是，此空间玉镯，早已被苏莫炼化，苏莫就是这片空间的主宰，别说这些人，就算人数再多十倍，也难伤他分毫。
<br/>　　苏莫意念一动，整片空间如同凝固了一般，众人的攻击，立刻被无形的空间之力所束，而后开始缓缓消散。
<br/>　　啊！啊！啊！！
<br/>　　惨叫声依旧在不断的响起，每分每秒都有人在不断的死亡，几乎是眨眼之间，那些武皇境之下的武者，便已经死亡殆尽。
<br/>　　只剩下了数千名武皇境的强者，在苏莫的吞噬之力下，还能勉强支撑。
<br/>　　无数死亡的武者，体内的玄力、武魂和气血，浩浩荡荡，向苏莫狂涌而去。
<br/>　　苏莫开始疯狂的吞噬了起来，疯狂吸收玄力和气血。
<br/>　　“攻击！”
<br/>　　“攻击啊！”
<br/>　　“杀了他！”
<br/>　　二魔老、殷棕等人不停的怒喝，个个满脸惊骇，他们体内的玄力和血液不断流失，就算以他们的修为，不需片刻也会支撑不住了。
<br/>　　众人疯狂的出手，不断的向苏莫攻击，但根本无济于事，连苏莫的毫毛都伤不到一根。
<br/>　　时间缓缓流逝，约莫十几个呼吸之后，一些低阶武皇都变成了干尸，只剩下百余名强者，还在苦苦的坚持。
<br/>　　这百余人也不再攻击了，因为根本起不打任何的作用。
<br/>　　他们开始极力压制体内的玄力和气血，让其减缓流逝。
<br/>　　“苏莫，放过我们，放过我们吧！”皇甫擎虚弱的向苏莫喊道。
<br/>　　此刻的皇甫擎非常的虚弱，他整个人已经骨瘦如柴，形如干尸，狰狞无比，连站立都快站不稳了。
<br/>　　皇甫擎虽然体内没有玄力，但是却有血液和气力，几乎被吸干了。
<br/>　　没有血液，就算是以皇甫擎炼体武者的体质，也活不下去。
<br/>　　苏莫仿佛没有听到皇甫擎的声音一般，双眸紧闭，极力的吞噬，冲击准帝之境。
<br/>　　毫无意外，两个呼吸之后，皇甫擎死了！
<br/>　　又过了十几个呼吸之后，活着的人，已经不足二十人了。
<br/>　　“要死我们一起死！”
<br/>　　黄泉魔宗大长老阎泰，面露疯狂之色，身形如电，向苏莫冲了过去。
<br/>　　待接近苏莫数十丈之后，阎泰的身躯急速膨胀，而后轰隆一声爆炸开来，化为漫天的冲击波。
<br/>　　武皇境九重武者阎泰，自爆了！
<br/>　　他想以自爆来伤及苏莫，但是，他的自爆根本伤不了苏莫分毫。
<br/>　　又过了片刻之后，殷棕死了、阳鼎天死了、大魔老死了，所有的人全部都死了。
<br/>　　整个玉镯空间之中，便只剩下了苏莫一人。
<br/>　　无数驳杂的玄力，浩浩荡荡的血河，各式各样的武魂，将苏莫完全包裹了起来。
<br/>　　苏莫体内闷响不断，无数的玄力、气血被他炼化，涌入了神丹之中，让得他五枚神丹不断的震动。
<br/>　　很快，所有的玄力、气血和武魂，便全部被苏莫吞噬一空，他的修为也达到了武皇境九重的极限，距突破只差最后一线了。
<br/>　　“还差一点！”
<br/>　　苏莫心中大急，若是此次冲击境界失败，下次还不知道什么时候，他才能收集够资源。
<br/>　　随即，他扫了一眼远处满地的尸体，立刻开始收集储物戒。
<br/>　　他大手连抓，立刻将一些强者的储物戒收取到了身边。
<br/>　　哗啦啦！
<br/>　　一个个储物戒，被苏莫全部打开，其内的物品全部倾倒而出。
<br/>　　滚滚灵石，堆积成了小山。
<br/>　　所有人的储物戒内，加起来有极品灵石十数亿。
<br/>　　这些人在玉镯空间中待了这么久，资源都快被消耗光了，灵石实在是少的可怜。
<br/>　　苏莫立刻开始吞噬这些灵石，但很快这些灵石，便再次被他消耗一空，他还是没有突破境界。
<br/>　　“灵晶！”苏莫想到储物戒内，还有一块仅剩的灵晶，立刻便取出了出来。
<br/>　　而后，他手握灵晶，开始吞噬灵晶中的灵气。
<br/>　　轰隆！
<br/>　　灵晶受到吞噬之力的影响，其内的灵气之海顿时炸开，精纯至极的灵气顿时向苏莫体内狂涌。
<br/>　　“好精纯的灵气！”苏莫面现惊讶之色，难怪这种灵晶如此的珍贵，一块就相当于十亿极品灵石！
<br/>　　这灵晶之内的灵气，或许远不如十亿极品灵石那般浩大，但胜在精纯，比极品灵石中的灵气精纯了数十倍。
<br/>　　有了灵晶中精纯灵气的加持，紧紧是几个呼吸的时间，苏莫便冲破了境界壁障。
<br/>　　轰隆！
<br/>　　只听苏莫体内，五枚神丹猛然一震，便开始急速暴涨，眨眼之间便扩大了数倍，他身上的气息变得更加磅礴、浩大。
<br/>　　呼！
<br/>　　少顷之后，灵境中的灵气也消耗一空，苏莫睁开了眼眸，眸中精光璀璨。
<br/>　　终于突破到了准帝之境，接下来的时间里，他只需提升武魂便可以了。
<br/>　　随即，苏莫手臂连连挥动，道道气浪席卷，将玉镯空间中无数的尸体，全部都卷到了一起。
<br/>　　而后，他动用火属性玄力，将这些尸体全部焚烧成了灰烬。
<br/>　　一切搞定，苏莫便立刻离开了玉镯空间，回到了外界的阁楼中。
