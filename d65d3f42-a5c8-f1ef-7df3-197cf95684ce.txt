　　“姑娘，轮回神朝有六大王宫，各自为政，为何没有联合在一起？”苏莫向白裙女子沉声问道。
<br/>　　他的问题，问的很巧妙，既能了解到情况，又不至于让三人怀疑他的身份。
<br/>　　“天尊大人进入了轮回，轮回神朝无人统领，无人有此威望，能统领六大王宫！“白裙女子骆妤淡淡的说道。
<br/>　　这种事情，在轮回神朝人尽皆知，就算她说出来，也没有什么影响。
<br/>　　苏莫闻言，心中惊疑不定，轮回神朝无人统领，那么，这就说明帝一魂还没有掌控轮回神朝。
<br/>　　但是，这里面是什么原因呢？
<br/>　　是帝一魂没有去掌控轮回神朝？还是没有能力？亦或是六大王宫生出了异心？
<br/>　　苏莫想不清楚，心中立刻急思，第二个问题。
<br/>　　少倾，苏莫问道：“这样的话，轮回神朝在各大神朝之中岂不是最弱，难道六大王宫之中，没有人想一统神朝？”
<br/>　　“当然有，在轮回神朝，天尊大人有两具轮回之身，分别在天王宫和地王宫！”骆妤轻声道，虽然不知道眼前的青年为何如此关心轮回神朝的局势，但她也没有隐瞒。
<br/>　　苏莫闻言，面色不变，心中却是大震。
<br/>　　一瞬间，他的心中闪过万般念头，他终于明白，轮回神朝六大天宫为何没有被帝一魂一统了。
<br/>　　那是因为另一具轮回之身，拓跋青云。
<br/>　　因为拓跋青云，得到了其中一个王宫的支持，有和帝一魂抗衡的实力。
<br/>　　苏莫不清楚，那位拓跋青云是出生在神域，还是也是飞升之人。
<br/>　　他也不清楚，那位拓跋青云，有什么机遇，得到了一个王宫的支持，但是，其能抗衡帝一魂，这是肯定的事情。
<br/>　　不然的话，其也不会活到现在。
<br/>　　不过，苏莫心中也有疑惑，这个疑惑就是，帝一魂是主魂，为何六大王宫，不全部支持帝一魂？
<br/>　　“你还能问最后一个问题，问完立刻离开！”骆妤见苏莫沉默不语，淡淡的道。
<br/>　　苏莫闻言，微微的吸了口气，继续问道：“天尊的轮回之身，定然无比强大，那两位如今都是什么修为？”
<br/>　　“都是中极神王！”骆妤道。
<br/>　　“明白了，告辞！”
<br/>　　苏莫点了点头，取出一个储物戒，丢给白裙女子，微微抱拳，便转身离去。
<br/>　　既然答应给报酬，他自然不会食言。
<br/>　　骆妤结果储物戒，稍一查探，便收了起来。
<br/>　　这点源石，虽然对于她来说不值一提，但既然对方给了，她也不会弃之。
<br/>　　“骆妤姐，这人好奇怪啊！”待苏莫的身影消失，娇小女子秀眉微皱道。
<br/>　　“不错，此人如此打听我们轮回神朝的情况，定然有什么企图！”褚彦点头道。
<br/>　　“不必理会，他一个虚神境武者，能有什么能耐？”骆妤摇头，旋即又道：“我们准备好六日后进入时空之城才是首要任务。”
<br/>　　“嗯！”褚彦和娇小女子点头，随即三人便各自回房了。
<br/>　　他们从轮回神朝人王宫来此一趟不容易，必须不虚此行。
<br/>　　……
<br/>　　苏莫回到了房间之中，他盘坐在蒲团之上，眸光深邃，久久无声。
<br/>　　六大王宫，天王宫、地王宫？
<br/>　　苏莫心念电转，他想到了一个计划，这个计划很大胆，但是，却能险中求胜。
<br/>　　这个计划，就是去轮回神朝，其它的四大王宫。
<br/>　　帝一魂和拓跋青云，能各自得到一个王宫的支持，他也同样有希望。
<br/>　　只要得到了一个王宫的支持，他就能拥有抗衡帝一魂的底气了。
<br/>　　当然，此法太过冒险，他对于六大王宫的具体情况，并不清楚。
<br/>　　一旦计划失败，那就是生死道消。
<br/>　　目前，摆在他面前的，有四条路。
<br/>　　第一条路，自然是去找绫裳，有绫裳的关照，他会安全很多。
<br/>　　第二条路，自然是自己独自修炼，默默的蛰伏。
<br/>　　第三条路，是去混沌神朝，他修炼大混沌五行神诀，继承了混沌神帝的道统，加入混沌神朝问题不大。
<br/>　　第四条路，便是去轮回神朝。
<br/>　　苏莫安静的盘坐，心中不断的思索，思索四条道路的利与弊。
<br/>　　第一条路是首选，第二条路最自由，第三条路有保障，唯独第四条路没有保障。
<br/>　　不过，苏莫反而觉得，第四条路，有最大的保障。
<br/>　　因为前三条路，并不能保证他的安全。
<br/>　　神王的神通，广大莫测，估计帝一魂想要找到他，难度并不是很大。
<br/>　　所以，苏莫有一种感觉，只要对方确定他已经来了神域，他始终会被对方找到。
<br/>　　而一旦被对方找到，生死就由不得他了。
<br/>　　目前来看，别说几十年了，几百年之内，他怕是都很难抗衡帝一魂。
<br/>　　那么，若是想要保命，就必须要靠外力，或者说有靠山。
<br/>　　这个靠山首选混沌神朝，毕竟混沌神朝够强大，混沌神帝隐隐有神域第一强者的威名。
<br/>　　只不过，苏莫对于混沌神朝，还有迟疑。
<br/>　　混沌神朝，会因为他修炼了混沌道统，而全力庇护他吗？
<br/>　　苏莫仔细考量这一点，混沌神朝会不会庇护他？不能因他的天赋来衡量？
<br/>　　而是要看混沌神朝，想不想让轮回天尊回归？
<br/>　　乍一看，混沌神朝貌似不可能希望轮回天尊回归，因为轮回天尊回归，轮回神朝将再次重铸巅峰。
<br/>　　但是，事实好像并非如此，因为帝一魂和那个拓跋青云都还活着。
<br/>　　若是混沌神朝，或者是其它神朝，不希望轮回天尊回归，那么只要对帝一魂和拓跋青云下手便可。
<br/>　　但是，帝一魂和拓跋青云，却是顺利的成长到了神王的境界。
<br/>　　那就说明，其它神朝，没有对这两人动手。
<br/>　　不然的话，那轮回神朝六大王宫虽强，但是，决不可能挡住那些太古至强者。
<br/>　　这又说明了什么？
<br/>　　这说明，其它神朝，不介意轮回天尊回归，甚至是希望轮回天尊回归。
<br/>　　苏莫不知道，这是因为什么原因，但是，他知道，自己的这个猜测，应该不会有错。
<br/>　　而一旦是这样，他去轮回神朝的话，那就危险了。
<br/>　　“若是混沌神朝不介意、或者是希望轮回天尊回归，那我就不能去混沌神朝！”
<br/>　　苏莫低声自喃，摆在他面前的路，决定着他的生死，也决定着他的命运。
<br/>　　“去轮回神朝！”
<br/>　　少倾之后，苏莫狠狠的咬了咬牙，不入虎穴焉得虎子，他要拼上一次。
<br/>　　而且，最危险的地方，或许是最安全的地方。
<br/>　　就算去了轮回神朝，不能达成目的，也能为他争取到成才的时机。
<br/>　　想到此处，苏莫松了一口气，仿佛心中的大石，轰然落地了。
<br/>　　有了计划，便好前进了，这比他像无头苍蝇一般，在神域左冲右撞要好的多。
<br/>　　虽然风险很大，但是，一旦计划达成，就能扭转局势，获得生机。
<br/>　　不过，去轮回神朝，还要待他去过时空之城再说。
<br/>　　片刻之后，苏莫抛却心中的杂念，开始了修炼。
<br/>　　时间匆匆而过，六日的时间，转瞬即逝。
<br/>　　清晨，苏莫早早的便离开了客栈，向天空之城而去。
<br/>　　当苏莫来到城中央，那片辽阔的空地之时，发现此地的人，比六日之前多很多。
<br/>　　熙熙攘攘，乌压压一片。
<br/>　　“这么多人都是要进时空之城？”苏莫暗暗惊讶，不是说时空之城，危险很大吗？这些人都不怕死吗？
<br/>　　“褚彦哥、骆妤姐，你们看，那人又来了！”
<br/>　　不远处的人群之中，娇小女子看到了苏莫，轻声向身边的褚彦和骆妤道。
<br/>　　“沁儿，无需理会，此人估计也是要进入时空之城！”
<br/>　　褚彦轻声道，言罢，他嗤笑一声，又道：“此人修为不足真神，居然也敢进时空之城，真是不知死活！”
