　　第414章
<br/>　　李行哉愣了好大一会儿，随即发出一声狂笑。
<br/>　　“宋伯康啊宋伯康，我真有些佩服你了，事到如今，你竟然能想到这样的鬼话，我信了你才是有鬼。”
<br/>　　“殿下心中已经信了，不是么。”宋伯康道。
<br/>　　李行哉外表的伪装被击破，他立在两军中间，显得有些孤单，随后他狠狠的摇摇头。
<br/>　　“不，这不可能，你在诓我。”
<br/>　　“臣怎敢欺瞒殿下，消息的确是昨夜才得到。就在七天前，贼匪攻破长安城。”宋伯康。
<br/>　　“我父王呢？”李行哉失声问。
<br/>　　“明帝陛下已死在贼匪手中。”
<br/>　　“啊！”
<br/>　　李行哉失声发出一声尖叫，像是被踩住尾巴的猫。他猛烈的摇头，想要否认这个事实。
<br/>　　“不，这绝无可能，长安不可能破。”
<br/>　　“殿下，长安已经破了。”宋伯康。
<br/>　　李行哉陷入巨大的恐惧中，周围是浓稠挣不开的黏稠黑汤，压得他无法呼吸。他已触探到真相边缘，但那真相就是深渊。他不想靠近，却又鬼使神差的走向最深处。
<br/>　　他目光看向程大雷，希望从程大雷身上得到否定的答案。这明显是开玩笑么，长安城坚，凭着区区十万乌合之众，怎能攻破长安。
<br/>　　蒙蒙天光中，程大雷看着李行哉，无可奈何的点点头。
<br/>　　算下来，程大雷和宋伯康几乎是同一时间得到消息的。阿喜在收到情报后，第一时间就向程大雷汇报了。
<br/>　　半个月前，正义教开始对长安城开始全面进攻，这也是没有办法的事情，弹尽粮绝，正义教面临的情况是要么退兵，要么攻城。
<br/>　　破釜沉舟，死而后已，正义教爆发了强悍的战斗力，帝国兵也拼死抵抗，战斗进行了七天七夜，长安城下堆满了尸体。
<br/>　　就在七日前，伤亡惨重的正义教终于敲开了那座天下第一城的城门，随之就是巷战，天下第一城变为天下第一杀戮场。
<br/>　　正义教首领林少羽亲率亲卫攻入皇宫，与帝国鱼龙卫正面接触，林少羽展现出前所未有的战斗力，亲手用长枪捅穿明帝的心脏。
<br/>　　他报了杀父之仇。
<br/>　　随着程大雷的点头，李行哉一颗心如同冉冉升起的气球，在高空被飞鸟的尖喙啄破，鲜艳的红气球瞬间干瘪下来，变得毫无生气，垂垂落地。
<br/>　　时间过去得太久，长夜将尽，日头在东方出现个角，天快亮了。然而李行哉却觉得周围从未像今日一般黑暗。
<br/>　　没有希望的黑暗。
<br/>　　在长安，李行哉常常被看做废物王子。沉迷酒色，放浪形骸，在关于皇位的争夺赛中，他很早就被淘汰出局。如李乐天等皇子身边，早就有了一批支持者，可从开始到最后，李行哉身边永远是他一个人。
<br/>　　可无论李行哉是不是废物，但他绝不会认为自己是废物。不过是，有些道理他想不明白，有些别人告诉他的事情他不信，非要自己尝试了才接受。
<br/>　　明帝的子孙，骨子里都有种以天下为己任的基因，李婉儿如此，李行哉也是相同。很小的时候，他就面对这样一个帝国：戎族横行无忌，诸侯阳奉阴违，百姓生不如死，皇权一日一日衰老。
<br/>　　归根结底，就只剩下一个问题：帝国怎么会变成今日这个样子？
<br/>　　为了寻找问题的答案，他走过许多地方，也拜过很多老师。他去幽州、青州、并州、草原，如一个乞丐般流浪。将流民当老师，将逃兵当老师，将戎族当老师……将程大雷当老师。
<br/>　　他在蛤蟆寨待了三年，但他什么时候开始这过程，却是无人知道。
<br/>　　三年前，戎族大举入侵，李行哉想力挽狂澜，却败得一塌糊涂。如今已经是三年后，在这三年内，他当过逃兵，做过苦工，与戎族做生意，至于失败、被鄙视、落魄对他来说更是家常便饭。
<br/>　　李行哉本不在意失败，能从一场场失败中活下来的人，总会比其他人更强大些。
<br/>　　如今，他已觉得自己足够强大，已有足够的能力去改变一些事情。
<br/>　　然而，现实狠狠给了他一巴掌。
<br/>　　长安城破，明帝被乱贼杀死，自己毕竟什么都没有做，被人关在凉州城，当成一个废物。
<br/>　　或许，自己本就是一个废物。
<br/>　　李行哉形单影只的立在那里，衣衫褴褛，浑身上下充斥着从阴沟里爬出的恶臭，身上的确没有半点身为王子的体面。
<br/>　　“殿下，长安城破，各路诸侯已打出破贼的旗号，长安正等您主持大局呢。”宋伯康。
<br/>　　李行哉失声狂笑：“破贼，如果你肯出兵，贼早破了。现在，你舍得出兵了！”
<br/>　　宋伯康一辑到地，毕恭毕敬道：“只等殿下一声令下，臣便率兵去长安破贼，鞠躬尽瘁，万死不辞。”
<br/>　　李行哉眼中的怒火要把宋伯康燃为灰烬，如果不是他把自己拖在此地，自己哪怕是单枪匹马赶到长安城，就算不可破贼，但总能做些什么的。哪怕是陪着明帝共存亡……可结果却是什么都做不了，李家百年天下，一朝覆灭。
<br/>　　宋伯康似乎没有觉察到李行哉眼底的愤怒，依旧平静道：“殿下，凉州十万精兵，就等着殿下下令了，我们愿以您为主，共讨逆贼。”
<br/>　　宋伯康身后，人越聚越多，他们渐渐已有了阵型。同时拱手，异口同声喊着一句话。
<br/>　　“奉天为主，共讨逆贼。”
<br/>　　赵子龙眉头皱紧，趁乱出击，己方还有胜算，可时间这么拖下去，等对方大军集结，想要获胜就不容易了。
<br/>　　他目光看向程大雷，询问他要不要主动出击。程大雷摇摇头，目光遥遥望着李行哉，等待着他的抉择。
<br/>　　李行哉的确面临艰难的抉择，从他本心出发，他恨不得将宋伯康挫骨扬灰。但帝王之道是无情道，不仅是杀戮他人时的无情，也包括杀戮自己的感情。
<br/>　　如今两支军队对垒，三千龙字军人数虽少，但却气势如龙。凉州兵虽多，但却显出乱象。
<br/>　　但强大的龙字军不是自己的兵，凉州兵至少奉自己为主，他们日后未必不能成为自己的人。而离开此地，想要拉起一支人马谈何容易。
<br/>　　两军对垒，一东一西，李行哉不知自己该何去何从。
<br/>
