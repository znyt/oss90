
<br/>　　<br/>　　    王惊蛰没有反抗，被鬼王和夜游给扣住之后，直接就被押上了黄泉路，顺着黄泉路往奈何桥方向走，准备过桥就进入地府了。<br/>　　<br/>　　    一路上，由于鬼门已经关上了，黄泉路上有很多孤魂野鬼，浑浑噩噩的排着队，特别是奈何桥前等着喝汤的就更多了。<br/>　　<br/>　　    王惊蛰夹杂在鬼群里，面无表情的看着四周，这些都是最近刚死没多久的人来地府报道的，那死样千奇百怪，什么样的都有，就像他身前有个鬼整个右半边身子都碎了，明显是被大力道给砸的，并且对方看着还挺年轻，也不过就二三十岁的年纪。<br/>　　<br/>　　    王惊蛰打量对方几眼，叹了口气，问道：“兄弟，是不是想不开走的啊？跳下来的吧？”<br/>　　<br/>　　    对方没抬头，低着脑袋“嗯”了一声，声音干涩嘶哑还有点呜咽。<br/>　　<br/>　　    “何苦呢······”王惊蛰摇了摇头。<br/>　　<br/>　　    自杀的人进了阴曹地府，是不被阴间正常的死亡状态所承认的，从规矩上来讲，自杀死的几十年内都不能投胎转世，要受尽刑责才行，他们这一群人会被关进炼狱中受苦，直到孽障消除才可。<br/>　　<br/>　　    这是为啥呢？<br/>　　<br/>　　    人之性命，身体发肤受之父母，是被大道所认同的，你以自尽的方式结束了自己的性命，无疑等于是在反抗天道人伦，上天不责罚你罚谁啊？<br/>　　<br/>　　    生老病死是人之常情，除此以外都不在常情之列，很多事实就是如此，自杀者不入轮回，溺水而亡者心有不甘多数会化为水鬼，被人杀了的则就更不甘了，九成九都会变成亡魂厉鬼，所以啊，人还得是生老病死才行，否则遭罪的不光是你自己，还有你身边的人。<br/>　　<br/>　　    王惊蛰跟着排队的鬼群挪蹭着上了奈何桥，过了奈何桥就是望乡台，台上有个老妇人端着一碗汤，这就是孟婆汤，喝了可以忘记前生所有的爱恨情仇都一去不复返，能够让你彻底的忘了一切，然后再来一回重新做人。<br/>　　<br/>　　    据说孟婆汤是从十殿被发配到各处的鬼魂还有一些药材调制而成的，至于真假那就无从得知了。<br/>　　<br/>　　    排到王惊蛰的时候，孟婆刚把一碗汤递过来，就皱了皱眉，看着鬼王问道：“生人？”<br/>　　<br/>　　    “孟婆，此人魂魄不受拘束，不知为何我们的拘魂索根本拿不出他的三魂七魄，所以才送到奈何桥上，让你为其喝一碗孟婆汤，看看能不能再把他的魂魄给抽出来”<br/>　　<br/>　　    孟婆摇头说道：“这不符合规矩，生人不可”<br/>　　<br/>　　    夜游说道：“他触犯了阴间条列，擅自闯入阴曹地府，打伤阴差，还盗取了忘川河边的彼岸花，也私自放走了忘川河中的亡魂，这个罪罚足够他被押往炼狱受刑了”<br/>　　<br/>　　    王惊蛰鄙夷的说道：“上下两嘴唇一碰，随便你们怎么说咯？”<br/>　　<br/>　　    鬼王说道：“你还有什么可不承认的？鬼门开，你们擅闯阴间这事有没有吧？”<br/>　　<br/>　　    王惊蛰转过脑袋给他们来了个神之蔑视的眼神，压根也不想解释了，这个时候他说啥都没有用，在阴曹地府始终还是阴差和阴帅说了算的，他就是长了一百张嘴也说不清，毕竟确实是他擅闯阴间的。<br/>　　<br/>　　    夜游跟孟婆说道：“孟婆汤你先给他喝了，到时我再带他去判官那里断案，有罪他是肯定的了，就看怎么断了，但不管怎么说要进地府总归得是带着他的魂魄去吧，生人是不能进的”<br/>　　<br/>　　    “你的拘魂索都没能把他的魂魄抽出来，你确定我的一碗孟婆汤可以？”<br/>　　<br/>　　    鬼王和夜游顿时一愣，确实这个问题没有考虑过，万一也不行呢？<br/>　　<br/>　　    王惊蛰笑了，摆着手说道：“我也想低调，可是实力不允许啊······”<br/>　　<br/>　　    鬼王咬了咬牙，说道：“给他喝汤！”<br/>　　<br/>　　    王惊蛰端上一碗孟婆汤的时候，其实他也是犹豫的，心里也直敲鼓，这碗汤他到底该不该喝？<br/>　　<br/>　　    坦白的讲，王惊蛰也有点怕，他没听说过活着的人有谁喝过孟婆汤自然也不知道是什么结果，那么一切就只能是赌了？<br/>　　<br/>　　    那王惊蛰有没有反抗的机会呢，也有，他甚至能保证的是自己全力一搏的话，此时也有可能从奈何桥上逃脱出去，甚至鬼王和夜游都未必能留下他，但王惊蛰没想这么做。<br/>　　<br/>　　    对于一个受了天谴，三十岁的时候就有可能毙命的人来讲，三十岁之前所受的磨难其实都是过眼云烟，反正我都有可能活不过三十了，我还用在乎这些么？<br/>　　<br/>　　    所以，王惊蛰想要搏一把，他想试试看，这一碗孟婆汤喝下去后会不会对他的三魂七魄有什么影响。<br/>　　<br/>　　    如果有，那更好。<br/>　　<br/>　　    如果没有，那他就再跑也不迟。<br/>　　<br/>　　    王惊蛰端着孟婆汤，凑到嘴边，先添了一口，无滋无味的没有什么感觉，就跟一碗清水差不多。<br/>　　<br/>　　    “咕嘟，咕嘟······”王惊蛰随即仰头就干，一碗汤不过眨眼的工夫就被他一饮而尽了。<br/>　　<br/>　　    当一碗孟婆汤喝下，王惊蛰忽然之间有了一种很奇妙的感觉，就好像他的人生在以一种倒放的形式，缓缓的倒退着。<br/>　　<br/>　　    王惊蛰先是想起了鬼门之前，他送茅小草和陈三岁离去的那一瞬间，然后想到了在良水城中的时候，他和小草姑娘并肩散步的一幕。<br/>　　<br/>　　    在长安城，在陇西，在渭河上。<br/>　　<br/>　　    “少年听雨泛舟水上，红烛昏罗帐，壮年听雨客舟中江阔云低，断雁叫西风”<br/>　　<br/>　　    “而今听雨渭河下，鬓已星星，悲欢离合总无情，一任阶前，点滴到天明”<br/>　　<br/>　　    “叫我小草就行了······”<br/>　　<br/>　　    “我叫王惊蛰，我生于惊蛰那天”<br/>　　<br/>　　    王惊蛰脑海里的倒放越来越快，一幕幕曾经过往的画面，快速的翻动着。<br/>　　<br/>　　    直到，王惊蛰呼吸一窒，两眼悄然闭上，然后缓缓的一头栽倒在了地上。<br/>　　<br/>　　    鬼王松了口气，说道：“孟婆汤看来是奏效了，拘拿出他的三魂七魄，然后送往十殿”<br/>　　<br/>　　    “谁让你们动我弟弟的了······”忽然之间，自奈何桥下，一声幽幽清脆的嗓音传了过来。
