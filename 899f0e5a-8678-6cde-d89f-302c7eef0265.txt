　　住所外，虽然被封锁，不让外人开进，但依旧有大量的学员汇聚，没有散去。
<br/>　　大家都想看看气息如此恐怖的蛇妖到底是什么样子。

<br/>　　一个个竖起耳朵听住所内的动静，渐渐的，里面安静如斯起来。
<br/>　　副院长不愧是古学院一等一的强者，一出手就定乾坤，蛇妖没了动静，气息都散去了。
<br/>　　想必很快，大家就可以看到蛇妖的尸体了。
<br/>　　就在此时...
<br/>　　轰...
<br/>　　住所二楼处，一面墙壁轰然塌陷，只见一条庞然大物措不及防的砸了起来，众人大吃一惊，纷纷的闪躲，地面跟地震来了一样，震动了起来。
<br/>　　半响后，一个个学员撑着惊骇的目光看向地面一处，只见一条足足有近二十米的巨蟒遍体鳞伤的倒在血泊中，它通体麻花色，有云状斑纹，背面有一条黄褐斑，外貌凶煞可怖，一块块蛇鳞片触目惊心，看上去比铁皮还要坚硬。
<br/>　　猛然的，所有人倒吸了一口凉气，把嘴张得像箱子口那么大，一下子就愣住了，好大的一条巨蟒，简直闻所未闻。
<br/>　　纵然修武者的眼界，远非常人可比，但这般大的巨蟒还真的吓到了他们。
<br/>　　“这就是蛇妖吗，天哪，太吓人了”
<br/>　　“这蛇妖要是吃人，我等也就一口吞罢了”
<br/>　　“我很好奇，这蛇妖到底是怎么混进来的”
<br/>　　学员们窃窃私语，很多人看到巨蟒，脸色悄然的转了白，他们都是温室里的花朵，何成见过这等巨蟒。
<br/>　　大家都在不断的后退，这巨蟒虽然倒在血泊中，但无人知道它是死是活。
<br/>　　哪怕是装死，也是极有可能的，都是开了灵智的生物，非常狡猾。
<br/>　　住所二楼中，一位清秀的男子骤然掠下，直接落到了庞然巨蟒的身边，威风凛凛的一脚踩到了巨蟒的蛇颅上。
<br/>　　这一幕，让学员们惊为天人。
<br/>　　“蛇妖已除，大家莫要害怕”他朗朗高声，目光环视周围环境一眼。
<br/>　　“最强新生？”
<br/>　　有了早上的那一幕，大家都知道，这住所是最强新生的房子，如今他的住所有蛇妖霍乱，而他却一点事情都没有。
<br/>　　看这样子跟姿态，莫非蛇妖是他所杀不成？
<br/>　　那副院长？？？
<br/>　　住所一楼，副院长慢条斯理的走了出来，看了一眼装逼的涂小安，表情有点不好看，调整了下心态，对着外围的众学员道：“这蛇妖偷偷潜入古学院，如今已经被涂小安学员击杀，老夫很欣慰，真是后生可畏呀”
<br/>　　“什么..”
<br/>　　众人诧异：蛇妖还真的是最强新生所杀，敢情副院长是上去打酱油的？
<br/>　　“院长，这蛇妖应该是三阶妖灵吧”
<br/>　　那恐怖的蛇类气息弥漫古学院，让很多人都诚惶诚恐，可要这是三阶妖灵，最强新生也能杀？
<br/>　　还是说副院长故意将这个功劳让给最强新生。

<br/>　　如果这蛇妖并不是三阶妖灵，不过二阶精怪级别，那么它就不可能化形，如果不化形，那么庞然大物如何能混入古学院。
<br/>　　他们都是睁眼瞎？
<br/>　　这可不是一条小蛇呀，是将近二十米的庞然大物。
<br/>　　所以说，这定然是一条三阶妖灵，化形而潜入古学院。
<br/>　　要如此，问题又来了，最强新生的实力无容置疑，可说能杀死一条三阶妖灵，还是不能让人信服，你都那么牛逼了，你还来古学院做什么。

<br/>　　很多人针对这几个问题回来一思忖，最后总结出的结论就是，这最强新生跟副院长关系匪浅，之前副院长的磅礴气息都散发出来了。
<br/>　　不可能没出手，所以肯定是副院长击杀蛇妖，然后将功劳让给了最强新生。
<br/>　　莫非，最强新生是副院长的孙子？
<br/>　　或许外孙，又或者偷偷摸摸生出来的儿子？
<br/>　　最强新生，不，最强关系户...

<br/>　　副院长一脸的沉痛，戏份很足。
