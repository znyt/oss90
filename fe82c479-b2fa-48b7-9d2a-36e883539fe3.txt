　　　　出了萧家，萧逸到城外随意寻了个隐蔽之地。
<br/>　　开辟了个山洞，走了进去。
<br/>　　现在的萧家，一片狼藉，处处是断壁残垣。
<br/>　　连闭关室，或者是完整的房间，都没有。
<br/>　　迫不得已下，萧逸也只能到外面来炼药。
<br/>　　炼药前，萧逸打算先吞服妖兽精血，修炼修罗战体。
<br/>　　这次从北山慕容家得到的妖兽精血。
<br/>　　四级精血，三十万瓶。
<br/>　　五级精血，数万瓶。
<br/>　　待得10万瓶四级精血被完全吸收。
<br/>　　肉体力量，从破玄四重，飙升至破玄六重。
<br/>　　顺带一提，破玄境之后，四级妖兽精血的作用，已经不怎么明显了。
<br/>　　只是数量庞大下，量变才引起了质变。
<br/>　　每个境界的肉体力量，都只有相对应级别的妖兽精血，才能完全发挥出功效。
<br/>　　比如，破玄境肉体力量，就需要五级精血，才能真正有效。
<br/>　　地元境肉体力量，则需要六级精血，或者更高级别的精血，才能有效。
<br/>　　再服下10万瓶四级精血。
<br/>　　肉体力量，达到破玄七重。
<br/>　　如普通修炼的填满气泉不同。
<br/>　　修罗战体，越是修炼到后面，需要的精血力量就越是庞大。
<br/>　　所以，同样是10万瓶四级精血。
<br/>　　能提升得重数，也不一样。
<br/>　　越往后，提升得越慢。
<br/>　　将最后10瓶也吞服下。
<br/>　　肉体力量，勉强达到破玄八重
<br/>　　“四级妖兽精血已吞服完。”
<br/>　　“剩下数万瓶五级精血。”
<br/>　　“看来，修罗战体，可以突破到下一层了。”
<br/>　　萧逸自语着，满脸喜悦之色。
<br/>　　修罗战体，突破到下一层，肉体力量将达到地元境。
<br/>　　没有浪费时间，萧逸立马开始修炼。
<br/>　　数万瓶五级妖兽精血，快速吞服，并消化。
<br/>　　肉体力量，也在快速飙升。
<br/>　　直至达到破玄九重，还在飙升着。
<br/>　　“嗯？”萧逸忽然皱起了眉头，“不对劲。”
<br/>　　体内心脏位置处，出现了一个诡异图案。
<br/>　　图案，在不断吸收妖兽精血的力量。
<br/>　　待得图案完全被填满。
<br/>　　五滴液体状的诡异‘小水滴’出现。
<br/>　　正是修罗之力。
<br/>　　顺带一提，当初击杀幽骨老怪时。
<br/>　　施展崩界拳，将体内那滴修罗之力完全耗尽。
<br/>　　但随着时间，那滴修罗之力早已完全恢复。
<br/>　　现在，算上这新出现的5滴，加起来恰好6滴修罗之力。
<br/>　　“原来如此。”萧逸面露恍然之色。
<br/>　　本来这些妖兽精血，是足够自己突破到地元境肉体力量的。
<br/>　　但，现在却完全转化为了修罗之力。
<br/>　　“6滴修罗之力，如果我没有猜错的话，想要突破到地元境肉体力量。”
<br/>　　“起码要凝聚够10滴。”
<br/>　　萧逸暗暗自语。
<br/>　　他对自己修罗战体的修炼情况有着明确的感知。
<br/>　　虽说，没能突破。
<br/>　　但，修罗之力，是施展修罗变以及崩界拳必须用到的力量。
<br/>　　以自己如今破玄九重的肉体力量。
<br/>　　加上这6滴修罗之力。
<br/>　　实力，绝对又是一次飞跃。
<br/>　　这已经是北山慕容家几乎全部的妖兽精血了。
<br/>　　却还是无法突破到下一层修罗战体。
<br/>　　可想而知，修罗战体何等的难以修炼。
<br/>　　也难怪，当年以狂血玄君的本事，修炼了大半辈子，仍旧没有突破。
<br/>　　仍旧只是破玄九重肉体力量。
<br/>　　想必，他也是卡在了凝聚修罗之力上。
<br/>　　萧逸摇摇头，没有再多想。
<br/>　　摒去心中杂念，开始了炼制无杂质丹药。
<br/>　　天材地宝的数量，不用担心，绝对足够。
<br/>　　而妖兽内丹。
<br/>　　四品的，三十万颗。
<br/>　　五品的，数万颗。
<br/>　　萧逸甚至发现，连六级妖兽内丹，都有十数颗。
<br/>　　六级妖兽内丹，已经非常珍贵了。
<br/>　　六级妖兽，可没那么好杀。
<br/>　　想必，慕容家本身除了有六级内丹外，还有六级精血的。
<br/>　　只不过，精血都被慕容家强者用于浸泡肉身去了。
<br/>　　妖兽内丹，人类武者无法服用。
<br/>　　妖兽精血，人类武者却是可以用于浸泡肉身，提升一定实力的。
<br/>　　……
<br/>　　言归正传。
<br/>　　萧逸拿出八龙焚火炉，正式开始了炼药。
<br/>　　嘭的一声。
<br/>　　紫炎从手中冒出，而后打入火炉。
<br/>　　八龙焚火炉，炼药时，炉内之火，如火龙吐息，药材瞬间炼化，威力惊人。
<br/>　　果然。
<br/>　　萧逸将一份份天材地宝快速投入炉中。
<br/>　　不多时，已经可以开始凝丹。
<br/>　　再加入妖兽内丹。
<br/>　　不过数秒，无杂质丹药已经开始成型。
<br/>　　有了八龙焚火炉，自己的炼药速度，大大增加。
<br/>　　半天后，三十万颗四级内丹被消耗殆尽。
<br/>　　15万粒四品无杂质洞玄丹，被炼制而出。
<br/>　　又是半天。
<br/>　　数万颗五级内丹被消耗殆尽。
<br/>　　一万五千粒五品无杂质破玄丹，被炼制而出。
<br/>　　看着面前几乎堆积如山的丹药，萧逸砸吧了下嘴巴。
<br/>　　“这一次，不会有意外了吧。”
<br/>　　“如无意外，地元境，必能突破。”
<br/>　　萧逸自信地说说罢，丹药，如同糖豆般，大口大口吞入。
<br/>　　冰鸾剑武魂、控火兽武魂，两大武魂齐开。
<br/>　　丹药几乎是刚刚入腹，便已经化为精纯力量。
<br/>　　而后进入小世界。
<br/>　　小世界，虽然在体内。
<br/>　　但其实，他是一个自己开辟出来的，受自己掌控的空间，庞大无比。
<br/>　　按照萧逸的猜测。
<br/>　　当初，450粒无杂质破玄丹。
<br/>　　已能让自己的气泉填满两成。
<br/>　　修为突破两重。
<br/>　　现在先吞服洞玄丹。
<br/>　　洞玄丹虽比不上破玄丹，但数量庞大下，量变引起质变。
<br/>　　大概5万粒洞玄丹，就该可以填满气泉了。
<br/>　　孰料，吞服完后，修为才突破了两重。
<br/>　　从破玄五重，突破到破玄七重。
<br/>　　“不对劲，出问题了。”萧逸脸色一变。
<br/>　　他明显感觉到，自己填满气泉所需的力量，变得更多了。
<br/>　　而且不止是多一点，而是多大幅度变多。
<br/>　　不信邪似的将剩下10万粒洞玄丹吞服。
<br/>　　修为，又突破了两重，达到破玄九重。
<br/>　　“怎么可能？”萧逸满脸不可置信之色。
<br/>　　填满气泉所需的力量，一向是相等的。
<br/>　　怎么忽然开始忽然剧增了。
<br/>　　他怎么想都想不明白。
<br/>　　但，直到他将一万五千粒五品无杂质破玄丹吞入腹中后。
<br/>　　他便开始明白了。
<br/>　　脸色，也陡然间便得无比惊骇。
<br/>　　体内气泉真液，早已完全填满。
<br/>　　可是，此时的气泉，却忽然间剧烈翻滚起来，仿佛成了一个闹腾的大海。
<br/>　　而随着丹药入腹，再次化作力量。
<br/>　　这些力量，竟悉数被气泉吞噬。
<br/>　　不错，并非是吸收，根本就是吞噬。
<br/>　　气泉，仿佛成了一个无底洞。
<br/>　　待得所有丹药都吞服完毕。
<br/>　　如此大量的无杂质破玄丹，完全化为力量。
<br/>　　气泉，陡然间变了。
<br/>　　一股极度深寒的气息，不断萦绕在气泉上方。
<br/>　　不多时，化作了一座巨大冰山。
<br/>　　冰山的大小，恰好占据了一千六百多丈的空间。
<br/>　　同时，一股炙热可怕的气息，也在萦绕。
<br/>　　不多时，冰山之外的一千六百多丈空间，化作了一片熔浆火海。
<br/>　　三千三百三十三丈气泉，完全被分割。
<br/>　　一半冰山，一半火海。
<br/>　　气泉的最上方，是冰鸾剑以及控火兽武魂。
<br/>　　冰鸾剑之下，便是冰山。
<br/>　　控火兽之下，便是火海。
<br/>　　“冰山火海？”萧逸眉头紧皱，“我的气泉，怎会出现这样的异变？”
