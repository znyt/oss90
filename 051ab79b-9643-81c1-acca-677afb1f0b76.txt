<br/>　　闻言，涂小安蹙眉起来，看来自己是高估了这条魔鬼蛇的价值，居然还不能让自己到提升一级的地步。

<br/>　　宿主：涂小安。
<br/>　　身份：觉醒蛇种（十八米）
<br/>　　等级：十九级。
<br/>　　进化能量值：500000/1000000。
<br/>　　天赋：蛇族统治能力。

<br/>　　我去...
<br/>　　看了一眼最新的数据，涂小安一头黑线，原来自己要步入二十级，成为二阶的蛇精，既然需要一百万的进化能量。
<br/>　　涂小安诧异了，要不要那么夸张呀。
<br/>　　敢情自己吞噬了这条蛇精，才完成了二分之一的能量值。
<br/>　　看来二阶是一个大瓶颈，并不是那么好突破的。
<br/>　　涂小安的实力早就超越了等级，所以他现在每提升一级，都不是那么容易，更别说现在他是要步入蛇精层次。
<br/>　　不行，我必须要达到二阶蛇精层次，方可进入蛇沟深处。
<br/>　　他是要去对付吸血蛇，如今的吸血蛇不知道进化到何种地步，况且它已经在蛇沟深处了，说不定已经有了什么机遇。
<br/>　　涂小安要有万分的把握进去才行。
<br/>　　此时，吞噬完魔鬼蛇的涂小安高高的悬浮起来，神态冷淡地环视一圈，几十万毒蛇密密麻麻，匍匐于地，原来充满低沉嘶吼声的现场，因为这双蛇眸的横扫，顿时间变的鸦雀无声。
<br/>　　这几十万毒蛇吓的连一丝声音都不敢发出来。
<br/>　　三条敢挑战金银蛇的强大毒蛇，已经付出了惨痛的代价。
<br/>　　更别说它们，它们连对方的气息都承受不住。
<br/>　　“在找几条觉醒蛇种吞噬，必须要在进入蛇沟之前步入二阶蛇精”
<br/>　　涂小安打定注意，蛇眸锁定在一条躲在蛇群中的大蛇身上，眼睛浑然一眯，这条大蛇腾空而起，不由控制的飞到了金银蛇的面前。
<br/>　　“叮...十五级觉醒蛇种过山风，吞噬可获得100000进化能量值”
<br/>　　“算你命不好，化为我的能量值吧”
<br/>　　这条过山风在空中被金银蛇给吞噬了，它从始至终都不得反抗，涂小安强大的意识力让它犹如被定身。
<br/>　　进化能量值：600000/1000000
<br/>　　再来几条。
<br/>　　“叮...17级觉醒蛇种白腹蛇，吞噬可获得150000进化能量值”
<br/>　　整个开阔地蛇蛇自危，几十万毒蛇陷入了一阵死亡恐怖中，那条悬浮的金银蛇在它们的眼中赫然成了恶魔。
<br/>　　哪条蛇被他看上，下一步，就是被吃。
<br/>　　尤其是那些觉醒蛇种，更是得到末日审判的号角，等待死亡，反而是低等级的毒蛇变的无比安全。
<br/>　　当涂小安再次连续吞噬了五条觉醒蛇种后...

<br/>　　白光冲天而起，将金银蛇笼罩其中，半晌后，一股惊天之力慢慢的散发出来，搅动风云，在白光的笼罩下，一条二十米长的巨蟒露出峥嵘的一角，通体金光灿灿，耀眼无比，他的气息愈演愈烈，天地都快要为之变色。
<br/>　　开阔地几十万毒蛇发出惶恐不安的嘶吼声，大山之中，无数群鸟腾飞，各种动物不安的鸣叫起来，紧接着一只只一头头匍匐与地，诚惶诚恐。
<br/>　　二阶生物，蛇精层次，涂小安突破了。
<br/>　　二阶已成，三阶生灵还会远吗。
<br/>　　异象消失，变回一米的金银蛇。
<br/>　　涂小安目光深远的看着地面那条没有尽头的长沟，是时候顺着长沟去蛇沟深处了。
<br/>　　准备继续赶路，涂小安再次看了蛇群一眼，一条十米长的黑色大蟒诚惶诚恐的骤射而来，匍匐在金银蛇的面前，乖巧如小猫咪。
<br/>　　涂小安微微一笑，坐骑不能忘呀，这条黑曼巴还要继续完成它的使命。
<br/>　　重新落到黑曼巴的头顶，它的身子游曳了起来，几个呼吸便是一公里外，比千里马还要快。
<br/>　　身后几十万毒蛇大军也动了，浩浩荡荡，跟着它们的王。
<br/>　　如今的涂小安可谓意气奋发，一越成为二阶蛇精，身后几十万大军跟着，简直就跟御驾亲征似的。
<br/>　　这种阵仗，攻击一个人类城市都不费什么力气。
<br/>　　............
<br/>　　蛇沟深处，那是一片烟雾茫茫的沼泽地带，淤黑的沼泽地宛如一个会吃人的无底洞，透着死亡气息。
<br/>　　烟雾茫茫的沼泽地里面，络绎不绝的有一股异香弥漫出来，仿佛像少女的轻歌曼舞，楚楚动人，让人不由自主的想接近，想拥抱。
<br/>　　这种异香如毒罂粟般热烈，狂放，诱惑，只见一条条大小不一的毒蛇不要命似的朝着淤黑的沼泽地游曳而去。
<br/>　　可当它们疯狂游曳的时候，它们的蛇躯一点点的陷入了沼泽地中，一道道凄惨的嘶吼声传出，片刻，不复存在。

<br/>　　它像是恐怖的深渊，一入其中，性命无存。
<br/>　　但也不是所有的毒蛇都会被沼泽地吞噬，每每到了日落时分，这片沼泽地像是吃饱了，泥潭般的沼泽变的如履平地，不在吃人。

<br/>　　犹如风云之后就会见彩虹，没有人会知道透着死亡气息的沼泽地身后，是这样一片优美的环境。

<br/>　　蛇峡谷，蛇类真正的大本营。

<br/>　　一句话说出，他的身上激荡一股摄人的气息，气浪席卷。
<br/>　　人类觉醒者。
