　　归墟天，到底是有着多么庞大，现在的楚风眠都无法知晓。
<br/>　　尤其是在这归墟天之中，周围空间的法则，坚固无比，就算是以现在楚风眠的实力，都无法去穿越空间，只能够从其中飞行。
<br/>　　因此，这也就是将归墟天的范围，仿佛便是放大了无数倍一样。
<br/>　　一路上，楚风眠也是路过了不少的星球，漂浮的大陆。
<br/>　　在归墟天之中，几乎每一处地方，都是蕴含有着无数宝藏，楚风眠一路上，将这些星球，大陆上的宝藏，全部都是收敛一空，全部收取到了空戒之中。
<br/>　　不过宝藏虽多，但是危险也是并存的，楚风眠在收取其中一处星球上的宝物的时候。
<br/>　　突然就在这地底之中，一尊巨大无比的玄金蟒，便是突然冲了出来。
<br/>　　这一尊玄金蟒的实力，只怕是寻常生死境巅峰武者，四五个联起手里来，都不是他的对手。
<br/>　　在这归墟天中的武者，只怕是大部分人，碰到这玄金蟒，都是死路一条。
<br/>　　就算是曾经还没有度过生死大劫的楚风眠，要是真正对付这玄金蟒，都是极为困难的一件事。
<br/>　　只是对现在的楚风眠来说，这样一头玄金蟒，对他都甚至是算不上威胁，几招之内，便是将他斩杀。
<br/>　　这一头玄金蟒身上蕴含的玄金，都是被楚风眠收取走，融入到了黑玄剑阵之中去。
<br/>　　归墟天中，宝藏众多，但是这些宝藏，也蕴含有着无数危险，没有实力的武者，就算是进入到这归墟天中，都是无法收取任何宝藏，甚至是自己姓名都很有可能会搭进去。
<br/>　　九域规则，也是如此，实力为尊，实力才是最为重要的。
<br/>　　一路上碰到的所有宝藏，如今都被楚风眠收取，很快，楚风眠的遁光便是离开了外围星域，直接便是来到了归墟大陆前。
<br/>　　归墟大陆的外围，是一处云层，令人远远一看，根本是无法看清这归墟大陆的全貌。
<br/>　　这云层之下，就是归墟天的中心，归墟大陆。
<br/>　　楚风眠来到云层边缘，便是毫不犹豫，一道遁光，直接便是穿越其中，一股灵力，突然围绕过来，让楚风眠化为了一颗巨大无比的火球，俯冲而下。
<br/>　　最后这一颗火球，便是种种的落入到了归墟大陆的一处山脉之中。
<br/>　　“轰隆！”
<br/>　　周围的土地都在震动，似乎是因为楚风眠的撞击，山脉都是在颤动。
<br/>　　突然之间，一道巨大无比的身影出现，这一道身影，遮天蔽日，一对双翼，足足是有着百米之宽。
<br/>　　“鲲鹏？”
<br/>　　这是一只巨鸟，出现一刻，遮天辟日，令楚风眠一下子想起了远古异兽之中的鲲鹏。
<br/>　　不过鲲鹏，却是远远要比起这还要巨大无数倍，一双翼展，足有千里，眼前这巨鸟，跟真正的鲲鹏相比，简直就是一只小蝼蚁一样。
<br/>　　但是正常来看，这巨鸟，也是一只真正的庞然大物。
<br/>　　“嘶！”
<br/>　　这巨鸟察觉到楚风眠，突然尖啸一声，直接张开大嘴，要将楚风眠吞噬。
<br/>　　眼前这巨鸟的境界，已经是达到了生死境巅峰的地步，在他看来，楚风眠便是他的猎物，直接是要将楚风眠吞噬。
<br/>　　“找死？”
<br/>　　楚风眠瞥了一眼这巨鸟，手一挥，猛然一掌打了出去。
<br/>　　这巨鸟，直接被打飞出去数百米之远，但是却并没有被打伤，再次尖啸一声，冲着楚风眠飞了过来。
<br/>　　“原来是一只有着几分鲲鹏血脉的鹏鸟，虽然血脉稀薄，但是毕竟是拥有这远古异兽的血脉，的确不是寻常妖兽，可以比拟的。”
<br/>　　楚风眠刚刚的那一掌，虽然不能算是动用全力，但是一般生死境巅峰的妖兽，肯定是足以轰杀的。
<br/>　　但是眼前这巨鸟，只是受到了一些伤，这便是归功于他的血脉之力。
<br/>　　“给我死！”
<br/>　　楚风眠右手握拳，直接再度一拳轰击出去，砸在这巨鸟的背后，他的身上，五脏六腑都被楚风眠直接砸碎，从空中直接便是掉落了下去。
<br/>　　“这鹏鸟的心脏，倒是炼制灵器的好材料。”
<br/>　　楚风眠向着那鹏鸟，直接走过去的一刻，突然之间，就在楚风眠的身边，一股灵力，突然涌现出来。
<br/>　　这一股灵力，将周围全部封锁住，是让楚风眠，根本便是无法离开。
<br/>　　“什么人？敢靠近在一片山脉，这一片山脉，是属于我们炎阳门的了，谁敢靠近，格杀无论！”
<br/>　　只见三名武者，就在一旁，突然走了出来，这三人的身上，都是一身红袍，身上涌现这一股股炙热的气浪，席卷着周围。
<br/>　　“炎阳门的人。”
<br/>　　楚风眠心中默念一声。
<br/>　　炎阳门，便是在这万年间，先崛起的势力之一，跟北邙学院类似，都楚风眠曾经那个时代，没有出现过的门派。
<br/>　　九域之中的炎域，炎阳门便是其中第一大的宗门，炎域的整体实力，跟北域还要强大。
<br/>　　在九域之中，炎阳门，几乎跟玄天宗一样，算得上是七大宗门下，最为强大的势力之一了。
<br/>　　眼前这三名炎阳门弟子，都是有着生死境四重的境界，每个人的实力，虽然不如那公孙道，但是比起其他古月峡谷弟子，还是要强大几分。
<br/>　　这样的势力中，也许无法培养出像是公孙道一般的绝顶天才来，但是弟子的实力，也丝毫不弱。
<br/>　　就在楚风眠打量那几名炎阳门弟子，思考起来的一刻，这些炎阳门弟子，也都是看向了楚风眠，灵识议论了起来。
<br/>　　“看着小子的样子，不像是七大宗门的人。”
<br/>　　“宗门弟子，进入到这归墟天中，应该都是联合了起来。”
<br/>　　“此人居然落单了。”
<br/>　　“看他的样子，应该是从外面搜刮了不少好东西，才来到这归墟大陆的，看来就是他刚刚落到了归墟大陆之中。”
<br/>　　这些炎阳门弟子，看到楚风眠孤身一人，一个个的眼神，都是变化这，露出了几分贪婪。
<br/>　　在归墟天中，杀人易货，才是最快的方法。
<br/>　　这种独自一个人的，简直就是一个移动的宝藏，可以随时摘取。
<br/>　　“此人的实力，才生死境三重，看起来也没有什么了不起的势力，要不要杀了他。”
