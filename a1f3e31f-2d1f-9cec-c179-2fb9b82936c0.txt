夏振国的神色阴暗不明，最后转身离开，灯光将他的背影拉出几分怀念和感伤。
<br/>
<br/>        夏小蝶进了房间，趴在李玉兰床头大哭，“妈，爸是不是要跟你离婚，爸怎么可以这么无情呢，我们该怎么办？”
<br/>
<br/>        李玉兰将那份离婚协议书揉成团用力的丢掉，她不会离婚的，她不会离开夏家的，她不会像一个无用的垃圾般被人丢弃的。
<br/>
<br/>        这时有脚步声响起，夏小蝶喜出望外，“爸，你是不是改变主意了，你…”
<br/>
<br/>        夏小蝶一滞，因为进来的不是夏振国而是夏夕绾。
<br/>
<br/>        李玉兰现在恨毒的目光恨不得将夏夕绾给戳出几个血窟窿来，“你来干什么？”
<br/>
<br/>        夏夕绾坐在了椅子上，玉白的小手给自己倒了一杯茶，她勾唇，“当然是来看你笑话的，要不然呢？”
<br/>
<br/>        “小野种，我要杀了你！”
<br/>
<br/>        李玉兰拽着床单想下去，但是她身上遍体鳞伤，这么剧烈的一动只让她痛的双眼发黑。
<br/>
<br/>        狼狈至极。
<br/>
<br/>        夏夕绾看着李玉兰，“你说只要你一天还有价值，就不会倒下，可是现在你在娱乐圈苦心孤诣的口碑和人脉已经轰然崩塌，全网都在黑你，夏氏医疗这里，你已经成为了夏家的耻辱，令我父亲蒙羞，我将你从有价值变成毫无利用价值，果然，我父亲很快就甩了你一脸的离婚证书。”
<br/>
<br/>        “李玉兰，从乡下回来的每一天我都在想，早晚有一天我也会让你们尝一尝我十年前的命运，痛失所爱，被全世界背弃！”
<br/>
<br/>        李玉兰恨得咬牙，她真是失策了，她真的小瞧了夏夕绾，她回来并没有多少时日，但一双素手拨弄风云，已经将夏家绞的天翻地覆。
<br/>
<br/>        “夏夕绾，早知如此，我就应该在十年前杀了你的，我不该让你活在这个世界上的！”李玉兰咆哮道。
<br/>
<br/>        夏夕绾气定神闲的喝完了一杯茶，“你是应该后悔当日对我的不杀之恩，因为你慢慢的才会知道，今天仅仅只是开始。”
<br/>
<br/>        说完，夏夕绾放下茶杯，转身离开，“一个废物，不值得我再浪费时间，也许苟延残喘，摇尾乞怜还能够令你活下去。”
<br/>
<br/>        “你！”
<br/>
<br/>        李玉兰觉得喉头腥甜，她一口血要喷出来了。
<br/>
<br/>        不过她将那口血深深的压了回去，对着夏夕绾纤柔的身影大笑道，“夏夕绾，你以为你父亲真的会跟我离婚，告诉你一个消息，妍妍回来了！”
<br/>
<br/>        夏妍妍！
<br/>
<br/>        提到夏妍妍，这个夏家的骄傲，夏振国最喜欢的女儿，海城第一名媛，就连夏小蝶都一下子起死回生了，“夏夕绾，我姐姐要回来了，十年前你就不是她的对手，十年后依然不是！”
<br/>
<br/>        夏夕绾轻轻的敛了一下纤长的羽捷，再抬眼时，她澄亮的翦瞳里绽放了清冽的碎光，顾盼流转之间，光芒万丈。
<br/>
<br/>        “那真是太巧了，我等这一天已久，你们是我送她的见面礼，见到她替我问一声，这份见面礼她可还满意？”
