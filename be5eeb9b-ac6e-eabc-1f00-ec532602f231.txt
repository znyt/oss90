　　第629章
<br/>　　徐神机一席话，却是坚定了程大雷的心意。和珅的话也未必完全不可听，无论是单纯想要在这个乱世活下去，还是与诸侯争一争天下，都需要足够的实力。
<br/>　　而目前蛤蟆城已到顶点，没有更多发展空间，需要另开基地，支撑自己后续发展。
<br/>　　朔方城现在的情况的确很糟糕，但好好整治一番，未必不能令它重新繁华起来。
<br/>　　在程大雷坚定发展朔方城的想法，而不是抛弃它后，脑海中有一座城池的图像被点亮，程大雷现在已可以查看朔方城的信息。
<br/>　　城池：朔方城（破败的初级城池）
<br/>　　人口：5000
<br/>　　技能：诸神的祝福，建筑精通，江湖传言
<br/>　　程大雷逐字阅读着系统信息，发现在开启攻城模式后，各城池已共享同一套技能。也就是说一座城池开出的技能，所有地方都可以通用，而不像以前的山寨，每座山寨都有单独技能。
<br/>　　嘟，朔方城不足一万，请尽快扩充人口到一万，完成任务后可获得系统奖励。
<br/>　　这时候程大雷收到一条系统任务，要求他尽快扩充人口。
<br/>　　因为有建筑精通在，城池修葺并没有太大困难。最大的困难还是人，或者说，是养活这些人口的粮食。
<br/>　　如果真忽悠过来几万流民，就是几万张口朝程大雷要饭吃，程大雷攒下的哪些家底，总不能让他们消耗掉。
<br/>　　这才是程大雷目前急于解决的难题。
<br/>　　到第二天黄昏，和珅已经带着运粮队到了。以蛤蟆城与朔方城的距离，来回大概需要一天一夜。和珅可以来得这么快，大概是刘发财刚将消息递过去，他便准备出发了。
<br/>　　和珅的办事效率是极高的，这一点程大雷也不得不佩服。
<br/>　　程大雷在街道上支开粥锅，先给暂留朔方城的人一碗饱饭吃。这些人已经不知道有多久没有吃过饭了，听到有人舍粥，顿时群起而动，端着锅揣着碗就来了。如果不是有兵丁维持秩序，他们恨不得把粥锅踹了。
<br/>　　程大雷算是第一次见到自己治下的子民，昨天他进城的时候，根本没有人搭理他。
<br/>　　他本想说些大家一起努力，美好日子就在眼前的话，估计也不会有人搭理他。
<br/>　　程大雷直接进入主要程序，兵丁维持秩序舍粥，并给这些人登记造册。宋伯康原先留下的账薄，一点用处也没有，程大雷恨不得一把火烧掉，一切还得程大雷从零做起。
<br/>　　程大雷也大概分辨出这些人的主要组成部分，懒汉居多，还有一些逞强好胜的狠人，剩下的就是些走不动路的老弱妇孺。
<br/>　　见城主府门口的舍粥工作步入正轨后，程大雷、徐神机与和珅三人走进城府，他还有事要与和珅商量。
<br/>　　和珅整个人很兴奋，这从他脸上的表情就可以看得出来。朔方城的困难他未必没看出来，不过能够扩充地盘他就很开心。程大雷终于不在蛤蟆城窝着，迈出了争霸天下的第一步。
<br/>　　“大当家，刚才我已看过，目前朔方城的人口大概有五千余人。对于他们，愿意走的可以让他们离开，愿意留下的我们也不会不管。”
<br/>　　“对于留下的人怎么安排？”
<br/>　　“老弱妇孺最好是将他们迁往蛤蟆城，毕竟名声要紧，不能看着他们死在眼前不管。剩下的男人愿意加入军队的我们收留，不愿意从军的便在城外种田。至于其中一些作奸犯科之徒，也绝对不能纵容，当以雷霆之威，立一立军威。”
<br/>　　和珅侃侃而谈，对这五千人以有了安排。
<br/>　　程大雷轻轻点头：“目前主要困难是没有人，你有没有办法，把朔方城的人气聚拢起来。”
<br/>　　和珅皱眉思索片刻，道：“怕是不太好办。”
<br/>　　就连和珅都说不好办，看来这件事的确不容易。和珅有些话没有直说，其实这件事之所以这么难办，和程大雷还有些关系。
<br/>　　程大雷的名声糟糕透了。他在并州大杀特杀得罪了许多人，换来了残暴好杀的名声。而东海琅琊城的事情，江湖都传言是程大雷放火烧了藏书楼，如此又得罪了天下读书人。
<br/>　　现在朔方城要招贤纳才，可假若将程大雷蛤蟆大王的旗号打出去，估计除了能换来一片吐沫星子外，不会有任何收获。
<br/>　　程大雷手托着下巴，口中叹了一声：“民心是个好东西呐。”
<br/>　　怪不得蛤蟆城的人口没什么扩张，原来是这方面原因。李行哉在徐州为何会很快可以招兵买马，手下聚了一大票人。人家是正经的逍遥王，明帝的儿子，还有传言说，当初明帝本来打算传位于他。无论传言是真是假，总还是有人信的，为他做事大家也不觉得丢人。
<br/>　　而其他诸侯王再如何也是朝廷封的，所以旗号也有用。但程大雷出道是个山贼，大家为了不给祖宗蒙羞，一般不会投靠一个山贼。
<br/>　　程大雷本可以用恐惧值召唤一批人来，不问质量，只问数量。可恐惧值是个好东西，程大雷觉得不该浪费在这件事情上。
<br/>　　“招揽人口的事，慢慢想总能想到办法的。”程大雷道：“我为难的是，假若真有了人，咱们拿什么养。家中有粮，才能心中不慌，蛤蟆城的粮草是不能动的。”
<br/>　　“依我看这件事倒也不太难办。”和珅道：“咱们种不出来粮食，可以动手抢呐。”
<br/>　　程大雷一怔，他倒是忘了自己的老本行。自己出道就是个山贼，没有粮食自然可以从别的地方抢来。
<br/>　　“可咱们有能抢的地方么？”
<br/>　　“凉州城？”
<br/>　　“咱们刚从人家夺了一座城，再抢人家的不合适吧。况且赵子龙还在宋伯康军中，目前不方便。”
<br/>　　“百里胜？”
<br/>　　“百里胜活不了几天了，他的粮草都是由朝廷供应，目前比咱们更穷。”
<br/>　　“如果这两个地方都不行的话，那就只有向关内动手了。”
<br/>　　蛤蟆城位于帝国西北边关，而朔方城则是和荆州益州接壤，益州也比较穷，蛮夷居多。而荆州则是天府之国，如果要动手的话，荆州却是个不错的选择。
<br/>(本章完)
