　　黑暗魔龙气势毁天灭地，魔威滔天，逸散的气息，都让附近的空间和下方的山林不断的毁灭。
<br/>　　而巨大的象腿，则正好相反，所有的力量都凝聚在那一条腿上，并没有强大的气息波动。
<br/>　　很显然，光头大汉这一腿，全部是他肉.体的力量。
<br/>　　阎泰等五人，身形急速后退，他们不敢距离太近，免得被两人的攻击波及。
<br/>　　在五人震惊的目光中，庞大无比的象腿，轰然的和黑暗魔龙撞击在了一起。
<br/>　　轰！
<br/>　　一声惊天动地的巨响，震动了大半个蛮荒山脉，准帝强者互相硬撼，威力可想而知。
<br/>　　黑暗魔龙虽然强大，但是依旧被象腿碾成了渣，从象腿和黑暗魔龙相击之处，空间急速坍陷，并且成扇形向外飞速向外扩散。
<br/>　　所过之处一切尽皆化为了虚无。
<br/>　　而象腿微微一顿之后，继续向二魔老轰杀而去，速度快到极致。
<br/>　　二魔老避无可避，黑色大枪再次刺出，狠狠的和象腿相击在了一切。
<br/>　　轰！
<br/>　　又是一声爆响，劲力炸裂，二魔老顿感一股无上的伟力袭身，立刻全身剧震，犹如一颗炮弹般倒飞了出去，眨眼间消失在了天地的尽头。
<br/>　　“二魔老！”阎泰等人见此心中大惊，毫不犹如，他们立刻转身就走。
<br/>　　连强大的二魔老，都挡住这光头大汉一击，他们哪里还敢多留。
<br/>　　嗖嗖嗖！！
<br/>　　阎泰五人，个个爆发出了最快的速度，向二魔老消失之处飞去。
<br/>　　洞穴入口，光头大汉收回了腿，巨大无比的象腿，也瞬间化为了人腿。
<br/>　　“不堪一击！”
<br/>　　光头大汉摇了摇头，随即眼眸扫视四方，眉头微微一皱。
<br/>　　刚才一击，两人碰撞的余波，已经让得下方的山林都灰飞烟灭了，方圆数千里之内，大地塌陷，变成了一处辽阔的盆地。
<br/>　　光头大汉叹了口气，好在圣山有大哥布置的阵法，不会遭到丝毫损坏。
<br/>　　少倾，光头大汉转身，大步走进了洞穴之中，向圣山内部走去。
<br/>　　他并没有去追击二魔老，给对方一脚已经足够了。
<br/>　　若是对方想自寻死路，他随时可以出手。
<br/>　　十数万里之外。
<br/>　　二魔老身形伫立在虚空中，目光遥望着蛮荒圣山的方向。
<br/>　　此刻的二魔老，身上缠.绕的黑雾都被震散了，露出了其面容。
<br/>　　其脸盘宽大，面容苍老无比。
<br/>　　这些并不是最主要的，最主要的是，其额头之上，印刻这一个古怪的标志，这个标志仿佛是一个三叉戟一般。
<br/>　　此时，二魔老的嘴角，还有一丝猩红的血迹残留，显然刚才一击，他受了些内伤。
<br/>　　被光头大汉一脚踹出十几万里，二魔老既愤怒又无奈！
<br/>　　嗖嗖嗖！！
<br/>　　破空声响起，阎泰和四位殿主飞身而来，落在了二魔老的身边。
<br/>　　“二魔老，我们怎么办？”阎泰忐忑的问道，他也看出二魔老受伤了。
<br/>　　二魔老沉默不语，良久之后，叹了口气，他道：“回去吧！有这两位蛮荒大妖相助，就算宗主和大魔老来了，也是无计可施！”
<br/>　　“有这么强大？”阎泰五人闻言一惊，面带震撼之色。
<br/>　　不过，几人并不是太相信，只认为这是二长老故意夸大，好挽回战败的颜面。
<br/>　　他们都知道宗主和大魔老的实力，两人若是来了，都无计可施，那还得了！
<br/>　　二魔老瞥了阎泰五人一眼，见得五人的神情，他自然明白五人心中所想。
<br/>　　“你们有所不知，这蛮荒山脉的三只蛮荒大妖，都存活了近万年，要么是神兽后裔，要么是远古异种，在现如今的苍穹世界，基本上就是同阶无敌的存在。”
<br/>　　二魔老眼眸微眯，继续道：“刚才出手之人，名为象山，其本体乃是远古战象，拥有亿万生灵中最霸道的肉身力量，镇压天地万物，我败的不冤！”
<br/>　　“先前的那名黑衣中年人，如果我猜的不错，应该是翼瞳，本体为八翼飞龙蛇；至于最后一位，不知道叫什么，也不清楚其本体是什么妖兽，因为从来没有人见过！”
<br/>　　言罢，二魔老摇了摇头，转身离去。
<br/>　　既然无法再击杀苏莫，只能先行回去，再想其他办法了！
<br/>　　阎泰五人听的暗暗咋舌，他们对于蛮荒山脉中的三只蛮荒大妖，了解的倒是不多。
<br/>　　因为这三只大妖，基本上从来不离开蛮荒山脉，与苍穹世界一向井水不犯河水。
<br/>　　时间久了，很多人甚至都忘记了这三只大妖的存在。
<br/>　　今日一见，五人心中暗凛，这三只大妖果然恐怖，辛亏三妖不离开蛮荒山脉，不然的话，整个苍穹世界，必然会发生惊天巨变。
<br/>　　而后，五人便均是跟随二魔老，返回黄泉魔宗。
<br/>　　……
<br/>　　圣山内部，一片广阔的空间中，三座古朴的宫殿静静的坐落于此。
<br/>　　不过，与苏莫上次来时，有所不同的是，三座宫殿的旁边，又新建了一座略小的宫殿。

<br/>　　况且，他现在已经不是小八的主人了。
<br/>　　“那叫什么呢？”小八睁着晶亮的眸子，好奇的看着苏莫。
<br/>　　“就叫名字吧！”苏莫道。
<br/>　　小八闻言，瘪着嘴、撇着眉，思索了一番之后，笑道：“那我就叫你苏莫大哥吧！和青璇姐姐一样！”
<br/>　　小八认识宏青璇，知道宏青璇便是这么称呼苏莫。
<br/>　　“好！”苏莫笑了笑。
<br/>　　“嘻嘻！苏莫大哥，我带你参观我的行宫！”小八嘻嘻一笑，随即便小跑着进入了宫殿之中。
<br/>　　苏莫无奈的笑了笑，他发现现在的小八，不仅模样像个孩子，心智也差不多十个孩子。
<br/>　　也是，小八从出生到现在，也不过才五年左右的时间。
<br/>　　这对于拥有漫长寿命的妖兽来说，也只是幼年时期而已。
<br/>　　不过，小八如今的实力可是不低，据苏莫观察，小八分明是已经达到了六级八重妖兽的地步了。
<br/>　　这可是相当于武王境八重武者，这个提升速度，与他相比都是有过之而无不及。
