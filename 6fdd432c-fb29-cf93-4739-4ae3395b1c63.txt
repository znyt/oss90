　　空戒打开的一刻。
<br/>　　顿时整个房间之中，都是灵力暴起，一阵阵的血气，化为血风，席卷整个空间。
<br/>　　“这其中这尊血神的力量，只怕都不弱于那夺花尊者了。”
<br/>　　寒月漓感觉到这血神的力量，淡然开口道。
<br/>　　“不错，看起来这精血生前的主人，实力至少要达到生死境三重左右，死后留下的精血，才会有着如此力量。”
<br/>　　楚风眠笑着说道。
<br/>　　这无上血神的力量越强，也就代表这其中精血的力量越强。
<br/>　　他要将杀戮剑术，凝练到极致大成，这精血的力量，自然是越强大越好。
<br/>　　“一个御风境的蝼蚁，一个神力境九重的废物，也想要炼化了本尊？可笑！”
<br/>　　那无上血神开口，居然是可以吐人言，一团死去生死境强者的精血，化为的无上血神，居然都有着自己的意识。
<br/>　　这无上血神开口，打量起来了楚风眠一眼，眼神中突然露出几分光芒。
<br/>　　“哦？居然身躯中是有着远古战龙的血脉，好身躯！我夺取了你的身躯，便是可以重生！未来再临巅峰也不难！”
<br/>　　“天助我也，居然是这样两个废物，还敢将本尊放出来，那本尊便是就要了你的命！”
<br/>　　无上血神，大笑连连，看到楚风眠，好像是看到了一座宝藏一样。
<br/>　　他身形一动，顿时直接向着楚风眠抓了过去。
<br/>　　要将楚风眠的身躯，硬生生的给夺舍，好让他重生。
<br/>　　这种无上血神，若是让他得到一副身躯，他还可以重生归来的，可以化为真正的人类。
<br/>　　无上血神咆哮着，直接冲着楚风眠扑了过来，要将楚风眠的身躯，直接夺舍。
<br/>　　寒月漓看到这一幕，刚想出手，却是被楚风眠直接阻拦。
<br/>　　只见楚风眠脚下一动，大步上前，直接赢了过去，同时在楚风眠的手心中，猛然一股战龙之威，凝聚出来。
<br/>　　在他的身躯中，一千二百九十六滴远古战龙精血同时凝聚，化为了战龙之爪，猛然落下。
<br/>　　“给我破！”
<br/>　　“咔嚓！”
<br/>　　这无上血神的身躯，碰到这战龙之爪的一瞬间，便是开始破碎，一股股精粹血气，流露出来。
<br/>　　猛烈的撞击，令整个房间中，都是布满了血气。
<br/>　　这无上血神的力量，被楚风眠居然是一招击败。
<br/>　　“什么？你不过一个御风境的蝼蚁，怎么可能抗衡的了我的力量！”
<br/>　　那无上血神，不可置信的咆哮道。
<br/>　　他就算是只是一团精血，但是以他现在的力量，灭杀巅峰圣者，都轻而易举。
<br/>　　可是现在，居然是被楚风眠一招击败。
<br/>　　那一招战龙之爪，轰击下来，铺天盖地的力量，令这一尊无上血神都感觉到了恐惧，无法抵挡得住。
<br/>　　这完全不像是一个御风境武者的实力，就算是巅峰圣者，都不会有着如此力量。
<br/>　　“我就不信，你一个小蝼蚁，还真能翻天不成！”
<br/>　　无上血神长啸一声，一身血气，猛然凝聚，化为一根长矛。
<br/>　　这一根长矛，乃是无比精粹的血气凝聚而成，长矛一出，天地便是被无尽血气充斥。
<br/>　　这一根长矛，猛然向着楚风眠刺了过来，其中蕴含的无尽力量，更是划破空间。
<br/>　　化为了一道血色长虹，一矛而出，似乎是要将天地一切，都给贯穿。
<br/>　　“给我碎！”
<br/>　　楚风眠再次身体之中，凝聚出一千二百九十六滴远古战龙精血的力量，化为战龙之爪。
<br/>　　同时在楚风眠的背后，远古杀神的虚影，也是出现。
<br/>　　这远古杀神的力量，加持在战龙之爪上。
<br/>　　顿时令楚风眠的战龙之爪，又是可怕了无数倍。
<br/>　　“轰！轰！轰！”
<br/>　　一爪落下，那一根血气长矛，近乎是在一瞬间，就被直接折断，彻底被摧毁。
<br/>　　其中的无数力量，也被楚风眠尽数吞噬，融入到了远古杀神的虚影之中去。
<br/>　　“远古杀神虚影！”
<br/>　　看到楚风眠背后的远古杀神虚影，那一尊无上血色脸色骤变。
<br/>　　他猛然一动，直接化为了一道血气，向着门外逃去。
<br/>　　他已经感觉得出，楚风眠的力量已经远远不是他可以对付的了的。
<br/>　　顿时打算要离开。
<br/>　　但是楚风眠，怎么会给这无上血神机会。
<br/>　　他脚下一动，已经是来到了门前，手中赤炎剑突然出现，剑锋一闪，杀戮剑术尽出。
<br/>　　在这无尽杀意之下，这一尊无上血神，被彻底斩杀。
<br/>　　在空中，便是化为了一缸生死境强者的精血，楚风眠大手一挥，尽数吞噬了起来。
<br/>　　源源不断的血气，融入到了楚风眠的身躯之中。
<br/>　　他虽然吞噬过了许许多多的圣者精血，妖神精血，但是跟这生死境强者的精血相比，这还是截然不同的东西。
<br/>　　生死境强者，度过生死大劫，已经跟圣者不是同类了，他的精血，要比起圣者精血强大数百倍。
<br/>　　这一团团的血气，被楚风眠吞噬，全部凝聚在了远古杀神的虚影之中去。
<br/>　　这远古杀神虚影，都开始逐渐蜕变。
<br/>　　“砰！”
<br/>　　这最后的一部分，远古杀神虚影的头颅，也彻底凝聚完成。
<br/>　　这头颅凝聚完成的一刻，这一尊远古杀神虚影，仿佛便是活过来了一样。
<br/>　　如同死而复生一般的神奇。
<br/>　　力量方面，已经是跟刚刚，截然不是一个档次。
<br/>　　“远古杀神之力！加持！”
<br/>　　楚风眠眼神一变，冷喝一声。
<br/>　　这远古杀神虚影，便是彻底的融入到了楚风眠的身躯之中去。
<br/>　　顿时楚风眠的力量，都开始节节攀升了起来。
<br/>　　楚风眠如今的一举一动之间，都是蕴含有着无尽杀意。
<br/>　　仿佛现在的楚风眠，便是一尊真正的无上杀神一般，简直是可怕到了极致。
<br/>　　“你，真的还是楚风眠？”
<br/>　　寒月漓在一旁，都不由的出声询问道。
<br/>　　看现在楚风眠的样子，好像便是被一尊杀神占据了身躯一样的感觉。
<br/>　　不管是气质，还是力量，都是截然不同。
<br/>　　“自然是我。”
<br/>　　楚风眠轻声回答，忽然之间，他的心神一动。
<br/>　　“杀意，压制！”
<br/>　　就在楚风眠身上的无尽杀意，顿时消失的无影无踪，从刚刚一尊无上杀神，又变回了普通的楚风眠。
