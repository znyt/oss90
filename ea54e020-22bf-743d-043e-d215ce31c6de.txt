　　一开始，大伙都还沉浸在幻觉中，都很迷茫，叶少阳告诉他们真相，所有人震惊，七嘴八舌地回忆幻境里的经历，都跟叶少阳一样，经历了各种光怪陆离的画面，不过有一点一样的是，所有人都经历了非常凶险的局面！如果不是叶少阳出手相救，大概所有人都得死在幻境之中。
<br/>　　别人还没什么，法师之间，互相帮助也是理所当然，更不要说还是朋友，只有陈晓宇表情有点怪异，望着叶少阳，有些踌躇。
<br/>　　“你不要谢我，是卢晓清救的你。”叶少阳脱口而出。
<br/>　　“我就知道。”陈晓宇冷哼一声，“你怎么会有这么好心。”
<br/>　　叶少阳苦笑，如果不是自己先救了卢晓清，再让他出手，只怕陈晓宇现在已经挂了，自己虽然没有出手，也算是间接救了他，陈晓宇也不是智障，这一点肯定想得到。
<br/>　　如果他说一些感谢的话，叶少阳虽然不会跟他和解，但心里多少也会对他少一些鄙视，但他偏偏这样一副态度。
<br/>　　叶少阳冲他笑了笑，“你放心，如果有下一次，我保证，我不会救你。”
<br/>　　“你以为我会救你吗？”陈晓宇反唇相讥。
<br/>　　叶少阳没理他。
<br/>　　大伙转过身，朝那株巨花望去，还是之前的模样，并没有伸长到夸张的茎干，只是一个打开的花苞，中间有一层半透明的膜，中间包裹着不知道什么东西，虽然样子诡异，但是一动不动，看上去人畜无害。
<br/>　　“杀人于无形啊，这个牛逼。”叶少阳不由感叹。
<br/>　　“我知道这是什么了。”梧桐一脸的严肃和吃惊，抬头仰望着这株巨花，喃喃道：“这是水晶花。”
<br/>　　“水晶花？”
<br/>　　叶少阳怔了一下，觉得好像在哪里听过这个名字，仔细想了一下，找到了对应，立刻摇头道：“不可能，这不是水晶花。”
<br/>　　水晶花，并不是真正的花，而是一种巨型的真菌，说白了就是一种大蘑菇，不过形状像花，长得浑身透明，看上去像是水晶一样，有药用价值，但是极少，与血林芝差不多，是长在死人身上的东西，但也不同，血林芝是靠着人死后咽的那一口死气而生，水晶花却是靠吸收死人的血肉，一点点生长。
<br/>　　叶少阳在典籍上看到过这种东西，因为是在死人身上长的，所以带有几分邪性，会摄人心神，但也最多就是这样了，无论是形态上还是特性上，跟眼前这株巨花完全都不一样，叶少阳虽然没见过，也在典籍里经常看到，因此提出质疑。
<br/>　　梧桐道：“你只知其一，不知其二。我说的水晶花，并不是这人间的品类，而是它的母种：通灵佛子花，水晶花本是通灵佛子花在人间的一个品种而已。”
<br/>　　“通灵佛子花，这是什么鬼？”叶少阳道。
<br/>　　梧桐不知道“什么鬼”这个词是什么意思，一本正经地回答道：“跟鬼没关系，这种花，在佛经里提到，说是生在婆娑净土的一种花，就长在通往婆娑净土的必经之路上，凡是朝圣者，必须要从花下走，闻一口佛的香味，香味有三毒，会勾起人心中的贪嗔痴三念，进入自己欲望构建的幻境之中，如果道心不够，会沉迷在幻境中不能自拔，到不了灵山，只能回去重修。”
<br/>　　叶少阳听她这么一说，沉吟道：“听这么说，跟眼前我们看到这个还听相似的，不过这花的幻境可不是什么贪嗔痴三毒，那是真的能杀人的幻觉。”
<br/>　　梧桐道：“佛经上是那么说的，可能长在灵山下的那株，是被佛祖渡化过的，故意用来考验人的定力，所以才叫佛子花，灵山我是没去过，自然是不知道具体了，但是我知道在无量界，有几株通灵佛子花，形态上也与我们看到这株一样。”
<br/>　　大伙一听，都有震惊的目光打量眼前这株巨花，妙心惊诧道：“无量界不像是鬼域，就算是法师也绝不能轻易进去，更不要说把无量界的东西搬到这里来……这修墓的人，是怎么做到的？”
<br/>　　大伙都沉吟不语。
<br/>　　毛小方这时候发挥急智，说道：“对啊，没准真的是佛子花！”
<br/>　　叶少阳白他一眼道：“你又怎么知道，你一个道士，见过这玩意不成？”
<br/>　　毛小方道：“少阳子，你想想，这里的一切，都跟西域****有关，****跟藏传佛教的渊源，就不用我说了，佛子花也是佛门中的东西……这当然不是巧合了。”
<br/>　　大伙仔细一想还真是这样。梧桐道：“况且我只是说无量界中有佛子花存在，也不代表别的地方就没有。”
<br/>　　“行吧，不争论这个了，反正，这玩意摆在这可不是为了渡化我们，差一点命就没了。”
<br/>　　想起之前的幻觉，叶少阳还是心有余悸。
<br/>　　毛小方道：“别说这个了，现在怎么办？”
<br/>　　“既然都醒过来了，当然是弄了它，用火最好。”
<br/>　　大伙也都赞同。比火攻的话，首屈一指的当然是梧桐的五宝金莲，这法器对待邪物来说，简直就是一个喷火器。梧桐也不推辞，拿着五宝金莲走了上去。叶少阳等人也立刻跟上去，以免发生什么意外。
<br/>　　梧桐手持五宝金莲，对准了通灵佛子花，开始作法。五宝金莲喷出火焰，飞上了通灵佛子花的枝叶。
<br/>　　佛子花本是邪物，身上的邪气，是金莲业火最佳的燃料，看上去就像是浇上了汽油似的，火焰一下子就从叶片窜到了整株佛子花身上，一时间火光冲天，发出噼里啪啦的响声，将整个空间照亮。
<br/>　　佛子花之前合上的花叶再度打开，散发出了一股沁人的浓香。叶少阳闻到香味，心中一惊，急忙让大家屏住呼吸，往后狂奔到桥下，自己飞快地从包里拿出几张黄裱纸，把水壶打开，倒了水在上面，然后再涂上鸡喉骨粉，分给大家，每个人都往脸上贴了一张，上面留出一点空隙用来呼吸。
<br/>　　这鸡喉骨粉是来之前叶少阳专门去弄的公鸡的喉舌。
