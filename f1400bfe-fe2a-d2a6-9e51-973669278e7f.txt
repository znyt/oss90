        &lt;h3&gt;第三卷第九十一章  父子三人的决定&lt;/h3&gt;
<br/>
<br/>        张函从西城回来投入到紧张的工作中，他的时间也是刻不容缓的，这一段时间江俞轩没有在陈子昂是身边，他就好像少了一只眼睛，他对于陈子昂的消息只能是通过俩个孩子来了解。
<br/>
<br/>        陆玉走后，张函也不怕有人跟踪他了。
<br/>
<br/>        陈子昂忙的到处飞，但是每到周末她是必须飞回北京陪着两个孩子一起过周末的，这么几年孩子就是她坚强的支柱。
<br/>
<br/>        俩个孩子就像是大人一样会细心的问陈子昂的工作，陈子昂也不会隐瞒孩子，公司里的有些事情也会讲给俩个孩子听的，尤其是团团，小小的人动不动的就给陈子昂提一点意见。
<br/>
<br/>        张函就是从孩子的嘴里知道陈子昂的情况。
<br/>
<br/>        陈子昂今天飞郑州，明天湖北，后天青岛，张函都知道。他们父子三个已经是作战联盟了，都偷偷的隐瞒着陈子昂。
<br/>
<br/>        俩个孩子为了和父亲相见，自动的承担了教室值周生的任务，他们对财叔说他们要值周，打扫卫生，让财叔迟点去接他们，基本都会迟那么俩个多小时。
<br/>
<br/>        张函在北京的时候就会带着俩个孩子在附近的地方玩一会，偶尔父子三人还会打打游戏，当父子三个人又在一起的时候：
<br/>
<br/>        “爸爸，我觉得这样欺骗妈妈是不对的。”团团严肃的说。
<br/>
<br/>        “爸爸，你就说你到底想怎么样？”嘟嘟也说。
<br/>
<br/>        “我们已经欺骗妈妈很多年了，我心里愧疚得慌。”团团捂着心口，一副心疼的模样。
<br/>
<br/>        “你说，你的老婆是不是不会生孩子？你想骗我们给你当孩子？”嘟嘟一副受伤了的表情。
<br/>
<br/>        “我没有老婆。”张函心虚的说。
<br/>
<br/>        “切，骗小孩子是不对的，张函，张家张沛尧之子，某某部门的老总，娶妻陆家之女，陆玉。”团团小手敲着桌子。
<br/>
<br/>        “说，你和我妈妈是怎么回事？”嘟嘟盯着张函。
<br/>
<br/>        张函被俩个孩子盯得心里发毛。
<br/>
<br/>        “你们怎么知道这么多？”
<br/>
<br/>        “还当我们是三岁小孩呢？”
<br/>
<br/>        “嗯，你们现在是九岁小孩。”
<br/>
<br/>        “我们虽然只是九岁，但是我们有媲美你的情商和智商，我们俩个人的智商和情商加起来是你多一倍，别想瞒我们俩个。”嘟嘟虽然很喜欢爸爸，但是还是会维护妈妈的。
<br/>
<br/>        “想知道什么呢？”张函心虚的问。
<br/>
<br/>        “别装糊涂，说不说？不说就一别两宽，再也不见。”团团和嘟嘟很好奇爸爸和妈妈是怎么回事，但是因为一直隐瞒着妈妈，又不敢问妈妈，只好来逼着爸爸了。
<br/>
<br/>        “一两句话说不完。”张函不知道该怎么说。
<br/>
<br/>        “那就三四句也行。”嘟嘟大方的给了张函一点时间。
<br/>
<br/>        “得很久才能说完。”张函说。
<br/>
<br/>        “嘟嘟，走吧。”团团站起来准备走。
<br/>
<br/>        “捡重要的说，最后一次机会了，爸爸，你确定不说吗？”嘟嘟拉着团团的手。
<br/>
<br/>        “那你们坐下来？”张函看着俩个孩子认真的样子，他有点蒙，要是俩个孩子不理他，他怎么办？
<br/>
<br/>        “坐下来？”嘟嘟看着哥哥。
<br/>
<br/>        “说吧！”团团拉着妹妹坐下来。
<br/>
<br/>        “我和你妈妈认识的时候还没有你们。”
<br/>
<br/>        “废话，如果有我们，我们还用问你？”团团有些不耐烦了，他觉得爸爸在敷衍他们，合着他们这么多年培养的感情都是白搭？
<br/>
<br/>        “我和你妈妈认识的时候，你妈妈还不满18岁，你妈妈20岁那一年我们在一起了，然后就有了你。”
<br/>
<br/>        “我们只是想知道你和我妈妈为什么没有在一起？明白？”团团觉得很受伤，这么多年爸爸还要欺骗他们。
<br/>
<br/>        “我去国外读书，回来就找不到你妈妈了。过了几年之后我才找到你们。”张函不知道怎么给孩子解释家里的事情。
<br/>
<br/>        “那你为什么要和别人结婚？”嘟嘟很生气。
<br/>
<br/>        “因为你爷爷奶奶逼着我和别人结婚。”张函悲伤的说，他的悲伤是自然流露。
<br/>
<br/>        “爸爸，那个时候你还小是吗？你是被逼的呀？”嘟嘟伸手，摸了摸爸爸的脸。
<br/>
<br/>        “是的。”张函拉着嘟嘟的手“如果不是看到你妈妈和你们，我可能都坚持不下去了。”张函想起这么多年的隐忍，心里很苦，眼眶已经发红，声音哽咽，他害怕孩子不理他。
<br/>
<br/>        “爸爸，我们不逼你娶妈妈。只要你好好的。”嘟嘟已经开始流泪了。人们都说女孩子前世是父亲的小情人，嘟嘟的情感特别丰富，她似乎能感受到父亲的悲伤。
<br/>
<br/>        “我觉得你应该反抗，而不是偷偷摸摸的躲藏。”团团闷着声音说。
<br/>
<br/>        张函一时语塞，他反抗的资本是什么？
<br/>
<br/>        “我试图有过，但是失败了。”他并没有告诉孩子他病倒了，他的前半生都是掌握在陆玉的手中，只是因为他不想伤害，一味逃避、一味忍让。
<br/>
<br/>        团团说得很对，他确实是需要反抗。
<br/>
<br/>        甸城，他也是应该要去一去的，就算是陈子昂不去他也应该去，因为哪里还有一个陈子寒。
<br/>
<br/>        他能将闫欢带回来，他也能将陈子寒带回来交给陈家父母。
<br/>
<br/>        原来是有多憎恨他的身份，这一会就有多感激这个身份，陆玉临走的时候对他说：“要不是他是陆家的女婿，他的下场不是这个样子。”
<br/>
<br/>        有了陆家女婿的这个身份，是不是更容易做事情了？
<br/>
<br/>        俩个孩子见爸爸沉默不语，心里也有些难过，虽然爸爸说的很模糊，但是他们也查了爸爸的身份，一个小孩子怎么能反抗得了呢，如果妈妈逼着他们去做事情，他们怕是也会因为怕伤了妈妈的心会去做的。
<br/>
<br/>        “爸爸，虽然，我很想你和妈妈在一起，但是你也不要为难，妈妈忙她的事业，她已经习惯了一个人的生活，做不了夫妻你们就做朋友吧。”团团永远都是语出惊人，思维超出了一般孩子。
<br/>
<br/>        “对呀，爸爸，你可以和妈妈做朋友的，到时候我将你介绍给妈妈做朋友。我们可以光明正大的来往。”嘟嘟从悲伤中走出来，开心的笑着。
<br/>
<br/>        “唉，你不知道啊，这种提心吊胆的日子不好过，每次见了妈妈都怕被妈妈发现。”嘟嘟拍拍心口。
<br/>
<br/>        “对不起，是爸爸太自私了。”张函想到俩个孩子为了隐瞒和他来往的事情，心中一定是藏着愧疚的，他确实很自私。
<br/>
<br/>        “不用你们介绍，等你妈妈忙完这一段时间，我去找她，向她坦白。”该来的总会要来，该面对的总会要面对，他不可能一辈子偷偷摸摸的和孩子见面。
<br/>
<br/>        “你说妈妈知道我们和你来往回不回伤心？”嘟嘟有些担心。
<br/>
<br/>        “你妈妈是一个通情达理的人，只要我不和她争你们，她应该是能接受的。”
<br/>
<br/>        “那就这样说定了，找个时间我们向妈妈坦白。哎呀，终于不用提心吊胆了。好期待哦。哥哥，你期待吗？”
<br/>
<br/>        “嗯。”团团点点头。
<br/>
<br/>        “那就这样说定了，爸爸，你等我通知，我给你们创造机会。”团团站起来拉着妹妹的说。
<br/>
<br/>        “我们要回去了。爸爸再见。”
<br/>
<br/>        父子三人确定了要正面面对陈子昂，张函不知道面对陈子昂的结果是什么，但是他必须要和她共同面对一切。
<br/>
<br/>        陈子昂必定会和陆玉成为对立面，不仅仅是因为他张函。
