        连星儿微笑着点了点头。
<br/>
<br/>        童筱颖再次确认：“真的？”
<br/>
<br/>        “嗯，真的。”连星儿应道。
<br/>
<br/>        童筱颖接着说道：“选好了，就不能反悔了哦？”
<br/>
<br/>        “我才不反悔，要反悔也是他明司寒去反悔。”连星儿兴高采烈地说，或许是一时高兴忘了形，她又脱口而出道，“反正，到时候，他一定会跟他爷爷说取消联姻的。”
<br/>
<br/>        “你和他认识？”童筱颖倒是听出点了猫腻。
<br/>
<br/>        连星儿点头道：“他跟我一个大学啊！好像比我高一个年级还是两个年级去了。”
<br/>
<br/>        童筱颖顿时只笑不语。
<br/>
<br/>        连星儿这边一定下来，童筱颖便将情况反映给了傅夜沉。
<br/>
<br/>        傅夜沉也很意外，自己的宝贝女儿竟然会如此听话，一点都不像平日里翻天覆地的作风。
<br/>
<br/>        电话里，傅夜沉再三确认：“星儿那丫头真的答应联姻呢？”
<br/>
<br/>        “嗯，她自己选的明司寒。我问过她了，她确定。”童筱颖一边拿着手机回应，一边看着那边正坐在沙发上盘腿吃苹果派的连星儿。
<br/>
<br/>        “你把电话给她，我亲自问她。”傅夜沉接着说道。
<br/>
<br/>        童筱颖应了声“好”，然后回到了连星儿的身边，将自己的手机朝她递了过去：“你爹地的电话。”
<br/>
<br/>        连星儿见状，连忙放下手中的苹果派，拿纸巾擦了擦手指，接过手机，软糯软糯地唤了声：“爹地，你找我？”
<br/>
<br/>        “如果，你有喜欢的男生，这件事情，爹地会另外想办法，不需要你牺牲自己的幸福。”傅夜沉语重心长地说道。
<br/>
<br/>        连星儿趁机打趣地问道：“爹地，我若是跟你说，我喜欢洛羽，你会同意我跟洛羽在一起吗？”
<br/>
<br/>        “……”傅夜沉愣了半晌，答非所问，“爹地只是觉得他……”
<br/>
<br/>        “既然你不同意，那我嫁谁都无所谓啊！”连星儿故意调皮地说。
<br/>
<br/>        童筱颖站在一旁，不禁对连星儿使眼色起来。
<br/>
<br/>        傅夜沉深吸了口气后，接着说道：“那你先试着跟洛羽交往吧！明家的事情，我另外想办法。”
<br/>
<br/>        集团和女儿，到底还是女儿比较重要。
<br/>
<br/>        童筱颖倒是很欣慰，只不过，却不满意这小丫头故意刁难。
<br/>
<br/>        连星儿也见好就收，笑道：“好了啦！我就不逗爹地了！我至今没有喜欢的男孩子啦！不过，爹地，你放心好了！明司寒要是知道他爷爷要他跟我订婚，一定会反抗到底的。”
<br/>
<br/>        “你认识他？”傅夜沉顿时感到十分诧异。
<br/>
<br/>        连星儿解释道：“认识啊！他是我学长嘛！学校里的风云人物！”
<br/>
<br/>        “……”傅夜沉忽然觉得，自己似乎是真的老了，连这小丫头都“斗”不过。
<br/>
<br/>        “爹地，你放心好了。这件事情，我自己会搞定。”连星儿信誓旦旦地说。
<br/>
<br/>        傅夜沉虽然心存疑虑，但终究还是尊重连星儿自己的选择。
<br/>
<br/>        当明司寒从爷爷明护国那儿得知，连星儿选的人是自己后，不禁印堂发黑。
<br/>
<br/>        这丫头到底在搞什么鬼？
<br/>
<br/>        有男朋友了，还选他当未婚夫！
<br/>
<br/>        既然明护国来了滨江城，自然是会住在明司寒的私人别墅里。
<br/>
<br/>        面对爷爷，以及连星儿的选择，明司寒是一脸不满：“爷爷，你不知道，那丫头已经有男朋友了。”
<br/>
<br/>        “是吗？据爷爷派人所调查到的资料，连星儿这丫头就从来没有谈过一个男朋友。”明护国一脸淡定。
<br/>
<br/>        明司寒不解地看着明护国：“这种事情，爷爷你也查得出？你难道不知道，连星儿的男朋友是洛羽吗？”
<br/>
<br/>        “你爷爷我招孙媳妇，可不是随随便便的。连星儿的里里外外，以及各种人际关系，我派人已经全部调查清楚了。那个洛羽是吧？洛羽和他的经纪人林苏，十有八九是知道了连星儿是傅夜沉之女的缘故，在利用她的身世背景，为洛羽在演艺圈里搭桥牵线。”明护国双手交叠，搭在龙头拐杖上，有板有眼地说。
<br/>
<br/>        明司寒顿时不可思议地看着明护国，一句话也说不上来。
<br/>
<br/>        他沉默了一会儿，不禁深沉地问道：“爷爷为什么忽然改变主意，要跟傅家联姻？”
<br/>
<br/>        “傅夜沉在发展暗势力，将来难免会对明家不利。”明护国语重心长道。
<br/>
<br/>        明家现在可以逞一时之强，对傅家下手，但是，这事一旦被傅夜沉给盯上了，他肯定会放长线钓大鱼。
<br/>
<br/>        傅氏集团没了，傅夜沉还有资本东山再起，但是他们明家，在军政界活动，若是一人倒下，连带着都会跟着遭殃。
<br/>
<br/>        明司寒不满地皱起了眉头：“爷爷还说洛羽在利用星儿，您自己也不过是半斤八两。”
