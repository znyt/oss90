　　“我凝炼圣地，本是打算来威慑七大宗门的人，没想到居然有人将注意打到剑源界的上面去？五位半步天人境强者？怎么可能，哪一方势力居然可以派出如此之多的强者，前往九域？”
<br/>　　楚风眠的身形，在星空之中连连穿梭，他带着海公主，将遁光速度催动到了极致，急忙的向着九域敢去，同时脑海之中，有些吃惊的思考这。
<br/>　　这剑源界发生的事情，楚风眠已经是通过心魔之王，全部知晓了。
<br/>　　楚风眠凝聚剑源界，也是为了剑道门发展，跟可以威慑七大宗门所用。
<br/>　　可没想到居然是有人将主意打到了剑源界的上面来，一方圣地，想要凝炼出来，需要大量的资源财富才可以做到。
<br/>　　楚风眠可以将剑源界凝聚成一方圣域，也是因为七大宗门之中的许多强者，都是陨落在了剑源界之中。
<br/>　　借助那些七大宗门强者的骸骨，楚风眠才可以一举将剑源界凝聚出一方圣地来。
<br/>　　对其他宗门的武者来说，一旦是将剑源界占据夺取，便是不需要在辛辛苦苦，耗费大量的资源去凝炼圣地了。
<br/>　　不过在楚风眠的预计之中，除了七大宗门，应该是没有人可以有能力打剑源界的主意才对。
<br/>　　哪怕是有着两三位半步天人境的武者，以剑源界之力，在加上心魔之王，都可以轻松化解。
<br/>　　可这一次攻打剑源界的，居然是足足五位半步天人境。
<br/>　　大罗天星域的四大宗门中拥有半步天人境武者最多的齐林山，也就只有五位。
<br/>　　“荒古门？六道魔门？到底是那一方势力？”
<br/>　　楚风眠眉头紧锁，七大宗门之中，也许可以有着这么多的半步天人境武者。
<br/>　　可现在九域剧变，无数强者聚集九域，七大宗门自身也是焦头烂额，不可能在这个时候，会排出如此之多的强者来围攻剑道门的。
<br/>　　可除了七大宗门，楚风眠也想不到其他势力，可以做到这一切。
<br/>　　“赶快回九域之中，剑源界绝不能丢。”
<br/>　　楚风眠眼神之中露出几分急迫神色，剑源界可是楚风眠千辛万苦才凝炼出的一方圣地，如果说不是七大宗门的攻打，楚风眠积攒数十年的资源，才可以做到这一切。
<br/>　　更何况现在楚风眠身边的所有人，寒月漓，洛汐，心魔之王等，可都是在剑源界之中，一旦是他们有所损失，那才是灭顶之灾。
<br/>　　“走！”
<br/>　　楚风眠将始祖天龙血脉的力量催动到了极致，他的身形顿时化为了始祖天龙的模样，浩浩荡荡的从星河之中穿行。
<br/>　　楚风眠化身为始祖天龙，瞬息之间便是穿行了千万里的距离，速度快上了无数倍。
<br/>　　始祖天龙，掌控空间之力，楚风眠化身始祖天龙，在这无尽星空之中，任意穿行，不会受到任何的阻拦。
<br/>　　不过几分钟的时间，就在楚风眠的面前，出现了一座笼罩这九重天火的大陆。
<br/>　　这一方大陆之上，笼罩九重天火，如此独特的一方大陆，在这无尽星河之中，唯有九域。
<br/>　　“直接回剑源界，我倒是要看看，到底是什么人敢来攻打剑源界！”
<br/>　　楚风眠始祖天龙的身形，瞬间化为了他本来的模样，旋即隐藏了气息，落入到了九域之中，便是进入到了虚空深处，来到了剑源界附近。
<br/>　　至于海公主则是直接被楚风眠放在了九域之中的剑道城之中。
<br/>　　临近剑源界，楚风眠都可以清晰的感觉到一阵阵的灵力波动。
<br/>　　正是有人在出手攻打剑源界。
<br/>　　楚风眠遥远的望了过去，一共五道身影，全部都是半步天人境的强者，正在联手攻打这剑源界。
<br/>　　这五道身影，都无比年轻，令楚风眠吃了一惊，不是外貌上的年轻，而是他们的真正年龄。
<br/>　　这五道身影的真正年纪，也就是在百岁出头的模样，在武者之中，这样的年纪能够踏入到帝尊境，都是极为耀眼的天才了。
<br/>　　能够在百岁左右踏入九劫古帝的，更是天才之中的天才，号称妖孽了。
<br/>　　而至于能够在百岁左右踏入半步天人境的，几乎都是少帝，荒天王这样的绝世天才了，足以是在古籍之中留名的妖孽天才。
<br/>　　这样的人物出现一个，倒也正常，可现在楚风眠面前这五道身影，几乎都是百岁左右的年纪，都已经踏入到了半步天人境的天才。
<br/>　　“这些妖孽天才，到底都是从哪里冒出来的？”
<br/>　　楚风眠都是吓了一大跳，再三确认，才确定他没有看错。
<br/>　　这五道身影身上的气息，绝不是七大宗门的人，而是来自于其他星域的妖孽天才。
<br/>　　可就算是整个大罗天星域，四大势力加起来，都未必可以找到如此之多的妖孽天才来。
<br/>　　“这些人身上的灵器，也都不是平凡之物，虽然都是最近时代炼制的，但是这炼器手段，绝对是天人境的强者炼制的！”
<br/>　　楚风眠仔细观察着，看着那五名年轻半步天人境武者身上的灵器，都是无比吃惊。
<br/>　　他们身上的灵器，虽然都是天级灵器，可却是几乎都是蜕变之前巨阙神剑那个层次的灵器了。
<br/>　　天人境强者炼制的灵器，可是要比起帝尊境，半步天人境武者炼制的灵器，还要更加强大。
<br/>　　虽然达不到极道仙兵的层次，可也称得上是半仙器了。
<br/>　　如此妖孽的天才，在加上这一身半仙器，这一群人的来历，绝对惊天。
<br/>　　“不管是什么来历，居然敢来攻打剑道门，便是只有死路一条！”
<br/>　　楚风眠眼神无比冰冷。
<br/>　　这五名年轻半步天人境强者，就算是有着在惊天的来历，他们敢来攻打剑道门，也只有死路一条。
<br/>　　就在楚风眠思考之际，那一群人联手攻打剑源界的攻势，又一次是被逼退了出去。
<br/>　　剑源界蕴含的力量，虽然不足以完全抗衡五位半步天人境强者，可加上心魔之王，寒月漓，洛汐等人的联手，也足以抵挡一二。
<br/>　　“好难缠，这小小的圣地虽然只是刚刚诞生的圣地，但是已经接近八品圣地的层次了。”
