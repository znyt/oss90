　　陵端快步走上前来，脸上带着温柔的笑容，道：“芙蕖师妹，你怎么会来这里，是师傅让你来的？”
<br/>　　完全不理会陵端殷勤的模样，芙蕖目光反倒是落在风晴雪身上。
<br/>　　“你为什么上天墉城？”
<br/>　　“我想学天墉剑术，行侠仗义。”
<br/>　　风晴雪一脸认真的说道，虽然她对天墉城剑术不感兴趣，但为了找到自己要找的人，只能先说点好听的蒙混过去。
<br/>　　“天下门派那么多，想学剑术不一定来天墉城啊。”
<br/>　　“天下门派那么多，可也不是每一个门派都像天墉城，对待女孩子公平啊。”
<br/>　　说罢，风晴雪继续道：“我听说这里有女弟子，更有女真人，看师姐你如此英姿飒爽，所以天墉城对待女弟子，肯定是一视同仁喽。”
<br/>　　谁都喜欢听好听的话，芙蕖也不例外，当下就展颜一笑，道：“天墉城确实没有禁止女弟子报名入门的先例，那你就试试吧。”
<br/>　　闻言，陵端回过神来就想要阻止，道：“师妹，这要是让师傅知道了可怎么办？”
<br/>　　掌门真人可是让他负责今年招收弟子的事宜，其他弟子也都吩咐下去，一定要将此事做好，得到掌门的重视。
<br/>　　可现在芙蕖的介入，可能会让他的任务出现差池，到时候掌门真人责怪的可是他啊。
<br/>　　“没事，我跟我爹说一下就行了。”
<br/>　　芙蕖没好气的说道，她心思聪慧，自然知道陵端在担心什么。
<br/>　　话语刚落，直接将他接下来的话堵死，陵端也是无可奈何，谁让芙蕖有一个做掌门的爹呢？
<br/>　　就在这时，一道剑影划破天际而来，最后化作一道白袍青年，落在天墉城门口，顿时引起了所有人的注意。
<br/>　　“你是何人，竟敢在天墉城上空御剑飞行。”
<br/>　　瞬间，陵端将腰间长剑罢了出来，将芙蕖护在身后，面露警惕之色。
<br/>　　来者一袭白衣，身材修长，长发随风飘扬，仙风道骨，容貌更是令人看一眼就难以忘怀，一双星眸在众人身上来回打量。
<br/>　　但凡被星眸扫过，弟子们纷纷觉得，在青年面前，完全没有秘密可言，同时，心中也升起一股浓浓的嫉妒，世间竟有如此俊俏的男子。
<br/>　　来者不是别人，正是外出修炼顺带做任务的慕清霄。
<br/>　　慕清霄来到天墉城已经有十年左右，除了天墉城出现大事，或则需要炼丹，才会偶尔出现一次。
<br/>　　至于平时，都在宫殿中修炼，从不外出，哪怕是缺少炼制丹药的灵草，都是吩咐侍女准备。
<br/>　　正是因此，整个天墉城弟子当中，也只有陵越与他最熟悉，至于其他弟子，也只是听过他的威名而已。
<br/>　　“你是......芙蕖见过凝丹长老。”
<br/>　　望着映入眼帘的青年，芙蕖眼中有些迟疑，当看到他腰间的玉佩后，眼中的迟疑化作尊敬与崇拜。
<br/>　　“你是掌门真人的女儿，都已经长这么大了？”
<br/>　　目光落在清雅动人的芙蕖身上，慕清霄嘴角牵起一丝笑意，微微点头。
<br/>　　芙蕖可以说是他看着长大的，当年他刚来天墉城的时候，芙蕖还只是个几岁大的小女娃，没想到时间过的如此之快。
<br/>　　“凝......凝丹长老，芙蕖你没开玩笑吧？”
<br/>　　芙蕖的话语，如同惊雷般在陵端脑海中炸响。
<br/>　　手持长剑的他，顿时有些傻眼了，他拜入掌门真人门下后，自然听说过凝丹长老的威名。
<br/>　　若非听他的师尊，天墉城掌门涵素真人亲口所说，打死他也不会相信，有这种妖孽的存在。
<br/>　　比如，凝丹长老二十岁的时候，便是练虚期强者，更是七品丹师，就算打娘胎里就开始修炼，开始炼丹，也没有如此恐怖吧？
<br/>　　不仅只有陵端，任何天墉城的弟子，对于慕清霄有的只是尊敬与敬畏，至于超越和追赶，那只能在梦里实现。
<br/>　　他们都有自知之明，哪怕执剑长老，紫胤真人都承认，凝丹长老的资质远超于他，在不久的将来，估计就会追赶上他的脚步。
<br/>　　当初到现在，已经过去十一年，谁也不清楚后者有多强，炼丹术有多么的高深。
<br/>　　现在，整个天墉城，知道凝丹长老底细的，恐怕也只有掌门与执剑长老了。
<br/>　　后者容貌虽看似只是一名二十岁的青年，但当本人站在他们的面前时，弟子们都感受到一丝强烈的压迫感。
<br/>　　这种感觉，他们只在紫胤真人与涵素真人身上感觉到过。
<br/>　　此时，慕清霄的目光落在陵端身上，剑眉一挑，道：“你是掌门门下，陵端？”
<br/>　　“弟子陵端，拜见凝丹长老。”
<br/>　　感受到慕清霄的视线，陵端一个激灵，快速将长剑收了起来，恭恭敬敬的行礼，不敢有丝毫马虎大意。
<br/>　　凝了凝神，可能是因为原著，慕清霄对陵端的印象可谓是非常的差。
<br/>　　此人狂妄自大，常目中无人，心胸狭窄，手段残忍，哪怕是对待同门师兄弟，都敢栽赃陷害，屡次下狠手。
<br/>　　微微点头，慕清霄也不再理会他，陵端如何，他实在没兴趣知道。
<br/>　　旋即，目光跃过陵端，最后落在一旁俊逸的青年身上，神色如常，但心中却暗自心惊。
<br/>　　俊逸青年给他的感觉，就是一口深渊，探不出深浅，整个天墉城，也只有掌门与紫胤真人能带给他这种看不透的感觉。
<br/>　　如果不错，青年恐怕就是他要找的人，欧阳少恭（长琴太子）。
<br/>　　太子长琴被贬为凡人，永世不得为仙，实力恐怕不及紫胤真人，否则想要焚寂，直接出手抢夺即可。
<br/>　　他非常清楚太子长琴性格，如果拥有绝对实力，便会直接碾压过来，而不是设下各种计谋。
<br/>　　虽然不及紫胤真人，但差的恐怕也不是很多，毕竟欧阳少恭身上可是有一半的仙灵。
<br/>　　除了欧阳少恭，在场最引人注目的，便是一身绿裙的风晴雪了，美貌自然不用说，绝对称得上绝世美女。
<br/>　　慕清霄打量几人的同时，弟子们也都纷纷打量着他。
<br/>　　“他就是天墉城最年轻的凝丹长老，我在幽都的时候，就在婆婆口中听说过，传闻他在丹道上成就也非同小可......”
<br/>　　风晴雪一身修为不低，已经迈入金丹期，可在慕清霄身上，她却感觉到一股非常可怕的气息。
