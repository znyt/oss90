　　澎湃而疾速的气势，不断肆虐。
<br/>　　夹杂着的剑气、流光，锋锐至极。
<br/>　　莫悠仗剑而挡，以一敌五，丝毫不落下风，反倒游刃有余。
<br/>　　一手握剑，纹丝不动。
<br/>　　反观顾非凡、令狐忘等人，显然已实力尽出，气势不断爆发。
<br/>　　照这般情况下去，顾非凡五人，休想奈何莫悠分毫。
<br/>　　不过，莫悠身后的一众学教弟子，此刻却在周遭肆虐的气势下，衣袂猎猎作响。
<br/>　　忽然，嗤…
<br/>　　一声轻响。
<br/>　　轻响声，来自于一位学教弟子衣袖上的忽然划裂。
<br/>　　“好锋锐的气息。”这个学教弟子脸色一变。
<br/>　　五大学宫首席，加上莫悠的剑意之下，周遭肆虐的气势，早已达到惊人地步。
<br/>　　莫悠无碍，青麟、童烨等少数修为较强者，亦可轻松抵挡。
<br/>　　但不少实力较弱的学教弟子，却开始身体摇动，在周遭气势下渐渐身体难支。
<br/>　　莫悠微微皱起了眉头。
<br/>　　他能挡顾非凡等人。
<br/>　　可现今肆虐的气势，蔓延四周，这种如同无差别攻击一般的大范围锋锐气息暴走，他一己之躯如何能悉数挡下？
<br/>　　半晌，莫悠看了眼身后青麟。
<br/>　　“各院普通弟子你可能抵挡？”莫悠轻声问道。
<br/>　　青麟笑笑，“这还需要问？莫悠师兄太少看我了吧。”
<br/>　　莫悠闻言，笑笑，点了点头。
<br/>　　“好，那便交给你。”
<br/>　　话音落下，莫悠原本仅仅是随意握剑的手，头一次动了。
<br/>　　淡然的星目之下，爆发出一抹锋锐之色。
<br/>　　莫悠淡然看了眼五大首席，手中之剑一震。
<br/>　　“这里地太窄，不适合你们战斗。”
<br/>　　锵…锵…
<br/>　　五大首席，瞬间被震退十数步。
<br/>　　下一秒，莫悠脚步一动，手中之剑，剑影重重，顷刻封锁了五人。
<br/>　　“给我上来。”莫悠轻喝一声。
<br/>　　一剑之下，竟是瞬间牵引了五人气势。
<br/>　　嗖…莫悠一跃而起，飞跃至半空。
<br/>　　五人在剑意牵引下，亦一并来到半空。
<br/>　　很显然，莫悠将他们六人的战斗转移到了半空，以免波及黑云学教弟子。
<br/>　　这时，原地只剩黑云学教弟子。
<br/>　　以及周遭呈包围之势的各院队伍。
<br/>　　莫悠离去。
<br/>　　一道道不善的脸色，顷刻纷纷加诸在青麟等人身上。
<br/>　　“莫悠不在了，我看你们还嚣张什么。”石惊涛冷笑一声。
<br/>　　各院弟子，霎时蠢蠢欲动。
<br/>　　司空宇踏前一步，高声道，“天藏九玄果、千风皇果、火皇芝…如此多的珍稀天材地宝。”
<br/>　　“不仅是此次比拼的积分，之后更是归自己所有。”
<br/>　　“我想，黑云学教的弟子，除了青麟外，其余人也收获颇丰吧。”
<br/>　　司空宇只是寥寥几句话。
<br/>　　但周遭原本仅仅是蠢蠢欲动的各院弟子，霎时变得面露狂热之色。
<br/>　　“上。”各支队伍中，忽然一道暴喝声响起。
<br/>　　简单的一个字，却仿佛是彻底打破了平衡与静谧的音符。
<br/>　　各院队伍，瞬间出手。
<br/>　　青麟连忙看向秦翊和童烨二人，“我身上重宝最多，那些除却首席之外的最强弟子，应该都会找上我。”
<br/>　　“找上你们的，应该只是些土鸡瓦狗，可能应付？”
<br/>　　秦翊和童烨二人撇撇嘴，“你都说其他弟子只是土鸡瓦狗了，我们若再应付不了，岂非连土鸡瓦狗都不如？”
<br/>　　嗖…青麟一个闪身，拉开了与众人的距离。
<br/>　　嗖…嗖…嗖…
<br/>　　果然，各院队伍中的强者，几乎都是冲他而去，瞬间追向了他。
<br/>　　司空宇、莫忧、石惊涛、独孤情、等等。
<br/>　　五大学宫圣王境七重修为以上武者，几乎清一色包围了青麟。
<br/>　　其余各院队伍，亦以包围了黑云学教弟子，瞬间出手。
<br/>　　天藏秘境内，连续10天的比拼，今日，终于爆发了最激烈的一次大战。
<br/>　　只不过，这场大战，属于不对等的两方。
<br/>　　一方，是五大学宫带头下的所有各学院队伍，逾数百天骄；一方，仅仅是黑云学教49人。
<br/>　　嘭…嘭…嘭…
<br/>　　剧烈的爆响声，激扬的元力流光，一瞬间爆发至惊人地步。
<br/>　　双方的战斗，几乎一瞬间便是彼此实力尽出。
<br/>　　只不过，结果却并没有意料中的快速结束。
<br/>　　反倒是双方激烈战斗不断，无半分停歇。
<br/>　　青麟瞥了眼一众学教弟子的战斗，点了点头。
<br/>　　一众学教弟子，虽人数较少，却能应付；特别是童烨等人，以一敌数，轻松至极。
<br/>　　反倒是青麟这边，以一敌十数，稍稍落了些下风。
<br/>　　不过，也仅此而已了。
<br/>　　半空中，战斗同样激烈。
<br/>　　莫悠一人力战五大学宫首席。
<br/>　　天藏首席，顾非凡；五痴首席，令狐忘；天阙首席，莫问山；金辰首席，金尘；明幻首席，林渺。
<br/>　　天藏、五痴、天阙、明幻、金辰，便是中域五大学宫。
<br/>　　不过，莫悠一人战之，显然游刃有余，甚至有空打量下方战斗。
<br/>　　莫悠点了点头，淡淡一笑。
<br/>　　下方战斗，看似是48人对战数百人；实则，这是妖孽与天骄间的争锋。
<br/>　　而结局，也显然易见。
<br/>　　青麟这边，力战各学宫强者，倒是吃了些亏。
<br/>　　“噗。”青麟微微吐出一口腥血。
<br/>　　“明幻学宫的幻心决吗？有些本事。”
<br/>　　刚才，便是明幻学宫一位圣王境八重修为的弟子，忽然轰了他后背一掌。
<br/>　　“哼，刚才不是很狂吗？”莫忧不屑地看着青麟。
<br/>　　“现在该知晓自己是什么东西了？”
<br/>　　“没有我哥的庇护，你们屁都不是。”
<br/>　　“是吗？”青麟没有反驳些什么，只是，冷漠的眸子中，已然泛过一道狰狞的凶光。
<br/>　　若是萧逸在此，一定会瞬间发现，这是青麟即将暴走的征兆，生人勿近。
<br/>　　可惜，萧逸不在此。
<br/>　　而这些各院强者，也不知晓。
<br/>　　咔…咔…咔…
<br/>　　一块块青色鳞片，渐渐涌现于青麟的双臂。
<br/>　　嗖…青麟的身影，瞬间消失在原地。
<br/>　　“好快。”在场各院强者，瞳孔一缩。
<br/>　　哪怕是石惊涛、司空宇、莫忧等人，亦丝毫跟不上青麟的速度。
<br/>　　嗤…
<br/>　　一道血爪，陡然在空气中闪过。
<br/>　　那位明幻学宫强者，瞬间被轰飞，胸膛上，数道狰狞爪痕汩汩留着鲜血。
<br/>　　“噗。”那位明幻学宫强者一口腥血喷出，竟是瞬间重伤。
<br/>　　另一边。
<br/>　　学教队伍处，一道距离爆炸声响起。
<br/>　　一片火海，顷刻环绕；一缕疾风，陡然吹袭。
<br/>　　火借风势，风增火力。
<br/>　　一条条若风若火的巨大游龙，顷刻游走。
<br/>　　童烨与秦翊的联手下，数十条巨大游龙瞬间冲散了各院队伍的包围。
<br/>　　剩下的数十学教弟子，如狼似虎般激跃而出。
<br/>　　“老子忍你们很久了。”
<br/>　　一道道桀骜而冰冷的话语，从学教弟子口中响起。
