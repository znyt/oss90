　　萧家大型矿脉内，府邸、兵器堂、议事堂、药堂等等，应有尽有。
<br/>　　族人们也忙碌得很，挖矿的、站哨的，族人足有二、三百，长老两位，执事十数。
<br/>　　这里，俨然就是个缩小版的萧家。
<br/>　　但大型矿脉如此重要，也难怪萧家非常重视，在这里安排了这么多的武者。
<br/>　　议事厅内，四长老、六长老坐于首位，萧逸坐于一旁，三人边品茶边聊着天。
<br/>　　“好茶。”萧逸抿了一口，称赞一声。
<br/>　　萧逸明显感到，茶水入腹，一股暖意沁人心脾，似乎还有一股微妙的力量，让人神清气爽。
<br/>　　“呵呵，喜欢喝就多喝些。”四长老笑道，“这紫叶灵茶，外面可喝不上。它有暖脉温体之效，武者长期服用，可让经脉更为通畅，大有裨益。”
<br/>　　六长老也道，“这紫叶灵茶，茶业生长于紫云铁矿脉周围，受紫云铁土壤的滋润，带着一些火属性灵气，倒是应运而生。”
<br/>　　萧逸再次抿了一口，暗道，“这茶业味道不错，又有奇特功效，迟些离开的时候，顺手多带些回去给依依尝一下。”
<br/>　　“对了。”四长老忽然放下茶杯，正色道，“萧逸，听说你这次来陨星山脉，是要找慕容千军那厮的。”
<br/>　　“那家伙仗着那不知所谓的谣言，可把你诋毁惨了，你找他算账也是应该。”
<br/>　　六长老用杯盖推了推茶业，再次抿了一口，道，“可是，据说慕容千军最近消失了，也不知去哪了。待会儿我让萧家在外的探子打探一下，看能不能找到他的下落。”
<br/>　　“那小人，一边在抹黑我们萧家的名声，一边又带领慕容家子弟多次针对打伤我们的子弟，真真欺人太甚。”
<br/>　　六长老面露怒色。
<br/>　　萧逸耸耸肩，道，“能找得到便找，找不到也无所谓。”
<br/>　　萧逸来陨星山脉，最主要的目的根本就不是慕容千军。他只是想来外面看看，见识下所谓的凶残妖兽，见识下陨星山脉的危险。
<br/>　　至于替萧家找回场子，打败慕容千军，那只是顺带的事。遇到了，便战一场，遇不上，也无所谓。
<br/>　　说到底，萧逸根本没将慕容千军放在眼中。
<br/>　　区区一个慕容千军，还不值得他煞费心机，到处奔波寻找，花那么大力气。
<br/>　　他更想去那些险地闯荡历练一番。
<br/>　　作为一个武者，提升修为，领略更高武道境界的精彩，才是萧逸想要做的。
<br/>　　四长老和六长老忽然对视了一眼，两人都是人精，似乎看穿了萧逸的心思。
<br/>　　而他们也从彼此的眼神中读到了对方的意思，“这小子是个不安分的主儿。”
<br/>　　“萧逸，我可告诉你，陨星山脉很危险，你可别打算四处溜达。”四长老率先警告道。
<br/>　　六长老则语重心长地吩咐道，“萧逸，大型矿脉这边有我们看着。你有空便去那些小型矿脉四处巡视，也好照顾一下其他武力弱小的子弟。”
<br/>　　“好。”萧逸点点头。
<br/>　　“臭小子，别答应得这么爽快，别以为我们不知道你心里想什么。”
<br/>　　四长老严肃道，“这些年，来陨星山脉的年轻子弟我们见得多了，一个个都年少轻狂，没少吃亏。”
<br/>　　六长老刚想嘱咐一声，这时，一个执事快步走了进议事厅，面露忧色。
<br/>　　“萧虎，怎么了？”六长老问道。
<br/>　　萧逸知道此人，萧家第二执事，后天九重武者，在萧家颇有权利，他是四长老的心腹。
<br/>　　“禀六长老。”萧虎沉声道，“之前在小型矿脉回来的子弟，个个带伤，有些伤势比较严重...”
<br/>　　“他们的伤势不都稳定下来了吗？”四长老打断道。
<br/>　　“之前是稳定了。”萧虎道，“可是疗伤药都用完了，要是现在忽然复发伤口，那就麻烦了。现在回家族运丹药过来也是麻烦，我想是不是要先护送他们回家族疗伤呢？”
<br/>　　这次来陨星山脉的一百多个萧家子弟，大部分都在被慕容家子弟抢夺小型矿脉时打伤了，早早地来到了萧家营地这里养伤。
<br/>　　萧壮、萧子木他们还是萧逸及时赶到，才避免了他们也受重伤。
<br/>　　四长老皱起了眉头，道，“他们是来这里历练的，这才十来天时间就回去了，我怎么向三长老他们交代。”
<br/>　　萧逸也皱起了眉头，问道，“矿脉这边，没有其他炼药师吗？”
<br/>　　“嘿。”六长老摆摆手，道，“那些炼药师一个个都自持身份尊贵，哪里会来这里受苦。再说了，他们一个个都是家族的宝贝，不容有失，怎么可能会让他们来这里呢。”
<br/>　　“诶，不对。”六长老忽然眼前一亮，道，“萧逸你不就是一品炼药师吗？”
<br/>　　“不行。”四长老摇摇头，道，“七八十个子弟，需要的疗伤药数里太大了，萧逸不过是一品炼药师，速度有限，现在炼药也来不及，家族里的丹药库存倒是足够。”
<br/>　　“萧虎，准备一下，明天一早就送那些受伤的子弟回家族。”四长老吩咐道。
<br/>　　“哎。”六长老叹了口气，“他们才出来十多天时间，这就回去了，我怕会损了他们的自尊。而且，大好的历练机会，恐怕要耽搁很久了。”
<br/>　　“如今我们萧家的年轻一辈，整体实力本就比慕容家和江家差。现在再出差池，十来年后，恐怕他们就远远比不上另外两家的子弟了。”
<br/>　　作为萧家长辈，两位长老为家族年轻一辈的事没少烦忧。
<br/>　　毕竟日后挑起萧家大梁的，还是年轻一辈。他们这些长老执事，日后也会退下来。
<br/>　　“慢。”萧逸叫住了准备离开的萧虎。
<br/>　　“少家主有何吩咐？”萧虎停下脚步，问道。
<br/>　　萧逸看向四长老，问道，“四长老，矿脉这边，可有足够的炼药材料？”
<br/>　　四长老点头，道，“当然是有的，陨星山脉，最值钱的除了紫云铁外，还有生长在各处险地的天材地宝。”
<br/>　　“我们萧家在这边的族人，一部分开采紫云铁；一部分则是搜集天材地宝，定时运送回家族。家族里有超过一半的材料都是我们提供的呢。”
<br/>　　“那就好。”萧逸点点头，看向萧虎，道，“不用安排那些子弟回家族了，直接给我准备一百人份的疗伤药材料，一百人份的淬体丹材料，送到我房中。”
<br/>　　“萧逸你想做什么？”六长老惊讶地问道。
<br/>　　“呵。”萧逸淡淡一笑，道，“明天早上，你们会看到足够的疗伤药。”
<br/>　　说罢，萧逸起身，离开了议事厅。
<br/>　　“这小子，他有把握一晚上炼制这么大数量的丹药？”六长老满脸惊讶。
<br/>　　四长老对萧虎吩咐道，“听萧逸的，马上给他准备材料。就算他失败了，明日再送家族子弟回萧家也不迟。”
<br/>　　.....
<br/>　　萧逸作为少家主，矿脉这边专门给他准备了一座舒适的府邸，供他居住。
<br/>　　房中，萧逸拿出了之前三长老送的乾坤袋，里面，有一个三长老提前给他准备好的三旋火炉。
<br/>　　三旋火炉，一品炼药师的标配，价格昂贵，算是紫云城内最好的炼药炉了。
<br/>　　看着那堆满了房间的材料，萧逸没有任何迟疑，立刻开始了炼药。
<br/>　　他的炼药术在一品丹药中，接近完美。所以当初他炼制各种丹药的速度都极快，仅仅三天便炼制了数百粒九转淬体丹。
<br/>　　如今，他在看了大长老赠予他的炼药心得后，完全可以炼制出二品丹药。也就是说，他是个准二品炼药师。
<br/>　　那些疗伤药，还有淬体丹，不过是一品丹药罢了。
<br/>　　以二品炼药师的能力，炼制一品丹药，其速度自然是更快的。
<br/>　　加之，控火兽武魂已从橙阶进化为黄阶，控火能力增强了许多，炼制起丹药来更加得心应手。
<br/>　　如今的他，一晚上炼制一百人份的一品丹药，轻而易举。
