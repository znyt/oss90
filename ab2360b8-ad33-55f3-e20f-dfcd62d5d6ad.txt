　　他在掌心用自己的血画的这道符，名为唤尸符，可以利用血气，激发一切僵尸的凶性，用一个比喻来说，普通的人血，对僵尸来说，就好像是一个饿到极点的人看到米饭馒头之类的，如果没有别的食物，自然也是扑上去狼吞虎咽，但是在唤尸符刺激下的人血，就好像是酱肘子、猪耳朵这样的肉菜，僵尸见了，自然是放弃那些不好吃的，来追叶少阳。
<br/>　　“是你！”女尸在叶少阳面前站定，第一次开口发出了声音。人形煞是没有灵智的，但是被女鬼附身之后，现在成了鬼尸，当然是有灵智的，不过也女鬼也被人形煞的煞气反噬，虽然有灵智，但目前也只是一味地想要吸血，知道看到叶少阳——确切说是感受到了他身上的法师的气息，这才短暂地清醒，认出他来。
<br/>　　“是我，冤家路窄哦，吸血吸得挺舒服的吧。”
<br/>　　叶少阳冷冷一笑。这个时候，近处已经一个人都没有了，连小三等人也不知道去哪了，没有人能看得见自己的容貌，不会露馅。叶少阳就从容多了。
<br/>　　又念了一遍咒语，将符文的血腥气更加激发出来，让女尸闻到。
<br/>　　女尸眼神立刻变得恍惚，口中流出涎水，朝着叶少阳扑过来。
<br/>　　叶少阳……转身就跑。
<br/>　　这一幕让那些躲在远处观看的土匪们大跌眼镜，本来所有人都跑了，就他一个不跑，还对着女尸比划了半天，本来以为他真有什么本事对付这女尸，结果居然跑了。
<br/>　　一般人在女尸速度这么迅猛的追击之下，不可能有机会逃掉，但叶少阳施展出天罡步，在空地上奔跑起来，女尸在身后追赶。
<br/>　　叶少阳一路跑进了树林里，女尸也追了进去，叶少阳跑到一个地方，猛然站住，回头看去。
<br/>　　女尸怔了一下，被血腥味冲昏了头脑，没有多想，直接扑了过去。
<br/>　　叶少阳站着不动，看着她，女尸冲到她面前三四米远的时候，突然脚下一空，坠落下去。
<br/>　　原来，这里是叶少阳设计好的一处陷阱，不过不是他挖出的，而是……茅房的茅坑。不过这个茅坑有点特别——可能是因为山上人多，人多，拉屎尿尿的就多，这里有没有环卫工人清理粪便，所以叶少阳发现的这个茅坑，下面是空的，是两座岩石的夹缝，中间有空隙，直通山谷，人如果掉下去的话，并不会坠落到山谷里，会被两块岩石夹在中间。
<br/>　　叶少阳当时看到这处茅坑的时候，立刻就计上心头，当时在等待中，也没什么事可做，就从附近找了一些蓬草，盖住了茅坑，做好记号之后，引女尸过来……
<br/>　　叶少阳冲过去看，女尸上半身正好卡在夹缝中，张牙舞爪，狠狠的望着自己，口中发出咆哮声。
<br/>　　她是鬼尸，虽然力大无穷，终究不能飞天遁地，处在这种境地之下，也是无可奈何。
<br/>　　叶少阳微笑着看着她，小心翼翼地趴在地上，以免踩到粑粑，然后拔出七星龙泉剑，对着她的脑袋砍下去。
<br/>　　女尸挥舞双手，抓住了七星龙泉剑，用力一拉，差点把叶少阳也拉下粪坑。还好七星龙泉剑是九段光的无上神兵，就算没有罡气加持，自身的灵气之强，也不是一般邪物能够触碰的。
<br/>　　在她双手接触到剑身的瞬间，立刻腾地一声，冒出了一股尸气，女尸惨叫一声，立刻松开了双手。
<br/>　　叶少阳对着脑袋刺了一剑，这一剑刺中了眉心，女尸尖叫起来，不过当叶少阳再用力刺下去的时候，却将女尸的身子往裂缝下面推进去少许，七星龙泉剑的剑刃够不到了。其实也能够到，只是……叶少阳需要往下面再爬一点，而这里是粪坑，粪坑的四周都是粑粑，自己要往下爬一点容易，但势必会沾染上一些脏东西……
<br/>　　叶少阳犹豫了一下，看着粪坑里的粑粑，实在还是没能下足，对着咆哮不止的女尸耸了耸肩，说道：“没办法，本来是想给你个痛快的……”
<br/>　　说完，从背包里拿出一包硫磺，一包糯米，用一张大的黄裱纸裹住，点燃之后扔了下去，被女尸一抓，散落开来，不过流火和硫磺一起洒落在女尸身上，遇到尸气，立刻燃烧起熊熊火焰，叶少阳又画了一张地火符丢进去。
<br/>　　因为女鬼已经与人形煞合二为一，挣脱不得，等肉身被完全烧毁、煞气清除，她能够从尸身里离开的时候，她也已经被地火符烧毁了三魂七魄。
<br/>　　“啊……我做鬼也不会放过你。”女鬼感受到了自己魂魄燃烧的痛苦，怪叫起来。
<br/>　　鬼也是人死后之物，她也没去过阴司，没有接受过阴司的文化熏陶，一直在人间，情急中之中还是冒出了人活着时候经常说的话。不过在叶少阳听来还是挺搞笑的。“你已经是鬼了，当不了鬼了，只能当精魄了，撒扬娜拉！”
<br/>　　叶少阳说完，直接跑走了。
<br/>　　反正女尸也是必死，自己没必要在这浪费时间，不然一会肯定会有土匪赶过来。
<br/>　　叶少阳看了一眼空地那边，已经有人打着火把过来了，于是快速地穿过树林，翻过围栏，跳了下去，一路往回狂奔。
<br/>　　人形煞就这么被灭了，没有任何打斗，也没有什么险情，这其实也是叶少阳的原则：以他现在的法力，如果硬拼，就算能打赢，也要费不少力气，还要面临着被那些土匪发现的危险。
<br/>　　这件事也还是印证了人间法师对敌的原则：利用一切可以利用的东西，能尽量省劲就尽量省劲，更不要说叶少阳是最会利用这些外在的因素来制敌。
<br/>　　事实上，叶少阳真正跟对方拼到兵刃相间的地步，还真的极少。他自己也知道，自己不是道风那种擅长硬上的人，因为肉身所限，也没那个实力。真到那一步的话，会非常吃亏。
<br/>　　“等下！”叶少阳刚翻过围栏跳下去，身后突然传来一个声音，叶少阳吓了一跳，看了一眼身后，有个人居然跟着自己跳下了围栏。仔细一看，还不是别人，是那个小三……
