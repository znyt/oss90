　　紫霄宫中，道祖鸿钧收下三清与女娲为弟子，并赐下圣道之基鸿蒙紫气后，便不再言语。
<br/>　　眼见如此情况，在场众修士心里咯噔一声，暗叫不好。到了如今，他们岂能看不明白？这鸿钧道祖的弟子，就不是对应前方六个蒲团之上的人吗？而若是那鸿蒙紫气，是成圣的必备条件，那么，三清、女娲他们注定成圣，自己等人岂非永世为蝼蚁，难以翻身？
<br/>　　当下，帝俊、太一、镇元子、冥河等人纷纷脸色变得难看无比，一个个望着最前方的六个蒲团，神色之中露出深深的懊恼，至于鲲鹏更是神色怨毒地看向红云道人，若不是他，自己早就成为道祖的弟子了。
<br/>　　若是早知道六个蒲团，乃是代表着道祖鸿钧弟子，未来的圣人，自己当初说什么也要抢夺一个啊！如今，要如何是好？如何是好啊？
<br/>　　道祖鸿钧在给了女娲鸿蒙紫气后，便沉默不语。见此，就在帝俊、镇元子等人后悔无比之时，蒲团上，接引与准提眼看鸿钧沉默，二人顿时大急，纷纷跪下来，朝着鸿钧老祖开口恳求道：
<br/>　　“老师，吾等二人一心向道，还请老师收下弟子。”
<br/>　　沉默许久，看着二人，鸿钧老祖深吸一口气，叹息道：
<br/>　　“罢了，罢了！你二人与贫道缘分浅薄，却也可为贫道记名弟子，这两道鸿蒙紫气，就给了你们吧。”
<br/>　　大袖一挥，又是两道鸿蒙紫气融入到接引、准提体内。却是鸿钧想起了当年西方道魔大战之时所欠下的因果，这不得不还。他知道眼前这两人日后会离开玄门，重立他门，但他不得不这样做。
<br/>　　“弟子多谢老师，叩谢老师的恩德。”接引与准提大喜，叩谢道。
<br/>　　就在此时，人群之中，妖族的太一却是忍不住了，只见他站了起来，朝着鸿钧道祖恳求道：
<br/>　　“道祖，我大哥为妖族帝皇，可否得一道鸿蒙紫气？”此话一落，顿时其他人反应过来，纷纷恳求。
<br/>　　“九为极致，天道之下，当有九位圣人。我玄门占八位，已去七位，还剩一位。哎！也罢，此处还有一道鸿蒙紫气，便让它自行选择吧。”
<br/>　　鸿钧见此情况，面色淡漠，不动声色，对众人的恳求视而不见。可最终不知何原因，终究是叹息一声，大修一甩，一道鸿蒙紫气飞出，在紫霄宫中盘旋一阵，众修士急忙死死盯着，不放过，有的甚至还用手去抓，但鸿蒙紫气透体而过，他们并没有抓到一点边角。
<br/>　　最后，众修士看到，那一道鸿蒙紫气绕着大殿转了一会儿后，突然降下，并迅速落入红云道人身体之中。
<br/>　　红云道人见此，脸色狂喜，急忙叩拜，感激涕零一番，鸿钧道人却是始终脸色冷淡，不悲不喜，毫不理会。
<br/>　　其余众修士见此，纷纷看向红云道人，面上露出羡慕之色，但也有一些人，眼中闪过一丝丝冷光，心中却是打起别样的心思。如帝俊、太一、鲲鹏等人，特别是鲲鹏，他见到红云得到鸿蒙紫气，眼中怨毒之色更胜了。
<br/>　　倒是镇元子道人，他见此，便知不妙，心中为朋友焦急万分，担忧不已。
<br/>　　“第二次讲道结束，万年之后，第三次，亦是最后一次讲道。”鸿钧不管众修士反应，淡淡开口说道。
<br/>　　话音落后，他的身影便渐渐淡化与虚空，却是已经消失不见了。而在此时，紫霄宫大门也被打开了。
<br/>　　众修士见此，都纷纷出了紫霄宫，朝着洪荒大地而去，此次讲道太过深奥，他们还需要更深的领悟。镇元子更是第一时间抓起红云道人，飞快地往万寿山飞去。
<br/>　　“鲲鹏道人，还请留步，吾等有事相商。”
<br/>　　就在此时，帝俊突然叫住就要离去的鲲鹏，而后他们两兄弟与女娲两兄妹一同朝着鲲鹏走去。
<br/>　　鲲鹏见此，却是停了下来，看向四人，心中疑惑不解，这四人来找自己干什么？莫非——
<br/>　　“见过四位道友，不知四位道友邀我所为何事？”
<br/>　　帝俊毫不拖泥带水，直接开口说道：“鲲鹏道友，我等同为妖族，如今妖族统一是为大势，吾等在此邀你北冥宫，加入妖族之中。如何？”
<br/>　　果然，鲲鹏听到此话，眼睛微微一缩，心中叹息一声，沉声道：
<br/>　　“多谢四位道友的抬爱，贫道北冥宫地处偏僻，势力弱下，不想加入到洪荒争斗之中，就此告辞！”
<br/>　　说完，还不等四人再说话，鲲鹏便展开翅膀，直接向着洪荒之中飞去。
<br/>　　见此，四人面色有些阴沉，望着鲲鹏离去的身影，眼中闪过一丝寒光。
<br/>　　“这鲲鹏还真不识抬举！”太一怒声喝道。
<br/>　　“好了，区区一个鲲鹏，吾不信他能翻起多大浪花，吾等先回去斩尸，之后再共商妖族大事。”沉默一会儿后，帝俊开口道。
<br/>　　“善！”其他三人没有意见，纷纷同意。之后，驾起祥云，向着洪荒而去。
<br/>　　元始三人并不知道这四人之间的事情，但就算他知道，也会毫不在意。
<br/>　　三人穿过混沌进入洪荒之后，便径直返回昆仑，此次听讲三尸之道，他们收获颇丰，元神之中，三花跳动，恐这怕是斩尸在即的征兆，急需闭关。
<br/>　　回到昆仑后，三人便宣布闭关，他们需要炼化鸿蒙紫气，还有参悟斩尸之法，早日斩尸。徒弟们见他们急匆匆的样子，也就没有来打扰。
<br/>　　元始最先看的并不是鸿蒙紫气，而是参悟三尸之道。反正鸿蒙紫气就在那里，它又不会跑了，急了干什么。如今，斩三尸证道才是重中之重。
<br/>　　正所谓成道之法有三，一曰以力证道，二曰斩三尸证道，三曰功德成道。以力证道自然是不用多说，盘古以及三千混沌魔神，都是以力证道。在元始看来，以力证道并非是要开辟大千世界。以力证道，是要参悟属于自己的大道，之后不断积累道行，大法力，以大法力、大神通强行劈开境界的禁锢，到达混元境界。开辟世界只是以力证道中的一种。
<br/>　　功德成道，自然是对天地有大功，天道眷顾后赏赐所得，这是一种天地果位，不是混元大罗金仙，也不是所谓的圣人，而是至人，亦或者称作功德至人、功德圣人，这其实是一种取巧的做法，所以以功德成道最弱。
<br/>　　但，据他所知，后世的圣人，除了鸿钧道祖，其余的都是以功德成圣者。
<br/>　　这斩三尸成道之法却是鸿钧道祖独创的，通过先天之物斩去善恶执三尸，心境清明，然后三尸合一，混元一体，力量达到巅峰，证道混元大罗金仙，倒是有独特之处。
<br/>　　而且正是因为这三尸之法，所以才会有准圣这一境界，原本洪荒之中是没有准圣这样的说法，而是自从出现了鸿钧，才有准圣境界。原本这些都是被称作大罗金仙。
<br/>　　就如元始如今，他虽然还是大罗金仙境界，但他的法力道行早已经到了准圣圆满了，距离成道也只差一步之遥。
<br/>　　不再多想，他很快就沉浸在斩三尸之法中。
