　　第278章
<br/>　　蛤蟆寨，聚义厅。
<br/>　　云中龙被绑成粽子一样跪在地上，一左一右，是满身浴血的高飞豹和张肥。
<br/>　　程大雷看着这一幕，感到一阵阵头疼。
<br/>　　“谁的主意，究竟谁的主意？”程大雷将桌子拍得邦邦响。
<br/>　　“大当家，你先消消火，他们不是也没事么？”刘悲过来道。
<br/>　　“私自行动，自由散漫，无组织无纪律！按寨规第九章第三十二条该如何处置！”
<br/>　　“大当家，寨规第九章是什么？”
<br/>　　“咱们好像没有寨规这种东西。”
<br/>　　“呃……”程大雷感觉头越来越疼了，组织性纪律性要抓一抓了。
<br/>　　“高飞虎张肥，罚你们两个去砍树，一人给我砍一百棵树，好好想想自己错在哪里。”
<br/>　　“是。”
<br/>　　二人灰头土脸的退下，可程大雷分明看到，二人刚走出门口就昂起头来。
<br/>　　“高二爷，好样的啊。”
<br/>　　“下次再有这样的事，可别忘了兄弟我们。”
<br/>　　“一般也就一般啦。”
<br/>　　……
<br/>　　程大雷揉着发疼的太阳穴，目光落在面前的云中龙身上。
<br/>　　“大当家，他怎么处理？”
<br/>　　“你们是什么人，知道我背后是谁么，不把我放了，会让你们死无葬身之地！”云中龙恶狠狠道。
<br/>　　程大雷撇撇嘴，还真是没文化的反派台词。他挥挥手：“丢进地牢关起来，派人去送信，让他的手下拿银子来赎人。”
<br/>　　一山不容二虎，程大雷知道山寨发展下去，迟早会和莫鸣米碰上的。毕竟，莫鸣米是鸠占鹊巢，自己才是琴川关守将好不好。可现在山寨初建，还没有和莫鸣米硬碰硬的资格，可张肥和高飞豹的行动，毫无疑问将这个进程加快了，程大雷也得早做安排。
<br/>　　不过，靠他们二人，就能闯进对方老巢，杀人放火如履平地，最后还把人家老大绑了。干得委实是有点漂亮。
<br/>　　看来，自己必须重新估量手下人的实力，当然，还包括山寨的制度建设，也必须抓一抓了。
<br/>　　……
<br/>　　琴川关。
<br/>　　莫鸣米身着官袍，坐在太师椅上。在他面前站着周差海，方闻雷，田倜三人。三人都垂着头，像霜打的茄子一般。
<br/>　　“云中龙被人绑了？”
<br/>　　“是呐，莫将军一定要救救我们老大。”周差海道。
<br/>　　“对方有多少人？”
<br/>　　“不超过五十人。”
<br/>　　“五十人就能在你们老巢抢走人，你们是干什么吃的？”
<br/>　　周差海三人都低着头不敢说话，他们还没说，一开始动手的只有两个人呢。
<br/>　　“对方是从哪儿冒出来的？”
<br/>　　“还没查到。”
<br/>　　“那对方老巢在哪儿？”
<br/>　　“这个……也不清楚。”
<br/>　　“呃……那你们该不会连他们是谁都不知道吧？”莫鸣米微倾身子。
<br/>　　“这个知道，这个知道。”周差海擦擦额头的汗珠，从怀里掏出一封信：“这是他们送过来要赎金的信，最后面写着……”
<br/>　　“谁呀？”
<br/>　　莫鸣米伸出手时，下意识问了一声。
<br/>　　“蛤蟆寨蛤蟆大王。”
<br/>　　没人察觉到，听到这个名字后，莫鸣米眼底一道寒意闪过。
<br/>　　莫鸣米没有看信，而是倒扣在桌上，道：“我知道了，你们先回去吧。”
<br/>　　“莫将军，那我们大哥……”
<br/>　　“我已经知道了，自然会处理。”莫鸣米道：“对方不是要赎金么，你们如果着急，可以先把银子送去。”
<br/>　　周差海三人面面相觑，惺惺离开。
<br/>　　等三人离开后，莫鸣米恍然如一头睡狮惊醒，一把抓住搁在桌上的信。
<br/>　　脸上神色阴晴变换，双目圆睁爆射着寒光，心底只有一个想法。
<br/>　　他，来了！
<br/>　　从程大雷离开长安，莫鸣米就得到了消息，毕竟，在朝廷那里，他只是琴川关的副将，琴川关名义上的守将是程大雷。相府派出重金悬赏程大雷的人头，为的就是不让程大雷活着抵达琴川关。
<br/>　　可千防万防，他还是来了。算他聪明，没有拿着官凭来琴川关就职，毕竟，琴川关的兵丁都是周差海原先的当强盗时的手下。但他却拉了一捎人马，又做起了强盗。
<br/>　　莫鸣米心底对程大雷是有些惧意的，对方连战神杨龙停都被他杀了，又在草原上搞出那么大乱子，还能全身而退。对上这样的魔头，莫鸣米不得不慎重。
<br/>　　因此，他才不愿轻易出兵。
<br/>　　这时候一军师模样的人走进大厅，莫鸣米将信递给他，开口只说了三个字。
<br/>　　“他来了。”
<br/>　　名叫严笛的军师缓缓将信读完，半晌长叹一声：”来者不善呐。”
<br/>　　“我派人去查过，占据蛇岛的那伙人就是他们，我在岛上东西现在也应该落到他们手中了。”莫鸣米道：“对方人数虽少，但还是小心为妙。”
<br/>　　严笛沉思片刻，道：“不如我们传下令去，谁如果能平了这伙人，就将云中龙的底盘给他，我们也可以借此看看这伙人的虚实。”
<br/>　　莫鸣米除了自己的亲信，还控制着十八股势力，云中龙就是其中之一。严笛这条计策，不仅能试探出程大雷的虚实，也可以让双方互相消耗。
<br/>　　上位者，绝不能让手下的力量无限制膨胀，领导，有时候就是控制。
<br/>　　莫鸣米的军令很快便传遍大河沿岸，各管一地的十八家势力都得到了消息。
<br/>　　此时此刻，程大雷也在紧张的准备着，本来他是打算稳扎稳打，可张肥和高飞豹的行动让他的进程不得不加快。
<br/>　　系统已经奖励了二级山寨建设图纸，程大雷也放弃初级山寨的翻修，直接开二级本，只要能将点将台开出来，程大雷就有资格和莫鸣米硬碰硬。
<br/>　　当然，在这之前，程大雷还有一项事情要做。
<br/>　　“不顾集体，个人英雄主义爆发，这两个人就是你们的下场。”
<br/>　　徐神机慷慨激昂，张肥和高飞豹立在众人眼前，有些灰头土脸。
<br/>　　“从明天开始，每天早上十里跑操。”
<br/>　　山寨建设要做，精神建设也要做，来到琴川后，程大雷一直疏忽了这件事。现在看来，组织架构，赏罚纪律，精神荣誉这些都得做起来。
<br/>　　没有这些，队伍永远都是一群乌合之众，而乌合之众，则意味着更多的伤亡。
<br/>　　
<br/>(本章完)
