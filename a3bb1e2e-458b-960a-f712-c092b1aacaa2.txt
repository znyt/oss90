　　中年男子容貌好似钟馗，满脸乱髯胡须，背负长剑，给人的第一印象就是邋遢，浑身没有任何特别的气质，倒有一丝江湖人的洒脱。
<br/>　　慕清霄依旧盘坐于木板上，对来者仿若未觉，平心静神，宛如老僧入定，浑身不漏一丝的气息。
<br/>　　“现在的年轻人真是不爱惜自己的性命，镇里的客栈不住，偏要住这鬼窝。”
<br/>　　见慕清霄一袭白衫，腰佩玉饰，面如冠玉，盼顾生辉，根本不似普通人，反倒是像世家公子出游。
<br/>　　穿着不菲，也不像是没钱的样子，居然住这种地方，难道有特殊癖好不成？
<br/>　　在他眼中，世家子弟中，哪一个不是嚣张跋扈的个性，居然愿意住在这种脏乱阴森的地方。
<br/>　　微微摇头，将僧舍大门关上，找了个干净的地方盘坐而下，将背负的长剑与包裹放在一旁。
<br/>　　取出几块烙饼和一壶酒水，胡吃海塞起来。
<br/>　　片刻后，中年男子见慕清霄依旧如磐石一样，忍不住开口问道：“小兄弟，在下燕赤霞，敢问尊姓大名。”
<br/>　　慕清霄剑眉一挑，渐渐睁开双目，其中光泽流转，瞥了眼燕赤霞。
<br/>　　“难道，你师傅没告诉过你，在别人在修炼的时候，出声干扰，是一种很不礼貌的行为吗？”
<br/>　　两者目光相对，燕赤霞猛的一颤，迅速移开目光，双手握住眼睛，眼角狠狠的抽搐了一下。
<br/>　　刚才目光相对的一瞬间，他的双眼，仿佛被一柄利剑刺了一下似得，疼的眼泪都快出来。
<br/>　　眼睛虽刺痛，却不足以令他叫出声来，但心中却早已掀起一片惊涛骇浪。
<br/>　　原本，他以为后者只是一位不知天高地厚的世家子弟，却没想到有眼不识泰山。
<br/>　　眼前的青年，哪里是什么世家子弟，修为简直恐怖如斯，光凭那一眼，两人的差距就体现了出来。
<br/>　　“多谢前辈手下留情，晚辈燕赤霞，不知前辈在此，多有得罪。”
<br/>　　回过神来，燕赤霞连忙道歉，面前的青年气息收敛自如，一举一动都透露着某种意境，恐怕连他的恩师都有所不如。
<br/>　　更何况，他确实理亏在先，问也不问就闯入僧舍，而且自顾自的喝酒饮食。
<br/>　　他也是修士，非常清楚刚才的行为是何等的白痴，修士修炼中受到干扰，绝对是禁忌。
<br/>　　轻则走火入魔，重则生死道消！
<br/>　　闻言，慕清霄挥了挥手，语气平静道：“道歉就不必了，对面有间僧舍，你去隔壁吧。”
<br/>　　“是，晚辈先行告辞。”
<br/>　　恭恭敬敬的令起包裹与长剑，退出僧舍并将大门关上，燕赤霞依旧是一脸懵逼。
<br/>　　他修习的时日不短，自然知道人外有人，天外有天的道理。
<br/>　　世界上不仅有山野精怪，孤魂野鬼，更隐藏着强大无比的修士，而他的师傅就是其中一员。
<br/>　　但僧舍中的青年，给他一种极其恐怖的感觉。
<br/>　　完全看不出一丝端倪，本似普通人，但只是一道目光就刺痛他的双眼，这种恐怖的实力，简直匪夷所思。
<br/>　　刚才的事情，告诉了他一个道理，千万不要以貌取人。
<br/>　　看似普通的青年，可能是妖，可能是鬼，更可能是驻颜有术的强大修士。
<br/>　　见关闭的僧舍大门，慕清霄随手一卷，四面墙壁上的长明灯就诡异的亮起光芒，将整个僧舍都照耀的灯火通明。
<br/>　　望着紧闭的木门，慕清霄闭上双眼，瞬间进入修炼状态。
<br/>　　燕赤霞的到来，对他来说根本没有任何意义。
<br/>　　对于宁采臣那种手无缚鸡之力的书生来说，燕赤霞是个不错的保镖，但对他来说，却连蝼蚁都不如。
<br/>　　攀关系，简直是国际玩笑，一个筑基九重的修士，也值得他去攀关系？
<br/>　　至于结交，他现在可没有时间浪费在这种事情上面，倒不如用在炼化焚寂与玉衡上面。
<br/>　　如此一来，他的实力提升也会迅速许多。

<br/>　　...
<br/>　　深夜以至，皎洁的月光被黑黝黝的云层遮掩住，黑幕笼罩大地。
<br/>　　兰若寺外，阴风四起，枝叶杂草响起窸窸窣窣的声音，旋即一双白嫩的双手拨开灌木丛，来到兰若寺门口。
<br/>　　来者是一名青年，身穿白衫直缀，负笈而来，头戴儒巾，面容清秀，生。
<br/>　　青年书生看了眼阴风阵阵，布满灰尘与蜘蛛网的兰若寺，心中有些凉飕飕的。
<br/>　　可当看到两座僧舍中都有光亮时，目光一亮，显然是有人住在这里，当下心中一喜，抹去额头上的细汗，负笈走上前去。
<br/>　　“咚咚...”
<br/>　　阴森森的夜晚，响亮的敲门声，僧舍木门打开，映入眼帘的是一名满脸乱髯胡须的中年，看似凶神恶煞，将青年吓了一跳。
<br/>　　“在下宁采臣，正想找个落脚地，不知道这里方不方便。”
<br/>　　瞥了眼宁采臣，燕赤霞不屑的瞥了瞥嘴，道：“小子，难道你想死啊，居然来这种鬼窝。”
<br/>　　虽然刚才看走了眼，但他可不相信，面前的细皮嫩肉的书生是什么高人前辈，当下也没给他好脸色看。
<br/>　　“鬼窝？这世上哪有什么鬼怪，都是你们这些人胡诌出来的。”
<br/>　　瞥了瞥嘴，宁采臣满脸不屑，当下就要去隔壁的僧舍，却急忙被燕赤霞拦了下来。
<br/>　　“小子，你想干什么？”
<br/>　　闻言，宁采臣道：“你不让我住这间，总不能赶我走吧，我去别的僧舍还不行吗？”
<br/>　　“等等......这间僧舍足够大，你就住在里面吧，千万不要去隔壁，知道了吗？”
<br/>　　怪异的瞥了眼燕赤霞，又扫了一眼隔壁的僧舍，宁采臣微微点头，才走进眼前的僧舍中，嘴中却道：“莫名其妙。”
<br/>　　不再理会宁采臣，燕赤霞取下背负的长剑，看了眼黑漆漆的树林，目光凝重，回头看了眼僧舍，缓缓走进其中。
<br/>　　并不是他要拦着宁采臣，而是不得不拦。
<br/>　　僧舍中的前辈，刚才已经被打搅一次，若是在打搅一次，换做任何人恐怕都会生气。
<br/>　　他只是不想被宁采臣这种无知的毛头小子连累而已。
<br/>　　夜色渐暗，阴风呼啸呜咽，残旧的寺庙屹立不倒，燕赤霞手握长剑，消失在视野的尽头。
