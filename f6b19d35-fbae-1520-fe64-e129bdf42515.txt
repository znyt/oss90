　　“咳……应龙哥哥，我……恐怕不……不能陪你了，咳……”
<br/>　　女魃一身血红的躺在应龙怀里，口中鲜血直流，怎么也止不住。她断断续续地开口说道。
<br/>　　“不！怎么会这样？”
<br/>　　应龙满眼泪水，满脸的不可置信，刚刚还好好的女魃怎么转瞬间就成了这个样子。
<br/>　　“你不必悲伤，咳……就算没有这件事，我也活不了多久了。”
<br/>　　女魃伸出一只手想要抚摸应龙的脸庞，只是原先那鲜活粉嫰的巧手如今早已变样，变成一只血肉干枯，宛若枯柴似的老手。她的手刚升到一半，就停住了，想要重新放回去。可应龙却急忙拉住那只手，放在自己的脸庞上，满眼泪水的看着女魃。
<br/>　　他懂得女魃的意思，女魃自从之前被上古神兽“犼”的三魄夺舍融合后，她便无时无刻不在承受着煎熬，要不是有着一点神识，她的身体都不受她控制了，她表面上的笑都是她装出来，她不想让人看不起她。如今就连最后那一点神识都快要消散了，所以说，就算没有这突如其来的偷袭，女魃也活不了多久了。
<br/>　　女魃的手伸到一半，却又想要放回去，却是因为她看到了自己手的模样，不想吓到应龙，而应龙却没有在意这些，将她那干枯的手放到自己脸上。
<br/>　　“咳……”女魃的脸上突然涌上一股潮红，她好像有些害羞，看着应龙的脸庞，她喃喃自语道：“应龙哥哥，还记得你……你之前说过的话吗？咳……对不起，你一定要……好好……好好活……着……”
<br/>　　声音到这里就断了，女魃的身体快速变得干枯起来，连之前的红发，却突然燃烧起来，应龙突然不知从何处取出一大块千年玄冰，将之炼成一玄冰棺材，之后轻轻地将女魃送入其中，看着棺中的女魃，应龙脑海中浮现出一副画面：
<br/>　　画面中，女魃靠在应龙怀中，应龙坐在一山头巨石上，两人就这样温馨的看着红日落山，这时应龙说道：“女魃，等打完这场战争，我就和你隐居山林，过着平凡人的生活，一生一世……”
<br/>　　“好啊，应龙哥哥，你去哪，我就去哪……”
<br/>　　“嗯？”
<br/>　　“……”
<br/>　　“应龙哥哥，我一定会等你回来的……”
<br/>　　……
<br/>　　如今，女魃离去了，后人记忆中只有那个善良的美丽女子，在黄昏中，仰望北方，那是她义父征战的方向，还有她心爱的人，在血战沙场。她低声吟唱起一首忧伤的歌谣，歌声渺渺，思绪纷飞，歌声渺渺，带着思念，歌声渺渺……
<br/>　　脑海中那一幕幕，现在清晰浮现在心头，最后定格在“应龙哥哥，我一定会等你回来的”那一句话上，应龙强忍着悲伤，将冰棺收了起来，他轻声低语道：“我也一定会等你回来的。”
<br/>　　应龙满目嗜血地盯着远处那正在大战的身影，他知道现在是什么地方，也知道自己该做什么，但他现在不想去想那些，他只想好好发泄一番。
<br/>　　他心中的悲愤，只有用鲜血才能洗刷！
<br/>　　“昂——昂——昂——”
<br/>　　应龙仰天怒吼，吼声中，带着深深的悲痛，淡淡的思念，以及刺骨的杀意……
<br/>　　“杀！”
<br/>　　应龙怒吼一声，手中不知何时出现了一柄暗金色龙枪，全身爆发出恐怖的气势，冲进九黎一方阵营中，大杀特杀。
<br/>　　此刻的战场中，早已混乱无比，没有多少人注意到应龙这边的情况，况且，就算注意到了，他们还能说些什么，战场，早晚会死人的。
<br/>　　战场中，再次归来的蚩尤与公孙轩辕大战在一起，其余众神仙也都各自大战起来。而这场混战中最引人瞩目的也就是轩辕与蚩尤的大战，广成子与刑天大战，孔宣与数位大巫大战。
<br/>　　战场上，孔宣独自一人面对九位大巫，他丝毫没有胆怯，面对这些大巫的拼杀，他身后五色神光闪烁不断，每一道神光闪过，便会有一位大巫被镇压，赤、黄、青、白、黑五道神光美轮美奂，在虚空中绽放，犹如孔雀开屏，让人忍不住沉醉在其中，心甘情愿地被镇压。
<br/>　　刷刷刷！
<br/>　　一连串的冲刷，神光一过，无可匹敌，不知镇压了多少大巫。
<br/>　　比起孔宣战斗的优美，而广成子与刑天的战斗，只能算是太野蛮了。
<br/>　　刑天手持一柄战斧，化身出数百万丈，广成子也施展法天象地神通，化身数百万丈，拿着一枚大印，狠狠地砸向刑天。
<br/>　　大印化出一座神山，正是不周山的模样，向着刑天狠狠地镇压下来，刑天手持巨斧，使劲砍伐神山。
<br/>　　没有什么敌人是一印解决不了的，如若有，那就两印！
<br/>　　砸！
<br/>　　没有什么技巧，就是使劲地砸，不断地砸，砸死刑天。这场战争刑天从头到尾，都很憋屈。
<br/>　　战场中心，蚩尤突然不知从何处取出一枚黑色战旗，向着战场上摇动起来。
<br/>　　“杀啊——”
<br/>　　“冲啊——”
<br/>　　只见片刻时间不到，九黎一方的勇士们便个个都像是打了鸡血一般，满脸亢奋，眼中满是嗜血的冲动，不怕死一般地向着轩辕一方冲杀过来。而轩辕一方的勇士们，却好似遭受到什么打击似的，完全摸不清方向，耳中满是杀伐之音，敌人到处都是，但就是攻击不到，像是没头的苍蝇一般，到处乱飞乱跑。
<br/>　　这是蚩尤利用上古时大战剩下的残破戮妖幡所炼制成道无上战争魔兵——蚩尤旗！
<br/>　　蚩尤旗，上古魔器，不仅能够增幅己方战力，还能祸乱地方，迷惑对手，让人无处可打。
<br/>　　“怎么办？诸位，如今该如何是好？蚩尤不知从何处弄来的魔兵，威力竟如此诡异。”
<br/>　　战车上，公孙轩辕看着地方威势大涨，而己方却已经自乱阵脚，摸不清对方虚实，这仗该如何打，向着身边的众位仙神问道。
<br/>　　“人皇莫急，我有办法破了蚩尤的魔兵。”
<br/>　　这时，一道清脆悦耳地声音在人群中响起，至少在轩辕耳中，这声音清脆悦耳。他顺眼看去，人群中走出一个一身白色仙裙，头扎碧玉仙簪，面色红润，肌若白雪的仙子，这人就好似从画中走了出来，人间哪有如此美丽的女子。
<br/>　　这正是九天玄女。
<br/>　　“哦？玄女，你有何办法助我破了蚩尤魔器？”公孙轩辕挑了挑眉头，说道。
<br/>　　“人皇，你看——”说着，九天玄女手中多了一个战鼓，此鼓数丈大小，散发出一股厚重，凶猛、彪悍的气息，一看便知不是凡品，她一边将鼓递给轩辕，一边解释道：
<br/>　　“此鼓乃是我用神兽夔牛之皮所炼，夔牛出，风雷变。人皇可敲响此鼓，不仅能够鼓舞我方士气，更能慑服地方猛兽坐骑。”
<br/>　　“好，此次多谢玄女。”
<br/>　　轩辕大喜，郑重感谢道。接着好似想到了什么，又说道：“我记得先代人主在部落中留下一根雷神腿骨，若是取来敲打此鼓，配合风雷属性，那威力岂不是更甚！”
<br/>　　轩辕大喜，派人前往部落取来雷神腿骨，在战场上架起高台，敲响夔牛战鼓。
<br/>　　人皇公孙轩辕登上高台，亲自擂起战鼓：
<br/>　　“咚——”
<br/>　　“咚——”
<br/>　　“咚——”
