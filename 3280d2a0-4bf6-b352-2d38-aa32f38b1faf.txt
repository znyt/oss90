<br/>　　“这股力量。”
<br/>　　感受到这北邙宫的变化，楚风眠的目光，都是微微变化起来，有些惊讶，也有些怀念，这怀念之色，一瞬息之间便是闪过。
<br/>　　夜皇所看见的，只有楚风眠眼神中的惊讶。

<br/>　　“北邙宫中，居然还隐藏有着如此手段！”
<br/>　　“夜皇不惜动用如此底蕴，看来也是被楚风眠逼到了极点了！”
<br/>　　“一尊生死境的人物，居然可以将北邙学院逼迫到如此地步。”
<br/>　　周围的老祖，都在感慨，感觉到北邙宫的气势，他们不少人的心中已经是明白了。

<br/>　　不。
<br/>　　应该说在他们的心中，都从未想象过，楚风眠足以走到今天这一步。
<br/>　　楚风眠，将夜皇都逼迫到了如此地步，斩杀了夜家无数老祖，这一切所做，都已经是足以称得上是惊世骇俗了。
<br/>　　不过可惜，即使这楚风眠在妖孽，在逆天，这样的局面，也没有任何人还觉得，楚风眠还有着任何的机会。
<br/>　　“快逃！”
<br/>　　玄北圣者都是急忙回头，冲着楚风眠大喊了起来。
<br/>　　在这北邙宫中，北邙宫的威压，几乎是无敌的存在，在强大的武者，硬抗这威压，都是极为吃力的事情，更何况仅仅只有生死境境界的楚风眠，又能如何抗衡？
<br/>　　“楚风眠，你的实力的确妖孽，甚至是在整个九域中，都找不出像是你这样妖孽的人物来。”
<br/>　　夜皇看着楚风眠，一字一字的说道。
<br/>　　“不过今日，就是你的死期，你的一切机缘，都留下吧！我会先将你镇压，在慢慢的折磨你的！”
<br/>　　话音一落，夜皇便是猛然动了，他的手心中，直接凝聚出了一道极为庞大的禁制，跟这北邙宫的力量，融为一体，同时向着楚风眠笼罩而去。
<br/>　　这庞大无比的压力，已经是化为了一尊七层宝塔，被这七层宝塔死死的锁定，甚至是没有任何人可以从中逃出。
<br/>　　在这一招之下，就算是天命之境的武者，都唯有被镇压的下场。

<br/>　　楚风眠面对这压力，脸色平静，只是口中，突然自言自语了一声。
<br/>　　这声音极小，甚至是没有任何人听到。
<br/>　　只是这北邙宫之中，一个意念，突然之间好像是听到了楚风眠的声音一般。
<br/>　　“轰隆！”
<br/>　　就在这瞬息之间，一道金色灵剑，却是从这北邙宫的深处，突然飞了出来。

<br/>　　“北邙宫的器灵？”

<br/>　　夜皇，还有另外风皇，阳皇等人，看到这金色灵剑出现的一刻，都是愣住了。
<br/>　　别人不知道，可是他们却是清楚，这金色灵剑，正是北邙宫的器灵。
<br/>　　一直以来这北邙宫的器灵，都是隐藏在北邙宫的深处，从未出现过，可现在，居然是出现了。

<br/>　　这金色灵剑，来到楚风眠的面前，突然停了下来，在楚风眠的脑海之中，响起了极为稚嫩的声音。
