　　　　上官昊残碎的尸体中，残留的血气少的可怜。
<br/>　　几乎只是一瞬间，便被苏莫吞噬的一干二净。
<br/>　　苏莫稍稍感受了一番，他的血脉力量的确强了一些，大日神体的威力，虽然没有强太多，但也增加了一两成左右。
<br/>　　这顿时让苏莫暗喜，大日神体的威力增强，也算是让他的实力有所增强了。
<br/>　　轰！
<br/>　　就在此刻，一道恐怖的惨白色刀气，从天穹之上劈下，狠狠的斩向苏莫的脑袋。
<br/>　　浩大的刀威如同高大大岳，无边的的锋芒还未降临，就让得苏莫有种身体即将被撕裂的感觉。
<br/>　　是殷棕杀过来了，八柄本命灵剑，也最多缠住其两个呼吸左右的时间而已。
<br/>　　“杀！”殷棕的怒喝声如同一道惊雷，他心中的怒火和杀意，简直要爆炸了。
<br/>　　这么长时间了，他不仅没能杀死苏莫，反而让对方在自己的眼皮子底下，将上官昊斩杀了。
<br/>　　上官昊的死活，他虽然不在意，但发生此种情况，让他感到无比的耻辱。
<br/>　　苏莫见此不避不退，再次催动了大日领域，恐怖的刀气进入领域之中，威势不断的被消减。
<br/>　　而后，苏莫一剑逆斩而上，便再次破灭了这一刀。
<br/>　　“可恶！”
<br/>　　殷棕气的连连怒吼，随即他的背后虚影升腾，一柄硕大的战刀武魂升腾而起。
<br/>　　这柄战刀武魂，通天银光熠熠，乃是天级三阶武魂。
<br/>　　在苍穹世界，器武魂和兽武魂乃是最长见的武魂，如同幽冥魔瞳武魂、大日武魂这类特殊的武魂，极其的稀少和罕见。
<br/>　　当然，像苏莫这种逆天武魂，估计苍穹大陆的历史上，都没有出现过。
<br/>　　咻咻咻！！
<br/>　　动用了武魂之后，殷棕气势增长了不少，手中长刀不断的斩击，刀影翻飞，铺天盖地的向苏莫劈杀而去。
<br/>　　“战！”苏莫丝毫无惧，全力迎击，同时他心神召回其余的本命灵剑，不断的四面八方向殷棕攻击。
<br/>　　一时间，两人战的难分难解，恐怖的波动如同惊涛骇浪，不断的向四方涌动。
<br/>　　两人的实力相差无几，战了数十招谁也奈何不了谁，根本无法分出胜负，更别说斩杀对方了。
<br/>　　当然，苏莫还有大三宝术这个超强底牌，但他知道就算动用了大三宝术，也很难斩杀对方。
<br/>　　除非抽取体内六成及以上的精气神，但这代价太大了。
<br/>　　而且，苏莫相信，对方作为准帝强者，估计也会眸中强大的秘术。
<br/>　　但对方没动用，估计也是有同样的顾忌。
<br/>　　啊！
<br/>　　就在此时，远处一声惨叫声响起，只见二魔老的身形如同一颗炮弹，倒飞向远处，其口中鲜血狂喷，染红了苍穹。
<br/>　　二魔老，赫然是被东海龙王击伤了。
<br/>　　“走！”
<br/>　　这时，大魔老的口中发出一声尖锐的厉喝，随即身影闪烁，向二魔老倒飞的方向急速离去。
<br/>　　殷棕、北堂平华、阳鼎天三人见此，顿时纷纷舍弃了对手，同样急速逃离。
<br/>　　即便殷棕无比的不甘，但还是果断离去了，因为他清楚的知道，再继续战斗下去，也不会有什么结果。
<br/>　　“杀！”海明江满面杀机，还在追杀阳鼎天。
<br/>　　“海阁主，穷寇莫追！”苏莫见此，顿时向对方呼喊了一声。
<br/>　　随即，海明江便无奈的停下了身形。
<br/>　　望着急速消失在视线中的殷宗等人，苏莫无奈的叹了口气，这一次真的是白跑了一趟。
<br/>　　不过，要不了多久的时间，他必杀这些人。
<br/>　　“龙王前辈！今日多谢出手相助！”苏莫来到了东海龙王庞大的蛟躯身前，抱拳沉声道谢。
<br/>　　唰！
<br/>　　东海龙王瞬间化为人形，古板的脸上露出了一丝淡笑。
<br/>　　“苏莫小友不必客气！”
<br/>　　东海龙王摇了摇头，道：“本王一生从未服人，但今日，对苏莫小友却是心服口服！”

<br/>　　“小友莫要谦虚，你现在才武魂境三重巅峰修为而已，待你达到武皇境五重修为，便可无敌于天下。”东海龙王沉声道。
<br/>　　东海龙王心中有些唏嘘，他一直在东海龙宫，也早已听闻苏莫之名，虽然知道苏莫的天赋冠绝天下。
<br/>　　但今日一见，还是不免心中震撼。
<br/>　　这或许是老天爷，送给他们，乃至整个苍穹世界的一个机会。
<br/>　　如此时候，在大劫将至之前，苍穹世界诞生出如此逆天的天才，实属苍穹世界之幸。
<br/>　　而且这个天才心志不错，最主要的是和他东海蛟龙一族较好。

<br/>　　苏莫笑了笑，随即看了一眼来到身边的帝释，问道：“对了，不知岳父和龙王前辈为何会在此地？”

<br/>　　“苏莫，我们去你苍穹神宫吧，有些事本王要与你详细商谈！”东海龙王沉声说道。
<br/>　　“好！”苏莫点了点头，他对于东海龙王和蛟龙一族，还是比较信任的，毕竟对方出手救过他多次。

<br/>　　随即，苏莫向几人摆了摆手，意念一动，虚界神图中的无数武者，瞬间出现在了下方的大地上，漫山遍野，黑压压的一大片。

<br/>　　“还有我，我一定要加入苍穹神宫，通过考核！”
<br/>　　“我要成为苍穹神宫弟子，为苍穹神宫效力，以报苏宫主救命之恩！”
<br/>　　一声声高呼声随之响彻而起，浩荡不休，声音中充满了坚定与诚恳。
