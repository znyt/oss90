　　“什么弥天大祸？”苏莫皱眉问道。
<br/>　　“你杀了如此多的人，不仅太史世家派系不会放过你，估计学府也会重罚你！”木璃说道。
<br/>　　“重罚？”
<br/>　　苏莫闻言略作沉吟，道：“我可没有违反学府规矩，他们可都是自愿与我一战！”
<br/>　　苏莫没有什么担心，他虽然杀了很多人，但都是在规矩之内，并没有逾越。
<br/>　　虽说杀得人数有些夸张了，但学府也不能给他强行定罪吧！
<br/>　　再说了，他现在有火元圣师这张虎皮，还有这么多派系想拉拢他，应该没有什么大碍！
<br/>　　“你好自为之吧！”木璃导师淡淡的说了一句，便不再多言，飞身离去。
<br/>　　苏莫无奈的摇了摇头，随即转身走回洞府。
<br/>　　随即，他一边凝练修为，一边慢慢等待体内血脉力量的融合。
<br/>　　目前，他身具天玄武体、大日神体、琉璃宝体、金刚之体，四种特殊体质，无论是天赋还是战力，都又稍稍的提升了一个档次。
<br/>　　就在苏莫安心修炼之时，整个神武学府，也是沸腾一片。
<br/>　　七八十名弟子被杀，这绝对是轰动的大事，连学府的高层，都全部被惊动了。
<br/>　　而整个太史世家派系，更是动作连连，欲将苏莫置于死地。
<br/>　　一时之间，学府中风气云涌。
<br/>　　苏莫安静的在洞府修炼，没过多久，聂乾和韩盈回来了。
<br/>　　“苏兄，大事不好了！”两人急冲冲的走进苏莫的洞府，满脸焦急之色。
<br/>　　“怎么了？”苏莫结束了修炼，疑惑的问道。
<br/>　　“太史世家派系，正在向学府执法堂施压，要将你处死！”聂乾满脸担忧的说道。
<br/>　　“是啊！太史世家派系圣子太史爵，更是扬言，若是执法堂不处死你，他就亲自动手，将你镇杀！”韩盈道。
<br/>　　“是吗？”苏莫闻言眉头一皱，这事怕是有些麻烦了。
<br/>　　“苏兄，要不……你跑吧！离开学府！”聂乾迟疑的说道。
<br/>　　“跑？”
<br/>　　苏莫闻言摇了摇头，估计他现在已经被太史世家派系的人严密监测住了，能往哪里跑！
<br/>　　虽说他现在有隐身战魂，但并不能瞒过那些武尊境的武者。
<br/>　　一旦被发现，必死无疑。
<br/>　　“两位莫要担心，车到山前必有路！”苏莫叹了口气，心中却是有些焦急了起来。
<br/>　　其它派系的人，不是都要拉拢他吗？
<br/>　　为何这么久了，都不来了！
<br/>　　“哎！”聂乾和韩盈亦是叹了口气，两人的心中也是认为苏莫这次玩大了。
<br/>　　杀了这么多太史世家派系的人，人家不疯狂才是怪事了呢！
<br/>　　现在没有直接杀过来，估计也是看在火元圣师的面子上。
<br/>　　但是，火元圣师，会帮助苏莫吗？
<br/>　　两人心中充满了担忧。
<br/>　　“两位，你们先回去吧！放心，我不会有事的！”苏莫向两人说道。
<br/>　　“嗯！”聂乾和韩盈点了点头，现在他们也帮不上什么忙，只能静等事态的发展。
<br/>　　随即，聂乾和韩盈各自返回了自己的洞府。
<br/>　　苏莫则是在洞府中，静等了起来，等待其它派系的人前来。
<br/>　　不管是哪个派系，他都可以加入。
<br/>　　若是没有人来，那他只能厚着脸皮到山上找火元圣师了。
<br/>　　时间缓缓流逝，约莫过去了一刻钟的时间，一道人影无声的落在了苏莫的洞府之前。
<br/>　　苏莫立刻抬眼望了过去，便见到来人是一名身穿蟒袍的青年，身材匀称，俊逸不凡，最主要的是此人的气质，无形中给人一种高贵之感。
<br/>　　苏莫见此双眉一挑，此人绝对不是火元峰上的人，居然直接降临在了他的洞府前，难道是已经知会了火元圣师！
<br/>　　不然的话，没有火元圣师的同意，任何人都不能上山的！
<br/>　　“苏兄，可以请我进去坐坐吗？”蟒袍青年面带和煦的微笑，轻声说道。
<br/>　　“请！”苏莫点了点头，立刻起身，将对方邀进了洞府之中。
<br/>　　洞府中有石桌和石凳，而后，两人在桌前相对而坐。
<br/>　　“苏兄，久闻你的大名，今日一见，果然人中之龙，不同凡响！”蟒袍青年向苏莫微微抱拳，朗声赞道。
<br/>　　“阁下过奖了，不知阁下是？”苏莫摇头，随即疑惑的问道。
<br/>　　蟒袍青年闻言微微一怔，显然有些意外苏莫居然不认识他。
<br/>　　无奈一笑，蟒袍青年自我介绍道：“在下古天意，皇祖圣朝派系的圣子。”

<br/>　　“圣子谦虚了！”苏莫轻笑一声，眸光暗暗打量了一番对方，此人的修为他看不清，但此人绝对不简单。
<br/>　　不说修为实力，此人这为人处世的本事，就圆滑无比，无形中就能让人心生好感。
<br/>　　而且，此人没有任何作为圣子的那种高高在上的傲气，心机怕是不简单。

<br/>　　“知道！”
<br/>　　苏莫点了点头，随即问道：“那太史爵有这么大的能量？”
<br/>　　苏莫有些不理解，所谓的太史世家派系，毕竟只是一群弟子，并不是真正的太史世家，一名圣子有这么大的能量？能向学府执法堂施压？

<br/>　　苏莫点了点了，心中恍然，若是比势力与关系的话，这些各大派系的圣子，比神子都要厉害的多。
<br/>　　毕竟，神子虽然天赋逆天，在学府地位崇高，但并没有加入任何的派系。
<br/>　　这时，古天意叹了口气，告诫道：“苏兄啊，若是我所料不错的话，执法堂定然抗不住压力，会对你动手！”
