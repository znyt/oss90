　　小鬼的盘算打的很好，可他们只算计自己，却忘了算计别人。
<br/>　　能安排给他们的火车车厢，怎么可能没有任何的机关埋伏，所以就在他们议定接下来要做什么的时候，守在外面的情报科人员便已经通过遍布车厢里的铜管将他们的话听了个一丝不差。
<br/>　　李承乾从来没想到这些倭人竟然猥琐至此，还特么把主意打到了老头子的身上。
<br/>　　虽然说以现在老头子身边的力量来说，并不怎么乎那些倭人，可万里还有个一呢，如果真的出了岔子那可怎么办？
<br/>　　“哥，依我看这些倭人就是养不熟的狗，不如杀了算了。”李佑，从打倭国回来之后，这家伙疯狂的玩了很长时间，这次到皇宫里面来见李承乾，结果便遇到了这样的事情，让这位李五爷尴尬不已，总觉得像是自己没有把事情办好一样。
<br/>　　“陛下，老臣请命，望陛下起兵伐之！”候君集这老家伙典型的老杀才一个，听到打仗立刻红了眼睛。
<br/>　　他本和李佑是一条绳子上的蚱蜢，倭国的事情当初全部由他们两个负责，秦吉胜敢打李二的主意，和他未没能脱得开关系。
<br/>　　所以这老货的建议与李佑如出一辙，同样想着去倭岛大开杀戒，好好让那些倭人长长记性，让他们知道什么才叫宗主国。
<br/>　　李承乾原本还对此事有些犹豫，但现在看来，倭人似乎真的是另有打算，颇有要自立门户的意思，如果这样长此以往，就算老头子那边不出问题，将来也会在其它地方出问题。
<br/>　　想到这里，李承乾摇了摇头，看了一眼李佑和候君集两人：“这件事情你们两个先不要参与了，朕对你们还有其他安排。”
<br/>　　“哥，难道这事儿就这么算了？”李佑现在已经不是当初的孩子，但与李承乾的关系却并不差，听了他话之后不疑有他，只是觉得理解不上去。
<br/>　　好在李承乾也不想瞒着他与候君集，招手示意他稍安勿躁：“这次我打算另外调一批人过去，在倭国打仗我们的人不行，根本不能适应那里的环境，就算是打的再狠，到时候他们往林子里一钻，你们也是拿他们没有办法。”
<br/>　　“那怎么办？就这么放任不管？”
<br/>　　李承乾哼了一声，敲了敲桌子说道：“你别忘了咱们还有靺鞨人，他们有一部分就生活在丛林里面，只要给他们一些简单的装备，搞定那些倭人还不是跟玩儿一样。”
<br/>　　“靺鞨人？黑水靺鞨？”候君集愣了一下，但转念一想似乎李承乾说的似乎并没有错。
<br/>　　靺鞨人生活在大兴安岭里面，可以说就是生活在丛林里面，单就生活环境来说与倭国并没有任何区别，如果给靺鞨人一些精良的装备似乎还真是用来处理倭人最好的手段。
<br/>　　另外还有一点就是靺鞨人并不好控制，调一批去倭国也正好可以保证大唐境内的稳定。
<br/>　　想到这里，候君集点了点头不再说什么，也不再强调自己要去荡平倭岛之类的说法。
<br/>　　黑水靺鞨眼下是控制在李绩手里的，那家伙并不好对付，从他手里抢‘生意’弄不好会被那老小子报复。
<br/>　　李佑同候君集息火了，眼睛眨了眨对李承乾问道：“那，哥，你想让我干啥？”
<br/>　　“去西域，戍边！”李承乾看着李佑，不急不缓的说道：“朕打算调换一下防区，正好你小子也在长安休息这么长时间了，去西域驻扎两年没有问题吧？”
<br/>　　“呃……”李佑没想到竟然是让自己去西域，闻言愣了一下，想了半天才说道：“去西域是没有问题，不过，真的两年就能回来？”
<br/>　　“对，就两年，两年之后我安排别人去替你。”李承乾正色点头说道。
<br/>　　“那尉迟那小子怎么办？他不是在西域么？”
<br/>　　李承乾叹了口气：“朕会把他先调回来，他和仁贵已经在外面驻扎十余年，也该让他们回来了，再不调他们回来，估计就要有意见了。”
<br/>　　“这样啊！”李佑苦着脸咂咂嘴似乎有些不情愿，不过看李承乾好像已经打定了主意，他也知道自己再说什么都没用，于是只能硬着头皮点点头：“哥，那你可别把我给忘了，两年以后一定记得把我调回来，否则你可别怪我装病给你看。”
<br/>　　“呵呵，放心吧！到时候就算你不想回来，朕也会把你调回来。”李承乾呵呵的笑着说道，末了看了一眼候君集：“候老将军，不知你是什么意见？可愿去西域？”
<br/>　　“老臣这些年身体已经不比当年，若陛下同意，老臣便不去了吧！”候君集也是心眼极多之辈，他可不认为李承乾这样突然换防是真的体谅尉迟宝琳和薛仁贵，所以机智的选择了缩在长安以观察接下来的动静。
<br/>　　李承乾则好像并不意外候君集的选择，见他不想去，便点点头：“这样也好，老将军那就留在京中帮帮朕的忙，这段时间事情太多，朕还真是忙两脚不沾地。”
<br/>　　“是！”候君集知道李承乾说的这些只不过就是一时的客气话，心里并没有当真。
<br/>　　要知道，李世民已经出海去了，李承乾继位之后却一直没有什么大动作。
<br/>　　但显然他不可能一直都任用老皇帝的人，这就是所谓的一朝天子一朝臣，老一辈已经到了该退出历史舞台的时候，一些新皇当初培养起来的新人，正在大批量的出现在朝堂之上，新旧交替的时候，终于开始了。
<br/>　　说来老候还真是很意外李承乾竟然能忍到现在，想当年李世民上台之后，几乎是在月余之内就将朝堂上的大臣换了一遍。虽然这其中有一些人是因为在玄武门时站错了对，但当时换人换的也的确是太过勤快了些。
<br/>　　……
<br/>　　李佑和候君集离开之后，李承乾靠在窗前，望着外面湛蓝的天空，直到半晌之后才缓缓开口：“传令钦陵，让他调十万人出来，朕会在近其给他安排一批装备，告诉他，如果倭岛的海面没有血染十里，他就不用再回来了！”
