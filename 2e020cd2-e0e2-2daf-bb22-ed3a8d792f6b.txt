　　第271章
<br/>　　程大雷将剑拍在船栏上，砰砰响声，震得众人头皮发麻。
<br/>　　“现在劫你们的是蛤蟆岛蛤蟆寨蛤蟆大王，好好想想，你们是要吃馄饨还是要吃板刀面？”
<br/>　　“琴川根本没有山贼……”林掌柜大着胆子说了一句，在触及程大雷的眼神时立刻闭嘴。
<br/>　　“现在我来了，不就有了么。”程大雷道：“看来你是要吃板刀面了，好，我第一个拿你开刀。”
<br/>　　“不，不，我吃馄饨。”
<br/>　　说完，林掌柜迫不及待的跳进河中，接下来，在程大雷的威逼下，船上的伙计一一都跳了下去。
<br/>　　“现在轮到你了？”
<br/>　　程大雷面向西门春长，此刻船上就剩下西门春长和那名叫瓶儿的美妇人。
<br/>　　“我，我不会水啊。”
<br/>　　“那我就请你吃板刀面……”程大雷提着剑过来。
<br/>　　“不，不要。”
<br/>　　西门春长大叫一声，翻身跃下船，迎面只看见滔滔河水，一颗心差点从腔子内跃出来，他当真是不会水。
<br/>　　突然，西门春长发现自己并未坠入水中，而是被半吊在空中，程大雷一只大手如抓小鸡一样将他抓回船上。
<br/>　　“你，你不杀我了？”西门春长。
<br/>　　程大雷摇摇头：“不，我突然想到，你这种人更适合当肉票啊。”
<br/>　　西门春长一愣，只见程大雷大手一挥：“绑了。”
<br/>　　瞬间，二人就被绑成粽子相仿，丢在船舱内。
<br/>　　“大当家，弟兄们已经在岛上做好准备，雄黄一到，就可以行动。”刘悲。
<br/>　　程大雷点点头：“出发！”
<br/>　　船开波浪，一路顺风顺水，很快便接近蛇岛，不，现在应该叫蛤蟆岛。
<br/>　　货船在浅滩靠岸，蛤蟆寨的弟兄都在浅滩等候。程大雷事先已经吩咐过，轻易不让他们进入岛屿深处。岛屿上盘踞的蛇群，毕竟不是人力可以抗衡。
<br/>　　程大雷将雄黄从船上卸下来，早已准备好引火之物，众人将稻草束成丛，裹上雄黄后点燃。
<br/>　　刺鼻的气味泛开，众人脸上都裹着湿布，只见草丛中的毒蛇像是感觉到天灾的讯号一般，纷纷都窜了出来，向四周逃去。
<br/>　　毒蛇的数量不是一只一只，而是一层一层，就像涌起的蛇浪一样，蛇皮摩擦地面发出沙沙的声响。便是这些杀人如麻的悍匪，此时也觉得身上一层层的泛起鸡皮疙瘩。
<br/>　　动作必须快，如果被人知道，尤其是被琴川关的莫鸣米知道，必定派兵来围剿。所以程大雷必须在他们得到消息前，布好山寨的第一道防御。
<br/>　　所以必须快。
<br/>　　程大雷、秦蛮、赵子龙、张肥、高飞豹等这些手快的走在队伍最前方，偶尔有两条慌不择路的毒蛇冲向队伍，也被程大雷一剑斩为两段。
<br/>　　在他们之后，大家手里熏着雄黄烟，驱散草丛中的毒蛇，在之后，则是有人把雄黄撒入土中，以免蛇群卷土重来。
<br/>　　蛇群慌不择路，大多遁入水中，大家走过的地方，可以说寸蛇不生。
<br/>　　一天时间，程大雷差不多也将岛屿的地形查探清楚，整座岛屿大概是一个葫芦形，葫芦底部距离岸边最近，约有五里地，其他位置都被滔滔河水包围着。
<br/>　　最好，大家进入岛屿腹心，眼前出现几座残破的建筑，演武场，聚义厅……这应该就是被莫鸣米遗弃的山寨，此时也长满荒草。
<br/>　　“啊！”
<br/>　　当关鱼推开聚义厅破旧的大门时，忍不住发出一声惊呼。关鱼是怎样性格，天行大，我行二，除了大哥刘悲谁都不服。能让他发出尖叫的又会是怎样情景……
<br/>　　程大雷也走过来，只往里面看了一眼，全身上下瞬间被冷汗湿透，握剑的手瑟瑟发抖。
<br/>　　大门前站了一排人，每个人的表情都不比程大雷好上多少，膝盖发软，似乎随时都会跌倒。
<br/>　　聚义厅内盘着一条大蛇，可将它称作蛇，实在是对其的一种侮辱。
<br/>　　它有十几米长，房梁粗细的身躯，更诡异的是，通体白色，那种白就像是石头被敲破的磷白，或许，它本身已经石化。
<br/>　　此时它盘在聚义厅正中央，熹微的光芒从房顶落在它身上，一颗硕大的头颅，居高临下注视着诸人，不像一头畜生，倒像是一位阅尽沧桑的智者。
<br/>　　“这是龙吧。”
<br/>　　有人发出一声惊呼，某种程度上，这是在场所有人的想法，也包括程大雷的想法。
<br/>　　是呐，这是个一切都很原始的冷兵器时代，在现代社会很多物种都灭绝了，当没说这个时代，就不存在一些古怪都生物。
<br/>　　程大雷心脏砰砰打鼓，他伸出手向身后，道：“香，香呢……”
<br/>　　大家今天还正备着火烛香炮，毕竟搬家还要点鞭炮，何况是今天要安营立寨。徐神机小心翼翼将大香递到程大雷手中，看得出来，他也很紧张。
<br/>　　程大雷将大香点燃，插在地上，口中念念有词。
<br/>　　“蛇王大人，我等兄弟被官府所逼，无家可归，只能落草为寇，日后劫富济贫，替天行道，还望您给腾个地方吧。”
<br/>　　程大雷说罢，带着众弟兄向大蛇拜了三拜。
<br/>　　第一拜，蛇王不动。
<br/>　　第二拜，蛇王不动。
<br/>　　第三拜，蛇王动了。
<br/>　　动如脱弦之箭，以极快的速度向程大雷扑来，程大雷吓得魂飞魄散。
<br/>　　咚！
<br/>　　关键时刻，关鱼将大门关上，硬生生将大蛇堵在里面。
<br/>　　程大雷这才回过神来，发觉脸上布满冷汗。
<br/>　　“斧头，我的斧头呢！”程大雷大吼：“我真是给你脸了，给脸不要脸的东西，今天本大王就宰了你。”
<br/>　　手下面面相觑，心里都是同一个想法：这样的怪物，或者说神物，真的能杀嘛！
<br/>　　程大雷看到这一幕，举着斧头大吼：“大丈夫当行天下，何惧之有，给我……干丫的。”
<br/>　　大门撞开，露出一颗硕大的蛇首，程大雷一马当先迎了上去，使出身无彩凤双飞翼避开大蛇的攻击，同时在空中命令其他人。
<br/>　　“退，都给我退出去！没本事的别上来送死！”
<br/>　　以这大蛇的速度，甩着墙都是屋毁房塌，何况人的血肉之躯。程大雷仗着身子灵便，还能与这大蛇周旋，其他人上来也只是送死。
<br/>(本章完)
