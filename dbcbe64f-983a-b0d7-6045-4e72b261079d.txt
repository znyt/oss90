　　荒圣君的身躯，在这造化之力下，顿时都被全部剿灭。
<br/>　　他的身躯，再也无法从新凝聚，在这瞬间破碎。
<br/>　　楚风眠张开嘴，这一身精血，都被楚风眠吞噬。
<br/>　　“碎命之境的精血！果然强大！这荒圣君，居然足足是得到了十几次的机缘！”
<br/>　　楚风眠吞噬了荒圣君的精血，这精血在楚风眠的身躯之中，便是化为了无数精粹的灵力，充斥着楚风眠的身躯的每一个角落。
<br/>　　一个荒圣君的精血，比起一百名荒古门弟子的精血都要强大的多。
<br/>　　从他的精血之中，楚风眠都能够察觉出来，荒圣君曾经所得到过的无数奇遇。
<br/>　　这些奇遇，纷纷融入到了荒圣君的精血中，成为了他的力量。
<br/>　　不过如今，这一切，也都属于楚风眠了。
<br/>　　“生死境七重！”
<br/>　　几乎的一个瞬间，楚风眠身上的气息便是节节攀升起来，他的境界，也在这一瞬之间，达到了生死境七重的地步。
<br/>　　吞噬了这一个荒圣君，绝对是不亚于得到了任何一座宝藏。
<br/>　　荒圣君的空戒，也被楚风眠所得到，在其中的宝物，堆积如山，从这云羽天宫之中所得到的所有宝藏，现在也从新回到了楚风眠的手中。
<br/>　　“好，有了这云羽天宫的宝藏，也足有是为我超越生死，突破碎命之境多多做着准备！”
<br/>　　楚风眠不动声色间，便是将这空戒给收了起来。
<br/>　　楚风眠现在的境界，虽然还只是生死境七重，但是他已经是在计划这如何突破碎命之境的事情了。
<br/>　　对其他武者来说，其实生死境的每一重境界，想要突破，都是没有个三年五载，无法突破。
<br/>　　但是楚风眠却不同。
<br/>　　他修炼的战龙决，只要是灵力足够，便是没有瓶颈，他现在虽然只是生死境七重的境界。
<br/>　　可楚风眠若是愿意的话，楚风眠现在将云羽天宫之中的宝藏吞噬一部分，他瞬间就可以达到生死境巅峰。
<br/>　　这对楚风眠来说，没有任何的难度，只要是有着足够的灵力，他随时都可以突破。
<br/>　　不过楚风眠却是没有打算这么做，他的宝藏都是遗留下来，在准备突破碎命之境的时候动用的。
<br/>　　楚风眠平常突破虽然容易，可是他的瓶颈，却是比起寻常武者而言，要难上千倍，万倍之多。
<br/>　　他度过生死大劫，便是经历了千辛万苦，这超越生死，踏上生死台阶，更是比起碎命之境还要难上数百倍。
<br/>　　因此楚风眠不得不早早做打算才行。
<br/>　　“又突破了。”
<br/>　　无数武者看着楚风眠突破，如今都已经没有什么稀奇的了，对其他武者来说想要突破一次的千难万难，在楚风眠的面前，却是那么的轻松。
<br/>　　楚风眠刚刚来到天羽神域之中的时候，不过只是一个小角色罢了，谁能想到，居然是楚风眠得到了这云羽天宫的传承。
<br/>　　在场，除了天枢殿弟子之外，在场所有七大宗门弟子，还有刚刚对楚风眠出手过的武者，便是全部都被轰杀，没有任何一人还活着。
<br/>　　“多谢楚兄不杀之恩。”
<br/>　　人群之中，天枢殿的星筠，突然走了出来，对着楚风眠开口道。
<br/>　　七大宗门的弟子，全部都被斩杀，除了他们天枢殿的弟子。
<br/>　　这的确是楚风眠特别开恩，才会放过他们一命。
<br/>　　“无妨，这一切只是还你的恩情，你救了月漓一次，所以这一次我就放过你们天枢殿。”
<br/>　　楚风眠看着星筠，平静的扫过那些天枢殿弟子。
<br/>　　“这恩情，只有一次，下一次若是你们于我为敌，我一样会杀了你们！”
<br/>　　楚风眠的话音虽然平静，但是其中的杀意，依然是足以让人不寒而栗。
<br/>　　要是之前，敢有武者这样对天枢殿弟子说话，众人只会觉得是一个疯子，绝无可能的事情。
<br/>　　可现在从楚风眠的口中说出来，在场武者却是没有任何一人敢不相信。
<br/>　　楚风眠杀荒圣君，灭其他六大宗门，这一切在场众人都是看在眼中的。
<br/>　　“无论如何，这一次楚兄放了我们，便是对我们天枢殿大恩，我星筠，一生不会与楚兄为敌。”
<br/>　　星筠看着楚风眠目光闪动，开口道。
<br/>　　“若是楚兄日后有何吩咐，我星筠必定来报。”
<br/>　　“走吧。”
<br/>　　说完这一切，星筠便是带着其他天枢殿弟子离开了。
<br/>　　这风雷谷，也被楚风眠从新开启。
<br/>　　在天羽神域之中的无数武者，都是纷纷离开了。
<br/>　　“这星筠，倒是一个聪明人。”
<br/>　　听着星筠的话，楚风眠笑了笑道。
<br/>　　这星筠话中的意思，便是也带着几分拉拢。
<br/>　　不过倒也无妨，一个天枢殿弟子，楚风眠还不放在心上，他的真正仇敌，还是那些七大宗门中的老古董。
<br/>　　至于其他弟子，只要不跟楚风眠为敌，楚风眠放走几个也无妨。
<br/>　　等到这天羽神域中所有武者，都离开一刻，楚风眠才缓缓的起身，他大手一挥，那风雷谷便是再也消失不见。
<br/>　　这云羽天宫，从此之后便是只属于楚风眠一人，绝不会再让任何人进入其中。
<br/>　　在将这天羽神域的入口封锁之后，楚风眠这才最后身形一动，从他的面前，化为了一道空间隧道，一步踏入到了其中去。
<br/>　　不一会。
<br/>　　归墟天之中。
<br/>　　便是一道裂痕，突然出现，楚风眠的身形，降落到了归墟大陆之上。
<br/>　　一般武者，就算是超越生死的人物，都无法从这归墟天中撕裂空间，凝聚空间隧道。

<br/>　　楚风眠扫视了周围一圈，便是感觉到不少无上剑气的踪影。
<br/>　　楚风眠回到这归墟天之中，第一件事，便是要继续收集这无上剑气。
<br/>　　之前因为云羽天宫的原因，也浪费了他不少的时间，现在一个月的时间，已经过去二十多天。
<br/>　　虽然楚风眠拥有云羽天宫，可以随时从归墟天中离开，但是最多也就待上一个半月，若是时间太久的话，也许其他人也会发现云羽天宫的秘密。
<br/>　　这云羽天宫，不用等待三年时间，可以随时出入归墟天，绝对是七大宗门都想要得到的宝物。
<br/>　　现在的楚风眠，还不能暴露。
<br/>　　“只有拥有足够的力量，才不用这样躲躲藏藏！走！”
<br/>　　楚风眠身躯一动，便是直接化为一道遁光，向着无上剑气所在的方向，飞了过去。
