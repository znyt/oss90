　　进入山神庙的时候，她就差不多恢复了真实容貌，因此在这些士兵到来之后，她一直低着头，希望自己不被注意到。
<br/>　　“呦！还是个俊姑娘！”
<br/>　　几个士兵看到翠云的样子，立刻调笑起来，有的还吹起了口哨，露出肆无忌惮地垂涎模样。
<br/>　　一个士兵伸手想去摸翠云的脸蛋，被小三伸手打掉，转头问吴连长：“喜欢吗？”
<br/>　　吴连长怔了一下，盯着翠云看了一会，眼中也是闪烁着异样的光，说道：“我们这是去入伙的，带着家眷，这不合适。”
<br/>　　小三道：“也没什么，再者，这样的美人，就算你自己不要，送给山上大当家的做个礼物也是好的，先带走再说吧。”
<br/>　　吴连长听他这么说，点了点头，道：“是，兄弟们，先带走再说。”
<br/>　　于是有两个士兵上前抓翠云，翠云起先死死抓着叶少阳的手不放，随后猛然想到这样可能会连累他，这才松开了手，被两个士兵押着胳膊提起来，送到吴连长面前去。
<br/>　　“不错，好货色。”吴连长看了一会，哈哈大笑起来。
<br/>　　翠云心里头只是绝望，也没有反抗，只希望叶少阳能逃过一难。
<br/>　　吴连长伸出手，想在翠云脸上摸一把，突然身后响起一个声音：“连长，不对！”
<br/>　　吴连长停了手，转头找到那个士兵，说道：“什么不对？”
<br/>　　那士兵眼睛望着破庙的门外，怔怔地伸出手，指着门槛位置说道：“连长，石头的尸体不见了！”
<br/>　　吴连长和其他人都懵了，顺着他目光看去，可不是，地上另外两个的尸体还在，只是之前那个叫石头的躺的位置，只剩下一滩血，他的尸首却是不见了。
<br/>　　“娘的，这怎么回事！”吴连长惊呼起来，死人当然不可能自己爬起来走掉，尸体失踪，必有缘故。
<br/>　　“会不会是没打死！”小三说着走过去，盯着地上那一滩血迹看去，血迹中还有些白色的，是脑浆，连脑子都打出来了，人肯定是活不了了。
<br/>　　小三和身边几个士兵面面相觑，左右寻找起来，山神庙的前厅和后厅就那么大，大伙一眼就看出没有尸体。
<br/>　　这还真是怪了！
<br/>　　大伙互相张望，一个个都感到很是诧异，虽然一个人的尸体并不重要，不过这这件事却是诡异的很，不得不弄个明白。
<br/>　　小三站在那一滩血迹前，四周看去，这才发现了一个疑点：血迹是在门槛边上的，旁边就是门框的立柱，之前小三等人只顾在附近寻找，却忽略了门框，这时才发现门框上有血，情不自禁地抬头看去，只见门框上一直都有血，视线一直往上，终于看到了一直寻找的石头的尸体，居然拦腰躺在破庙的大梁上。
<br/>　　“怎么会这样！！”
<br/>　　大伙都失声叫了起来，左右看去，谁也不知道为什么会发生这么离奇的事。
<br/>　　有人放上去的？
<br/>　　大伙都觉得不可能，毕竟大梁有至少三米高，下方光秃秃的，就算是一个活人想要爬上去都不太容易，更不要说手里还抱着一具尸体……
<br/>　　最重要的是，大伙一直都在这里呆着，就算是没有注意石头的尸体，但如果有人想抱着他爬上房梁，肯定会闹出动静，不可能完全没人注意到吧？
<br/>　　一时间，场面笼罩着一种诡异的气氛，一个士兵喃喃说道：“总不能是他自己爬上去的吧？”
<br/>　　这句话一说出来，更是让大伙心头一紧。
<br/>　　吴连长深吸一口气，命令两个士兵爬上去，把石头的尸体弄下来研究下怎么回事。
<br/>　　两个士兵找了一些杂乱的东西垫在窗户下面，然后爬上窗户，其中那个身材瘦小的用两腿勾住了大梁，对着挂着石头尸体的位置爬了过去……
<br/>　　结果一只手刚抓住石头的手，想要把尸体扔下去，诡异至极的事情猛然发生了：
<br/>　　石头原本挂在房梁上的身体猛然间弹坐起来，一只手反手抓住那个士兵，脑袋转过来，冲他笑了一下。
<br/>　　“啊！”那士兵双手一松，掉了下去，啪嗒一声摔了个结实。石头的尸本来是挂在大梁上的，随着坐起来，他跟那个士兵一起掉了下来，本来他全身上下被打了至少十几个窟窿，脑袋都打开了，这么一摔，一团脑花子直接磕了出来，眼珠子也悬挂在眼睛外面，模样看上去十分狰狞。
<br/>　　“他刚才……动了？”吴连长不敢相信自己的眼睛，盯着地上的死尸，试探地问道。
<br/>　　没人敢回答这个问题，大伙都在安慰自己，一定是某些原因造成了尸体的移动，看上去像是动了似的。不过，像是为了证明大家都猜错了，地上的石头的尸体竟然撑着脑袋爬了起来，脑袋也抬了起来，直勾勾地盯着吴连长，然后撑着双手，向他慢慢地爬了过去，嘴里同时还发出了类似呻吟的声音。
<br/>　　“什么！”吴连长吓的当场跌坐在地上，一个劲往后退，差一点就要叫起来。
<br/>　　在这诡异的时刻，小三突然出手，手中步枪瞄准了尸体，啪啪啪就是几枪，直接将石头的脑袋打得稀烂，倒在地上不动了。
<br/>　　小三望着地上烂成一团的尸体，喘着气说道：“大伙也看到了，不管是什么精怪作祟，现在也没法动了。”
<br/>　　话音刚落，身后突然传来一阵呻吟声，众人一起回头，顿时又看到了惊悚的一幕：之前跟石头一起被杀的虎子的尸体，也从地上爬了起来，甚至用力扣下自己两个眼珠子，朝着小三砸了过来。
<br/>　　“开枪，开枪！”小三骇得魂飞天外，一个劲命令大伙开枪。
<br/>　　一通枪弹打过去，虎子的尸体也被打成了稀烂，像一团肉烂在地上，自然不会再动了。
<br/>　　吴连长等人静静地望着虎子的尸体，没有一个人开口说话，然而，这时候边上那最后一具尸再度尸变，更是直接把自己脑袋拧下来，朝吴连长等人砸了过来。
<br/>　　十几个人一起放枪，再度将这具尸体打得稀烂。
<br/>　　“都死光了，总该没事了吧……”一个士兵紧张得喃喃自语。
