　　神图空间。
<br/>　　苏莫伫立在地面上，目光冷漠的望着眼前的屈覃。
<br/>　　“说出我想知道的一切，你才有活命的机会！”苏莫冷冷的说道。
<br/>　　“苏宫主，我真的不知道，我与黄泉魔宗没有半点关系！”屈覃脸色难看，脚步微微后退，和苏莫拉开距离。
<br/>　　他没有选择逃跑，虽然他不清楚这是什么地方，但也能够猜测的出来。
<br/>　　“就知道你不会乖乖招来！”
<br/>　　苏莫冷笑一声，继续道：“那你就尝尝火炼的痛苦吧！”
<br/>　　言罢，苏莫陡然出手了，大手一张，掌心中火焰喷涌，瞬间将屈覃包裹在了熊熊烈火之中。
<br/>　　这火焰乃是他的火属性玄力所化，汹涌澎湃，威力巨大无比。
<br/>　　“啊！”
<br/>　　身形被火焰包裹，屈覃口中立刻发出一声惨叫，而后身形腾空而起，便想向远处逃离。
<br/>　　不过，这里是虚界神图，他根本不可能有半点反抗苏莫的机会。
<br/>　　周身的空间陡然凝固，如同铁索一般，立刻将屈覃的身形束缚在了半空之中，任他如何挣扎，都没有半点作用。
<br/>　　滋！滋！滋！
<br/>　　汹涌的火焰完全将屈覃包裹住了，炙热的火焰之力，不断的焚烧着他的身体，发出滋滋的声响。
<br/>　　啊！啊！啊！
<br/>　　屈覃立刻便惨叫了起来，惨叫声震人心魄，痛苦无比。
<br/>　　“若是你不说，我便将你生生炼化，生死魂灭！永世不能超生！”苏莫淡漠的声音响起。
<br/>　　他已经极力控制了火焰的威力，不然的话，对方瞬间就会化为灰烬。
<br/>　　要知道，他现在掌中喷出的火焰，乃是他修炼的帝级上品火属性功法《天火玄经》的火属性玄力，威力强大绝伦。
<br/>　　别说屈覃只是武皇境二重武者，就算是武皇境七重武者，都不一定能挡的住他的火属性之力。
<br/>　　“苏宫主，我真的不知道啊！你就是杀了我也没有作用！”屈覃大声的吼道，依旧不承认。
<br/>　　“不知道？那我就继续炼！”苏莫厉声说道。
<br/>　　随即，他稍稍加强了火焰的威力，让得屈覃的惨叫声更加的惨烈。
<br/>　　在熊熊火焰之下，屈覃身上的衣袍，早已灰飞烟灭。
<br/>　　他的血肉在火焰之下，被缓缓的焚化，冒出阵阵白烟，空气中居然还弥漫着淡淡的肉香，让人闻之作呕！
<br/>　　屈覃倒也是强硬，惨叫了片刻之后，居然停止了惨叫，咬紧牙关，强行的忍住了痛苦。
<br/>　　滋！滋！滋！
<br/>　　火焰炙烤血肉的声音，让人心底生寒，不多时，屈覃全身的血肉几乎被焚化一空了。
<br/>　　血肉被焚化，森森的白骨裸露在外，五脏六腑都清晰可见了，都开始冒出阵阵白烟。
<br/>　　虽然屈覃极力的用自身玄力抵御火焰的焚烧，但根本抵挡不住，他的玄力在熊熊烈火之下，只有被焚化的份。
<br/>　　而屈覃已久死不松口，不承认与黄泉魔宗有关系。
<br/>　　“屈国师，你再不说，就没有机会了！”苏莫冷冷的说道，即便对方是武皇强者，再炼下去对方也活不了了！
<br/>　　“苏莫，我已经说了，根本不认识什么黄泉魔宗的人，更不知道他们在哪里！你杀了我了吧！”屈覃声音低沉的吼道。
<br/>　　苏莫闻言皱眉，此人真是硬骨头，居然完全不怕死。
<br/>　　这就有些棘手了，问不出消息，就算杀了对方，也没有任何意义。
<br/>　　随即，他收回了手掌，停止了炼化。
<br/>　　他的目的是得到有用的消息，并不是要杀对方。
<br/>　　火焰消失，屈覃凶厉的眼眸死死的盯了一眼苏莫，随即立刻盘膝而坐，开始了疗伤。
<br/>　　苏莫低头沉思，思索有什么好办法，能撬开对方的嘴。
<br/>　　不过，思索了良久，他也没有找到什么好办法，最后便离开了神图空间，回到了天涯海阁分阁之中。
<br/>　　一处偏殿之中。
<br/>　　苏莫端坐在主座上，下首两侧，则是坐着十余名天涯海阁的武者。
<br/>　　“诸位，我已经擒拿了屈国师，但此人死不开口，不知诸位有什么办法？”苏莫向众人问道，他没有什么办法，只能询问众人，集思广益了。
<br/>　　“宫主，用尽各种残忍的手段，抽魂炼魄，不怕他不说！”一名青年武者阴狠的说道。
<br/>　　“无用，此人死都不怕，抽魂炼魄也不会有用！”苏莫摇了摇头。
<br/>　　“动用迷幻之术，影响他的神志，看能否让他自己说出来！”
<br/>　　“不如让他有机会逃走，我等再暗中监视着他！”
<br/>　　“……”
<br/>　　众人七嘴八舌的开口，说出了一个又一个办法，虽然乍一听确实不错，但都不一定能奏效。
<br/>　　这时，一名年纪稍长的老者沉吟了片刻，向苏莫抱拳，道：“宫主，天下间有一种秘术，名为搜魂术，可以直接搜索他人的记忆，您若是能习得此术，或者找到修习了此术的人，就能得到想要的消息！”
<br/>　　“搜魂术？”苏莫闻言眉头一挑，心中惊讶不已，居然还有如此霸道的秘术？
<br/>　　“哪里有这种秘术？”苏莫立刻出言问道。
<br/>　　“回宫主，据我所知，这种秘术只有黄泉魔宗的人会使用！”老者沉声说道。
<br/>　　苏莫闻言无语的摇了摇头，如果能找到黄泉魔宗之人，那还用找这种秘术吗？
<br/>　　显然看出了苏莫的想法，老者解释道：“宫主，据我所知，大多数黄泉魔宗武王境以上的武者，都会此秘术；目前虽然黄泉魔宗的山门，和各处的据点、分部都已经毁灭，但还有不少黄泉魔宗据点的武者散落在外，东躲西藏，找到这些人不成问题！”
<br/>　　苏莫闻言点了点头，不错，虽然现在黄泉魔宗在外的几处分殿，以及各大据点都被灭了，但人显然没有死光，余孽众多。
<br/>　　这些人，很难和巫森等一众黄泉魔宗的高层汇合，找到这些人或许能得到这种秘术。
<br/>　　不过，苏莫低头沉思了片刻，便立刻想到了两个人，天辰和安暖。
<br/>　　这两人在黄泉魔宗待了这么长的时间，而且两人的修为也不低，都早已达到了武王境，说不定就会这种秘术。
<br/>　　而后，苏莫没有在青俞岛多留，立刻离开了青俞岛，向苍穹神宫返回。
<br/>　　他准备回去询问安暖和天辰，若是两人会这种秘术，那就再好不过了。
<br/>　　若是不会的话，那就再安排人手，擒拿黄泉魔宗四散在外的余孽。
